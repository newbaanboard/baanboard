Hi Mr Roland

In the main, my company tends to develop native Baan reports using tools. This way all Baan users can have access to this facility. Since Crystal requires a licence for each client, this can get expensive.
We tend to use Crystal mainly for central reporting, "nice" external documentation and ad-hoc reports. As a techie,I tend to use Crystal to interrogate data when fault finding. Very easy and quick to change parameters, selections, sorts etc, since its all GUI based.
