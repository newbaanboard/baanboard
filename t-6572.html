I think the API functionality should already be on your system. I would try to run ttstpcreatdll and see if you can create a library. You do not have to use this session, but I like the libraries. When you create a DLL it will compile it for you in most cases. Sometimes there are problems with the compile. Also just because ttstpcreatdll does not work does not mean that the stpapi functions will not work. On our system ttpstpcreatdll aborts, but I can still code with the stpapi.commands.

To get the latest and greates solution for the API objects I would contact Baan support.
