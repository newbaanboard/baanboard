Hi, as mentioned you have to switch on at least one of the mentioned parameters in tccom0500m000. The project field on the PO line is not just an informational field. It can only be used in combination with PCS or TP. If that is too heavy in usage for you then you can not use the project field on the PO line.

However on the TP project it is possible to mark the project as Capital Project. Maybe this is something you can use in the future.
