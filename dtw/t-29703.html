Hi,

The session at which blocking has to occur is missing in the order type(tdsls0594m000) and hence the message pops up.

If you don't want the system to block automatically based on certain criteria, uncheck all the block reasons in sales order parameter. If you don't want the signal to pop up either, uncheck them too.
