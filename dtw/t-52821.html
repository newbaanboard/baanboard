Yes there is parameter/master data setting for both Sales and Service.

In case of sales it is Item-Sales data where you do the setting for committment and in case of Service it is Work Order Parameter (order control tab, committment section) where you set the committment.

This is based on LN, but I believe it should be same for Vc also.
