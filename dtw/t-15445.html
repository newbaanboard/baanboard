You can use the Sales Price Calculation in Baan to Add a Surcharge Amount or Percentage to the Standard Cost of the Item by Item Group Codes.

To do this you must Use Session ticpr1500m000 Price Calculation Code (S01) Example.
Session ticpr1510m000 Item Surcharges using S01 Sales Price Calculation Code, What Item Group you want to use, Insert Description, Calculation Line Type Surcharge, Surcharge Method, (% of Total Cost), Surcharge Percentage 1.0000 would be 100% Markup of Standard Cost.
Then Run Session ticpr2250m000 Calculate Sales Price by Item.
Look at help on these session and ask for information on them from Baan.
