Hmmm...Page Mode (set on the User parameters) may help you some. They can tab around the page at least. However, all those checks will just happen on the 'save', but you can see if it helps you guys.
I think (very roughly) what page mode does is take the entire form (session?) entry like a batch job. So the check inputs, etc, do not get performed as you tab through the fields, but only once when you either go to the next form or choose save. Not sure of all the details.

Also, some sessions and actions are not available using page mode, but you should find that out pretty quick. It gives you an error message.

benito,

check the parameters in Session tdsls4100m000 (Maintain
Sales Order Parameters)

Parameter:
- Block if Credit Limit is Exceeded
- Signal if Credit Limit is Exceeded

I think in your case these parameters (one or both) are enabled.
If you don`t need these checking, disable the parameters.

Additional you can check:
- Block if Invoice is Overdue
- Signal if Invoice is Overdue

In our company all of them are disabled.
