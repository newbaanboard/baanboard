Hi there,

we simply use printers that support barcode (brother hl series) for example, but several others do that too.
All you have to do then to print barcode in a Baan report is to study the manual and write the escape sequences into Baan 'barcode drivers' (simple textfiles containing these sequences). Then you can print fields in barcode.
Before we realised the simplicity of this functionality, we used other tricks to modify the data sent to the printer, but the way described above is a lot simpler and helps of course if you have to change a printer model.
No additional SW is needed, just make sure that the printer is able to print the barcode you need (code 128 shouldn't be any problem though)

regards

Norbert
