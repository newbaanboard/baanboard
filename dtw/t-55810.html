To be more accurate:

Is it possible in the ERL ln 6.1 to import a table, who will fill automatically in details this table.

Hello,

yes, you can do it. But a little bit tweaking around is neccessary.

1. Download: http://www.iso.org/iso/list-en1-semic-3.txt to your PC and open it with Excel.

2. Modify the Excel table for fit with the structure of your target table (tcmcs010 ?)

3. Export the Excel table as Text or CSV-file.

4. Import it into ERP LN with Exchange or with "Create Table fom Sequential Dump" ttaad4227m000. If you use ttaad4227m000 then the file shoud have the correct header. To get the header for your environment create a dummy entry in tcmcs010 and run "Create Sequential Dump of Table" ttaad4226m000 first :-)

Good luck.
