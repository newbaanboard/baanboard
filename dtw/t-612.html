Hi-
you can set the tab order through the tools module , application customization / forms / maintain forms. In your case it would be form tdsls4102s0001. Go into Edit Form. Once there, click the Form pull down menu and select Field Order. You can then set the tab order for the fields. Don't forget to do a form 'dump' when you are finished.
hope this helps

your supposition is right. What you want is not possible

The function to.field(xxxxx) is heavy use in the script, for
example after input of the item.number. So for non generic
items the field sequence is item -> quantity due statements
in the script.
