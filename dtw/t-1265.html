I would check if there are double lines in the sales order history. (The sales statistic update reads from the sales history tables).

There is a correction program that apparently corrects double entries in the sales order history (comes with solution #102752), but I haven't checked, what it does exactly.

I would also delete the statistic with "Archive/Delete Sales Statistic", then make sure that the statistic tables are empty and afterwards run a "full" update, to rebuild it.

But be carefull: if you already deleted sales history, the statistic might not get rebuild completely.
