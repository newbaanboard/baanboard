This may seem a wierd solution to the question, but it should work....

Rather than sales orders for gifts, promotional items etc., use Service Orders. Set up a Service Configuration for the customer, and a Warranty to provide 100% relief against materials. Against the Service Order Activity, record an Estimated Materials line of type 'From Warehouse'. Release the Activity, and the warehouse order will be created in a similar manner to Release to Warehousing in Sales. When the warehouse order is shipped, complete and close the Service Order. This would normally create an invoice, but the warranty arrangements will suppress it. If you really want to simplify things, then making the Activity Estimated Material 'From Service Inventory' rather than 'From Warehouse' allows you to show the item as having been sent without the need for ever having it in stock!
