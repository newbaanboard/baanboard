Hi Matt,
if you don't use ILC you can generate hard allocations by running tdinv2201m000 or use the maintain session tdinv2101m000. This has the same effect as generate outbound advice. You can run this session at any time during the sales order process (but you have to define it as a step in your sales order procedure tcmcs0142m000).
