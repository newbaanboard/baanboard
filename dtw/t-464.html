Hi,

In session Maintain Order Steps (tcmcs0139m000), find the record for Print Packing Slips (tdsls4403m000). At the bottom of the form you will see the re-execute sequence. Simply add the numbers of the order steps that come after Print Packing Slip for the order type that you are using and it will allow you to re-print the packing slip when the order is at any of these steps.

The numbers (order steps) can be found in Maintain Order Types (tcmcs0142m000). You will need to find the record that corresponds to the order type that you are using, it will then show all the order steps that form the procedure for that specific order type.

E.g. We have an order type SC7

The order steps that form the procedure for this order type are:-

1. 2 Print Picking lists
2. 3 Maintain Deliveries
3. 4 Print Packing Slips
4. 6 Print Sales Invoices
5. 7 Process Delivered Sales Orders

Therefore in Maintain Order Steps for Print Packing Slips we would add (step) 6 and 7 to the re-execute list to allow us to reprint the Packing Slips when the order is at these order steps in the procedure.
