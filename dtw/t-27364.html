I have found that the simplest method for sales return orders with ILC is to make the return order type a collect order as well as a return order.

Then, when you book the order, the maintain outbound line will pop-up immediately the order line is saved. If you book the stock into the default receipt location for the warehouse, you can then move it wherever you want after that.
