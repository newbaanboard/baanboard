Hi,

I guess you wont be having backup's. Anyway, if you just want to create this Sales Order number again, first make sure none of the dependents tables have the same order number. Now, if Baan 4 has first free numbers concept, you can change it and make it similar to the one you deleted. Now, if you create a new SO, you will get the lost number. Now, change the first free number back to the original.

Please note, do it in single user mode else it will cause problems if some other user creates SO before or after you.

I would not suggest you to do this unless it is very important. Try it in a test environment first!
