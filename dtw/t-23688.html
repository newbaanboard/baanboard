Hi Dnish,

there is two ways to deal with your problem: One is to use the Quality Management System in Baan and set up order specific inspections. This would allow the QC to enter measurements and / or qualtitative inspection results, block and release incoming materials and do some more QC stuff. It requires quite a bit of set up and overlaps with a specialized QC system.

The second approach is much easier. Set the 'inspection' flag in Maintain Suppliers form 4 to yes and set the the items to be inspected to 'inspection' = yes (form 3 in Maintain Item Data). Only the combination of both flags set to yes will then force the purchase order position to go through QC. Hence, you can control that stationary does not need to go through QC.

You MUST do Maintain Receipts BEFORE Maintain Approval. You do the Maintain Approval using the receipt number from Maintain Receipts. Maintain Receipts puts the goods in the inspection location in the warehouse where they were received if you are running location control.
