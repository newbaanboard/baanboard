Flip_J
27th November 2003, 11:02
It depends what the valuation method is. This is selected in Warehousing and set by Item Group. Can change by warehouse and seletected in Item Data by Warehouse.
CPR module calculates the FTP value.
Purch - Calc code will identify the source needed.
Manuf - Calc based on BOM and ROU. Bom = Manuf and Purch Item cost. Rou = Operational cost. Depends on Rou Parameter where the operational rate is linked. See ticpr2420m000 to see the detailed calculation of the FTP cost.
