I had the same problem some time ago and solved as given below.
You can also try the same.

Pl. go to session tgbrg0530m000 "Enterprise Units"

Select the enterprise unit that was linked to your Warehouse entered in your Order line.

Go to specific and choose "Entities by Enterprise Unit"

You should see all the enities linked to that Enterprise unit including your Warehouse in your order line.

If you do not see any enities, that is the root cause of the problem.

There is a work around solution to solve this situation.

Delete the subject Enterprise unit (system will allow this as it was not linked to any entities)

Add the Enterprise unit with the same code again.

you will see all entities now and you may not get the error messages any more
