Hi Rini,
sure this is possible.
In Item Base Data (tcibd0101s000) you have to define the item type as KIT and item specification either as Set, Option or Menu which means:
- Set: the components are fix. In your example 1 sofa is always a set of the defined components. (define item sales data for sofa)
- Option: while maintain sales order you are able to select only one component from the KIT.
- Menu: while maintain sales order you are able to select one or more components from the KIT.
in the two lower cases you have to define item sales data for all components.

Hope this helps,
Andreas
