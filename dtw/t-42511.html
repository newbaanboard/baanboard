Hi,

as far as I know this option is only relevant and therefore enabled if you have partially shipped goods and are using location control. As a result a transfer order will be generated that allows to create inbound advice to move goods from staging location back into storage locations. If you don't have locations there is no need to do as much in the system. Of course physically you still might move goods back into stock but the ERP admin is not influenced by that.
