Well, if they don't want to remove it from the PO and insist on making a 0 delivered, 0 back order receipt, then I think you need to read tdpur045.

When tdpur041.dqua = 0 and tdpur041.bqua = 0, then I would read to see if there is a tdpur045 record with srnb > 0. It should, of course, also have dqua and bqua = 0, but it probably isn't necessary to check that.
