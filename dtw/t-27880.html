Hi Jeffers,

You can find it in tfgld410 (or session tfgld4510m000). For example, when the transaction type in tdinv100 is 'inventory adjustment', look for transaction origin 'warehouse order' in tfgld410. Or when the transaction type is 'purchase order', then look for transaction origin 'Purchase (INV)'.

Hope this helps,
Frank
