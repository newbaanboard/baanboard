OmeLuuk is right.

There are several patches for inventory adjustment - session tdilc1121s000 - Maintain Stock Transactions by Lot and Location.

E.g.
With BAAN IVc3 SP3 when you make inventory adjustment and quantity is positive, the lot size is changed. If quantity is negative, the lot size is not changed.

With BAAN IVc4 SP8 the lot size is changed if quantity is positive and lot size is zero.
