Hi there,

No, you are not allowed to change any currency in the currency setup in tcemm170. You can only do that by the currency initialization module, i.e tccri.

You can imagine that there are a lot of financial tables which store the transactional data in the home currencies. If you change the setup of your home currencies, then all that data becomes corrupt. Also a lot of costing data etc. is stored in the local currency of that company.

Of course, because this is a demo company, you can hack tcemm170, but you should then throw away all transactional data in all tables and you are likely to forget some (costing data, some amounts in master data should also change etc.), so please use the currency initialization module for that.
