There is nothing that says- cycle counting turned on for location. Basically if a location for an item will come for cycle counting depends on three factor

1. Item ABC classification
2. The cycle count frequency
3. Last counting date and this is present in ilc101

Like in our case the reqmt was that if a location is counted then all the location for that item should also be counted. For this we did a modification which updates the last counting date.

Also, say if the item had no stock (means no ilc101 record), then when purchase or production receipt happens then the item location was coming on cycle count, again we updated the last counting date, so that the location will not come.
