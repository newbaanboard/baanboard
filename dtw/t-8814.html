shah_bs
11th February 2003, 03:56
The only reason I can think of is that there is a provision to Maintain Backorder, which is in a way 'changing' of the order quantity. After a partial receipt, one can Maintain Backorder and can reprint the purchase order such that it prints the Back Order quantity (which becomes the new Order Quantity) instead of the original Order Quantity.

Yes, it IS a bit a of a problem, not being able to change the orginal Order Quantity.
