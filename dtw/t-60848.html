Hello,

System will not allow if source and destination items is having different base units.Even if you apply conversion factor,will not allow to transfer inventory to the destination item. Only solution is to adjust the inventory through Adjustment order.

Always inventory unit and Purchase unit in Item Puchase data should be the same.

That item might have conversion factor, thats why even if base units are different system is allowing you to change the unit in Purchase order session.
