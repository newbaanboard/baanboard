Dear Faditimo,

I had the same problem like you in allocating the cost to the specific workcenter or machine.

The way I am doing it now is using the cycle count order to issue the material and use the reason to allocate the cost to the specific machine.

In this way, you have to set up many reason code (at least equal to number of machine/workcenter).

Then you can setup the integration with reason code as a key, and if you put your workcenter as a dimension, so you have to set up the option by integration element.

I hope this can help you.

Harley
