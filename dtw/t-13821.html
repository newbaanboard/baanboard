You should check at what point in your order steps you have told the system to allow the printing of storage notes. Unfortunately I believe the only way to change the status back at this point is GTM in table tdpur045. To avoid this in the future you should change the Print Storage List order step to be re-executable right up to order step 9. See session tcmcs0139m (Maintain Order Steps).
