Hi,

The warehouse is not available as an input field in the contract lines but it IS stored in the table tdsls301. When determining the warehouse, first the user defaults are looked for (see tdsls4123m000) and in case none is defined there then the warehouse is taken from the item master (tiitm0101m000).

Hope this helps.
