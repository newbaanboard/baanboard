I assume that you will be creating the inquiry header manually. If not this gets trickier. You must import into tdpur002 and tdpur011. You will need an item number. There is no description in the tdpur002 or tdpur011 tables. You could use a cost item and then put the description into text. In order to put the description into text this requires a special formatting of the text or description line. Of course if you use a cost item then you will not be able to transfer the inquiry into a PO if the PO is to be for a real item number. Hope this makes sense. All of this discussion above assumes that you will be using exchange.
