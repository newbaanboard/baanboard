Hi:

An "easy" way to solve your problem is to define an "intelligent" code of address.

For example, you have 9 characters for the address' code so you can define the first characters related to the coustomer and the final 2 or 3 characters as a consecutive.
Example.
My customer is Baan and i have 3 differents destinations, so you can do this:
BAANMEX01
BAANUSA02
BAANNED03

This only will help you to order by customer and it will be easily for you to choose the correct address.

Maybe this is not the best solution but it works, in the other way you will need a customization.
