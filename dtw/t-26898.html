Hi,

you need to run the session tdinv6220m000 in Baan IV, in ERP LN 6 it is session whina6220m000. From the help on LN:
"A slow-moving analysis calculates the turnover rates, which are compared with the slow-moving percentage as specified in the Inventory Analysis Parameters (whina0100m000) session. This calculation results in a classification of items into ten categories in which the best category has the highest ratio of actual issue/inventory on hand."

In Baan IV under Distribution Parameters go to maintain INV Parameters and on Form 3 enter the Slow-moving Analysis Parameters. Should be able to do Baan Help on the fields if you can't understand them. After you set these parameters then run the session tdinv6220m000 Perform Slow Moving Analysis by Item.
