Hello Lindah,

Inventory figures like "Inv on order" and "allocation" is sometimes wrong in Baan (and other ERP-systems). This will influence the output from different tools (MPS, MRP, SIC) you use for planning.

Try to run tdinv0250m000, Rebuild Planned Inventory Transactions, but do it when there is no one in the system.
This session delete all records with planned Inv transactions and rebuild them because of the orderlines in the system.
This might help!
