As promised the answer (i'm using another login name for my other PC):

In WHina 0510m000 you define a warehouse order type. Within this ordertype you attach the warehouse procedures for inboud/ outbound etc.

In Purchase order types (tdpur0594m000) you define Purchase order types and you link the needed warehouse order type to the used purchase order type.

It works.
