Hi,
The only way to force a cycle count daily is to write a small program that will do following:
1. Make the last counting date blank in ilc101 for concerned warehouse and locations,
2. Create a fixed location for item (if Item has zero inventory in that location).

Thats it. This can be run in batch mode before dailt cycle count.
