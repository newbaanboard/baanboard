You could use ttaad4229m000 to see where the old country code has been used and then amend those records using General Table Maintenance (at your own risk!). Then delete the incorrect country code. However, Print References to Parent doesn't pick up finance data so you would either have to accept those records would refer to the old country code or try to find those also.
