Hi ....

Although I have never yet implemented EDI between Baan IV and Baan V, I have worked on both versions for quite some time.

As far as I can guess .... the EDI formats for Baan IV and Baan V are different and so are the libraries that assist the EDI process. In Order to create an EDI link between these two versions, you will need an EDI mapper / Translator to create a link between the two formats.

I dont think ... a direct EDI interface between Baan IV and Baan V is yet supported.

Since the companies are not using the same bshell I think it is not possible to take advantage of multi site edi. However, if both sides are mapped identical it is possible to use edi to export orderinformation, then have a shell script run to transfer files to another directory and import the order information.
