We did for one last company recently in a week. This is mainly used to train the users how to deal with it. Good preparation makes the success.

Here are the main steps:
1. Training all the users;
2. Update all users' menu and permissions;
3. Switch the system with session tdilc0230m000;
4. Change location system for warehouses with session tdilc0231m000;
5. Generate locations for each warehouse;
6. Cycle Counting to move the stock to new locations;
7. Update other settings if applicable such as order types, reports, etc.
