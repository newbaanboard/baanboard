Complete Warehouse cannot be blocked

In the Baan application it is not possible to block complete warehouses. Neither in BaanIV, nor in the later releases. Baan only allows users to block locations, lots or stock points.

The advantage for this is clear: Baan does not allow to block the default receipt and staging (=Baan 5) locations either: it must always be possible to receive and ship goods.

Note from moderator:

In b50 it is possible to block an entire warehouse using session whinh6210m000 (Global (Un)block Warehouses/Zones/Locations).
You need to select the checkbox for Locations, make sure this applies to all transactions, enter a reason code, the warehouse number(s) and leave the selection for locations wide open so that this applies to all locations.

The original post concerned b40 so please keep this discussion clean and only reply regarding this version.

You can always generate a dummy Cycle Counting order for all items in the warehouse. This will block all transactions provided you have set the "Block during Cycle Counting"-Parameter in Maitain Warehousing Parameters.

Delete the Cycle Counting order(s) when you're done!
