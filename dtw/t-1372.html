As far as I recall that code is used as the item code put into the Intrastat tables, if there is NO "derived from" standard item given in the customized item. So, if the customized item is derived from a standard item, Baan will use the code of that particular item, and not the parameter value.

When using Baan Intrastat, keep in mind that the country of origin may be given in the item code only. So if you source the same item from multiple suppliers from different countries, the Baan Intrastat will pretty soon come to it's limits.
