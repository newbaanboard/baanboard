Looks like, that in a table the value for the financial or logistic company is missing, therefor is "0".

This could be e.g. the business unit or the warehouse (table tcmcs003).

The Select uses this company field to query the correct LN company.

Can you please check e.g. Session tgbrg0530m000 or related Tables for handling the receipt if there is a field for company having a "0" value and possible change this to your correct logistic company e.g. "100".

You can get some related tables with <CTRL>+<SHIFT>+8.
