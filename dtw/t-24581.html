Hi Ganesh,


Go to ttaad4134m000 maintain Table Data Authorisation

Do a group Insert Select a user

Select all Companies

The table which Stores the purchase Order Header tdpur040
The Data Authorization select "Not Authorised"


Insert a record

Type Condition as 1,Type Sern also as 1,Field as orno, operator between, Value 1 as 1, Value 2 as 499999, And/Or/End as and

In the Second record

Type Condition as 1,Type Sern as 2,Field as orno, operator between, Value 1 as 600000, Value 2 as 999999, And/Or/End as End of Expression


Now this User will be authorised only for Series 50.
