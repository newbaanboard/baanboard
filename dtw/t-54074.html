Hello,

maybe this will work:

In Parameter (tdltc0100m000) :

Generate codes = No
Series Number = No number

In Serials (tdltc0105m000) .

No ID - structure

So you can define your own number by receipt of purchase order if article is set to ltc.

Hope this is what you expect.

Hi Brend,

Sorry, as I havn't mentioned that I'm facing this problem on ERP LN 6.1

But the problem is resolved, This Manual Serial Control can be handled by following procedure:
1. In COM PARAMETER - don't define any default Mask
2. In Warehousing Order Type - Don't select the option "Generate Serials Automatically"
