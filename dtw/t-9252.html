One way of translating the BaaN application EDI file to X12 ANSI standard file is to use third party translators like Gentran, Harbinger, etc. This is the easiest and efficient way of translation.

If the EDI file need to be created in X12 ANSI format directly from BaaN, then a lot of changes has to be done to the EDI conversion setup in BaaN. The person doing this should have through knowledge of BaaN EDI processing. Also The interchange controls in BaaN need to be used to create ISA/GS segments.

In this board you can find threads, someone earlier mentioned that they created X12 EDI files from BaaN.
