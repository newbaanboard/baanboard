Hi,

you can use table tdsls011 to look up the sales order and position generated from the quotation (tdsls011.sorn, tdsls011.spon).
In table tdsls051 you can see the reference to the quotation (tdsls051.qono, tdsls051.qpon).
A standard session which can help is tdsls1511m000. You can see the reference to the sales order here. It is useful to activate Index1 of table tdsls011 for a better search.
