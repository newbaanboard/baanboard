This is how we map consignments in Baan IV .

1. Create a dummy consignment customer code with separate fin group and control a/c as Consignment stock . If u want to track consignment customer wise , u need to create separate consignment customer code for each customer .
2. Create separate order type for having different consignment sales invoice series .
3. Create SO and invoice it to dummy consignment customer.
4. On return / sales, create return sales order on the same consignment dummy customer with negative qty.
5. On sales , create regular sales order on the regular customer code with regular order type .

Lot of work . But u can automate using AFS to a greater extent . Good luck .
