Hi Eltjo Renken,

In BaaN IV, there is no functionality like automatically charging the surcharge amount if the Order value < $150 and no surcharge if the Order value >$150. If you would like to add the Surcharge amount in Sales Order, you have to insert a cost Item/ service Item with the Price.
