The status - Deleting order line means that you have fully processed this order line in the Distribution Module. You have selected to delete the order line when you run the session Process Delivered Sales/Purchase Orders. The order lines will be removed from this table when it has been processed in the Finance Module. Untill then the status will be deleting order line. Once processed in the Finance module the order line will be removed by the system but the info will still be visible in the Sales/Purchase History tables.

An orderline can be removed via tdsls4223m000 if:

1. there is no backorder quantity
2. the orderhistory/balance unpaid orders is updated
3. the project (if present) has to be archived
4. backorder created if full quantity was put in backorder
5. if all instalments invoiced
6. no rpl order present for orderline
