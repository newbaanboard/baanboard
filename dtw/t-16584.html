There is a session n Baan IVb4 to modify the outbound priority of an item / group of items. You can find this under
Location control --> Inventory Locations --> Miscellaneous --> Maintain outbound system by item.

Standard Baan allows us to change the location system by item just as it allows changing from lot to non-lot and vice-versa.
