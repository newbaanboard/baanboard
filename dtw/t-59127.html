To be undo only the release (so, the advise will not be removed), you have to enter the runnumber associated with your order.

Via the outbound advice you can easily navigate to the "Release Outbound Advice" (whinh4202m000) session. It will default the run number from the advice.

The "Release Outbound Advice" (whinh4202m000) works run based. The same as the "Generate Outbound Advice" (whinh4201m000) session.
