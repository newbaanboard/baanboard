Hi,

Yes, I have done for Customer master. It works fine.

Procedure is very simple.
1. Login as BaaN user.
2. Run the session ttaad4110m000
3. Enable the Audit trail for the database '001' (Default it is enabled)
4. Run the session ttaad4111m000
5. Insert the table name tiitm001
6. Enter the company No.
7. Enter the database No. "001"
8. From Special menu run "Convert to runtime"
9. Run the session ttaad4162m000
10. Assign audit security to user (User to view the audit details)
The above steps will start audit on itemmaster table.

Finally run the session ttaad4461m000 to view audit details.

Details you may see as follows :

Table Name: tiitm001
user : xxxx
Date : xxxx
Time : xxxx
