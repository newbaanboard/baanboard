It is not advisable to change the types of warehouses in a live environment in case these warehouses has been used. As you will probably know, this field is important for the different planningruns. You will find difficulties when you change the types and use for instance mrp runs.
Jim Ritchie
26th February 2002, 19:03
You can change them but you must check all planned transactions for the warehouse etc. If you plan to change back to non-nettable then you must change destination warehouse for all mrp items as the destination cannot be non-nettable.

You would be well advised to run an mrp report for the items effected both before and after the switch to make sure planning demand is correct.
