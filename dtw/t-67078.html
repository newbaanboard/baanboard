Hi,

I guess, when you make the receipt final you will have a new sequence of backorder line created in Purchase Order lines. Now, if automatic confirmation of backorders off parameter is off, you can change the Planned Delivery Date else the system will confirm it and will become a part of receipt which will not allow you to change the date.

I think there is no way you can modify it without Final flag is checked. The only way is to use Receipt Correction Option in whinh3512m000 which will generate a backorder line.
