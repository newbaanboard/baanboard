In Session "Maintain deliveries" there is a field for back order quantity, where the user can insert 0, when he do not want to have a back order quantity.

Second possibility is, that in the customer data the field "tccom010.back" has the value "no", then the field "back order quantity will be set to zero automaticly.

I hope this help You.
