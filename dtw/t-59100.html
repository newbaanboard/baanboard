Hi,

To revert the excess quantity reported against the order the user has done report order in negative.

Qty del is 8.2
Qty on last operation is 5.8

to make Qty del = 5.8 you need to issue back the manufactured item (8.2 - 5.8 = 2.4), the qty being shown in qty to deliver field. Process the warehouse order to issue this material and then you can close the order.
