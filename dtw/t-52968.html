Hello Umesh,
Line supply tells the system how the item in that warehouse is replenished...
So depending upon the method you choose, system will plan for the whenever the inventory goes down.

Example: If you choose TPOP method and then select to replenish the item from another warehouse, the planning engine (TOPO, in Inventory Planning folder) will create a warehouse transfer order. Similarly if you decide to replenishment the item from a supplier, TPOP planning engine will create a Purchase order advice...

Use the help and you should be able to simulate this...
