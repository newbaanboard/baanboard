We have a number of customers who have exceeded their credit limit. However, we still raise orders for these customers. But when creating a sales order header for that customer a message appears informing that the customer has exceeded his credit limit. This is ok.

However, at the next stage, when we enter items the same message continues to appear every time we add a new order line. Is there a parameter anywhere that will prevent this message appearing eash time we enter a new item for that order.

Hi NazrulTanni,

There is a Parameter in Sales Order Parameters form - 2, signal if credit limit exceeds- Yes/No. If you say "yes" system will pop up a message every time during the header & Lines entry. If you don't want the message to appear every time then make it as "No".

there is no other parameter setting for Customer Credit limit in BaaN IV.
