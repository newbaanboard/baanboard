Hi,

In standard BAAN the PRP run by itself will not create customized items in the BOM.
If the Order Policy of the item is "To Order" the system will make the item customized as part of the BOM but this is usually done before the PRP run is executed.

If in your case the PRP does create customized items you are dealing with a customization on your system.
Please check the VRC of session tipcs5201m000 with ttstpsessinfo.

Regards,

Edwin
