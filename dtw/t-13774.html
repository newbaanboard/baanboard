Flip_J
1st December 2003, 09:21
Hard allocation is stock allocated to a specific order. Soft allocation is allocation on a warehouse/company level and not order specific which means that any order can use the stock.

Hard allocation = Outbound data.
Commitment functionality is mainly used if you do not use location control and you want to hard allocate stock.
