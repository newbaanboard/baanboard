One workaround.
Set Table Data authorisations for tdpur041 as read only
For a user
all companies (or specific company)
Table Authorisation : D/I/M/R
is Read

if plev not equal to Supl.item (end of Exp)
-------------------------------------------------------------------------------
Again for the same user, specific company or all companies
table Authorsiation D/I/M/R
is D/I/M/R
if plev equal to Supl.Item


You can introduce PO Series if you desire in the expression.

This will ensure prices are entered
in the session tdpur0101m000 for Supplier. item
for the given PO series if specified.

DRAWBACK:
You may have a number of
tdpur040-Headers having no tdpur041-details.
