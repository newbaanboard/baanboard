Hi Lidea,

If you want to do some adjustment in your Inventory whose Outbound method is FIFO/LIFO, you need to select the the exact stock point in which you want the adjustment.

For selecting accurate stock point inventory you can zoom the button next to "INVENTORY DATE" in "Easy Adjustment Entry" session.

If still you have any doubts, refer the Thread "Negative Inventory Adjustment - logged by Mr. Bhushan Patil" in the same forum.
