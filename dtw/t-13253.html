Dear Charly,

You can separate slow moving from fast moving items on 2 level:
On company level (i.e. for this company, which items are fast and which are slow moving). To do this, set Slow Moving analysis by warehouse=No
On warehouse level (i.e. for this particular warehouse, which items are fast and which are slow moving). To do this, set Slow Moving analysis by warehouse=Yes
