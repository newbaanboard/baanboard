Hi Nicole,
Standard Baan will block the order every time a record is saved either in header or lines- this includes adding or removing or updating text notes. In my setup we have customized so that if an order is not blocked and if Baan blocks it and there is no change in the "Order Amount" system will unblock it. Also I note that your blocking step is "Print P/L" , so how is that going to work for a direct delivery sales order. In my set up we block at Order Ack. And we don't allow the PO sent to supplier if the sales order is blocked.
