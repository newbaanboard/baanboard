What do you want to block? Inventory in the warehouse or make this a non-nettable warehouse so Enterprise Planning will not consider this inventory? If you want to block this inventory, you can generate force cycle count. Make sure inventory is blocked during cycle count (inventory handling parameter). If you want Enterprise planning not to look at the inventory, then uncheck the Include in Enterprise Planning flag on the warehouse record in tcmcs0503m000 session. Hope this helps.

Block during cycle count will only block the existing stock points. It will still be possible to receive goods on new stock points.

In FP3 it is not possible to fully block a warehouse. Only as of FP5 it is possible to block a complete warehouse for inbound and/or outbound.
