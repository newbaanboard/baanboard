To open Enter Inventory Transaction (tdilc1120m000), first user must have permission for this session and the permission is given in the session tdinv1110m000 (Maintain User Authorisation to Inventory Transaction)

In this session, you can authorise a user for the following types of inventory transaction i.e.

Purchase Order
Sales Order
Production Issue
Production Receipt
Inventory Adjustment
Inventory Transfer
