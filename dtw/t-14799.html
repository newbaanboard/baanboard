Hi Manu,
Good point on GTM. I forgot that. As far as a log of archiving goes I'm a functional guy, not tools so I'm not sure. When you archive (see session tdinv1201m) you select the company to which you want to archive the records. Typically this is a company set up to be used only for archived records. However, you will notice that the drop down selection will list all the companies you have set up. Assuming you have more than one company there are two ways to search. In regular BaaN select the Options button on the main menu browser and change company. Then use the regular session (tdinv7510m) to search for the missing transactions. You can also just go in through GTM, change the company on the header screen and go to the transactions table to see if it's empty or has records.
Hope this helps,
Gary
