A Product Structure is defined as follows:
A Project (subproject, budget, or single project) with a project structure. This means the project is divided into main projects and subprojects). The following options are available:
BaanERP Generates for each sales order line a single project.
BaanERP generated one single project for all sales order lines.
After you have run this session, BaanERP creates a new project and new items. In the case of standard-to-oreder itmes, the BOM and ROU structures assigned to the original items are copies to the itmes created by Baan ERP. You can use the project engineering procedure to alter the standard structures.
Recommend that you get a copy of BaanERP Manufacturing Project Control System Document UP055A. send me a message if you need me to send you a copy of the document.
