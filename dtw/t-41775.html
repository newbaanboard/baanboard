Hello,
If you are using Baan IV MCR you can do this by customizing a session which should be run for a particular PO and changing tdpur040.ccur, tdpur040.ratp(1,2,3) tdpur040.ratf(1,2,3), tdpur045.mcrp(1,2,3), tdpur045mcrf(1,2,3),tdpur050.ccur, tdpur050.ccur, tdpur050.ratp(1,2,3) tdpur050.ratf(1,2,3), tdpur051.mcrp(1,2,3), tdpur051mcrf(1,2,3). In Baan IV standard: tdpur040.ccur, tdpur040.ratp, tdpur050.ccur, tdpur050.ratp. We did customize this session in Baan IV
In ERP-LN in the same way but there are different tables.
It's a good ideea to have this posibility in tdpur4201s000
