Hi Windham,

Transportation is a standard module in Baan IV but the integration to the other modules is sometimes very poor. For example Transportation does not refer to the item data (tiitm001). A second table as to be maintained (trtcd120).

We are running Transportation since almost five years now. It can be a useful tool, but you will need some time until it's working properly.

Hi All,
In case you have a performance problem in copying sales orders to transportation, please install solution 132320 (made on 6th feb 2003) to significantly increase performance. This solution is mainly for level 2 databases, a full table scan was done as a result of a missing order by clause. Result of installing the solution was that it took less then 10% of the time to generate transportation planning.

In 2002 also a fix was made to improve performance in generating visits. Please check local support or knowledge base to find solution number.

Update your system to the latest weekly dumps and you will surely be suprised about the improvement of the copying process.
