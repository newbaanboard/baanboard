So I was right that you guys forgot to request for production material in the first place. The purpose of generating outbound advice is for BaanIV to help you look for the most appropriate lot (or stock unit) to issue out of warehouse. If you'd like to specify manually which lot (or stock unit) to take, then create your own outbound advice, don't ask BaanIV to generate it. If you only want to follow some of the advice generated by BaanIV, then delete the advice you don't like and create new ones according to your liking

Hi Jordi,

In BaanERP Item Warehousing Data, there's outbound priority setting. There, on per item basis, you can choose whether BaanERP will suggest oldest lot to go out first (FIFO), newest lot to go out first (LIFO), or whichever lot happens to be in high(er) priority warehouse location. If I'm not mistaken, there may be similar setting in BaanIV. I haven't touched BaanIV for long time now

Depending on the outboundpriority as set up in item master data it will generate outbounds based on FiFo (first in first out), LiFo or by Location. Last one is perhaps easier when stock dates are irrelevant for your business and saves you lot of extra data handling.
