Anurag,
The parameter setting you changed would only make Baan stop an issue when there is a chance of the stock going into negative figures. What you are trying to do is to block a negative entry being used for the Tran. Type "Production Issue".
For this you might have to modify the script running behind this form. Another way, which i am not sure will work, is to customize the form tdilc1120m000 and use the [Input/Ref. Expression] option of the field "Planned Qty" to block quantity below zero. This however will have to be done only when the Trans. Type is "Production Issue", otherwise it will stop you from entering negative figures when you do other transaction for e.g Inventory Adjustment..

I hope i am clear...
