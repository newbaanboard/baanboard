There should be total 5 inventory tables for Inventory:
1. Item
2. Lot
3. Location
4. Warehouse
5. Planned Inventory transactions.
Condition is Inventory on hand, Inventory on Order, Inventory Allocated and Inventory Blocked for a given item should be same in all the tables. If it is different, BaaN Usually gives this kinda error message. This situation will come when you have a conversion factor such that the transaction goes into too many digits after decimal. Also if the server date is modified be someone (Which is a CRIME) same situation may occur.
There is a solution available somewhere in BaaN site. you can modify the inventory to match all the tables through GTM. Try to get the inventory tables list.
Good Luck
