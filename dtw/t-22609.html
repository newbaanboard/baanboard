15th December 2004, 23:28
Generate outbound is a Baan standard step. Release Outbound is not a step. You should have the steps set:
Generate outbound
Maintain deliveries

The reason you cannot proceed is that if you created a step Release Outbound manually, this session does not update the nest order step. If you want this as a step it must be modified to update the next step correctly in the tables.

You must run Release Outbound (even though it is not a step) prior to maintain deliveries. This is similar to the order entry steps.
