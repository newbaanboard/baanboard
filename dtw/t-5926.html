In Baan V it is not easy to follow the inbound line status. There is no clear overview session. In B52a there will be a beautiful session for this, but that is no solution for you.

In B50b/c there are two possibilities:

1) Check the warehousing order type and execute the procedures defined for it.

2) Use the three "...by activity" sessions. Change the index of these sessions by going to View, Sort by. Select an index starting with Order Origin.
The sessions are:

"Inbound Order Lines by Activity" (whinh2510m500)
"Inbound Advice by Activity" (whinh2515m000)
"Approvals by Activity" (whinh2511m500)
