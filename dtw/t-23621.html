Hi Luyen,

Each customer can have multiple addresses which can then be used to assign delivery of SO to correct plant/warehouse belonging to the customer. And of course, to use different delivery address in PO, you simply need to define your warehouses and use the warehouse codes accordingly in each PO (or PO line). I don't have direct access to B4 system right now, so I can't point out the session number. But the functionality is there
To display tax amount in Display Purchase Order session, you might need customisation at program script level, which can only be done if you have developer license
