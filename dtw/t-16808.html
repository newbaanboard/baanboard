In BaaN IV, 997 message is not present. If required you have to create one.

If the Direct network session is used to genarate EDI messages and Semaphore file processing is used by the third party EDI transaltion software to pick those generated files then the status will be Read by Relation.

The Functional Acknowledgements are being recevied by us in the RIP N READ format through emails. It is not integrated to BaaN.
