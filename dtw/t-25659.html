Hi Nicole,
It would be a bad idea to implement Location Control just to see inventory of customized item by warehouse. Location Control brings with it additional administrative headaches and certain change in the flow of data. If my understanding of a project is right, then each project is nothing but a new item with a new warehouse. Warehouse X in a standard item and in a customized items are two different entities even if they share the same name which, i guess you already know. The same is applicable to Warehouse X in Project 1 and Project 2. Baan treats it as separate entities because of which, i presume, there is no session to see inventory of customized items by warehouse.

If you still want to see the inventory of customized items by warehouse, i suggest you create something outside baan like through some reporting or query tool..
