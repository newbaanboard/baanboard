Hello,

Not sure about Baan IV but in ERP LN, you can generate a Production Order automatically upon Sales Order Approval. You need to make sure of the following.

1) Generate Production Order Activity must be present for the Sales Order Type
2) Also this activity must be selected as "Automatic"
3) Delivery Type field of SO Line = "Production"

Further, You can see the Linked Information from table tdsls402 (Field tdsls402.apdo)

Hi,
from where the delivery type defaults from on Sales order line.
I have created a SO line for item type of manufactured. but the delivery type got is 'warehouse'. it is not letting me generate production order & SO line not letting me to modify the Delivery type.

Check if "Allow Production Orders" checkbox is selected for the Sales Order type that you are using for the SO.
SO type will be defaulted from your "User Profile", but u can change it on the order after it is defaulted.

About the delivery type column becoming read-only, I guess this is because you are using a "Sales Order type" that allows only "Warehouse" Delivery Type. As there is no other Delivery Type allowed for your Sales Order type it becomes readonly. So If you use Sales Order type where "Allow Production Orders" checkbox is selected you will get to Change the Line Delivery Type to "Production".
Hope this info helps :)
