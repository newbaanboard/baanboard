First you should get planned orders for 120.

Now for the Sales Stats Question:
You can create a Display Code in Maintain Sales Stats Display Codes. In the mode field use Figures. You can then go into Display Sales Stats (Graphical) and you can use the find <ctrl> F to find the item number. Another option is to create Sales Stats Layout Codes and the Print Sales Statistics. In this session you can specify a single item or a range or all. You will get totals by item here. Hope this make sense.
