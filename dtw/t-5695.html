No.

Within BaanERP it is not possible to inspect manual order types. So, in case of a transfer (manual) order, inspection is not applicable. If inspection is part of the order steps, it will be skipped.

(By the way: technically a transfer order is considered as a manual order, hence also for a transfer order created from e.g. the project or manufacturing module, it is not possible to perform inspections.)
