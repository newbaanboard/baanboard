Hi,

Below given is the standard Baan solution,

1. Inbound needs to been Actualized (tdilc4204m000) or needs to be removed (tdilc4213m000).

2. Outbound needs to be removed (tdilc4211m000)

3. Warehouses have to be changed to warehouses without locations again (tdilc0231m000)

4. In case a ordertype has been used with "Generate Outbound Advise" (step 8), the procedure mask (Field: pmsk) of the table tdsls045 needs to be changed to 'n'. This only is required if the steps are:
1,2,8.

Also the orderstatus (Field: ssls) of the table tdsls045 needs to be changed from status 8 to 3 ("Maintain Deliveries").

5. Also the procedure mask of the table tcmcs042 needs to be updated if step 8 became part of the procedure. Step 8 has be removed and the other steps need to be moved upwards.

6. To be sure no incorrect data will remain in ILC and other tables , the session tdilc0250m000 needs be executed.

NOTE: The session tdilc0250m000 can only be executed, when there are no user on the system. This is very important to prevent errors occurring.

7. Finally the parameter ILC needs to be changed back to "NO" again. This is not possible by using the session tdilc0100m000, but can only be changed manually in the table tdilc000.
