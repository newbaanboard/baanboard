Hi Tanya,

I am not sure if I understand the question clearly. But I think that you are locking for a session like "Global (Un)Block Stock Point" (whwmd6230m000). The session can be started via run program, but is also available in the specific option of the "Items - Warehousing" (whwmd4500m000) session.

With this session you can block all available stock points for an item in one or a range of warehouses. Note that this will not block you from using the item on new orders. It is possible to receive the item on new stock points.

Another option is to use item signals. These can be entered in the "Item - General" (tcibd0501m000) session. Zoom from the "Item Signal" field on the "Grouping Data" tab. The opened session is clarifying. You can set a blocking status for different usages.
