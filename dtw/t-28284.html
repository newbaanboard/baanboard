Hi,

Did you get any other error message? It has probably nothing to do with your settings. This message is given only when the program did found some PO's to be commingled. What the program does is creating a new order with the lines of the 'to be commingled' orders and then it will delete the old orders.
One thing you can check before you try it again is: are there existing application locks on these PO's you try to commingle.
