It sounds like you are on Baan 4. Fields that could be used are Ref A, Ref B and Cust PO Number. In 5 you have extra fields. Cust Order Number, Orig. Doc Type (Can select Order,Shipment or Invoice), Orig doc number and Return Reason.
Another option is to create a standard text template that could be filled in by the person entering the order. The template could be called return info and filled in to save the info needed.

Approval could probably be handled by using workflow as the std. software does no allowe for approval allthough in 5 you could create a manual activity (Could be called Approval) that needs to be released and added to the order type. Give access to specific people to the session to release the manual step.
