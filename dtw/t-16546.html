We have situation where we generated 810S OUT (Sales Invoice) for one of our customer. The customer faced some problems while retrieving the message and therefore they are requesting to resend the message. Originally the message is generated with Printing of Sales Invoice (tdsls4404m000) but if I reprint the already printed invoice, no new message is generated.

Does any body know how can I regenerate the edi message for Sales Invoice?

For order acknowledgment I found the flag in table tdsls040.agen but is there a similar flag for Sales Invoices?
Chandra
18th September 2004, 00:26
Use session 'Restart Generated Messages' (tcedi7221m000) to resend the 810 EDI message. This session will be found under menu Communication under Electronic Data Interchange.

You should know Our reference number to restart the message.

We just store Sent messages (in EDIFolder\store_sent). If something needs to be sent again, you can just pull it out (edit if necessary to extract particular lines from the text file) and push it to the external gateway to be sent again.
