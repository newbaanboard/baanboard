It sounds like the cycle counting order for this item did not get processed. I would go to cycle counting and print all of the cycle count orders. You should only get open orders so you can leave the selection for cycle counting orders wide open. You may find this as well as other open items. Be sure to set "Include Already Printed Items" to Yes.
