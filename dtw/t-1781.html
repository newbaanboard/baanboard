There will not be much difference between Baan Internal & External EDI set up's. At the network directory level you will find directories appl_from, appl_to instead of appl_comm.
There are different EDI translators available in market. Gentran,Harbinger,netscape,etc. to name the few.
Iam using Gentran transaltor. Thay have prebuilt maps and scripts for standard Baan EDI messages.
