We have implemeted QMS module for Purhcase receipts. So when goods arrive they go to inspect location. Inspection orders are opened and sample test details entered and the Inspection orders come to the status "Processed". After this the "Maintain Approval" is done which automatically changes the Inspection order status to "Closed" in the QMS module (this is in table tcqms120).

In some cases we find that "Maintain Approval" step is completed and the Inspection order still has the status "Processed" and it is not changed to "Closed". In such cases the session "Check and closed standard inspection (tcqms1205m00)" also cannot be used to close the Inspection orders. Hence we have many such cases and all these Inspection orders cannot be "Closed" and taken to history and adding to confusion.

Is there a solution to this problem? Is there something wrong in the procedure done in the system to have such records. Please let us know!!!
