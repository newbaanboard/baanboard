Hi,
You set 'Number Group for Purchase Orders' in 'Parameter-1' of 'Purchase Order Parameters' (tdpur0100m400). Here in Parameters it will show you the actual & effective sets only.
Next you have to set the Number Serieses which you will set in,
'Purchase Order Series' for specific Purchase Dept in 'Purchase Offices' (tdpur0112m000) & for specific users in 'User Profiles' (tdpur0143m000).
Next when you place Purchase order, the system will pick the 'Purchase Order Series' with reference to 'Number Group' set in Parameters.
I hope this will serve the purpose. Regards.
