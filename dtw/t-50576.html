Change to Serial in Inventory can be done only if there is no inventory recorded against the item in any warehouse. In a scenario where the inventory is already present, the option to enable "Serail Tracking" is still possible. This would enable the possibility of specifying the serial during issue/receipt of the item in any warehouse. Enabling of serial tracking can be done using the session whltc5201m000 (Convert (to) Serialized Item).

Hope this helps.
