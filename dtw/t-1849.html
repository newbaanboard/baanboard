Jay,

I understand this is the way the text is handled when Sales Quotations are generated from PCF.

1.Create Sales Quotation Text : flag is Yes in PCF Parameter.

2. Whilst changing the Options for a generic Item with theProduct Configurator the following question will appear:"Generate sales quotation text for product variant? No"
If you choose" Yes" an Option Text is generated within the Text Manager.

This Text will appear in table tipcf510 with the same Text number as in the Text Manager.
Next session tipcf5250s000 Generate Product Variant Structure for Sales Quotations is used again a Text is generated which is the same but with another Textnumber.

This is the PCF Article Text which will be written into tipcs002.txta as an Calculation Parts Text.
Also registered in tipcs0110m000 Maintain Calculation Parts, second form.

With help of tttxt1400m000 Print Texts you also can find where Text used.

Mind the fact that the user has a Default Text group as defined in tttxt1113m000 Maintain Default Text Group by User.

Hope this info helps you.
