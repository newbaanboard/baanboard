The only other EDI maps that are currently available are in the sch0/1 (supply chain) and nas0 (NA automotive supply chain)extensions. These are the 824, 830, 843, 846, 861, 862, 866. The 867 is not currently supported.

Even if someone did have a conversion setup written, you will obviously require a BAAN DLL to process it.

The only other thing I could suggest is that if the report is outbound, you could write a session to pull the data from BAAN and output it in an EDI format (or something similar).

If the report is inbound, but not required to update anything functionally in BAAN, then simply read in the 867 report from the VAN, and set up a session / script to format it and email to the appropriate person. I have done that at several clients with strictly informational EDI messages not supported functionally by BAAN.

Hope this helps.
