Yes you can.

Use ttaad4134m000 Table data autorithations

You can enter this:

User: user login name
Company : Select all or a especific one
Table: tdilc410
Data Authorization: Not authorized
Condition : 1
Serial Number : 1
Field Name: logn
Operator: Not equal
Value1 : user logname
Value 2: Empty
And/Or/End : end of expression

Now you have to save this and convert to runtime user in ttaad2100m000,checking Session and Database Authorizations.

Next time user is logged, he will not be able to use any runnumber created by other user.

Hope this helps.
