All three tables are connected by 3 fields -

Order Origin, Order Number, Order Set.

You can check it using -

Session -> Warehousing Order(whinh2100m100) -> Specific -> Satellite Sessions - > Now, if you double click on individual sessions, you will get the related fields for connection.
