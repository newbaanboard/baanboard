Hi LuyenDT,

In order to be able to print the sales order acknowledgement and the packing slip you need to have printing the sales order acknowledgement and printing the packing slip defined as a step in the procedure linked to your sales order type (check tcmcs0142m000). Besides that you need to have reached that point in the procedure (check tdsls4533m000 - if 'ready for' for the order is set to a step before the SO-acknowledgement, it won't be printed yet. The same goes for the packing slip). Next to that check your customer data (tccom1101m000): if the field 'acknowledge by exception' in form 6 is set to 'no' there'll be no problem but if it's set to 'yes' the step will be skipped in case you don't have an 'acknowledgement code' assigned to the sales order line.

VAT should be printed on the SO-acknowledgement.

Hope this helps.

Frank
