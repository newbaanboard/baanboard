Hi,

If its a customized session, yes, its easy. If its a standard session, there is not an easy way to do that. First, I will suggest you to add tccom100.nama field directly on the form and check if its getting any value. If yes, then we are good.

If no, then in that case, you will have following options -

1. Add Customer Defined Fields(CDF) to the table
2. Add a Menu Option to show additional Information
3. Create a wrapper script for the session

You will need to decided a way which is feasible for you.
