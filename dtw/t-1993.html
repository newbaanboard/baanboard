What is the purpose of field "Freight value" in the session "Receipt header" (whinh3110s000)
Kees de Jong
11th June 2002, 22:26
Not used

The field freight value is not used within the Baan application. Actually the session "Receipt Header" (whinh3510m000) does not give much added information. The session can be used for registering purposes.

Main reason to use this session, is by filling the required fields, which can serve as default values for the Receipt session. Example is the packing slip number.
