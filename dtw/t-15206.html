Hi

For fixing the problem go to session tdinv1550m000 (assuming you are on Baan IV). See under Order type - if it is sales / purchase- then it may be that somebody has maintained an order with negative qty (return order). If it is resulting from production it may be that material is to be returned back to warehouse from production due to incorrect production transactions etc. Based on the analysis the corrective action can be taken.
