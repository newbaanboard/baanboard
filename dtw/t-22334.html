When you zoom from Order Line, you can see the "Recalculate the order amount" (or in similar name. I don't have access to Baan system at home, hence cannot spell the exact name).

But that session is available to be run manually for each order. What you can do it is create a new session which in turn will call the existing session in the background for the range of orders you want to recalculate the order amount.

PS: But if your pricing in based on order date and the order date falls in the old bucket, then nothing will change.
