If you never purchase a "manufactured" item, I think there are two method in BaanERP5.0c:

1: Don't input any purchase info for "manufactured" item in Item Purchase Data, then you cannot find the "manufactured" item while you select an item while order entry.
If this doesnot work in IVc4, you may

2: In Tools Authorization, select "Table File Data Authorizations" for the purchase role, set that while the item type field data is "manufactured" item, the role cannot read/find the item.
