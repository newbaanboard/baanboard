Even if you issue by main item from the sales order line session you can still make sure the people in the warehouse that pick, pack and ship will be able to see all the components on their documents.
In session tdsls4100m000 (Sales Order Parameters) you can set the field 'Manufactured Item or Components on Picking List' to "Both" and the field 'Manufactured Item or Components on Packing Slip' to "Both".

Perhaps in this way you can still utilize the system without many modifications. Only when you invoice based on the components.

In b40_c4 it is not possible to issue components that are set to Lot Price in tiitm0101m000. This is hard coded in the script.
