If the Time-Phased Inventory Check parameter is checked, BaanERP performs a time-phase inventory check on the availablilty of suffieient inventory during the items's order lead-time. If this parameter is cleared, BaanERP checks the availability of the inventory by subtracting the allocated inventory from the inventory on hand. If inventory is not sufficient, BaanERP displays the following Options:

Planned inventory movements
inventory by item and warehouse
Alternative items
Generate Purchase Order Advice
Generated Direct Delivery
Create back order
Adjust quantity ordered
Restore old situation
No action

You can use these options to ensure that the sales-order line can be delivered.
Wording from Baan Sales Order Procedures. Hope this helps.
