If you are frequently sending items to subcontractor you can think of having WIP warehouse for subcontractor. Send him all material on Replenishment order. You can set DRP set up to plan RPL orders automatically. You can send him copy of sub-cont order as a intimation to start work for particular prod. order. All material can be Backflushed once operation is completed.

Many advantages are there if you explore this possibility !!
