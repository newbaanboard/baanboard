Hi Popeye,

only specific sessins may be used as order steps. The reason is that the session has to update order step specific data of the respective sales order lines, and only a few sessions in Baan do that. So you cannot use an arbitrary session as an order step, unfortunately. The sessions that may be used should be predefined as order steps already.

Popeye,

Use session 'Patch Objects after Error Solving' (ttiex3226m000) to patch your new session and then add it in the order steps. Now you will not get the error.

Uli, is right. Only few sessions in Baan is capable of updating the order steps after executing that step. You need to add this functionality to your new session if you want to make your session a mandatory step.
