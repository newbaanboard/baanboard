HI Jenny,

As per my knowledge there isn't standard session to get a consolidated status report against a Order Line having multiple delivery lines.

You can check the status of individual lines from the session tdsls4510m000, here you can also check the Line Type whether it is a "Order Line" or "Order/Delivery Line". If a line having the type "Order Line" this means this a actual order line against which there are multiple delivery lines.
