Hi,
You set 'Number Group for Purchase Orders' in Parameter 1 of 'Purchase Order Parameters' (tdpur0100m400).
The Number Group is the whole bucket of different 'Purchase Order Series' you intend to use. Hence in Parameters it will show you the actual & effective sets only.
According to use you have to set the Number Serieses which you will set elsewhere, like.
You set 'Purchase Order Series' for specific Purchase Dept in 'Purchase Offices' (tdpur0112m000) & for specific users in 'User Profiles' (tdpur0143m000).
I hope this will serve the purpose. Regards.
