As you suspected the purchase requisition module is included in the automotive (we run ags2). It allows you users to place electronic requisitions without item no, supplier and prices. These can be added later on by the purchasing department. An approval matrix can be set up and the requisition can then be converted into a PO. It is not a fancy tool but for companies with paper requisitions it is a step forward.
