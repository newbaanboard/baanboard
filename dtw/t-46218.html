Record type 1 means order line created/changed
Record type 3 means invoiced
Record type 2 means cancelled

Sequence number is used when 2 or more transactions are written top history for the same transaction time.

When changing an order line first a counterposting is made with a reversed quantity. secondly the changed line is written to history. (this explains entry 2 and 3 in your example)
