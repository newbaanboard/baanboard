Rush order definition in SAP....
Rush order type is used to create deliveries immediately as and when you save the sales order. For example with normal order type, when you save a sales order you get "Sales document XXXXX is saved" message. But if you save a Rush sales order, you get, "Sales document XXXXX is saved, Delivery YYYYYYYY created" message. So that you can directly call the delivery document in VL02N transaction and proceed.
