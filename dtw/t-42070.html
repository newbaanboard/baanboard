You can change this date in whltc110m000 form 2, Lot date field.
In receiving procedure, you have 2 ways to define lots:
-by generating lot based on a mask, and in this way the inventory date is inaccessible (is system date)
-define lot manually and in this way you can modify the default inventory date

Valuation should not make a difference. It is indeed relevant for lots but also if you use the outbound method FIFO or LIFO. The inventory date will be used while generating outbound advice on what materials should go first. Using FIFO or LIFO will result in storage of item inventory by date and thereby increasing the number of records.
