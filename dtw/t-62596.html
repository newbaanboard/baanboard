In LN the data that you need is in the following. Hopefully, similar data is available in your version of Baan:

Inventory Receipt Transactions - Table whina112
Inventory Receipt Transactions - Session whina1512m000
Print Receipt Trasactions - Report whina1412m000


The Transaction Date along with the 'To be Consumed Quantity' should give you what you need for an aging.
