In whina2200m000 you as user specify how safety stock should be calculated.
==========
Calculate Economic Order Quantity (Camp) (whina2201m000)
Q = Sqrt ((2 * D * F)/I)
Q= economic order quantity
D = expected annual issue
F = order costs
I = inventory carrying costs per inventory unit
==========
