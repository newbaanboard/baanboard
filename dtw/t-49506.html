Freight management requires the Rate basis number to find out the costs associated with the transport. This rate basis number is in turn used in Pricing module to set up costs associated with the carrier.

This is set up in the session "Rate Basis Numbers by Attributes"(fmfrc0120m000) which can be accessed from session "Rate Basis Numbers"(fmfrc0110m000). Note that the Rate Basis number can be defined for any one or combination of attributes.
