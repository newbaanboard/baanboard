This is on Infor ERP LN 6.1 FP7.

The material ( 6 no's ) transferred to Consigment Owned warehouse for Customer against the Consignment Replenishment Sales Order. Part of the Quantity ( 1 no ) is billed to the Customer thru the Consignment Invoicing Sales Order. Remaining 5 no's Qty is still present in the Consignment Owned Warehouse.

This 5 no's Qty will not be used and hence need to return back to the main warehouse of the factory. But the Inventory Transfer is not allowed in the system because the Consignment Owned Warehouse is not allowed in the warehouse transfer order.

Please note that the REturn type of Consignment Replenishment Order Type is not allowed for creation in the system.

Please let me know the process to bring back those 5 no's qty from Consignment Warehouse back to my main warehouse.

In Order to Return consignment owned inventory, please do the following:

Step 1

Create a Sales Order with Sales Order Type as "Consignment Replenishment" with a negative ordered quantity about the quantity to be returned.

Step 2

Approve the Sales Order.

Step 3

Release the Sales Order to warehousing module. It will create a warehousing order to transfer the Consigned Owned Warehouse to the Normal warehouse.

PS: Infor KB Article for similar issue (for Consigned Not Owned) : 1384791
