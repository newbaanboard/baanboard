Warehouse Inspection can be used with or without QMS. I assume you want to use without QMS.

You need to set up the parameters in Warehouse Inventory handling for warehouse inspections.

Next you can enable the inspection for Sales or Purchase or Production areas. The set up will vary accordingly for each of the above.

Warehouse inspection is applicable for incoming raw material procure from Purchase order. in order to activate this, you need to setup the inventory handling parameter for warehouse inspection and also in warehouse you need to activate the option "handle rejected goods". afterwards item purchase data you need to select as inspection is applicable. this will defaulted the inspection applicable on POL.
your warehouse order type should have inspection activity and at the time of receipt, the option inspection should be checked.
