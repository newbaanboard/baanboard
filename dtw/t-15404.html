Hi,

If the item and supplier is set for inspection, the PO line will also be set for inspection. This will block the inventory in the application system from being issued before it is cleared by QC in the Approval session.
There are integration entries to/from inspection coming from purchase receipts.
Of course you have to match this with controls in the physical system.
