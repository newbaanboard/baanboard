Hi,

Error messag 1202 means Database is locked. You can derive the error message as below:

Error code - 1000 = DB Error

Here 1202 - 1000 = 202. And as per baan error 202 means Database is locked. This is for Oracle and SQL database. I don't have any idea about other database.
