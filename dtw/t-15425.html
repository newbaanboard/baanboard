If you wish to create a price list that you can use for multiple customers, you must first create it using tcmcs0134m000, maintain price lists. Here you get the opportunity to link it to a currency.

Next you use maintain customers to link specific customers to a certain price list.

Then you create the prices by price list & item in tdls0104m000.

If you wish specific prices for an individual customer you would create them in tdsls0101m000.

There are a few other sessions in this area as well: for example tdsls0113m000 maintain prices by item. Go into one of these sessions and follow the help text to see Baan's method of determining a price.
