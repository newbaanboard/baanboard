1st April 2003, 19:02
You do need to confirm backorders. Use the above session to do so.
I found Baan's functionality to be a bit thin in this area, because there is no standard baan report that will show you everything that you are expecting to receive.

You can print purchase order back-orders (tdpur4430m000)(prints only things that have had the backorders confirmed.)

You can print purchase orders by delivery date,
but it does not show what is still outstanding to be received.

I ended up writing custom 'Expected Delivery List' reports, that incorporate this data including the various planned delivery dates, changed delivery dates, current planned delivery date fields in the purchase order lines.
