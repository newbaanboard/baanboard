Andy,

did you evaluate the return order functionality? You will have to set up an order type with the return order falg set to yes.

The rest of the procedure is pretty much like normal order handling, however with negative amounts / quantities. If you use ILC, you will have to manually maintain a (negative) outbound advice, to get the material back on stock. Printing the invoice for such an order will print a credit note.
