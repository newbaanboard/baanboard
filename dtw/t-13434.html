The through functional and technical knowledge is required to develop a new EDI mesage in BaaN. Developing the outbound EDI message is easier as compared to Inbound EDI message.
All the tables from which the data need to be pulled has to be found first. Based on that the new conversion setup need to be developed for the new EDI message. Also the new EDI DLL in lines of standard EDI Dll, needs to be developed which will pull data based on the new conversion setup.

Finally the pregen session/ report need to be developed, which will prepare EDI message i.e. creates a record in 'Messages to be generated' table. Create the EDI setup for this new EDI message. This generates the new outbound EDI message.
