Evert-Jan Bosch
29th August 2005, 16:44
You can use manual activities in the order types session, just as Paul said. However, don't use the session code, but take a self-defined string like "Approve Order".
At the moment the sales order is ready for this activity you can go to session tdsls4501m150 (Release Manual Activities). Search for the activity "Approve Order" and select the sales order and via specific menu option you can release the manual activity.

Good luck.
