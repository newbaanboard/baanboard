Hi !!!

There can be two possibilites in Sales Order

1) Fully Pending ( means no delivery has been made agt that Order)
2) Partial Pending ( Some Qty has been Delivered agaist that SO line)

For Fully pending Sales Orders either u make the Ordered Qty as Zero , or delete that SO Line

while to close the Partial Pending Sales Orders , Change the Back Order Qty to Zero.

U can also do that same byu writing a session , but this session should update all the tables like

inv150
itm001
sls41
com11
back order table ( forget its table no)
sls045 ( for back order qty)

there are other tables also + some more tables needs to be updated if PCS , Transporartion etc is implemented.
