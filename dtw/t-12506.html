rupertb
6th August 2003, 16:48
You create a signal code in session tcmcs0118m000 and you assign a blocking status to the relevant section. NOTE: blocking sales/purchase will prevent users from creating/maintaining orders - however orders already in place for the item in question can still have deliveries/receipts recorded against them.

Unfortunately, no you cannot put a signal code on a customized item in PCS on 4c4. The signal code field is not in the PCS Item Master session. You might try creating the item as a standard item with a signal code and then bring it into PCS on your BOM. since, eventually, the actual POs etc end up back in standard modules that may work. But, I'm not sure. We've just barely started to look at possibly using PCS.
