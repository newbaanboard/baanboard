EdHubbard
7th November 2003, 22:34
You need to run session tdltc0130m000, convert non-lot item to lot item. This converts the item master data to say "lot controlled" and will give any existing inventory in the same warehouse, the same lot number.

The session does one item at a time.


Good luck
Ed
