Hi,
use separated item code with special item group. Or you should create warehouse for this stock. With this separated warehouse you can detach value of consignation as well.
With appropriate item group you can detach not only inventory values but transactions as well.
If you have purchased (paid) stock and consignement with the same item code then you can try lots with zero lot price.
Unfortunatelly, in baan IV you can handle and detach cosignement value manually...
