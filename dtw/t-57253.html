In the MMT of Purchase Orders (tdpur4100m900) you can enter a project specific item. If the project segment is filled of the item then also tdpur401.cprj will be populated. If you want to buy a standard item for a project then just open the line (tdpur4101m000). There you can find the project field on the first tab.
The project field can probably also be added to the grid via personalization.
