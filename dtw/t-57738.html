The "Maintain Deliveries" is used only when ILC is not implemented and if you have parameter set to confirm the deliveries. The quantities cannt be decreased,but a partial order can be cancelled in this session. If you have ILC implemented the change the outbound quantity, otherwise use the maintain deliveries as suggested inearlier post.
