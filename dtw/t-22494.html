Firm release

A firm release means that the order will not be checked again on occurance of the blocking reason.

Soft release
When an order is released with a soft release, the user can continue to work with the current order step. But if the user wants to continue with a next order step, the check on the occurance of the blocking reason is checked again.

E.g. an order line is blocked during order entry and soft released. When the user tries to release the order to warehousing, it is checked again whether the order line should be blocked.
