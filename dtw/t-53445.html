Do you have location control (ILC) switched on? If so, you should not be using hard allocation as the Generate Outbound Advice order step will do the same task when it runs.

If you are not using ILC, then Hard Allocation should come just before the Picking List step. When it runs, it makes a specific commitment of inventory to demands up to the available stock limit. Order priorities are observed if there is not enough stock to go round.
