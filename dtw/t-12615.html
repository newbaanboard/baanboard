Should have no problems using GTM going from Non-Nettable to Nettable.
It can cause problems going the other way (Nettable to Non-Nettable) if you have items in the warehouse with MPS/MRP order system.

I would also recommend that you rebuild planned inventory transactions after you have carried out the change.

Hi,

The warehouse field for nettable is not accessible because there are items with stock in this warehouse. If the stock of such items in the related warehouse is cleared, then the system allows to change this field in the session "Maintain Warehouses". The system checks for data in the table "tdinv001" "Maintain Items by Warehouse".

However, changing through GTM should not give any problem.
