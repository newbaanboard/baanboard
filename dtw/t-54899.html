HI,

You can use the standard ERP LN's help.

But still if you have any doubts, Follow the below mentioned Steps:

1. Define Matrix definition (tdpcg0110m000)
Here you have to define the Matrix Type(Purchase Price) & the attributes on the basis you want to define the criteria. There are lots of criteria available while defining the Matrix Definition.

2. Define Matrix Definition (tdpcg0120m000)
Define the sequencing (priority) of the Matrix definition defined by you.

3. Define Price Matrix (tdpcg0130m000)
Select the Matrix Definition defined by you, and the PRICE BOOK to be used for the defined Attribute.
define the Exact Combination of Attributes on the Purchase Attribute Tab.

In this Similer way you can define the Discounting Matrix Also.
