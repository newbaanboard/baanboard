Hi,

You can use tdsls4409m000(Print Sales Order Lines) session and customize the standard report to show only released lines.

For excel, use excel device to print or just create a new session to print the required data. Wont take much time.

Or simply use whinh2400m000(Print Warehouse Orders) session to print released sales order lines.
