Since summer 2002 you can overdeliver a salesorder.
There is a new parameter in warehousing, where you can maintain the percentage for overdelivering.
This is only for new generated warehouse-orders and only for sales.
For production orders baan will not be able to overdeliver a warehouse order from type material issue,
also not in the next versions !!!
