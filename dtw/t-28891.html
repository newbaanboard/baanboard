Hi,

First of all, check the item warehousing data (session whwmd4100s000) for this item. Select the tab "Issuing" and check the value of the field "Outbound Priority". It should say "FIFO". If it doesn't, you can change this by making use of the session "Define New Outbound System By Item" (whwmd3202m000) to change the outbound priority.
Also, check if the "Minimal Picks" option is selected in the "Inventory Handling Parameters". When selected, this option overrules FIFO and tries to generate minimal picks.

Hope this solves your problem.

Best regards.
