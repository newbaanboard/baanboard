This happens in Maintain Deliveries when a user selects "Global" on an order marked as "ship complete" whose current delivery line has not had its Maintain Deliveries step completed.

If you say yes, it does some checks to make sure it can deliver all of the remaining lines at the same time. If so, it performs the delivery and/or confirmation of the delivery.

I suppose the question is there as a check in case someone forgot the order was ship complete.

I hope that helps.
