First you have to create a session for Confirmation of Acceptance of Order Acknowledgement.

Then add this session in Maintain Order Steps (tcmcs0139m000) after this add the step in Maintain Order Type (tcmcs0142m000).

Before adding step to order type, make sure pending orders for the particular order type should be deleted & re-booked again after adding the step.
