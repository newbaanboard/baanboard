Inventory commitments are only created for requirements within in the supply lead time of the item ('commitment date'). When the requirement is outside this lead time, there is enough time to either buy or make the item.

In FP2, using inventory commitments require the setup of lead times.

In FP3, it is possible to set the "Inventory Handling Parameters" (whinh0100m000) option "Generate Inventory Commitments based on Planned Delivery Date". With this option the inventory commitments are generated based on the planned inventory transactions, without using the commitment date.
In FP5, the parameter is moved to the "Inventory Planning Parameters" (whinp0100m000) session.
