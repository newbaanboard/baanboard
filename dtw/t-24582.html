Hi,
ASN is nornaly used in EDI environment. You can use EDI set up to generate ASN (when packing slip is printed). You can then take the record from EDI table(generated message) and can generate your custom message if you are not intending to use EDI. However you will need to set up the required customer in EDI. If you donot intend to do any EDI setup then you will have to customize at the reqd session- "Release Outbound" or "Print Packing slip".
