When you create the item first time, by default Item supplier data is created so that you have at least one item supplier data. Then you may manually create more Item Supplier data based on number of suppliers for that item.
When you change the Item Purchase Data of pre-existing item, the system would not know as which of the item supplier information should be changed among many item supplier data. Hence you have to make the change manually.
If you have always one supplier for an item, then you can develop a small customization when you change the item purchase data.
