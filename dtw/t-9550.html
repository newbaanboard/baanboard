Once the sales order has been financially processed you can delete the order using the process delivered sales order session. Until then you can add sales order lines to the order.
At this point you can only see the sales order info in the history tables.

In Baan system, each sales order line is treated as seperate sales order in the sence each line can have different status. The sales order line is closed at Process Delivered Sales Order status. After this the line can not be modified. In Process Delivered Sales Order session, there is an option to Delete the SO's / SO Line's, once the complete SO is deleted (all lines deleted) one can not add any lines to the sales order.
