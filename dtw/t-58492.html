Hi Puneet,

You can check/put the below mentioned solution to resolve the issue :

"Delivery date does not fall within the availibilty of warehouse calendar : F of warehouse 11”

(1) Check the Calendar code with availability type with working hours
(2) Put the Calendar code in Role of Sold to BP
(3) Put the Calendar code in Role of Ship to BP
(4) Put the availability type of the calendar in Sales order Parameter
(5) Put the availability type of the calendar in ‘Warehouse Master Data Parameter

"Reason date ................ (time zone NZ) is not available in Calendar F & availability type STD10"

For above problem, with above said mentioned solution, check the period table, there might be ‘Time Zone NZ’ is not linked with period table code, calendar, and availability type.
