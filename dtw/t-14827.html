Maintain delivery addresses for the customer thru the session "tccom1102m000" and attach the address code to specific delivery adresses before delivery (say. adress1)

if Customer changed the delivery addresses to address 2 after delivery, it is not possible to change the code in specific delivery address in tdsls04101m000, whereas we can change the adresses in the session tccom1102m000 - name field and adresses for the same code.

Hope this will works.
