Hello,

It's not possible to link multiple cities to a single ZIP code due to the fact that tccom136 table is having Zip Code as an Index field.

When we had the same issue, we have created the city codes with a suffix "-01" (Ex: 1831-01) that are having a same ZIP Code and attached a default city code in the tccom136.

During the creation of an Address, user can easily zoom & change the required City code.

Hope the information is useful to you.
