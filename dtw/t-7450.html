One way I can think of is to have the parameter block warehouse for cycle count turned on and to do a cycle count on the warehouse (should zero all stock in the warehouse first). This only works if you have an entry for all your items in the tdinv001 table and if you have more than 999 items (and there are not many of us who don`t) then you would need to create multiple cycle counts as that is the limit for one cycle count order. not a pretty solution but should work.

Might help to know the reason for wanting to blocking the warehouse.

Darren,
for us there are two main reasons to block a warehouse:
1) you are using a warehouse for a certain time - typically an offsite warehouse
2) you are not able to change the warehouse type or whether it's nettable or not - the only thing you can do is to set up a new one (and try to block the old :) )
