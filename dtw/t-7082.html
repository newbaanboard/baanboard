9th October 2002, 15:16
You do not have to create inventory history. Transactions (Sales/Issue) needed for forecast is saved on the system. Demand forecast in Inv Module is only used to update the Inv Parameters - SStock, Re-Order etc.
Calc demand forecast - Enter info in Item Master (Forecast Method etc) and run Cal Demand Forecast. Cam also run Global Update of Inv Parameters and you can specify the formula used to update SStock. Switch Update inv/Order Data to yes in the Item master.
