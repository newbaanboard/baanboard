Hi,

As soon as you check the box "'Blank' Availablity Type Used' in session tcccp0110s000 (Calendar Codes) in the <Structure> tab you will start seeing this kind of problems. Not only in tdsls but also in Service when creating service orders.
The only way to resolve is to set up a blank availability type as suggested by BAAN Support (undocumented feature).
