26th March 2008, 16:44
Hi Ulrika,

I never worked with internal EDI (only to external BP) but I think it will not work with the Baan standard EDI setup regarding the messages.

The conversion setup for the messages in BaanIV and LN is complete different.

Example:
The message of type ORDERS has 6 levels in BaanIV, whereas LN use 14 levels with mich more fields resulting from the complete different table structure.

So without some own customized messages it will not work.

Regards,
Juergen

For Internal EDI to work, the Buy/Sell companies should be on the same Baan application server. The out EDI conversion setup should match the Inbound EDI conversion setup. The EDI message name may not matter, but the EDI file name created by outbound EDI message should be acceptable to the inbound EDI message.
