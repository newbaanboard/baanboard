Hello,

In EDIFACT, the composite data element is made up of two or more pieces of data (known as components) which form a single data unit. Typically the first data element is the value, which is being qualified. The second data element is typically the qualifier. These are typically ID (code values) fields. The qualifier gives additional definition to the value.
