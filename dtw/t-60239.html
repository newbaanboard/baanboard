this problem comes when any one of the unit against which your are maintaining the records are not come under main units which we define in parameter.

example if in parameter i define unit for weight as 'gm' so system will always ask reference and conversion against 'gm' for all rest weight units like 'mg, kg, lb' etc. but if i want to maintain conversion factor between lb and kg in that case i need to give either item code/item group in "maintain conversion factor" then only system will allow.
