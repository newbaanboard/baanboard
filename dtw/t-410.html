Dmitry, you could try Table Field Data Authorization (ttaad4135m000). For example, if you want to prevent moving items from warehouse "W1" to warehouse "W2", you can set condition:
Table filed: tdrpl105.dwar (Destination Warehouse)
Data Authorization: Not autorized
Conditions:
1. rwar equal "W1" AND
2. dwar equal "W2"
