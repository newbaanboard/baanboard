Hi,

It is not recommended to change the Inventory Unit of an item if the transactions are already done. Because, once they are done, multiple tables are updated and economic stocks are booked.

Changing the unit can create a huge inventory corruption. The other way can be to duplicate the current item and create a new one with the required inventory unit and start all over again. You can block the other item then.
