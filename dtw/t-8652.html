believe that if you make the item Lot Controlled (Lot for Lot) your problems are solved. each receipt is a new lot number and that specific lot then has an expiry date. next receipt is a new lot.

but - be careful using lot controlled because you then have to specify from which lot you are issuing to work orders, etc.
