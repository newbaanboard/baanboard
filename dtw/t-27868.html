In Baan IV I can achieve it quite easily
1. Create Item X as phantom and then define A,B, C its components.

One can always sell Phatom or component whatever the situation demands.
If you want to create Item A, B and C from X then I would create a rework order for ItemX and it would call from Item X, Item A, Item B and ItemC. Item A, B and C would have negatibe quantity. When material is issued, it would decrement the stock of Item X and increment A, B and C.

You may have to do some customization, in order to make process smoother. Bit I don't see any issue, here. I guess in Baan V you may be able to better.
