The easiest way to load intial inventory balances is to:

- Create an Adjustment Order
- Add Adjustment Order Lines
- If the item is serialized you will be prompted for a serial number
- Process the Adjustment Order

If you have thousands of balances to load and you have an excel or text file with the initial balances, you could:

- Manually add an Adjustment Order
- Use Exchange to populate the following tables:
- Adjustment Order Lines (whinh521)
- Adjustment Order Lines Lots and Serials (whinh526)
- Then manually process the Adjustment Order

If you decide to use Exchange, attached are the entity relationships that you would need to consider.
