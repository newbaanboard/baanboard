Also, depending on how you are defining the orders to which the engineering costs apply you may be able to use the additional order cost functionality.This would mean that the cost item is automatically applied to the sales order perhaps at a price of zero. Since additional order costs can be set to always appear on a particular order position is would then be possible for finance to have a report which prints out these positions depending on order status thereby ensuring that the correct value is assigned before invoicing.
