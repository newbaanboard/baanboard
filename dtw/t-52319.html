hi,

please check what is the searching crietrea of tax in tax parameter session tctax0100m000.ERP Ln uses various methods to determine the tax code and the tax country of taxable transactions for example standard,exception by contry and so on.After selecting your parameter,check the master setup of tax in following session.

1. Tax Handling (tctax0138m000)
The tax code for the tax country and item combination, if you specify one.
2. Item - Sales (tdisa0101s000)
The tax code for the item if you specify one.
3. Tax Handling (tctax0138m000)
The tax code you define for the tax country and an empty Item field.

One important thing please go through the Taxation Documents.
