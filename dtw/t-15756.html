Hi,
When u maintain sales order system is inserting a records in tdsls040, tdsls041, tdsls045, tdsls050 and tdsls051. some localize tables are also updated at the same time.

As soon as you maintain the delivery tdsls045 table is populated with the delivered qty. When you run calculate taxes session it will update session tdsls901.

While printing the sales invoice tdsls049 table is updated with the order and invoice reference.

PDSO consider table tdsls901, tdsls049, tdsls040, tdsls041 tables and localize tables for integrations purpose.
