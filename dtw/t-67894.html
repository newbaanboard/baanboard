In one of my implementation we have used pricing control as follows
1. Set the Matrix definition for Purchase Price with Matrix Segments as 'Buy From BP' - Suppliers.
2. Set the Price books for Purchasing in Price books sesseion(tdpcg0531m000), each price book will act as a supplier price book and will contain list of item as per their defined/OEM or Individual Pricing.
3. Create Price Matrices for the combination of Matrix Type as 'Purchase Price' and Matrix Definition created in step 1.
4. For each combination define Purchase Attributes in session tdpcg0130s000.

This will give you seperate pricing for same item with diffrent suppliers, you can customize the attributes as per your requirement.
