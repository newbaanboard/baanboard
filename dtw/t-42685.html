With session tdsmi1201m000 it is possible to global add features to Customers. Is there an easy way to global remove these features?

In Dutch:

Oude kenmerken zijn niet meer van toepassing en dienen te worden verwijderd, maar aangezien deze collectief zijn toegevoegd (dit kan wel in standaard Baan), zou het heel veel tijd besparen als deze ook collectief verwijderd kunnen worden (maar dit kan niet in standaard Baan).
Het gaat om gegevens in tdsmi101. Deze zouden op selectie van kenmerk (tdsmi101.cfea) verwijderd moeten kunnen worden bij alle debiteuren.
