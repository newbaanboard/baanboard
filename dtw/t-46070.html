In Enter Material Issue for Production Order you need to enter a negative quantity in the issue column for all returns. Be sure to zero out the Subsequent Delivery if you are not going to outbound more. Go to Maintain Outbound and enter each line individually with a negative quantity. When all return lines have been entered then you have to Release Outbound. Outbound will not be generated automatically for negative lines.
