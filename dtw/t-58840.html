Hi,


The system will generate outbound advice base on the Outbound Method defined in the Items-Warehousing session. Please refer to example below.

Example:

Item X Inventory
Location A = 100 pcs, Inventory Date = January 15, 2010
Location B = 90 pcs, Inventory Date = January 1, 2010

Order Qty. = 80 pcs

Outbound Method and Advice that will be generated by the system
FIFO------>Location B = 80 pcs
LIFO------>Location A = 80 pcs
By Location------>Location A = 40 pcs, Location B = 40 pcs
