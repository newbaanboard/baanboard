tdinv0250m000 is for rebuilding planned inventory transactions and would not affect the inventory records in inv or ilc tables.

tdinv0252m000 would do that. but please be careful when running this session . it checks data in the following order

Lot
ilc
inv
itm

so if your lot / ilc inventory records are wrong, then all the inv and itm records will get updated with wrong inventory.
Also please run it after logging out all the users from the system. some wrong advise on this issue caused inventory corruption for us.
