Dear Baanbab,

It is possible in Baan Vc through session Change Prices/Discount after Receipts (tdpur4122m000).

But you can cannot change the price/discount, if :
 The order quantity is zero.
No receipts are yet recorded or receipts recorded with 'Final Receipt' as 'No'.

The order is a subcontracting order, but the status of the production order is not correct (the order is not yet in production or is already closed).

One or more receipts for the order line are already registered in the order history. In this case you are too far in the procedure to change prices and discounts.

Purchase invoice matching has already taken place.
