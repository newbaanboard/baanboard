tdssc019.plvl and tdssc019.ptyp are filled in from the subsession tdssc0119s000 - Maintain Delivery Packaging which is accessed from the subsession tdssc0118s000 - Maintain Planned Deliery Lines.

The values in this session default from the table tdssc065 - Sales Contract Packaging so if the user does not access the session to amend the delivery packaging then these values will be same as in table tdssc065.
