It's possible to link the subreports on a formula field. The formula field could be your numeric field converted to a string field through the use of Crystal's To_Text() function. You're the best one to make the decision as to which field is converted how. Anyway, the goal here is to make the two fields match in type so that a link can be made.

There is some information in chapter 23 of the Crystal Reports User's Guide in a section called Using subreports with unlinkable data and Linking to/from a formula field. The user guide is usually included documentation with the installation CD.
