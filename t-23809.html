Pretty sure the AFS will not work in b2, unless some unique patching has been done (I have not seen this). Look for the tools library ostpapihand in your tools directory ($BSE/tools/ttB40_b/ottstp directory), it's not on our b2 system. That at least is required to run, and there is not a version for b2 readily downloadable. You may contact Baan to see for sure, but the api documentation says it's supported in IVc and higher only.

An alternative is the wrapper programs, these work on any version. You can create a library that calls session logic without having the source code with the wrapper programs. These are 3rd party pieces, such as QKEY, RMCgen (My company..) and the shareware piece 'rush' on these boards (look in the code and utilities section).

Or upgrade :) The b2 - c4 upgrade is not too painful, depending on how much you have modified, and if you are a company that uses the tools, c4 is greatly improved over b2.
