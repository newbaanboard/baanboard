Dear Baan User,

We like to introduce the CRP  Central Resource Planning solution. This Solution integrates the resource scheduling system Open Plan with several modules of SSA / Baan.

This solution is created to provide a centralized planning solution for TP Project, PCS  Project and SFC Manufacturing driven processes. The main objective is it to provide companies with solid planning data for the financial control of Projects and large Manufacturing orders. Project margins and cash flow can be obtained based on dynamic data coming from Open Plan combined with static budget and cost data in Baan.

Baan Project and Manufacturing orders can be linked in a Planning Network using resources from OPP and Baan to schedule everything within their availability parameters.

Supported by the Multi  Project Capability of Open Plan, it is possible to link active Projects and Orders with planned Orders to verify the future Resource Availability.

The solution is created by S&T Services and Solution / Slovenia and PMS- Project Consulting / Germany. The solution was created in cooperation with Skoda Transportation, Plzen, Czech Republic.

The Resource Scheduling Software Open Plan is created by Welcom Software Technology International USA, the leading Manufacturer of Project Management Solutions.

Detailed description of the CRP  solution being available. We hope that you will find this solution interesting. If you have any queries please do not hesitate to let us know.

Yours sincerely

Ulrich Hofmann

Hi Vladimir,

as Martin has stated: the CRP Interface is a new solution, since the privious Baan - Welcom interface is not supported any longer by Baan and Welcom.

Besides this, our approach in this interface is very different from the old one. The basic idea is similar to the first one, made for ABB / Boeing several years ago. But the link does not only connect Baan project with Open Plan, but also PCS - orders and now procurment positions as well. Furthermore, it also works with OPD (Open Plan Desktop), not only with OPP Open Plan Professional) from Welcom.

Then, for this interface clients have contracts and manitenance and we add new functions required by users in order to improve the exchange of data, costs covered by manitenance contracts

This interface works based upon Baan Functions, it does not need Open World Licences. Its installed in about an hour. On the Open Plan side, we use also standard import / export functions, just to keep everything as simple as possible.

If you need to know more, ask Martin or me for details.
