ION desk - Connect - File Templates - Add - Give Name and description, select format type XML and on Settings you can Upload a sample file or just copy paste part of the UserData with your XML content in the SampleContent box. Press Generate Metadata.

In the process further you will be able to select this document template and map fields, etc.
