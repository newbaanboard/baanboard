Do you mean with the "Integrated Baan Reporting" from Crystal? If so, then yes it does. There are a few steps that you need to do to get this to work properly, but yes it will work. Baan Support has a solution to help set this up and follow the installation documents provided with the IBR.

Obviously you can also use Crystal Reports to connect to the database with the ODBC driver or using the Baan Native Driver from Crystal.
