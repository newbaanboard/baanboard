Have you installed BaanERP 5.0c SP6 or higher?
Is the portingset 7.1c.02 or higher?

Passwords can be changed with the utility BnIRTConfig.exe in the \Crystal Enterprise Solution for Baan\IBR folder. Don't change these passwords via CMC.

Is there anything logged in the \IBR\ss_log folder? Note that you have to set the following registry value to 1.
\HKEY_LOCAL_MACHINE\SOFTWARE\Seagate Software\Baan IRT\Logging Level
