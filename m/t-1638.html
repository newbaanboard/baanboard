Here's an email we received recently:

Dear SupportEye user,


We like to inform you that a new version of the iBaan SupportEye portal is
available now. You can download this new version from solution 120511 on the
Baan Online Support website (www.support.baan.com).

This new version contains enhancements and additions to the scans. For
example, the database scans now include SQL Server. Also known issues have
been solved and the new iBaan SupportEye portal is fully compatible with
Netscape.

Another enhancement is a change in the licensing of iBaan SupportEye. The
expiry date of the new version of the portal is no longer fixed, but you can
obtain a license file at any time via the Baan Online Support website. This
means that only when major upgrades are issued, you will need to install a
new version of iBaan SupportEye.

Should you have further questions or need assistance using SupportEye please
read the information provided in solution 120511 or contact your local
support center.
