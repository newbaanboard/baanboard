Hi folks, FYI, the following solutions can work:
Set up the project or service, and in Finance - Chart of account, under the Miscellaneous tab, set the integration indicator to e.g. Project Costs for an expense (P&L) account (but don't switch on the integration flag).
You need to set up the sundry cost item in Project (as well as the element, activity, etc).
During data entry in GL, when you choose that account, after putting the amount in, the Project Sundry Costs session tpppc25911m000 will pop up, and you can enter in your costs.
Finalise the journal as per normal in Finance, then go to Project module, and approve the Project costs.
If, for example, you set the project up as being cost plus, then go to Sundry Cost transactions to be invoiced tppin4561m000 and release to Central Invoicing.
That cost can then be included onto an invoice via standard Central Invoicing.
I would appreciate any other suggestions of how to do this, without having to set up Projects or Service!
