From the Edit menu in the chart manger choose "Category +" and "Category -".

You may have to choose "Draw" from the edit menu to get it to refresh the first time, this looks like a little bug.

You can set the default number of periods to show in session "Maintain Runtime Resources" (ttdsk3160m000" under package "tg", resource set "enterprise", set the option "Nr of periods in chartmanager"
