Hi,

You can simply use whwmd215._compnr.

For instance, if you only want to know if a table exists, just write:

domain tcbool table.exists

select whwmd215.*
from whwmd215
where whwmd215._compnr = :company
as set with 1 rows
selectdo
table.exists = true
selectempty
table.exists = false
endselect

Good luck!
