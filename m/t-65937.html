Because DAL do not interact with the User-Interface this is normaly done by the UI-Script.
A common way for this is that the DAL use a global variable and provide a buisness method to set this global variable which can be called by dal.start.buiseness.method from the GUI-Script.
You can use the ttstpbaandoc programm for the dal to explore the buiseness method of the dal or you can use debug level and then call the session and see what happend in program flow after you answered the question with yes.
