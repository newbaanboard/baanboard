24th January 2014, 10:32
Hi Rashna,

It depends on what you want to customize.

If you want to make a customized version of the WarehousingOrder BDE you can use the following steps:

* Import the standard BDE from the backend via the Business Interface from Runtime Repository importer. The result will be a linked library in your project.
* Open the library in the interface explorer and search for the business interface implementation
* Right click on the interface implementation and choose specialize, use the same name as the standard. The result is a interface implementation in your project which you can customize.
* You can generate code for this implementation.

Regards,
