Solution Description
English Session for changing user password
TOOLS VERSION(S):
B40_c, 7.1_a, 7.6_a2, 7.6_a3

SITUATION IDENTIFIED IN:
"Change Password" (ttstpchangepw)

SITUATION DESCRIPTION:
ERP LN contains functionality to handle password aging, but does not contain a session which enables a user to change the password, when password aging is not enabled.

SOLUTION DESCRIPTION:
Session Change Password (ttstpchangepw) is introduced. With this session a user may change his password. Session is present in menu User Management/Miscellaneous.
