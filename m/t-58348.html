Krish,

If you are savvy programmer, you can create a device with your own 3-GL program which can take the report output, convert it into PDF (open source tools or Baan's XML-PDF option), take that output file and use CMF/SMTP functions to email out with attachment.

Alternatively, an easy way is to get the Nazdaq products listed by George :)
