I've got a question: Is it possible to change the default texteditor in ERP LN?
No, you cannot change the default Text Editor from notepad. There are some old threads on the forum, where they have renamed the notepad.exe to mynotepad.exe and other editor's exe to notepad.exe

Everything opens up in Notepad, but i want to open it in Notepad++
If you are referring to the Editing the Program Scripts, then you just have to create a new entry for Notepad++ in ttams1150m000 (Development Parameter Template) and set it for a user in the User Data, execute run-time & re-login to use it.
