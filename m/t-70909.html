If you are on the latest Tools release you can also use Report Designer to make changes to classic reports. One of the the options is to include a company logo in the report. The logo is to be stored using session ttaad1100s000.
If you have the correct Tools release while printing an option Personalize Reports will show. Starting the print to report will also start the personalize option. Using a company logo in reports is one of the options that is supported.
