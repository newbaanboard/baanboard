Ivo Ladage

Hi Srinivas,

Yes, Worktop 2.2 is available on BaanIVc4. Officially, you need Service pack 9, but you can already make it work on SP7 with some additional PMC solutions.
Worktop is a new User Interface on top of ERP, replacing the old menu browser, desktop manager, DEM browser, process viewer and BDN toolbar. It also hosts an internet browser, keeps an history of recently performed actions and supports central deployment.
So basically, it's an extended interface for BaanIVc4 (as well as for 5.0c, 5.2a and 6.0). If you need more info; just ask me.

Regards, Ivo

Ivo Ladage
16th December 2002, 13:58
Hi Paul,

I think what you mean with "decluttering" the menu is just an option in Worktop; the option is called "Only main process" (in the View->Tree menu) which is by default off. The same option is available in the DEM browser and when you enabled it in the DEM browser but disable it in Worktop, that explains the difference.
Anyway, as I said before Worktop replaces the DEM browser on 5.2a and later completely and therefore any functionality that was available in the old DEM browser is also available in Worktop 2.2 now.
If you have any additional Worktop related questions or want more information about the cluttering issue, please ask.

Regarding the problems you had with Worktop 1.x; 1.0 was released a long time ago so i would recommend to try 2.2 instead.

Regards, Ivo
