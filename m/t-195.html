Hi PJ,

Here's most of them, briefly described. This is an extract from the BaanIV Technical Manual (some of the text has been edited for clarity):


General
audit_srv6.1: Server which handles logging of actions on the database.
bdbpost6.1: Program to create database tables from a sequential dump.
bdbpre6.1: Program to convert database tables to a sequential dump.
bdbreconfig6.1: Program to reconfigure database tables.
dpt6.1: Test program of database actions. Program for internal use, not for the end user
gcommand6.1: Database test program. Program for internal use to solve problems, not for the end user.
refint6.1: Function to check the referential integrity of database tables.

BAAN Base
bclock6.1: Synchronization process which activates bmanager6.1.
bflusher6.1: Flusher for data and index buffers from shared memory to disk.
bisam_srv6.1: Database server, using the BAAN Base access method.
bmanager6.1: Manager process for one or more flusher processes. It receives messages from the user and sends them to the flusher(s).
tbase6.1: Administration tool of BAAN Base-server.

BAAN Base TP
btam_srv6.1: Database server, using the BAAN Base access method with transaction processing.
lflusher6.1: Flusher process for flushing log buffers from shared memory to the disk.
lmanager6.1: Manager process for lflusher processes.
smanager6.1: Process to swap the contents of shared memory to the disk
tbase_tp6.1: Administration tool of BAAN Base TP-server.
tclock6.1: Synchronization process which activates tmanager6.1 and watchdog6.1.
tflusher6.1: Flusher for data and index buffers from shared memory to disk.
tmanager6.1: Manager process for one or more flusher processes. It receives messages from the user and sends them to the flusher(s).
watchdog6.1: Transaction monitor

Informix
inf_admin6.1: The shell script for doing administration.
inf_exec6.1: Auxiliary file used by the INFORMIX driver.
inf_install6.1: The program used to install the INFORMIX driver.
inf_maint6.1: This is a program internally called by 'inf_admin' for doing various functions.
inf_srv6.1: This is the INFORMIX driver itself.

Oracle 7.0/8.0
ora7/8_admin6.1: The shell script for doing administration.
ora7/8_deinst6.1: The program to undo the installation of the ORACLE Driver.
ora7/8_inst6.1: The program used to install the ORACLE Driver.
ora7/8_maint6.1: The maintenance tool of the ORACLE Driver.
ora7/8_srv6.1: Database server for the ORACLE RDBMS.

Sybase
syb_admin6.1: The shell script for doing administration.
syb_install6.1: The program used to install the SYBASE driver.
syb_maint6.1: The maintenance tool of the SYBASE driver.
syb_srv6.1: This is the SYBASE driver itself.

Logic driver
bshell6.1: A virtual processor between the user interface, database driver(s), application(s) and the operating system. This program makes applications, including BAAN IV Tools, independent of the operating system. The bshell functions as a shell between the application and the operating system.
bshcmd6.1: This command can be used to change the log facilities of the Logic Server (bshell) or kill one or more bshell processes. The actions done with this command are performed to a running bshell.

Development
bic6.1: Program compiler.
repgen6.1: Report generator.
std_gen6.1: 4GL-program preprocessor.
bic_cstub6.1: Converts a BAAN library to a C-library.
bic_info6.1: Shows object information.

Licensing
brand6.1: Program to authorize an environment. You must be root to run this program.
hostid6.1: Program to print the host identification number.
licd6.1: Daemon process to watch over the license. Ethernet software must have been installed before using the license daemon, because of using sockets.
licmon6.1: Monitor to retrieve information from the license daemon.

Printer Management
filter6.1: Program to translate BAAN native codes into printer codes used by the pdaemon6.1 program.
lp6.1: System spooler interface for the pdaemon6.1 program.
pdaemon6.1: Daemon process to print requests from the environment. This program must be started as user 'root'.

Shared Memory
shmmanager6.1: Manager program for shared memory.
shmtimer6.1: Daemon process which stores and updates the current time in shared memory.
shmvalues6.1: Program to generate entries for the "shm_param" file.
srdd_init6.1: Program to initialize shared memory.

Network
client6.1: Test program of the client part.
fs6.1: Program to handle remote file I/O. With this program it is possible to approach files on another system. On the other system you must put the program fs6.1 in the ipc_info file.
ipc_boot6.1: Program to start remote processes
server6.1: Test program of the server part.

TRITON Super Set
tsscomp6.1: Compiles and checks TSS information file 'tss6.1'.
tsscvt6.1: TSS conversion filter.
tssinfo6.1: Gives information about current TSS settings.

Miscellaneous
binput6.1: Program to read input. This can be used in shell scripts.
bput6.1: Program to set or print terminal settings.
compress6.1: Program to compress data. (Equivalent of the UNIX compress command)
encrypt6.1: Program to encrypt passwords, which is used to fill the "r<user>" file.
kermit6.1: Serial communications program. (Equivalent of the UNIX kermit command)
mirror6.1: Program for demonstrations and courses.
nlsedit6.1: Editor to maintain input and output conversion tables for Native Language Support.
popup6.1: Program to handle popup screens. This can be used in shell scripts.
sort6.1: Sort program. (Equivalent of the UNIX sort command) It is possible to use the environment variable BSE_SORT to specify a path where temporary files will be stored during the sort process.
sum6.1: Sum program, used to calculate (and patch) executables
