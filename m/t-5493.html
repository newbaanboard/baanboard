Hi Ravi,

I am not aware of a direct option to do something like that -

But here are some leads to explore -

1. $BSE/lib/users/unprao

in this file you have the entries as -


timeam: am
timepm: pm
dateformat:MMDDYY
datesep:-
company:General_Data


2. Menu - BaaN ERP Tools -> Application Configuration -> Miscellaneous -> Languages - ttaad1510m000

When you maintain a language you can also maintain a specific date format.

If LTS(Language Translation System) is implemented on your BaaN installation, maybe you can create some dummy language code with a different date format and the users can be linked to this language.
