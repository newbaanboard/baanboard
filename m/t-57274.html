Some of this is really dependent upon who you ask. Your profile says 4c4 and your questions is on LN. My understanding is DAL is recommended on LN over AFS. Since I still use 4c4(No DAL) I can not answer how hard it is to learn. AFS is just a method to run sessions in the back ground and feed data to them. I use them to consolidate multiple sessions into one session and for mass transaction processing. So for example instead of the user generating and releasing outbound - we have one session to do all the steps for them. We automated several of the processes using the AFS funtionality. In the AFS/DDC/OLE forum you can see a sticky thread and read the documentation attached.
