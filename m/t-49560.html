1) Go to the particular users terminal and login with a user that has administrative rights.
2) Click on File Menu and choose Run Program.
3) Execute ttadv5245m000.
4) Select the packages (multiselect) for which you want to download the help file and then click on Download Button.

I hope this is what you asked?
