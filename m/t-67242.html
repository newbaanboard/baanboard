Hi,

Check the file dtt.pd under $BSE\tools\dd directory. Check if the length of domain ttaad.user has been converted to the required one or not i.e. search for ttaad.user in the file and check the length. If yes, then you can try manually changing the c & u files in $BSE\\lib\user directory i.e. changing the name with required length and try to login.

Else, you might need to run a Create Runtime for Application Server. Can't guide much on that part as I am not sure about the sessions and structure about your version of Baan.
