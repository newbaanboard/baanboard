Hitesh Shah
2nd March 2004, 08:55
Restarting license server licd6.1 again initializes license to 0 . There is some time gap between the stopping and restarting of the licence server . During this time no fresh login will happen . There may be problem with opening new sessions in existing connections. And finally if already existing Baan connection remains unutilized for some time and number of connections after restarting license service reaches maximum number , the existing user with unutilized BW will be refused to work with Baan when he/she resumes work.

So you can resort to this mechanism with these inherent issues. Alternatively you may restart licenses at every evening if possible.
