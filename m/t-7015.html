There are some other things to check:
1. registry entry (see attach.) in HKEY_CLASS_ROOT set as You see. For "Baan.Application" key "CurVersion" must be an entry that is active in your BECS configuration (in my case "Baan.Application.5"). "CLSID" must be the same number as in case of entry "Baan.Application.5".
2. Check to be sure that in BECS you have active-default configuration

Also you can try next: start Baan, then start Excel and try tocreate template. But normally problem is in regedit.
