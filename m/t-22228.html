Hi,

It is possible if you use OpenWorld. Through your webpage, you can call the java proxy function with necessary input values, which in turn through OpenWorld Adapters will call Baan. You can get the return values or error documents.

This is something which all e-products from Baan (like E-Sourcing, E-Procurement) does.
