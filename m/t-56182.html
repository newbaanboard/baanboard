Hi baan999,

Well, usually you can access reports in session "Reports - ttadv3530m000". If the report has a report script - you can open the script (I guess you must have obtained the license for using sources/scripts from Infor). Select "Layouts", "Report Layout Fields" and "Input Fields" in the "Specific" - menu to get further details of the report. To get all details, you have to analyze the script of the session which starts the report as well, because it prepares the data to be finally displayed on a linked report...
