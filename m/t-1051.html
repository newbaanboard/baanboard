If you set up your text groups to use an external editor (see session tttxt1116m000, Maintain Text Groups with External Editor), you can then use notepad to do standard Windows copy/paste.

There is some copy capability in the built-in editor, but it is more primitive.

You have to use escape keys, such as Escape-U. These are defined in your 'Text Editor Options' (session tttxt2100m000).

But the copy is very clunky. It expects you to use Escape-( and Escape-) to indicate the start and end of the block you are copying and then Escape-) to paste. To make matters worse, it can often cause errors with your text window, as it puts your text into Read-Only when you do the copy and undoes the Read-Only when you paste.

As 'evesely' has already stated, it would be recommended to use an external text editor (such as notepad)... (Interesting Baan consultant you must have, I take it they are not technical...).
