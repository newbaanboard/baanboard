Prerequisites for SSA Baan IVc4
Install tools solutions
Download the PMC solution 140164stnd from:
ftp://ftp.support.baan.com/updates/B40_c/140164stnd
Note:
This solution needs a number of pre-requisite solutions. Therefore, you mustrun the following sessions:
-> Generate FTP Script (ttpmc221m000).
-> Download, Scan, and Connect Solution (multi-Level) (ttpmc2210m000).
After these sessions, you can install the solution on the backend with session Process Solutions (ttpmc2101m000). The folder where the PMC solution is installed from contains the following three XML-files:
-> XFE_model_domain.xml
-> XFE_model_form.xml
-> XFE_model_session.xml
These XML-files must be placed on the backend in the directory:
-> $BSE/lib/XML_MODELS
If this directory does not exist, you must create it.
Install SLM client
On the backend you must install an SLM client. Refer to the SSA® License Manager 2.1 - Installation and Configuration Guide (U7631 US) for more specific information.
Register Webtop-enabled sessions
Before you can load the cache, you must run the session Register Webtopenabled sessions. Not every SSA Baan IV session is suitable to run in the Webtop. The session Register Webtop-enabled sessions adds a property to the ERP sessions, which indicates whether a session can be run in the Webtop. Take the following steps to initialize this property to the correct value:
1 In Worktop, enter the command Run program on the SSA Baan ERP back end.
2 Start session ttconwebtop and then click Continue.
