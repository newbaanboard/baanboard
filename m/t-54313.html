In a command window, you can navigate to your $BSE\bin directory. Execute the command "ntbshell.exe -v". This should output the portingset version for you. Also, if you navigate to $BSE\bin in Windows Explorer, right-click over the ntbshell.exe file, choose Properties from the context menu, and go to the Version tab, you should be able to find the portingset version there as well.

The easiest way to determine SP level is to open session ttaad0500m000 - Version Scan Tool. The SP level is on the first form.

I guess the session Version Scan Tool is only available after you install certain Service Pack. I dont recall which SP as in all our new installations we use the SP batch utility which brings you upto date to the current SP.

You could try running the session number ttaad0500m000 from the Run Program option under File menu to see if its a hidden session of sorts...

To check what SP you have right now, you could try running Process Solutions {ttpmc2101m000} which you would find under
Software Validation=>
Miscellaneous=>
Product Maintenance & Control (Recipient)=>
Maintenance

This again is only possible if you have pmc installed.
