Hello,

from BaaN IV on charts are createt by using the modul ttchm.

New charts are maintained with ttchm1100m000.

To programm dynamic charts the two includes are used:

Systeminclude : bic_chartman
Include : tcmcs0000 (chart main functions)


|*Description:
|* This include contains several functions that can be used in a program script
|* to communicate with the chart-manager.
|* These functions all start with "CHART". They use the basic chart-manager
|* functions that all start with "chm".
|* The CHART-functions have been created in order to make the interaction
|* between program-script and chart-manager easier.

e.g.
function long CHART_OPEN( ... )
function long CHART_DRAW( ... )
function long CHART_DATA_IN( long seriesno,
.........

Not to be discouraging, but at least in BAAN IV, it is not very easy to create and thereafter maintain sessions for charts.

My recommendation would be to write a session that will create output in a CSV format, and then use EXCEL or similar to create the charts. The session can be written in such a way as to move the CSV file directly to the user's computer in (say) the C:\temp folder. From there, the user can device a macro or such to create his charts.
