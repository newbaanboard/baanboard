Hi Sandy,

I am currently working on implementation of QMS module. Your question is too general, and it is difficult to answer in this post for me. It is very simple module and there is no much set up is required.

First set the QMS parameters. (All fields are self explanatory)
Then you need to set the QMS master Data.
The Inspection can be done for
* purchase receitps
* routing (during the operations)
* BOM (for component issue)
* after production quantity
* before sales delivery.

The help in the field "quantity to inspect" in report production orders completed gives you the idea.
If you have any specific questions, please feel free to contact me.
Hope this will give you a clue.
