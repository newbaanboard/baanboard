Hi Francisco,

Yes, i think i can help you further. The problem seems to be the same as some problems with cyrillic characters which we encountered in Worktop.
Important to know is the version of tgintworktop (backend library used by Worktop for DEM integration) and the version of Worktop itself.
The solution for the cyrillic problem has been solved in Worktop 2.2.22 and tgintworktop version 8 (for 5.0c). You can find the version of tgintworktop in Worktop itself; File->Properties dialog.
Additional language problems (but as far as i know only appearing in chinese, Japanese, so Multi Byte languages) will be solved in Worktop 2.2.25.

What Worktop version and tgintworktop version do you have, and do you have 5.0c?

Regards, Ivo
