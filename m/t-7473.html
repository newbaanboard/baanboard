Baan IV was never designed for Internet capabilities, Baan V was designed with "internet enabled" capabilities. iBaan is supposedly re-designed Baan V with better ability to be interfaced with the web.

In real life, only custom programming will really make any Internet intergration possible. Thus, it really wouldn't matter what version of Baan, they all can be adapted to for web based usage.
