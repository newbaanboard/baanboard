We use Crossroads/RMC's RMCtime integrated solution for:

- Time, Attendance and Payroll Interface
- Direct Labor Reporting - Hours Accounting for Production Order Hours
- Indirect Labor - Hours Accounting for General Hours
- WIP Tracking - Report Production Order Operation Complete
- RMC Middleware for controlling Shop Floor Bar Code Data Collection network (Intermec wireless bar code readers)

We use Hidox FormDesigner for printing bar coded ERP LN documents:

- Production Order Bills of Material
- Production Order Routing Documents
- Sales Order Packing List

Both products are fully integrated with ERP LN, perform as advertised and have good support people.
