Dear tools123,

We are currently testing solution for the same requirements; thin client for warehouses and shop floor. Our test system uses Win2K server terminal service and WinCE based thin client. We separate the terminal server from our Baan server but connect them to the same switch. We simply install B5c client on the Win2k terminal server and running Menu browser or DEM browser on the thin client is no problem. No need for extra software. Everything runs just as if it's a normal thick client

However, we decided that both the B5c browser might be too complicated for WH and shop floor personnel here and are developing our own middleware
