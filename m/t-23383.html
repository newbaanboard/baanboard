Hi ale,

Not sure if my answer is still relevant for you ;)

You need not need to try removing application locks for this problem.

Just run the session tfgld1116m000, set the status to "Free"
Go back to maintain transactions. There you see that the status of that particular transaction type is changed to "Free".
Just enter into that transaction type and come out.
The status is now changed to Ready.
Now you can finalize the batch

Tell me if this helped you (if not already resolved ;) )
