There are three broad categories of errors. Errors generated by the operating system (numbers 1 - 99), errors generated by the Baan database system (numbers 100 - 900) and errors generated by the underlying RDBMS (numbers 1000 and upwards ). In case of an RDBMS error, the actual (native) error code can be obtained by substracting 1000 from the Baan error code. See Infor ES errors and messages.and An error code of 1000 and above indicates a native error of the database. The value and the meaning of these errors are database specific. To get the actual error code of the database involved, subtract 1000 from the error number. Example: bdberrno=1604 is database error 604. Please consult the RDBMS vendor's documentation for more information. Then Error 13514 translates to Error 12514 for which you can try to find a solution (https://stackoverflow.com/questions/10786782/ora-12514-tnslistener-does-not-currently-know-of-service-requested-in-connect-d)
