When you are using webtop and when you start the session the first time, baan will create an XFE form dump for the session. Webtop needs XFE form dumps for each session to start that session. Now, Baan takes a lot of time to generate the session the first time. I am not sure of the root cause of the problem you guys are facing but you should try generating XFE form dumps for all the sessions using webtop.

XFE form dumps can be generated via the Webtop Administrator's login. The documentation for Webtop will tell you how to generate forms. The process takes anywhere between 2 and 4 hours to finish generating form dumps for Webtop.

This should resolve your problem.

Once the form dumps are generated, there is a particular directory in which xml files are saved. These xml files are used by webtop to start the sessions.
