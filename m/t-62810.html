Yes, LN FP7 has an integration with MS Reporting Services. For a number of native reports new MSRS designs are delivered by Infor and creating your own designs is also possible. See for example KB http://www.inforxtreme.com/espublic//EN/AnswerLinkDotNet/Solutions/ViewSolution.aspx?SolutionID=22947060 .
Note that you can also create your own reports directly on the LN database, but then you are missing the business logic that is already in the native reports.

i create both but probably 75% of the time i use ssrs reports. one big advantage is i can create a report linking all databases in my company, not just the LN database. in fact my LN db is DB2, i link it to sql servers within the org. SSRS also has the so-called subreport where you can drill down from the main report. i can take advantage of colors for status of sales orders for example. logo is easy to implement. one report i have done is for kitting on Production orders. the report will spit out the item numbers including the pictures. the pictures are not stored in LN database but on a separate sql server, i just linked them. SSRS has almost unlimited capability. it can do dynamic sql and if you're handy with VB you can create very complex functionality. i also advise to couple this skill with SSIS (both available from Visual studio) and you almost have unlimited capability in terms of reporting and data transformation. for example i read xml data from slm license server as a graph. this is accomplished through SSIS and read by SSRS.
