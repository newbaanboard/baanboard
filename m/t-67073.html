Hello there

if you want the week number in your report then you can use the function utc.to.week. The usage is given below



function long utc.to.week(long utc.date,ref long week_dayno,ref long week_yearno,ref long week_num,ref long hours,ref long minutes,ref long seconds,ref long year_no(optional))


Returns 0 if it is successful otherwise it returns zero.



This function will return the week_number out of the year of 52 weeks. If however, your date is not in utc function, you can change it to utc through date.to.utc function.
