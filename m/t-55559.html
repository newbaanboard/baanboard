You are using a very old BW as well as Worktop. The users company is read from their u files.

Have you tried putting in the following command in the command option of your BW:

-set PACKAGE_COMB=<package combo> -set BSE_COMPNR=<comp#>

The package combo is only needed if the package combination differ from what the user has in their u file.
