you can also use this standard dll:

long tccom.dll0350.date.time.to.utc( long i.days, long i.seconds, ref long o.utc )

Expl. : Get the utc day/time for a given date.
Daylight saving time error will be checked.
Pre : -
Post : -
Input : i.days - day
i.seconds - seconds
Output : -
Return : 0 OK
-2 daylight saving error
-1 other error
