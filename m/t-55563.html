We have found a solution for this issue on Infor's support page (infor365.com). That's what solution 233753 partially says:

"To use Infor ERP LN Studio (Application Studio, Business Studio, Reporting Studio) some products have to be licensed. Those products were not mentioned in the documents that were made available for Studio. When you used one of the Studios without the appropriate licenses, it was not always clear why the Studio did not work."

"SOLUTION DESCRIPTION:
The following table shows for each Studio the products that have to be licensed:

Studio: Product ID: License Type:

Reporting Studio 7046 - Connector for ODBC/JDBC Server"

So, it is really necessary to obtain new license to be able to develop/customize reports using Eclipse/BIRT - tech.
