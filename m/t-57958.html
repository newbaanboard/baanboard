You can change the format of date in windows Control panel.

Even if the cell is formatted yy/mm/dd, you still enter it as m/d/y. This order is defined by the Windows Regional and Language Options control panel settings. Examples: the English(US) setting for dates is m/d/y, English(UK) is d/m/y. You can customize it, but changing this setting will change it for ALL Windows programs.

Excel can display the date components in any order, but the entry order does not change with it.

Just googling on Excel Date Setting and you will get enough information on this.
