I've been offline for a while (busy busy busy), but got several questions in my PM about printing to Zebra printers in Baan.

We print receive labels in our warehouse on Zebra S600's. (These printers are connected to our network with HP jetdirect boxes.)
For this to work I have taken the following steps:

1) Create a label with Codesoft in which I have defined the text fields we needed. Then I've printed it to file with the Zebra driver. You get a file with ZPL-code.

2) In this file I had to modify this file to insert "Fieldmarkers". This fieldmarkers are used in the Baan report.

3) The next thing to do, is sending the file to the flash of the printer.

The ZPL-file looks like this:
^XA
^MCY
^XZ
^XA
^LRN^FWN^CFD,24^LH0,0
^CI0^PR2^MNY^MTT^MMT^MD0^PON^PMN
^XZ
^XA
^DFE:pick.zpl^FS
^POI
^LH36,0
^LT32
^A0N,28,28^FO487,76^CI0^FN09^FS
^A0N,28,28^FO487,156^CI0^FDMag:^FS
^A0N,28,28^FO667,36^CI0^FDBenodigd:^FS
^A0N,28,28^FO671,116^CI0^FDGeleverd:^FS
^A0N,28,28^FO12,36^CI0^FDArtikel :^FS
^A0N,28,28^FO132,36^CI0^FN01^FS
^A0N,28,28^FO12,76^CI0^FN02^FS
^A0N,28,28^FO12,116^CI0^FN03^FS
^A0N,28,28^FO292,116^CI0^FN04^FS
^A0N,28,28^FO12,156^CI0^FN05^FS
^A0N,28,28^FO12,192^CI0^FN06^FS
^A0N,28,28^FO487,116^CI0^FN10^FS
^A0N,28,28^FO551,156^CI0^FN11^FS
^A0N,28,28^FO551,196^CI0^FN12^FS
^A0N,28,28^FO487,196^CI0^FDLoc :^FS
^FO659,16^GB3,240,3^FS
^A0N,28,28^FO667,76^CI0^FN13^FS
^A0N,28,28^FO487,36^CI0^FN08^FS
^A0N,28,28^FO280,156^CI0^FN07^FS
^PQ1,0,1,Y
^XZ

4)In Baan I've made a report that generates lines with ZPL-code. This ZPL code calls for the label in the flash memory and passes the values of the fields. The printer then generates the layout and prints (what else).
The Baan output looks like this:
^XA
^XFE:PICK.ZPL^FS
^FN01^FD450.999.00054 ^FS
^FN02^FDIC SMD GS9035ACPJ PLCC-28 ^FS
^FN03^FDGS9035ACPJ ^FS
^FN04^FDGENNUM ^FS
^FN05^FD ^FS
^FN06^FD ^FS
^FN07^FD ^FS
^FN08^FDOntv: 248318^FS
^FN09^FD ^FS
^FN10^FD ^FS
^FN11^FD010^FS
^FN12^FD ^FS
^FN13^FD 78,00^FS
^PQ1,0,1,N
^XZ


Advantage of this solution is that I don't need BOW and I don't print using BWprint.

I hope this is of any use.
