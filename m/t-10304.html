Hi Gunther,

For Worktop you will also need to have BW installed. And you also need to create the BWC files as you were used to using BW. After that, you will see the configuration files in Worktop and can select them while creating a new Worktop document.
Note that Worktop supports central deployment, which means that using Worktop you do not necessarily have to install Worktop and BW on all clients. Instead you can share one centralized installation.
