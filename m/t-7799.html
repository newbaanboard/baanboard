Don't know about add-on packages, but that type of business could probably be coped with by using Baan Project; it would be simple enough to set up projects with a WBS element to cover the hire service to the customer. The duration of the element can be set to whatever you want it to be, there is a budget that can record what standard equipment is being used, and installation payment schedules can be set up to bill the customer.
