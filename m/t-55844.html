The documentation cd of 5c contains some difference studies 5b-5c.

I have done this upgrade in the past, it's not that complicated, Infor supplies software to facilitate it although it's a bit old (doesn't reflect latest SP's).
This software extracts some tables, than you import your tables with the 5b dd, recreate them with the new dd and the migration software updates some tables again from your export out of 5b.

5b is no longer updated, that should be enough reasons to upgrade.
