Hi Srinivas,

The message ttaad21014 is "Worktop not supported by this bshell" which means that the porting set you are using is too old to be able to run Worktop on. I investigated the porting set issue further;
At least porting set 6.1c.06 or later is required for Worktop. This porting set was released in August 2000. So to make everything work correctly, you should update your porting set. Futhermore, the current Worktop version also requires 6.1c.06.05 or later, but probably the next Worktop version will also be available on 6.1c.06.02 or later.
You said before that you are using porting set 6.1c.03, which was released in 1998. I strongly remommend this porting set.

Regarding the session data entering you mentioned, Worktop is just a navigation framework. Nothing has been changed in the Baan sessions itself, it just provides an easier navigation framework around the sessions and around other navigation metaphors (DEM, BDN) and external applications (Web pages, MS office).

Regards, Ivo

Hi Srinivas,

Ok, well with this old porting set it is possible to use the Menu Browser normally. The only thing you really need the porting set for is for the component authorization. So to be to also see the Dynamic Menu browser and Process Viewer in Worktop, you will need this update. If you just want to have a 'sneek preview' there is one workaround to enable these components. If so, let me know.

With the web page integration i mean that an internet browser is hosted insight Worktop, not the other way around. So you cannot run Worktop from within a web browser, but Worktop supports 'central deployment' which enables centralized installation and configuration of Worktop. This is very suitable within a LAN network but not over a telephone line. However, you can choose for a centralized configuration and a local installation. In that case you install Worktop and BW locally, but keep the configuration (things like hostname, BSE, username, etc. etc.) centrally.
So there are possibilities but i need more specific information to see if there is a suited solution for your particular case.
