As per Document code: U9581A US;

Infor ERP LN contains 8 predefined BIRT reports, which are in the /infor/bse/reports folder.

Template URL to run these reports
http://<hostname>:<port>/reportviewer/frameset?__report=<reports_location>/<report.rptdesign>

Example
http://sx32001:12000/reportviewer/frameset?__report=
/infor/bse/reports/InventoryValueByWarehouse.rptdesign

Available reports
OpenProductionOrders.rptdesign
OpenReceipts.rptdesign
OpenShipments.rptdesign
YTDPurchaseByBussinessPartner.rptdesign
YTDPurchaseByProduct.rptdesign
YTDSalesbyBusinessPartner.rptdesign
YTDSalesbyProduct.rptdesign
