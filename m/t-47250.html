Q: Any news on the BIRT integration with LN and Baan IV?
A: You can find the BIRT intergration with the Enterprise Server 8.4 ISO from the download center.

Q: Is there any integration with Eclipse? Can it be downloaded?
A: Reporting Studio 8.4 integrates with Eclipse and it is already released with Enterprise Server 8.4. Please see the document Infor Reporting Studio 8.4 Installation and Configuration Guide.

Q: Are there any pre-built homepages for webtop? Can homepages be built by users?
A: There are pre-built homepages for Baan ERP LN using ES 8.4. They are delivered via solution # 221768. Home pages can be assigned to a specific user (according the role).

Q: Are there any demos, manuals, or overviews of the BIRT integration?

Infor WebTop 8.4 SP1 for ERP LN - Installation and Configuration Guide
