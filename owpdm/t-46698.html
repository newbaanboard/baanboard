You do need to read some documents, as suggested by Dave.

The configuration file basically stores all the open world configuration data in central location. Actually it is your choice, you can use a file based configuration storage system or you can use an LDAP.

You can put an empty configuration file and name it whatever you want and then put the path to it in owconfig.properties file. Then when you will open Central Configuration Console, it will automatically format the file.

In any case you will need to go through the documents.
