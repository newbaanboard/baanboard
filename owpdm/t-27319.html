Just wanted to update this thread to mention that this problem was fixed. The problem was that I had different ldap.xml files on the server and client. The server was using bidisockets for the connection whereas on the client side I had it setup to use sockets. Since there was a mismatch in the protocol's the data wasnt being transferred back to the client.
