The Baan OpenWorld Adapter partly consists of Baan components (packages tm/cc), so for this a working Baan environment is needed. This means it is not possible to install the adapter on a local machine not running Baan.

If you want to install the client part of Baan OpenWorld, then you can install this locally. You then should install the Connector for Java and if needed any of the other connectors for other tooling.
