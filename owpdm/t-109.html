James,

The package "cc" became available first on Baan IV c3. I'm not sure if the BOI generator works for versions and releases prior to Baan IV c3. As of April, 2001, the generator was not available for Baan IV b releases. The BOIs can, however, can be used by versions back to Baan IV b2 by creating the "cc" package and importing the BOI objects. You have to do some GTM to create the "cc" package in the standard VRC.

Some editting might be needed for BOIs that are ported to an older Baan release due to data model differences.

I hope this helps,
