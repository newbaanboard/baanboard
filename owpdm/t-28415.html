Hi...,

Try to add java environment path (ex: C:\java\1.4.2\bin and C:\java\1.4.2.\bin\client ) to your system PATH or to your SSA Environment.

1. Open SSA Manager (baanman.msc)
2. Right CLick in your SSA Environment, and click Properties
3. On Environment's tab, add java environment path (ex: C:\java\1.4.2\bin and C:\java\1.4.2.\bin\client ) to value of your PATH
