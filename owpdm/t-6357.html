You will need to install Openworld adapter and Gateway on machine from where u want to access the data from baan through gateway.

Now having said that you have lot of options for configuring the whole environment.

You can have one machine with OW adapter and gateway and all other machines can access information via the gateway machine(via http-web, Sockets, MQ series or FTP). This is the way to run Gateway out of process this the way Nortel Network uses OW gateway.
