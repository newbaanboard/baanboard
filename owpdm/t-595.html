Hi Trenchard,
Drag and Relate is possible between crystal reports and bdn only if crystal reports are imported into BDN repository. When crystal reports are fired, they launch in the sa,me web interface.
Whenevere a report is queried, the request goes to baan database through the HRNP server of BDN.
I have seen that it puts a heavy load on the BaaN's database server and the whole Baan server slows down.
