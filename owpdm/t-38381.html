The examples below are taken directly from User Guide for Adapter Server for Baan IV, Baan ERP and ERP LN (U8931A US)

To start the bshell without the GUI, you must specify the -server flag. On Windows systems, use the following command:
ntbshell -server
Make sure that the environment settings are correctly set. You can use the $BSE/bin/fillenv.bat batch file to check the correct environment settings.

The following batch file can be used to start one or more BusComponents in three separate bshells.
rem example file StartOW.bat is a script to start components in various bshells
start c:\baan\bin\ntbshell -server otmboaserver -f c:\Buscomp1 -d -s -l c:\stat1.log
start c:\baan\bin\ntbshell -server otmboaserver -f c:\Buscomp2 -d -s -l c:\stat2.log
start c:\baan\bin\ntbshell -server otmboaserver -busc Archelon -d -s -l c:\stat3.log
The previous script refers to files BusComp1 and BusComp2. These component files contain the BusComponents to be started by each bshell (see examples below).
rem c:\Buscomp1, contains bus components with IDs 'Archelon' and 'Basilosaurus'
Archelon
Basilosaurus
rem c:\Buscomp2, contains bus components with IDs 'Coelurosaur' and
'Dunkleosteus'
Coelurosaur
Dunkleosteus
