3. If the fields are from different table, the changes are to be made in DCD file
are.
In the main Elementdef include like element the new elementdef for one occurs or like group for N occurs.
example:
A)<ElementDef Model="Elements" Id="ElementsHeader" Type="table"...>
<Group Occurs="Required">
<Element>field</Element>
<Element>Elementlines</Element>
</Group>
</ElementDef>

B)ElementDef Model="Elements" Id="ElementsHeader" Type="table"...>
<Group Occurs="Required">
<Group Occurs="ZeroOrMore">
<Element Reference="False">Elementlines</Element> </Group>
</Group>
</ElementDef>

And after define this Elementlines:
example:
<ElementDef Model="Elements" Id="Elementlines" Type="table"
<Group Occurs="Required">
<Element>field</Element>
</Group>
</ElementDef>

In QCD define this elementdef in way InOut:
example:
<Parameters>
<Element Scope="InOut">ElementsHeader</Element>
</Parameters>
