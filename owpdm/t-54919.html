The code needs to be put in program script and not report script . And the code could be like

extern long rptcopy |to track report copy
for rptcopy = 1 to 5
brp_id1 = brp.open( "rwhinh341212000", "P", 1 )
Spooler is opened automatically
brp.ready( brp_id1 )
brp.close( brp_id1 )
endfor

It is good practice to have minimum number of rprt_sends in order have fast throughput . So make use the above code judiciously .
