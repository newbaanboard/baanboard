Hi,
You can very well connect your java application with BaaN ivc using OpenWorld. OpenWorld server contains various components like OpenWorld Broker, OpenWorld Adapter,OpenWorld Studio, OpenWorld Gateway, OpenWorld Domains.

OpenWorld Broker is useful to integrate business
applications using both synchronous and asynchronous communications methods. You can use the Baan OpenWorld Designer to configure and customize the integrations. The broker applications consist of collections of objects, stored in a repository.

The Broker connects with BaaN ERP using Business Object Interfaces(BOI).OpenWorld studio helps in building BOI in a GUI environment.
