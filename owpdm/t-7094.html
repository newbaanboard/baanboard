Hi John,
TIBCO has various EAI products like TIBCO Active Enterprise, TIBCO Active Exchange, TIBCO Active Portal, TIBCO Extensibility. Which product do u use for BaaN integration?

You can use TIBCO Active Enterprise which has the readymade solution for BaaN integration. If u want to build your own adapter on top of TIBCO using BOI's then u need to go for TIBCO Adapter SDK which is part of TIBCO Active Enterprise. Actually TIBCO Adapter SDK provides readymade data transfer features and also API's to develop adapters in bar with the industry standards.
