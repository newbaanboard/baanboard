You install the OpenWorld Adapter on your Baan server and you also install one or more connectors on your Windows server running the add-on. In all cases you install the connector for Java and depending on the tooling used to develop the client you might need to install additional connectors (.NET, XML, COM etc).

The connectors are simply installed via an installer program that is part of the OpenWorld software.

After installation you will have to some configuration and that's about it.

Hope this helps.
