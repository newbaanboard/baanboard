Hi,

On a HP box, for server side adapter installation, you should use java1.1.8 version. you can download this from hp.com website. Please dont use the JVM from java.sun.com as it is meant for solaris. Or if you can find JVM for HP-UX on java.sun.com go ahead.

How ever, on the client side of the adapter, you can use java1.3 or even java1.4 without any isssues. again if the client side of the adapter is gonna on a unix server, then download from the respective OEM side.

for windows side of the client installation, you can use any of the Java1.X from java.sun.com. Same thing applies for server side as well.

The above expl is applicable both for OW2.1 as well as OW2.2, and also for OW3.0
