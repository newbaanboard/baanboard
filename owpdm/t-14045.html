A Baan session, in simple, is basically a UI that allows users to enter/edit/delete/process/view the data within the application.

There are different types of BOIs and only Function Server based BOIs are tightly integrated with the particular session(s).

If sessions on both BAAN installations are exactly same, then it is possible to create a BOI on one and then transfer it on the other and it should work without any effort.
