15th July 2004, 14:03
1.Get the Webtop CD's from Baan support as you need to sign some agreement(no license cost plus first year support is free).
2.Run the SSAWebtop.html and install the SLM(license manager) first.
3.Read importantNotes.htm
4.Stop the IIS server to test the tomcat web server as it listens on port 80.
you may cange port later.
5.Start the SSA Webtop installer for Win32 platforms go with defaults.
You can read the Installation & Configuration guide which shows step by step setup procedure.
6.setup creates few icons on the desktop, Run the tomcat first and see it shows no errors on the cmd window.
7.Click the webtop admin login shortcut (Webtop Admin login),type in the password you typed during the setup. You can follow the documentation to do the initial configurations which are quite easy steps.
8.If you havent got the webtop build GY0114 then download it from
/ftpdownload/updates/webtop/GY0115/SSA_Webtop__3.0_buildGY0115.zip
(Size: 98290 KB). update is run automatically and explained in documentation.
9.It is windows so reboot as many times as possible to clear unknown errors.

On the baan side
I have done with the following
1.Porting 6.1c.7.03 and sp15
2.Tools standard 143492.
3.Dem in webtop 144611.
4.webtop enabling of dem 144769.
5.Dem browser does not show in webtop 145125.
6.run ttconwebtop on baan to convert/enable sessions to be used with webtop.
7.In user management startup tab click the webtop for the user you want to test.

Please try this out and let me know if you have any issues.

Refer to the SSA Webtop 3.0 - webtop-faq

1. The login dialog box does not appear
This can have various causes. Please check the following points:

Check whether the Sun Java Plugin is disabled:
- Go to Control Panel
- Double-click the Java Plugin icon; if it is not there, the Java Plugin is not active.
- Check the Browser tab
- Clear Microsoft Internet Explorer

5. Does Webtop work with on the Mac through Internet Explorer?
Internet Explorer is available on the Mac. However, Webtop does not run successfully in this version of Internet Explorer.

6. Is Netscape supported by Webtop?
Provided the market requirements do not change, the Webtop team has no intentions to enable Webtop to support Netscape. .

And some more info in the release notes -

2. Internet ExplorerIE 6.0: Webtop only runs in Internet Explorer 6.0 (or higher).
Java Virtual Machine : Internet Explorer must use the Microsoft Java Virtual Machine. It cannot work with the Sun Java Virtual Machine. If you cannot run the Webtop, check your Virtual Machine settings: Tools menu|Internet OptionsÂ—> Advanced. Make sure the Java (Sun) is NOT checked.
