Greetings Razman,

There are certain Baan functions which are not supported in Webtop. Functions such as client2server, server2client, shell, etc (You can probably get a complete list from your account rep) are not supported in Webtop because there is no application residing on the client PC. I'm not familiar with the B2Win products, but I believe they probably use the server2client function to transfer the file to the users PC, to subsequently open in Excel, Word, etc.
