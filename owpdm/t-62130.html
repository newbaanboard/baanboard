How to know the supported verbs: that is indeed a thing that needs improvement. It is not (yet) documented.
The business object session gives you that PurchaseOrderBOD has the code bopur047. The supported verbs will have functions like bopur.bl047sb00.OnProcess.

Some of the supported incoming BODs are very specific and will work only for specific scenarios. So just a flat list of all supported incoming BODs will give a too optimistic view on the LN world.

In the Business object session only the object not 'From iStudio' will have additional data like the public method. Some BDE and all BODs will be from iStudio (Business Studio). So the session will not give you any additional information.

Hope this helps
Adriaan
