OpenWorld 2.0 has the following editions such as OpenWorld Basic Edition, OpenWorld Suite Edition and OpenWorld Professional Edition.

OpenWorld Basic Edition
- Contains OpenWorld Adapter. It integrates any third party application with BaaN ERP.

OpenWorld Suite Edition
-Contains OpenWorld Broker. Broker is an integration server which has two components. The one which sits on the server side is called as Broker and the other component which sits at the client side is called as Designer. Baically it does the job of data transfer.

OpenWorld Professional Edition
- Contains OpenWorld Studio. Studio is an interface to develop Business Object Interface (BOI).
