Issue:Where can I find SLM Product ID Codes?

Solution:VERSION(S): SLM

SITUATION IDENTIFIED IN: Licensing

ATTACHMENTS: n/a

SITUATION DESCRIPTION: Where can I find SLM Product ID Codes?

SOLUTION DESCRIPTION:
To find SLM Product ID Codes:

(1) Login to www.Infor365.com

(2) From the left hand navigation, select the "Software License Key" link

(3) This brings up a Software Keys webpage, with information on key request forms. There are 2 pages. Go to the second page (Page 2).

(4) From the second page, select the link for "Infor Baan Key Request Form"

(5) This brings up a software validation webpage. At the bottom of the webpage, click on the drop down list, select Infor License Manager(SLM), and then press the "Information" button.

(6) After pressing the "Information" button, click on the drop down list at the bottom of the webpage, and select the desired product to view the desired SLM Product ID code. After selecting the desired product press the "Information" button.

(7) A webpage appears showing the Product name, License type and Product ID.
