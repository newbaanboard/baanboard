I am trying to set up the BAAN CRM integration with Lotus Notes.
The installation process is OK. The DBSync Assistant is installed. It is also configured to accept the integration with Lotus Notes.

In BAAN CRM in the contacts form there is a new button: "Send to mail". Suppose I select a contact and I send it to mail.

In DBSync Assistant - Lotus Notes Tab , when I press "Syncronize" I get the error message

"Error setting locale in Lotus Notes"
Error: 91 (Object Variable or With block variable not set)

Do you have any idea why?

Due to the support we have I managed to solve the problem. It seems that CRM integration with Lotus Notes only works if the user has a local copy of his e-mail. It does not work if you are in the same LAN with the Domino Server and you read the e-mail directly from the server.

Therefore I had to make a local Replica of the mail database. This Replica is scheduled to be syncronized periodically with the database on the mail server.

Now the error message "Failed to initialize Lotus Notes API" is solved.
