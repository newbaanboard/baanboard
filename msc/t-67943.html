Since you are creating different production orders at each work station, I suppose you have different finished goods at each of the work-station.
Lot code is created in conjunction with the item code. Since your item codes are different, I suppose you can have same lot number for different items. However this needs to be either created manually or you have to create a logic for lot code creation in such a way that it produced always same set of lot numbers for various items.

Dear Pegaga1979:
BaaN gives quite a many options for you to configure the lots. e. g. I suppose if you have new lot numbers starting from 1 for every calendar year, then you can create them and apply this logic for all the items. Only thing is you have to close all the production orders at the year end and create new production orders for all the items in the beginning of the new year in the same sequence.
If the above does not suit you or find it cumbersome, then you may have some small customization to create lot codes as per your requirements.
