Following steps can be followed.
1. You can reopen "wrong Production Order"
2. Return material from "wrong Production Order" to warehouse
3. Issue correct Item to "wrong Production Order"
4. Issue material returned in step 2 to correct Production Order
In fact because both the orders are closed, posted to finance and there is no physical variable in inventory all above steps are not necessary. Variance posted in finance can be adjusted and reported.
