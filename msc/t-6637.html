Robert,

When you say updated the cost price components, I assume that you mean updated the std cost price of the components.

The official answer per Knowledge Quest is:
If the cost price for the end item or for a material component has been recalculated, it is not allowed to re-activate the order. Re-activating would result in a revaluation based on the cost price as calculated during the period that the order is closed. Consequently, a wrong production result is obtained because of a wrong work-in-process value.

So the short answer is "no." I dont think Ive ever tested, however, to confirm that it "works as advertised."

The other case worth noting is that if the SFC order is closed, if the order is linked to a project, and if the project has also been closed, then you stand no Buckleys even if the std cost of the end item hasnt been updated.
