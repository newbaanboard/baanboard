I agree with what Sasikanth has explained.....but if you still want to do it.....By Default you have functionality available in ERP LN where you select the operation you want to block and in actions you can find the BLOCK option by clicking on it you can block the operation, you should also give the block reason as "blocked for completion"........to stop the material from backflusing attached to that operation you can select "COMPLETED OPERATIONS ONLY"...in backflush material and hours (tisfc0220m000)
