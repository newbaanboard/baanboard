PCS is mostly the make to order solution. It is linked to manufacturing and the PCS item can be tracked from start to finish.
TP is used to manage a project. The project plan can be managed from budget to closure. If an item needs to be manufactured you can link the PCS project as well.
TP projects module is much more complex than the PCS solution as it has more functionality.
There is a standard interface between TP projects and MS Projects.
