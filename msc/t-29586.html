Hi,

It is possible to enter a completion date in the past, but no longer if the order is already completed.
If you enter a completion date in the past, don't automatically generate an inbound advice. Instead, receive the produced goods manually and change the receipt date, also in the past, and then put away the goods. The only thing is that the inventory date of the lot (if you are using lot numbers) is not correct.
If the receipts are already done with a wrong date, you can no longer correct them, unless you would start making changes via GTM.
