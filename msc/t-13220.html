1. In the general item data, if you specify an item as manufactured then the planning engine will create planned production order. If you specify as purchased then planned purchased order is created instead. In the plan item data, you can also specify the supply source, but I think this is more to the supply chain thing. CMIIW :)

2.For the horizon/scene, you should set it in the item plan data (order horizon for order based and planning horizon for master based)
