Hi Preethish,

You must check the session tdinv0100m000 (Maintain INV parameters), the option "Negative inventory allowed" should be "No". If you perform cycle counting the option "Block during Warehouse Cycle Counting" should be "Yes". As per your statement it could possible some other person might be performing some inventory transaction during your cycle counting process for which the inventory of the same item got allocated, otherwise during cycle counting process item's inventory never gets allocated.
