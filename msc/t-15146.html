See Baan Document UP128 on Baan Manufacturing - Engineering Data Management. Also Look at ODM (Object Data Manager). Standard Baan only Supports some file types that you can display using the Display Drawing Buttton that you see in the Item Mater and other Places, if you set the parameter and directory structue up. ODM allows more file types. You need to get both documents and review them. Hope this helps.
