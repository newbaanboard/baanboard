Rasi,

Just a guess, but take a look at the Costing tab in Shop Floor Control Parameters (tisfc0500m000 --> tisfc0100s000) and make sure you are using Actual Man and Machine Rates instead of Estimated Operation Rates.

Another thing you might look at:

Check your Cost Price Calculation Parameters (ticpr0100m000):

- make sure that the Standard Price Calculation Code is the one that you are using when you update your Operation Rates.

- it sounds like you are maintaining Operating Rates by Work Center, so make sure that Type of Operation Rates is set to 'Work Center Rate'
