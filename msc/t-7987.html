The fact is that any other table will not change just because tcmmcs023 is changed. Other than editing the description of the item group additions to the table will not affect any other table and deletions will not be allowed as long as the Item Group is used. If you are wondering where this table is used then run session ttadv4430m000 (print where used tables).
