tirou4204m000
SESSION OBJECTIVE
To create the default company calendar. This calendar is needed if the
value of the parameter "Company Calendar Type" is "Five Days a Week",
"By Work Center" or "By Company" and if
there is no calendar for the dates to be computed.

HOW TO USE THE SESSION
When you have chosen the "Continue" option, the system will first
delete the existing default calendar and subsequently create the new
calendar. This calendar spans the two dates specified on the form. By
default, 5 years in the past and 12 years in the future. When you try to
determine dates that fall outside the calendar, the system will respond
with the message "Calculated calendar times may be incorrect (Date %D002);
Create default company calendar first".
