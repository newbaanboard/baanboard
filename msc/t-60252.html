Hi,

In Baan IV, it's difficult to delete the whole list of planned MRP orders at a shot (in LN it's easy to select and delete), you need to choose one by one each order to delete, you need to open the session 'Maintain Planned MRP Production/Purchase Orders' and then select the required item code (top level) and delete the same MRP order of the top level item, followed by all below level items with the same procedure.

regards,
Roy
