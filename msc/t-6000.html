If you really want to change the inv. unit you need to and have changed Inventory on hand, hold ,order, allocated inventory, quotation allocation, economic stock to 0 but you allso need to change cumulative issue to 0 via table maintenace. You will then be able to change the value in the item master.
Be warned as your history will now be corrupt for this item.

The suggestion by Djie to change the BOM unit is by far the best option.
