Hi Josu,

welcome to the board!

Exceptions messages will be generated if the field "Generate Exception Messages" in Session Maintain MPS Parameters
(timps0100m000) is set to yes.

In your case please please check the field help in that Session for the fields "Demand Exception Message - Upper Boundary" ( for message "Actual demand exceeds planned demand by xxx %") and "Receipt Exception Message - Lower Boundary" (for message
"Actual and scheduled receipts are xxx % less than planned receipt"). Maybe the help text for these field can give you a
first idea.

Juergen
