Jason,
When was this production order closed? Did you run any Calculate Cost Prices for standard items after this? In short, was there any cost price change that took place for the item within this production order? Baan wouldnt let you re-activate a production order..
-If the item inside this production order has a cost price change that took place post production closure..
-If the project attached to the production order is either has status Finished or Closed.

There could be other reason why baan wouldnt let you re-activate a production order but these are the two reasons i came up against...Since the project scenario was ruled out by you, check on the first point..
Check the Last CP Trans. Date of the manufactured item, this production order was created for, and compare it with the production order closing date..If CP Trans.Date is greater than Closing Date, then you have crossed PONR (point of no return)..
