While maintaining employees via session tccom0101m000 several employees can be linked to the same workcenter. There is no restriction as to the number of employees that get linked to a workcenter. It is indeed not possible to indicate that the same employee is part of multiple work centers. However while recording actual hours people of several work centers can work on the order of a specific work center, not necessarily the one of the employee(s).
Please clarify what is your problem/limitation experienced?

As far as i know it is not possible to use multiple employees for the SAME work center while doing backflushing. Either you have employees record time or you backflush to one employee, often enough not a real person but an employee set up specificly for the use of backflushing.
