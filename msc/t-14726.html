Hello Neal,
I would start by confirming that there are no production orders \ sales orders \ purchase orders \ etc that are linked to these warehouses.

If there are 'open' orders against these warehouses, that would explain the 'allocated \ on order' quantities.

tdinv0250m000 'Rebuild Planned Inventory Transactions' will confirm that these planned inventory transactions are correct.

Even if there are not 'open' orders, Baan will not allow you to delete these warehouses if inventory transactions have been processed through them, due to database referential integrity.

Good luck,
John
