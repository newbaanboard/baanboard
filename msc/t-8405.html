It has been a while since I worked with Baan IV, but this is how I might do it.

Repair the part with a manual production order. Create the order for the part using the same item number. Using the Estimated Materials and Production Planning sessions, delete all defaults (based on the BOM and routing) and replace with what you actually use in terms of materials and labor for the repair. These costs could be put into seperate Cost Components for ease of reporting later.

The item will be put back into inventory at standard cost, but the production order will retain the actual cost of the repair. You could create individual lot codes for the repaired items to have the link between that particular item and the production repair costs. Additionally, if cost components are set up correctly, the reporting would be much simplier to query.

In Baan V this is made simplier with the use of the rework option on the production order. I'm not sure if Baan IV has this option.

Don't hesitate if yoy need further assisitance.
