Srinivas,

You are in the display operations session.
If you go into the tisfc1101m000 session. (Maintain Operations), then you are allowed to enter a text.

Operation texts are actually meant for instructions.
Therefor there is no option in report operations to maintain it.

There is a QMS module (in the TC -package), in which you can do things like this with rejections BTW.
