From your post it seems there is nothing that has been received under a PO and finally the back order has also been deleted. You can delete this, but it is not something that Baan offers. You will have to do this at table level.
1. First remove tdpur045 records
2. Delete record from tdpur041
Now you can remove purchase order header from Standard Baan session.
After removing tdpur045 if you use purchase order line session it will remove the record,but then it will mess up tdinv150, tdinv001, tiitm001 and also supplier order balance.

In BAAN IV c3, there is a session tdpur4223m000 - Process Delivered Purchase Orders. It has an option to "Delete Order Data".

You may want to read the session help for this session and then test it in a non-production environment to see if what the session does achieves what you have in mind.

Read the session help first.
