The functionality of alternative items:
We can define alternative items for an item through the sesion tiitm1101m000.

What it means is:
If the main item is not there than we can use the alternative items on its place.

How it is used:
It is used by manual intervention . Means system will not use it automatically on its own when the stock for the main item is not there. we need to use it manully except in PCS as given below:


The alternative item functionality is very limited and will not generate demand in mrp and mps planning. The following is an explanation of the alternative item functionality:

The system will never generate outbound for an alternative item. Alternative
items will be used in PRP generation only. It will substitute alternative items in the customized BOM. Standard functionality says that an alternative item will be automatically issued in place of another item only through PCS. If the item is flagged interchangeable through the session tiitm1101m000 Maintain alternative Items, the substitution will be made without user intervention if the original item is not available.

Through SFC and SLS, however, the substitution can be made only after
the user is prompted and instructs the system to make the change.

For example; through SLS when entering a sales order, the user will be
prompted that there is insufficient stock for the original item. A popup box
contains the option to Display Alternates. Upon selection of that option,
the sales order item will change to the alternate item selected in the
display. The same scenario holds true for SFC, for example, when trying
to issue materials (session ticst0101m000), a user can zoom on an item and
display alternative items, but must manually change the items on the
estimated materials.

In summary, alternative item use is only planned when generating PRP,
and not planned for in the case of standard items (in MRP, MPS, etc.)

Hi,

Alternate Material can not be backflushed automatically unless you add such item in the Estimated Material of the Production Order. Automatic adding of Estimated material depends on some setting which are as follows:

1). In Implemented Software Component under oncept folder, Use Up and Alternate Material must be checked.
2). In the Shop Floor Control Parameter, under Material/Hours folder, Automatic Select Alternatives can be kept either 'At Release' or 'At Generation'
3). Against the BOM line you need to maintain Alternative Item with Use Up or some priority.

Based on these three set up when a production order is created or released system will automatically replace the esmtimated material with the alternate item. However, some condition will be applied:
1). There should be no inventory of the original estimated material.
2) There should be full inventory of the alternate material.
Once estimated material is replaced with the alternate material, it will be considered for backflushing.
