MRP is standard functionality, not exclusive to Baan. If you understand the concepts of MRP, then setting up Baan is basic. On the Item Order Data, you must select the correct order system & method, then set your MRP quantities.

Try looking at APICS for basic MRP information and then Baan help for additional set up information
