Hi Raman,

Not sure if blocking of machine can be possible at LN, but you can replace the old machine with new machine. Select the old machine at session tirou0102m000 --> go to specific --> select 'Where used' --> you will get the session where the machine is being used at routing operation against the item code. Then with the help of a technical guy you can run the correction programme to replace the old machine with the new machine code.
