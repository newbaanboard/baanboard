I run into the same problem some days ago. It's caused by the default application that Windows is using for opening .gif, *.jpg etc. (Windows Image Viewing or something like that). Register (within Windows) a "regular" Image Viewer Application (like IrfanView or something like that) for the image file endings and it will work.
