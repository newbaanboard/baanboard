Reset the production order back to completed using session ticst0203m000 - Reset Production Order Status. Once the production order is completed, you should be able to use Enter Material Issue for Production Order with -ve qty. I've done this lots of times in Baan IV.

The only time you can't do this is if a cost price calculation has been performed that revalued the inventory after the production order closed. (Only seen this very rarely)
