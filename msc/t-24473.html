Solution # 5753

SITUATION DESCRIPTION:
Incorrect quantities are placed in Subsequent Deliveries field, when user modifies issue field and the SFC parameter "Manual issuing" (in tisfc0100m000) is set to yes. This occurs when the user uses the tab key or the arrow key. The value of the last displayed occurrence overwrites all the occurrences.

Hope this can help.
