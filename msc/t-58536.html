Process of Adding Holidays to Company Calendar


1. Start by opening the “Standard Calendar Template”


2. The Standard Calendar Template should look like the following images. One image for each day of the week (Sunday-Saturday).








3. Go back to Menu Browser and Open “Calendar Lines”



4. Use the binoculars to find the Company Calendar if it is not already displayed
5. Find the day(s) that you want to update.
6. Select “New”
7. Enter the date that you are updating


8. As you can see in the picture below I entered 12/28/2010 and set the Availability Type to “HOLIDY”, leave Time Interval Type blank.

9. Next select the other two lines (Sequence Number 1 & 2 for 12/28/2010) and hit “Delete”. A prompt will pop up, select “Yes”.

10. Your screen should now look like the image below.

11. Repeat Steps 6-9 until all updates are completed. Hit Save.
12. Select Specific →Refresh Derived from Lines

13. Once Refreshing is complete select Specific →Update Actual Calendar

14. Within the “Update Actual Calendar” Session select “Company” then tab so that it appears in both text boxes. Make sure the “Process Child Calendars” Check box is not selected (Has no check mark) as you will see in the image below.

15. Hit “Update” then “Continue”
16. Finished, the Company Calendar has now been updated.



Updating Work Centers to Reflect Company Calendar Changes

1. Start by grouping all work centers by Time Interval Type. This is done so that you do not have to constantly change the Standard Calendar Template for each work center.
2. Open the “Standard Calendar Template” Session
3. Select Monday as shown below, then select the appropriate “Time Interval Type”. In this example I have selected “1”, which equals one 8 hour shift.

4. Follow Step 3 for Tuesday thru Friday, you should never have to change Saturday and Sunday
5. The “Standard Calendar Template” should now look like the image below, take note that the template for work centers should only have one line unlike the way the Company Calendar was set up with 2 lines one with no “Availability Type” and one with.

6. After the Time Interval Type has been changed for all of the WORKDY’s close the template session.
7. Now open “Calendar Code” session, select the work center you want to update, choose Specific →Calendar Lines

8. Your screen should now look like the image below, next choose Specific →Refresh Derived from Lines

9. After refreshing is complete select Specific →Update Actual Calendar

10. Once the “Update Actual Calendar” Session opens choose the work center that you want to update, make sure the work center name is in both text boxes as shown in the image below. Also make sure there is no check mark in the box called “Process Child Calendars

11. Choose “Update”, then “Continue” on the next pop-up
12. Finished, you have now updated the Calendar for one work center.
