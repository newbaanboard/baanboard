The below mentioned text has been copied out of the solution from BaanOnlineSupport::

==============================================
SITUATION IDENTIFIED IN:
"Generate Planned (MPS)/MRP Orders/Batches" (timrp1210m000)

SITUATION DESCRIPTION:
A float exception error condition occurs when generating MRP orders. This error occurs on Hewlet Packard machines. This error occurs when the combination of the "order quantity multiple of" is a small number and the demand quantity for a MRP item is a large number.

SOLUTION DESCRIPTION:
When generating MRP orders, changes were made to change the way that the order quantity multiple is verified. These changes correct the situation that causes the float exception error.
==============================================
