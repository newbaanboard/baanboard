In BaanIVc4, the details of ABC analysis is as follows.
1) The ABC Classification criteria percentage is maintained in tdinv0100m000 form-2
2) The ABC Analysis is performed thru tdinv6210m000
3) The ABC Analysis performed as above, will be updated in
Itemdata tiitm0101m000 form-2 by system,
which is shown as the ABC code for that item.
