In Baan you can close a production order only if all components on the material list are already consumed.
If your SFC order is already closed, but some materials must be issued, then this is a bug to be notified to Baan Support.

Viceversa you can issue the materials and post hours on a SFC order completed. The materials requested by the SFC order create a planned transaction and a requirement for MRP
