Hi Brad,

If you use a dependent currency system (see 'companies': tcemm1170m000) then the conversion to the entered currency is done by using the internal currency rate (see same session). I can imagine that in your case 'DEF' is the internal rate as defined for the company.

As far as I understand in an independent currency system the cost prices of items are stored in one or more currencies so if the cost price of an item is present in the entered currency in ticpr2400m000, then I suppose no conversion will take place at all.
