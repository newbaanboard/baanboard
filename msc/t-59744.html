Hi,

There are only two tables (cprpd100 & cprpd120) which needs to be updated if you want to maintain the Item-Planning Subentity.

However if you are using script, Use AFS on cprpd1100m000 with the condition that Item Order System must be "Planned" in Item Ordering data. An AFS will automatically update all the relivant tables.
