Hi robby,
A bit late but anyways....In Baan4c4, when you click on copy option in item master (for an item with any text), the very first message that would greet you before the session tiitm0202s000 comes up is "Do you want to copy item text?". This is where you should click on "No" if you dont want the same text. Furthermore, if you want to delete an existing text, all you have to do is to open the text in Text Editor and then select "Disconnect..." from file menu followed by clicking on the key "Y" on your keyboard and pressing enter.
