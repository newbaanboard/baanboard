Hi Raman,

If you are talking about standard cost, then the standard cost not going to change as per the production Order.
If you are talking about actual cost then you can get the data from ticst010 table, sort by Item based cost and select based on main item, then aggregate data of ticst010.aamt this will give you the end unit Actual cost of Production Order, you can aggregate for all orders and divide by number of orders will give you the actual average cost of a main item.
The standard report gives you with respect to Production order. You can take it to Excel and further process or have customized query.
