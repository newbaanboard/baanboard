This may happen, if there are some generic items in the lines of your generic BOM.
You should check following fields for these lines in your generic BOM:

Planning quantity
Planning percentage

If one of these fields is not 0, the generic item in this line will cause a planned order in enterprise planning.
For more information please check the online-help for these fields.
