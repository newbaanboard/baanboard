Step wise

Maintain Production Order
Release Production Order
Generate Subcontracting Order (If routing is for subcontractor)
Maintain Receipts (For subcontracting Orders)
Report operations complete / Enter Hours Accounting if applicable
Report Production Order Complete (Also Post to inventory)
(Match & Approve to be done by Finance if order subcontracting)
Close Production Order

These are basic steps and will change from Company to Company depending on Parameter settings stc. Also if ILC implemented then inbounding and outbounding against production order necessary to close Production Order.

We can say the process is complete when Production Order is closed. To close production order, the Production Order must be reported complete. All inbounding / outbounding of materials & all finance related entries like matching receipts to subcontractors relating to the Production order must be complete.
