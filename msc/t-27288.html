For reporting hours to the production order through Hours Accounting, many times employee 0 is used, they set up a dummy employee 0 for this purpose. Then use 'Production Order' as the type and fill in the order number.

tihra1201m000 does process all, including production orders. You may need to run the 'cost roll up' or something like that to see the hours cost against the orders..
