I think it is Intelligent Resource Planner (IRP)
From Baan docs: IRP is a production planning tool, which is designed to operate in combination with an Enterprise Resource Planning (ERP) system, such as TRITON or BAAN. IRP is strongly coupled with this ERP system, allthough it functions independently. An interface layer takes care of the exchange of data between IRP and the ERP system. The Constraint Planning package contains the functionality of the Intelligent Resource Planner.
