Resetting the order status will only set the order to Completed. You cannot change the order quantity or re-set the status to active at this point. The only thing you can do is to issue materials or report labor. If you reported too few complete return any materials issued and open a new production order to make the additional items. If you reported too many complete than all you can do is make an inventory adjustment.
