Hi,

there's a session in Baan IVc4: Print Work-In-Process (pspmg0450m000)
It will give you a report per each order batch which status is not "closed" by the time you print the report.
It will show you WIP for each raw material, operation cost as well as overhead cost.

The weakness of this report is: you can't specify which period you want to print the WIP report because the options available are only production batch no & currency. So, at the end of the month, it will be hard to compare the WIP amount shown in this report to WIP amount shown in GL (Finance). Unless, you can stop the production people from updating the production batch data during finance "close period" time.
