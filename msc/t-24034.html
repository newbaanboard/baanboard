Hi sathasivan,
this functionality is available with Solution 129498 in 5c.
Refer to the Solution on Baan Support Site.
When the field 'Order Policy' in the Item Ordering Data (tcibd2100s000) is set to 'Anonymous' for a Generic Item, then the Generate Project Structure Session (tdsls4244m000), will now create a standard item
If i remember right you further have to set some parameters in the PCF-Parameter Session:
The 'identifying item code' Flag has to be set to 'No' and you must set the 'sequence number from position' and 'sequence number to position' so that a unique itemcode can be generated.

Hope this helps,
Andreas
