The Estimated Cost of the Order are based upon when the Order is created. You can replan the order until the Order is released. At release the Estimated Cost are frozen. If the Routing or Bill of Materials are modified you will have your variance between Estimated (Standard) and Actua Cost which is what Finance looks at. Hope this helpsl

In BaaN 5 you have the option of when to freeze estimated costs for production orders. In tisfc0502m000 - Default Production Order Data you can select to freeze estimates at Order Creation; at Order Release; or Before the first WIP transaction. So, it is possible to make changes to the order that would not get reported as variance if this is set and changes were made before the first WIP transaction.
