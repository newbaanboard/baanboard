Shalini,
If A is parent of B,C & D and if you want that all components of A should get be backflushed except say D then the flags should be set like this

For A - Backflush to be Yes
For B- Back flush if material to be Yes
For C- Back flush if material to be Yes
For D- Back flush if material to be No

System uses this to have default value in Estimated Materials. But one can change it the Estimated manually, if needed.
