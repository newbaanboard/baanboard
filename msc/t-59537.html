Hello Jeffers Yuan,

Before you can start, you first need to freeze your assembly (line station) orders via tiapl3203m000. Once this is done (your line station orders on the segment should have status 'frozen'), you can the start of the order via session tiasl6520m000, using the specific option 'start assembly order'.
