Murthy,

The only way you can tell the origin of a Production Order is to have different Order Series for Production Orders generated from EP and Manual Production Orders.

For example, this can be done in Shop Floor Control Parameters (tisfc0100s000) by setting the following Number Group parameters:

- Production Order Series for Production Orders (Manual) = M
- Production Order Series for Enterprise Planning = P

As far as I can tell, no record is maintained of the Logon ID of the user who created a manual Production Order.
