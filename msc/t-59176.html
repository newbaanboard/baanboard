Hi ,
yes u can use reverse bom to handle this case.

Return from customer , you can process item transfer to change the item code to assembly item code .
this assembly will have some stock.
Then you will create a production order for the child part, and input this assembly item in estimated material by prod ord.

Process the prod ord, then you will get child part inventory.
