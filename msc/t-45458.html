Both the solutions provided above are correct but are applied to two different scenarios.

1. De-assembly when the production order is not yet reported completed
2. De-assembly when the production order is completed and manufactured to be dismentled item is in inventory.

In ERP LN 6.1 it can be done as below

For the first scenario same production order is to be to be used to dismentle and return the components to warehouse. A task of De-assembly can be added in production planning to capture De-assembly work hours and related costs.
For second scenario a rework order can be created for the manufactured item to be dismentled and components to be returned are to be added in estimated materials with -ve quantity.

For both the scenarios production order is to be completed with NIL quantity.
