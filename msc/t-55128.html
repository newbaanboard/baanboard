But, you can set the Unit round coefficency to get the accuracy. i.e. you can set the kg as 0.000001. (the max lenght is 6).

While, in BOM line, the max lengh is 4. so , you can set 0.0001. Any value which lower than it will be display as Zero. for , in this field, the lenght is limited to 4(0.XXXX).
