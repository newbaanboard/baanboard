Luke,

Not sure about baan4, but in LN you should be able to do it with the Product Configurator.

For example:
Define Item A as a Generic Item.
Define the following Features for Generic Item A:
LENGTH - Length of Item
COLOR - Color of Item
Define the following Options for Feature LENGTH
All the possible lengths in millimeters
Define the following Options for Feature COLOR
All the possible colors... Red, Geeen,...

Then when a Sales Order is written for Generic Item A, the order taker will be prompted to pick the Length and Color of the Item.
