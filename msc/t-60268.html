Hello again,

I found the problem. Our Default Calendar("Planning Calendar" tirou410) was outdated (the last day was 28/08/2011). This calendar has the calendar for an empty workcentre which is used to calculate for "Five Days Week", despite the ROU paramenter Company Calendar Type = "By Company", which uses the same table (tirou410) but worcentre = "ZZZ". It calcualtes the dates twice, first for worcentre = "ZZZ" then does a second pass but for worcentre = " "(it was failing here).

Note that the Planning Calendar (tirou410) is different from the Company Calendar (tirou400).

I just created the default calendar (tirou410) with session "Create Default Company Calendar (tirou4204m000)" for the next 10 years and all planning orders got the right dates.

I would like to know what is the diffrence between these two calendars (Company Calednar "tirou400" and Planning Calendar "tirou410").
