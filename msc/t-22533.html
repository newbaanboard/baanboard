In BaanV you have the session whcor0200m000, however, you will be able to choose different things in this session. For instands, do you want a financial booking for the fixed corrupted inventory.... So be carefull, consult somebody from SSA-Global.
The session you ran is for planned inventory transactions, the cor0200 is written for the actual inventory settings. So, try this one in test mode, see if this helps.
