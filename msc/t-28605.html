The other way is using Intemediate Concept for subcontracting item

1. Make a semi finished item (with only subcontracting operation).
2. Add this item in Multilevel bom of main item
3. Raise a seperate production order on it & apply QMS On SFC

In this way u can do inspection on subcontracting as well as track it inventory ( if required)
