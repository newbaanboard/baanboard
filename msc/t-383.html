Hi Darren,

Baan has a module called Process. Which allows you to set up formulas and add materials to a formula. Basically this is like a recipe. It is often used in food and chemical indutries. You can set up a route for your formula which would be like the recipe instructions. You can then create production batches to manufacture your item.
