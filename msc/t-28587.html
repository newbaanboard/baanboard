Create a task in the routing of this manufactured item, which has the foam as the material, with a Subcontracting workcenter. After the production order is created for this manufactured item you could generate a subcontracting purchase order. When the purchase order is received, the operation in the production order gets completed and you will get the cost booked into your project. You would need to create a standard item code of type subcontracting which then can be included in the session Maintain Subcontracting Rates that helps in Generating the purchase order with this subcontracting item(more or less like a cost item)..

If the above is too much, you could use the General Project Cost session in Project(Pcs) control..

PCS Projects can only be created for Purchased and Manufactured item...as far as Baan4c4 is concerned...
