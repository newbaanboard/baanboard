Hi,

I've used array's in pcf-constraint's.
But you can only used array's of type long or double.
The array can also be used as a global value (just add global)

long sample(15)

sample(4) = 50
sample(5) = 75
sample(6) = 100
sample(7) = 125

c: [fs01] = sample(val([fl02]))
