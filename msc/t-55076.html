There are two ways to calculate the start/end date of the WO.
1)detail plan. ==>the way just you are used.
2)rough plan.==>only use the LT to calculate.

You can set the WO(Fishish production LT time as Zero). so, when the system use the rought plan, it will only set 0 days ahead. (the same day as your entry date)

then, how can the system use the rough plan way other than detail plan way. there is a parameter you can use.
in the item plan data, cprpd100m000, the Fixed LT of SFC. this value control which plan way to use, to use the detail plan way(user production rate and order qty to calculate )or rough plan way(just read the LT to set start date).
