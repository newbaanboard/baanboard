Hope the matter is clear.

You need to define a warehouse at your subcontractor. This warehouse needs to be a WIP warehouse. It needs to be linked to work centre defined for your subcontractor. You need to set the sfc parameters to allow the use of WIP warehouses (form 2 parameter 3).

When you create a new sfc order it will have the BOM warehouse in the estimated material warehouse. When you release the order the warehouse will be changed to the wip warehouse.
