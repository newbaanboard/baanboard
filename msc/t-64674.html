Hi,

See, the To Issue quantity is present, hence there exists some planned orders against it. Try to make To Issue quantity as 0.

Also, check in session whinp1500m000 (Planned Inventory Transactions) if any line exists for that production order.

A basic thing is that a SFC production order cannot be closed with any quantity (other than zero) in either the 'to issue' or 'subsequent delivery' fields. These fields can be maintained by the user, so if nothing more is to be issued on a given material line (ticst0101m100), simply update these fields to zero and then close the order.

Just make To Issue quantity and Subsequent Del. Quantity as 0 and it should work.
