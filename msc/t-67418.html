Hi,

What do you mean by Item Type Text?

tcibd001.kitm has Domain tckitm which has Multiple values -

Purchased, Manufacture, Generic, Cost, Service etc

The Enum/Set Data has this information. You can go to Domain - > Specific - > Enum/Set Data.

To get the text in script, you can use inbuilt function enum.desr$()

e.g.

string type.text(20)
type.text = enum.descr$ ("tckitm",tckitm.purchase)

This will return type.text = Purchased

or

string type.text(20)
type.text = enum.descr$ ("tckitm",tcibd001.kitm)

This will return the text of value for tcibd001.kitm i.e. if tcibd001.kitm = tckitm.manufacture then type.text = Manufactured

The format is -

enum.descr$ (string domain_code(12), enum_expr [, string language_code])
