Hi. I posted a similar message on Baansupport. See below for my question and their answer...

Case description:
In tirou0501m000 Work Centers I can have mulitple work centers setup, and link them all to a main work center.

What is the functionality of this main work center? It looks to me that I cannot see the parent work center in Capacity Planning or in the Production Order Planning Displays (such as tisfc1506m000 Work Center Utilization by Day Overview).

Solution description:
The capacity utilization for sub workcenters
does not aggregate into the main workcenter for this session.
Within BaaN ERP, the functionality to add sub work center aggregation was dropped because of the problems that arose with implementation. The ability to add sub to main work center aggregation may be added in a later release of BaaN
ERP. Essentially, ERP treats each resource as unique.

But the functionality is that each work center will be planned separately and their capacity will not be aggregated in this version.Essentially, ERP treats each resource as unique.

However if you want to see the aggregated capacity of main and sub work centers ( as to capacity use by planned orders, SFC order etc, you can see them in the session cprmp3410m000(resource planning).Also the same functionlaity can be used in tihra3450m000(print Hours history)
