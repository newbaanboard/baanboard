no replies... but just posting my finding so someone can benefit; Planning does not look at Time interval types' Day percentage or capacity percentage. Neither is it looking at Available time as per Availability type. It sheerly goes by start and end time as in Time interval type and generates calendar line which gets converted to Actual Calendar.

The capacity percentage is only used in the planning of the routing operations. Make sure that the availability type and calendar in Routing are correct.

The other lead-times, like safety time, inbound, outbound etc are not using the capacity percentage.
