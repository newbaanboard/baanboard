BaanIV
Even though you have ILC switched on you don't have to run inbound advice if you create a warehouse that does not have locations. If you have locations in a warehouse you will have to generate inbound advice.

BaanV
If you do not specify generate inbound advice, put away as steps in the warehouse order linked to production order receipt the system will automatically execute the steps. This means it will store it in the location with the highest priority. Could be a fixed location etc. You allso have an option when you report production orders complete to direct process inbound data. This means that even if the steps are specified they will be executed by the system but you have the oprtion to manually run the steps.
