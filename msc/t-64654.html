Hi,


You will need to extract the master data of that item for all its master tables like item general and sub entities. Now, you can create ascii files for every master table which has that item. Then use exchange scheme to to upload the item in the new company.

The second way can be to create a script which reads the data for that item in the master tables and write the same data in the new company.

If you are sound in programming you can try the second way. Make sure you understand the tables which needs to be updated for that item.

Another approach is to export data for master data tables Tools sessions to Create Sequential Dumps to pre data, Create Table From Sequential Dump to post data to new company after executing session tccom0214m000 to correct company numbers. Above mentioned sessions are Tools sessions and you might need Admin's help unless you have super user access in LN.
