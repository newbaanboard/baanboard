Iordofe,

Just a suggestion:

Create & release a rework production order for these end items.
Issue the Main Item materials.
Add material lines for the materials (that left in inventory) with negative quantities. Send these materials back to inventory by issuing (negative qty).
Report the order completed with qty completed := zero (0).


To my knowledge this should work.
Correction with Cycle Counting might be another option.
