Well one way to handle this in Baan IV is In Sales order user PartA, B, C and G. G will be a manufactured item. So based on the configuration one can change the BOM of G. Then process G's production order to get G in stock and ship all the orders. In Baan IV since it is std costing one can expect to get the production variance, but one can map this variance to one account to get actual BOM cost.
