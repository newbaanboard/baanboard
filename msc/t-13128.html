In B40c4 following is the situation which I feel is no different in B5c

1. If u wish the order of display is always item wise and not position wise, then permanent solution is to add an index on tibom010 (manufactured item , item , position number). Alternatively u need it only for a while , u can use query by form (Tool --> query) . And in the 2nd form use the sort sequence 1 for item as 1 .

2. If u are on latest porting set , u can configure hot keys (including function keys ) using key map groups (ttaad2108m000) , key map settings (ttaad2109m000) and user setting (ttaad2105m000).
