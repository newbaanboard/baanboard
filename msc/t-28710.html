Im using FP7, we can use unit effectivity for your problem.

In Baan 5, im afraid you can not combine those 2 things (bom and routing). but i think you can do this:

1. Use Engineering Data Management.
Create Revision 1 which is using C
and Revision 2 which is using B
Still the latest revision is used to calculate the standard cost price (revision 2).

2. Create Alternative Routing

Standard Routing will use Operation 10 = Assembly process
Alternative routing will use Operation 20 = COS Process.

Still the standard routing is used to calculate the standard cost price (Assembly Process).

3. Everytime Create Production Order, system will be by default taking the standard cost price option (Material B and Assembly Process).

So you can choose Revision 1 and then select Alternative Routing.

Hope it helps.
