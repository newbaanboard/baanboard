I'll be talking in terms of 5.0c. If all materials for an order are always issued in one go, then you can go to Material to Issue for Production Orders, choose to Transfer Subsequent Delivery to Issue, then perform Initiate Inventory Issue . This will instruct BaanERP to prepare the issue of all the remaining production materials for the production order. If some materials are found short, then record a negative issue for those materials to account for this matter. This will work well with the patch for 5.0c that allows SFC to correct the production material transaction in INH, i.e. update the SFC with the latest patch
