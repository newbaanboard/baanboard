Hi robertvg ,

In BaaN Iv c4, this functionality was not there, whenever you run the MRP planning engine system will consider all the Sales orders.
there was no check whether the Sales order was blocked or not.
Even you don't find any message in MRP exception messages.

If you don't want to include in MRP then exclude these orders by changing the Nettable warehouse in Sales Order line to Non-nettable warehouse.
