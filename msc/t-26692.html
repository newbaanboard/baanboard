you don't necessarily have to archive your project, though it would indeed help.
A project is definitely closed when you remove financial transactions (tcfin100 records can be removed with session tcfin1200m000). There's this flag "definitely closed" on the projects session that will be set to yes then. When having removed financial transactions, you cannot reopen, or rather reactivate the project anymore (so project is definitely closed).
