Do you know the microrouting in LN? They allow the improvement of routing operations by adding instructions, operation steps, configuration class and process variables.

They can only be used for standard routings. Generic routings do not have access to these additions.

We have made a request to INFOR for an improvement. It could be considered if it gets the necessary number of votes.

If you go through INFOR Concierge you can go and vote for the enhancement request 90843 -> https://ers.infor.com/ERS/SoHoXi/Request/XiDetails.aspx?tbv=1&erID=90843
