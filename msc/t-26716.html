Hi,

i just checked the help and it looks like you need to do some setup in the BOM as well:
"Alternative materials in BOMs
This topic introduces the concept of alternative materials.
Often, multiple items are interchangeable. Sometimes, you purchase these items from various vendors. You can use any of these items as material in a bill of material (BOM). You can define up to nine alternative material for any material in a BOM. If a shortage of a standard material is expected to delay a production order, ERP LN can automatically select one of the alternative materials.
The use of alternative materials has the following advantages:
ERP LN can handle material shortages for some items without user intervention.
The use of alternative items can reduce the average inventory level.
Important!
This functionality is available only if the Use Up and Alternative Materials check box is selected in the Implemented Software Components (tccom0100s000) session."
Hope this will help on your first question.
