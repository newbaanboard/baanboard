On your query re.: We can report order complete more than order quantity.(only pop up message appear).How to prevent this in LN?
Ans: Don't have the space to explain it all here but I suggest you the ffg:
1. check/read about the Reporting Method for Previous Operations field in the Shop Floor Control Parameters (tisfc0100s000) session.. set it to Manual
2. also read about "count point"... you can set selected preliminary operations as count point.


Q: We can input negative quantity in Material to issue , and also Hours Accounting.

Ans: I do not see any problem if user can input negative qty as well as inputting positive qty. in the material to issue session.

just ensure that once negative issuance is initiated, storage list should not be automatically "put-away" in order to control the posting (return) to inventory..
