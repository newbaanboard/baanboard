If you're talking about the Warehouse of the Production Order, you can change the warehouse code until you post the main item to the warehouse (report order complete). Once the quantity is posted, you can no longer change the warehouse of the order.

If you're talking about the Warehouse of the Material Position, you can change the warehouse code until you issue the item.

Hope this info helps you.
