Hi Brad,
The table timrp010 contains all the open order. OPen order here means order that are present in planning side, order that are actual and have beend transferred to execution module like sales, purchase, production etc.. The order type will tell what kind of order it is. The table tdinv700 contains only the order that have affected the actual inventory e.g. if material is issued to production order then one can find that issue transaction in tdinv700.
The table tdinv150 contains only planned order that are present in excecution model,hence it will not show MRP/MPS/SIC orders.
