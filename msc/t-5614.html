I was wondering if anyone knew if it was possible to have more then one standard cost for a purchased item in Baan IVb?
It's not possible to have more than one Standard Cost per item in IVb.
You can have other cost prices. Insert a new calculation code (set up Purchase Price Priority), build a cost price model for the new code (Maintain Surcharges) and calculate the cost price for the new calculation code.

Sri_baan is of course right, you can change the CPR Calculation Code in the parameters every day and have a lot of different calculation codes for standard cost prices.
It's still not possible to use two different standard cost prices at the same time, for instance if you want to calculate different standard cost prices for the same item at different warehouses.
