Did you check item cost price in Item Cost Data (tcibd0107s000)?

If cost prise is equal to zero, you have to calculate item cost for every item in BOM and also for Parent item (session: Calculate Cost and Valuation Prices: ticpr2210m000).

If cost prise is still quals to zero after calculation, you need to define material cost (purchase items) and operation cost (in routing section) for every operation.
