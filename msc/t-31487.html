Hi,

You can copy in using two ways. If the revision is not accociated with a change order, use the session tiedm1220m000 ( maintain EBOM Copy Data) and use the special opiton to copy the engineering structure to the production.

If it is thorogh a ECO, Create a change order., Maintain ECO Lines, Include / Exclude E-Items by ECO, Generate Eitems by ECO, Then the appoval proc ess, Generate EBOM Copy Data by ECO and finally process ECOs ( Process ECO will inturn call the finalize engineering data)

The above is applicable for Baan IV and from Ln onwards, MBC is used.
