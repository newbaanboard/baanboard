Rama,

This a little tricky, but I'll give it a go ...

The way WIP Operation Costs are calculated is dependent on your Shop Floor Control Parameters Costing tab ... Process Hours field. It can be set to Actual Man and Machine Rates or Estimated Operation Rates. Ours is set to Actual Man and Machine Rates, which implies that WIP Operation Costs get calculated by Hours Accounting using Man and Machine time.

There a two reports that you can compare to get an idea as to how the WIP Operation Costs are calculated in your environment ...

- Print Work-in-Process (ticst0450m000)
- Print Production Order Costing Overview (ticst0417m000)
