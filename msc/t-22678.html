Hi Sathasivan,

If you want to make full use of the Earned Value Methods (including Percent Complete) in BAAN V then you should consider to implement BAAN Project (tp) as well.
In BAAN V you can make an easy link between a project in tppdm and a project in tipcs. This will allow you to use BAAN Project (tp) for entering the work brake down structures (activities), project budgeting, project actual costing and Earned Value calculations. In tipcs in BAAN Manufacturing you will execute the production of the products concerned and the actual costs of making these products end up in the project in BAAN Project (tp) against one or more activities.

It takes courage to use BAAN the way it was designed for but in the long run you will have the best of both worlds.
