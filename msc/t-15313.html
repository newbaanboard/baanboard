Baan Document UP099A on Under Common Data you have to do the following:
Chart and Planning Board Data
Use the Chart and Planning Board Data business object to:

Define planning-board groups. A planning-board group is a set of default planning-board colors, fonts, and other presentation details. For example, BaanERP uses these settings for production planning boards and workcenter planning boards.

Define default settings for the charts that the users can create in specific sessions. For each session you can define which type of chart and which default data the Chart Manager must use.
