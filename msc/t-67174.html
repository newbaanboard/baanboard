Once the item is unlinked it behaves like an independent item.
As long as the PCS project is open the item can be re-used - though it is usual for the item to be customer specific.
We would tend to treat the item as a one-off order, and if the customer wants a repeat we would create another PCS project.
The PCS project should be closed in a reasonably timely fashion to keep the finance side in good order.
