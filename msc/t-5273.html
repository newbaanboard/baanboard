It is not there in B50c indeed.

But we built this in B52a.
It is not waterproof yet there, but we are fine-tuning this in B52b.
(on demand - maybe- also in B52a)

It is fully integrated in Warehousing - Production and Service.
As-Built structures can be maintained for Assembly and Shopfloor production. These structures can be copied to Service to do maintenance.

As we do not yet have much customer input on this we would love to get your input on this.

========================
Additionally we built something called "Unit Effectivity" in B52a/b
This serves two goals:
- Pegging to Sales Order Line (via Effectivity Unit)
- Unit effectivity enables you to model deviations from a standard end item. For example, deviations can be:
* The use of one component instead of another. This affects the EBOM or the PBOM in BaanERP Manufacturing.
* The way the end item is produced. This affects the routing or the routing operations in the Routing (ROU) module of BaanERP Manufacturing.
* The origin of the end-item's parts. This affects the sourcing strategies defined in BaanERP Enterprise Planning.

Unit Effectivity can be used in combination with Serialized Items.

As we are fine tuning this functionality, we would love to know:
- in what way this should work (working procedures).
- what it should support (functionality).
