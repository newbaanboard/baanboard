What kind of planned orders? If the orders have the status "Firm Planned" run MRP again and select the field "Delete Firm Planned Orders?" as "Yes". If they're simply "Planned" there is no mass delete functionality that I'm aware of. You might have to do a script unless you want to modify the MPS/MRP demand so they will be eliminated.
