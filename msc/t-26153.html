Joy:

Compiled objects are used in PCF to improve the performance when executing the product constraints for generic items. Either you need to compile the constraint for the item with session 'Compile Constraints by Generic Item' (tipcf2201m000) to get the missing object or change the PCF parameter (tipcf0100m000) ' Product Configurator Version' from 'Object Version' to 'Interpreter Version'.
