Hi Neal

The status active means some or all hours have been posted against that order. All you need to do is finish reporting against these orders. I have listed the status in order below.

Free - Should never see this one if you do call SSA Baan support.

Planned - Newly inserted/transffered

Documents printed - If documents has been determined to be required.

Released - Able to be reported against

Active - Hours have been posted

Completed - Inventory has been posted

Archived -

Cancelled
