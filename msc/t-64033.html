You can't bring customised inventory in or out of stock via the INV module.

To get it into stock, you have to :

1) Buy it on a purchase order
2) Make it on a production order
3) Receive it through a sales return order.

To get it out of stock you have to:
1) Sell it on a sales order
2) Return it on a returns purchase order
3) Issue it to a production order (NB: If you want to scrap something you can report the production order complete with a qty of 0)
4) Globally write off all the inventory in Close Project by entering a value in "Delete Project Inventory up to " Field.

You can transfer it on or off using tipcs5130m000 Maintain Planned PRP Warehouse Orders (Wrh to Project) using a spoof standard item that has the same units.

e.g. If I want to add a small amount to Project inventory (say, somebody has miscounted when they reported production) then I can transfer that inventory to the project using this session.
