When I print a production order; I only print the check list, the order status remains released. Is this correct? When does the order status become "Documents Printed"?

Hi,

The Production Order status will be Documents Printed when you print the Production Order Documents when the Status = Planned. After printing, the status becomes Documents Printed and then you will need to Release the order to make the status Released.

If you want to restrict the users to release the order without printing the documents, you can tick the check box in the SFC Parameters saying "Print Production Order Documents Mandatory".
