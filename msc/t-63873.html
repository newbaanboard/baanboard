I can't really speak for Baan5, but in Baan4 you cannot block a warehouse unless it is location controlled.
If it is non location controlled, then probably a lot of wrok arounds wil be needed like...
1. Remove this warehouse from all the master data setups, so that this warehouse is not defaulted in any transaction.
2.Change the warehouse description to "DO NOT USE", so that user does not do any transactions on this warehouse.

Remember, above all were just workarounds.
