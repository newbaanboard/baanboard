If you do not want to implement the Engineering Data Management which would enable the Drawing button for the Item(thus giving you a lot more features surrounding drawing and specification), the best other option would be Item Text. There are some informational fields in ItemMaster session but they wont appear in other sessions without a bit of customization to the form. But in the case of the text, it should go further as there is no constraint on the length of the information. The length would be as big as your Item Text Group.
