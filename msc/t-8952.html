Hi Paul,

We've done a similar requirement...
We print the Item Sketches along with the production order... (we print the Bar codes, Routing, BOM, Blueprint and other details).
These sketches are in bmp, gif or jpeg format.

The user maintains the bmp image name in a field of the item master.. and all the images are stored in a common network folder.
To display the image, I've created a button on the Item master form to launch the 'paint' (say) application with this image.
Its not a great solution, but works fine.

Hope this helps. If you need any further info, pls do let me know.
