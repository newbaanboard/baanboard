When u are using actual costing it is somewhat tricky.
It is mandatory to backflush all materials & hours before you execute " Report order complete"
so recommended procedure as follows:
1. set the backflushing Automatic.
2. Complete the last operation in "report operations completed"
3. Pop up message will come " Backflush materials & Hours". Ans to it " yes" always.
4.Report the order qty completed.
note: it is mandatory to issue all the materials befor executing " Report prodcution order complete"
