Your variant is having reference type 'sales quotation' and the reference order of the product variant will probably correspond with your sales quotation number. Such variants can only be used on that specific sales quotation (in order to prevent others from changing this variant (from outside the related quotation)). So you just cannot reuse your variant in other places.

You'd need a variant with reference type "standard". Such variants can be used on any quotation, order, project, budget. That means you'd have to create your variant through session 'product variants' (tipcf5101m000), which will make it reference type 'Standard'.
