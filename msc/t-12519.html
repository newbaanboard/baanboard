1. what is the importance of the relation between TP & PCS projects?
PCS is the execution of the production process. Items can be manufactured and then transferred to TP. You need the link Between TP and PCS that can be manually created or by adding the PCS item onto your Budget. Transferring the PCS item moves the item cost to TP as material cost

2. If a manual Purchase order is created on a PCS project can the information be transferred to TP project?
An item is transferred to TP. If the PO was for a component and then was issued to the item that has actual costing then the cost will be transferred. The other way of getting cost to the TP project is via journal. Can be done from Finance module
