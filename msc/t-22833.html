Hi,

If there are two machines doing the same job, you can just double the machine rate. If your question is, how to maintain routing for different parallel operations, it is called network routing. You have to use phantoms to achieve that. Please read the runtime help in the SFC parameter session fields 'Operation Numbering'. You can select the field and press F1 for the field help.
