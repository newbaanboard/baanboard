Hi,

Here the problem is the identity of the semi-finished item, after the 2-3 operations the raw materials are transferred to different item which is currently not existing in Baan/LN system, so until we have them in Baan/LN, we can not issue them against the second (another) production order. Again to make them existing within Baan/LN we need to define the item code for the intermediate items (semi finished items) and we need to receive the same item for a particular warehouse/location before we re-issue them for another production order. Hence even though physically you can maintain and issue semi finished items to another production order, when its time to maintain the same in Baan/LN, you need to create another production order for the semi-finished item.
We are helpless in this regard.
