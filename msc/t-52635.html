For the given production order number (which you want to print), and actual plan code check 'cprrp041'. Get the 'transaction number'. For this transaction number (as supply transaction), and actual plan code, from 'cprrp040' get the demand transaction. For this demand transaction number get the sales order number from 'cprrp041' based on actual plan code and transaction number.
