Start option in BaaN IV is stored in field ttadv200.sopt. So, start ttaad4100 and edit table ttadv200 for company 000. Find the session you want to change and put into ttadv200.sopt any value, you wish, for instance a zero.
Change become active after "Create Runtime Data Dictionary for Tools (ttadv5213m000).
I know, it's not recomended to modify tools tables maually, but that change is quite safe.
