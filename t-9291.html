I have it working now (again).

Things to try:

1. Check that the Crystal Baan Server is running
2. Check that there are licenses available
3. See if you can log onto e-Portfolio as BaanAdmin/BaanViewer
4. Try running the Win2K service as LocalSystem instead of a local user
5. Run \program files\crystal decisons\solution kit for baan\IBR\ibrconfig.exe (I think that's the path) and turn on logging. Restart the Crystal Baan Server, and check the log files
6. Check that proper XML is being generated (might need to recompile your reports)

Give either Crystal or Baan support a call, and they should be glad to help you.
