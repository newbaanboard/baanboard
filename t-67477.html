Hi,

Your question is not completely clear.
If you have already an application that can publish a BOD to the Outbox table and if you have an application that can receive the BOD in its inbox then you need to define both applications a connection points in ION Desk and define a document flow between those 2 connection points. At moment form activating the doc flow, ION will start reading the outbox for available BODs to be routed to the other application.
regards, Adriaan

Hi, At the infor support side you can download a document that describes the BOD which can be published by LN. See KB #22945150.
So if the BOD you need exist for LN then you are fine and you can use the configuration guide from the same KB#. This guide describes what to do in ION and LN in order to get the BOD actually being published.
I'm not an expert in connecting third party software to ION. Main thing is that you figure out how a connection can be made from ION to that application. For this there are multiple connection types available in ION desk.

Hope this helps you in getting started.
