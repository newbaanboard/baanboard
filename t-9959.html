New Baan User Manager!!

This product was developed in order to assist administration of Baan users. The product is developed using standard Baan tools. It is very easy to use and extremely friendly.

Installation process is similar to regular Baan component installations.

The User manager allows the administrator to preform several operations and procedures such as:

* Display a list of current Baan users with:
- Process Id
- Baan User
- CPU Time
- Department description

* Display a list of open activities per user (session modules)
* Send a massage to a specific user, group or all users
* Ability to "Kill" specified users or all users
* Ability to "kill" a session or sessions of a specific user
* Ability to limit the number of logins per user
* Ability to detect idle users and to "kill" them using a predefind timer
