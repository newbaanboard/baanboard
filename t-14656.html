so far there is only a Baan IVc version. I'm hopeful though that other versions will follow.

I got a lot of solutions for Baan 5 and Baan 4 late last year - things I had been writing calls for on behalf of clients since several years:

- All archive sessions now have a button "Archive General Data",
- There is a new session that archives the project records (tipcs020) when all references to a project have been archived (important if you want to re-use project numbers)
- All archive sessions now have a date field, including "Archive Production Orders" and "Archive Projects"

Things have definitely become better.
