Hi Ajesh,

I have done this long back. If my memory goes right, you need to do the following activity to let the custom BOD visible in the Documents List.

1 - Go to ION Folder (Ex: C:\Program Files\Infor\ION\Desk\bin\RegistryData\BODMetaData)

2 - Go to Custom Folder

3 - Register your BOD (xsd) in the BODs\Developer folder (You also need to place the xsd of Noun in the Resources Folder

I have taken Infor Support to do this activity as it was bit confusing. I would suggest to go for Infor support to get it done.

Please feel free to correct me if I am wrong.

The step described by Sasi is to register the new BOD in ION. Note that path described differs between major ION releases. So probably good to check the documentation.
Thereafter the connection point need to be updated, the new added BOD need to be added there. Now the monitor can be created and activated.
