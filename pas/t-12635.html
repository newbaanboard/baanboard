Pal,

I see your point... but having a different setting for each user does not really make sense. You can define different bshell entries in IPC_INFO file and use the $BSE/lib/default/bshellname file to obtain a setting for user groups. Read also HELPDESK 21 "Settings voor alle gebruikers" and HELPDESK 22 "Vrijgeven van niet-actieve licenties"... How to set it in bw client is documented in HELPDESK 11 "Commando regel opties in Baan".
(Sorry for those who cannot read these articles: turn to the relevant solutions: 118978, 121449, 123238, 123390 and threads on this board instead).
