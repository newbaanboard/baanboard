Hi,

You can use standard BOD's for creating alerts as well as create your own customized alerts using LN Message and add a mail alert when the condition's meet.

In your Outlook, you can create a rule to display a pop up message when mail arrives. Other than that, ION doesn't offer any functionality to display pop up alerts in your LN environment.

If you want, you will need to do customization which will be a bit complicated.
