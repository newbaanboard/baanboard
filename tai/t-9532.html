In order for Japanese to work correctly with our English system, we had to install everything as if it were a Japanese install. Not sure what OS/DB/Baan version you are trying to install but in our case it was UNIX/Oracle/IVc4.
UNIX needed Japanese fonts installed/ Oracle had to be installed as Multibyte/ Baan had to installed as Multibyte.
An important note is that Baan has to be originally installed as Multibyte... it was not possible to change Baan after the initial install. Understand that Japanese will run on an existing system but each character takes up twice the space (i.e a 16 character field only held 4 Japanese characters instead of 8). This made translation very hard is some cases due to the lack of space.
As far as documentation, we found very little.
