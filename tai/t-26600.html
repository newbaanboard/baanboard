In Portingset 7.1d.11 there is a known problem in the repgen of this portingset. The problem is that when compiling reports, an error message is given if the print expression of a field in the report contains a double-quote character. The error message is: Error: 'X' not expected (where X is the character between the double-quotes in the print expression).
