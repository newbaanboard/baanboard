i'm not familiar with the MCR Version but for standard and cnt0 the tabel tiitm001, tiitm100 and tdinv001 must be filled. Actually table tdinv001 would automatically filled when you the first time have stock on the item, but still it is better you fill it for the compination warehouse/item where the warehouse is equal the value from tiitm001.citg.

Keep in track to set the field tiitm001.ccit and tiitm001.reli with the value from tiitm001.item.

If you have inserted the rows, you have to run the cost price calculation for the items. Then you are ready to use the new item.
