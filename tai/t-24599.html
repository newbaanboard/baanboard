Hi Ganesh,

You should install filterbgsb.exe. The purpose of this is to control the users that can log in the baan. It allows to create groups of users and allow just a certain numbers of users of each group at a time. It allows to limit connections to 1 for each user. Here you can treat group as department. This solution is for NT Users. For Unix, you can check similar thread or my thread.

I think this will solve your problem.
