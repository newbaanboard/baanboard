17th March 2010, 15:15
-set BAAN_WIN_TITLE=[%S]-[%f]-[%C]-[%c]-[%h]

This should display also the user signed in and the current company number in Baan IV
The same command should display different information in ERP LN.
That is because a couple parameters apply to one ERP but not the other. You can play with them and see what suits you. If you notice the -[] are displayed as is and the %A are the actual parameters.

%S Menu Browser
%f username
%C Company description
%c Company number
%h hostname of the application server

Nestor.

Walkaround:

Is it possible to add the server in the title bar of the Baan GUI?
What are the possibilities to change the display of the GUI for different servers?

It is not possible to get the name of the server displayed in GUI.

However it is possible to change the color: e.g. Red for test, Green for Operational.
Download the $BSE/gif directory of the test server, edit the pictures (e.g the GIF:menu.normal in $BSE/gif/browser.gif) and upload it to the test-server.
Every user will now see a different color for the test server. This is very useful when both servers have the same company numbers.
