You don't need a different port, just a different BSE. The BSE value will direct the login to the right Baan Software Environment (BSE). You will be giving the database a different name. You might have Baan or BaanIV for your current environment, this one may be called ERP or ERPLN.

Give VMWARE a thought also. You can experiment, and when needed you can restore the VMWARE server back to the initial install and start again. If you are using it to train users, you can refresh it back after each class.
