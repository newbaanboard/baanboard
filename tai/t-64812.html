Hi,

Does this error shown for the normal user?

If its for your user,
Check these things:-

1. Do you have Super User type for your user?
2. Check your Roles
3. Check if you have access to all package VRC's.

Generally, to start with you should be logged in as BSP i.e Baan Super User. Later you can create the roles and create new users and assign them roles.

Also, specify what exactly the error is, if possible post a screen shot.

If its for Normal User,

Check the Table authorization for Tables/Company.

You can give full authorization to all the tables in all the companies. And try again.

Pls. Note:- You need to do Convert to Runtime after creating users and log out and log into baan again, else changes wont take effects.
