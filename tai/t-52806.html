You have a few options for users to access Baan/Infor LN without installing the software on their desktops:

1. Worktop Central Deployment
2. Webtop
3. Citrix
4. Terminal Server.

I cannot seem to find a way to install the Worktop Client on an End-user's machine, by way of Central Deployment.
Central Deployment does not install Worktop Client software on the end users desktops. If you are looking at other tools for end users desktop software installations remotely refer to the thread - software deployment bw, becs , .bwc integration (http://www.baanboard.com/baanboard/showthread.php?t=28524&highlight=microsoft)
