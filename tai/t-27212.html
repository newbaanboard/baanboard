A Company can be linked to a Package Combination and not VRC.

Check for other sessions -

ttaddv6255m000 - Purge Entire Package VRC

ttadv6250 - Delete Software Components by Package VRC

Then remove package VRCs (ttadv1511m000), software directories (ttadv1115m00), remove packages from package combination (ttaad1211m000), then remove the link of company & package combination.

When creating a company on its own PC (package combination)/VRC's and customizations you usually do:
- create VRC's
- create PC
- create the company and then its tables
- customize (can be also done before the company steps)

here you need to do the reverse order:
- delete tables and then the company
- delete customizations
- delete PC
- delete VRC's
