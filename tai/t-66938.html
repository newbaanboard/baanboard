Hi,

What I did was, create a bat file to run the Jobs. And added the bat files to the Scheduler.

It works with/without spaces:-

Sample:-

set BSE=D:\Infor\FP7LN\bse\
set USER=baan
set BSE_COMPNR=200
D:\infor\fp7ln\bse\bin\startjob.bat REBUILD BALANCES
TASKKILL /IM cmd.exe
