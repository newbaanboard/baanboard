PB,

Please be aware that the 6.1 porting set release has been discontinued and has been without support by Infor for a while. If you continue to use BaanVI, you should update to the 9.x porting set release (latest 9.3c), which replaces the old one. Unfortunately, the upgrade requires a few steps and also testing of the business processes, especially, if you are on an older version without the latest Service Pack and additional solutions.

Also heads up, BaanIV is slowly being retired with first Infor changing the support and eventually retiring it, e.g., there will be no solutions provided anymore at some time.
