Hi,

While the functionality of BW remains the same i.e. connecting with the server via bshell, Worktop adds feature to it. It simplifies the user interaction by providing you multiple functionalities. Following are some functionalities worktop adds -

1. Easy to use and intuitive keeping the underlying technology hidden.
2. Lets you configure the BW for multiple environments on a single window
3. Lets you connect multiple BW without any hassles
4. Multi-language support
5. Enables Central Deployment Mode
6. You can switch to multiple environments at a same time very easily

Other features are -
1. Creating your own shortcuts
2. Creating your own Navigation Area, Work Areas
3. You can manage the appearance and customize it
4. You can browse the websites through it

And there's more..

To sum it up, worktop is an enhancement to the BW which still requires underlying BW to get itself working.

Someone else can come up with other features as well. There are numerous ones.
