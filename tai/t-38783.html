Hi Darpan,
You got to install Webtop first and then follow the procedures listed below..
1) Insert the Enterprise Server CD and locate the folder which contains the file "erp_B61Uastnd_SP1_en_xmlhelp.zip"
2) Log on to the Administration Console of Webtop
3) Click on the Help Content option in the Navigation Area of the Webtop Administration Console.
4) Enter the directory and filename of the above mention file in the path under the heading "Install Help Content".
5) Click the upload button and wait till you get an "OK" message...thats when your help server installation be over.
It would take some time and so be prepared..
