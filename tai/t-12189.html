This session is used to copy the templates from the Crystal server to the Baan server. Error -3 indicates that the Crystal Server is unknown.

You first have to select a Device where the Argument field is filled with a string like: XML server=<servername> port=<portnumber>

For other devices it's not possible to upload templates from the Crystal server.
