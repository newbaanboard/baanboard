I think you're missing my point...

In ttadv5210m000 there is an option "Also Dump Data Definitions"
There is rarely any reason for this to be checked. If it is, It will create fd files, as well as other files.. (i.e., you are using for reasons OTHER than
making changes to Tables and Domains)

As for software directories. Look in session ttadv1115m000
