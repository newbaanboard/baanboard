Probably you are getting this error because your auditdef6.1 is not correct.

To fix it, go to session ttaad4116m000 (maintain audit file directories).
You must insert at least a record with all (other) tables option for all companies.
In the menu special, run the convert to runtime. It will generate a new $BSE/lib/auditderf6.1 file. I recommend backup this file before you run the convert.

Hope it helps.
