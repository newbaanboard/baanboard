You can migrate, but the real question is how do I migrate?

Since you are on the same Version, going to a new release usually has some conversion programs to go to the next release. 4c4 was the service pack release for you. Over the last several years, service packs became releases and the stoped incrementing the service pack number 4cX. So you need to review each service pack for new functionality and decide to install each service pack and run the conversion programs supplied. Or export and import the data into the new table structures and initialize any new fields and turn on new functions.

You could do a test and add the 4c4 VRC's and simply change the package combination and see what problems develope during a system test. It may be more cost affective.

I would come up to the latest porting sets as there are some new web enablement features, webtop, and webhelp.
