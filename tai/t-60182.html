We had the similar issue of hanging bshell and it took a while to figure out the cause of the problem. The problem got resolved when we changed the Heap size settings in windows registry. Basically each Bshell connectivity will consume some memory in the heap size and it need to be increased from the default value.
In your case, it might got resolved because the default value in 64 bit environment is more.
