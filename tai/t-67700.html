Hi,

If its a customized session, you can use brp.* functions to manipulate and print the report.

If its a standard session, you can set the defaults in ttstpsplopen(though it causes problems, if in future you are doing an AFS on this or clearing user defaults) ttstpopen pops up when you click the print button on the session.

Another way is to create a custom session which calls the standard session through AFS, with the required device.

If the standard session is a part of any standard activity, you can set the default device in the activities as well.

e.g. Print Sales Order Acknowledgement Report.

The last way will be creating a Wrapper.

Hope that helps.
