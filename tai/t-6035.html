Check out session ttaad4111m000 - Assign tables to database.

This should have a list of tables which are assigned against your audit db. The default tables are normally the parameter tables.

If this is the case then add sales table you wish to run audit on into this session aginst the AUDIT db and convert to runtime using application menu.

The audit should then start on that table.
