garfield ruhr
6th June 2002, 18:00
Hi,

you can use the session ttstppwdaging to chaneg the password on OS level and for the database configuration. The session ttstppwchange is only a subsession for the ttstpaging session. The password changing does only work if:
1) password aging is enabled on the server
2) the notification time frame has started (defined on the server)

I hope this helps,

Garfield
