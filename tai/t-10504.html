Check the size of the file m_help.log in the LOG directory in your Baan installation directory. Probably it is in the mega bytes. Empty this file totally, do not delete it. That will solve your problem temporarily until it fills up again in a few months depending on the usage of your user.
To permanently solve the problem, download and install bw_068_I386 (the most recent Baan client) that does not utilize the file m_help.log.

Good Luck,
- Ameen Wassif
