Thank U all for your suggestions
I've just solved my problem without using a 3gl.
Well, I've done the following:
-You must insert text code into tables tttxt001,tttxt002 and tttxt010. Field tttxt002.refcntd may have the number of instances of tttxt010.ctxt (diferents text translations).
- Insert the text into table tttxt010 which you want to associate on table tfacr200. This text DOES MUST have a new line character (ascii code 10) at the end. This is a very important thing, because if you don't add this character, BaaN will not recognize the text.
- Update tfacr200.text with text code on the record.
