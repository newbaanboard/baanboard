SOLUTION DESCRIPTION:

With the delivery of service pack 5 of the B50_b release, the new delivery tool PMC (Product Maintanance and Control) is also delivered.
From SP5 on customers need to install solutions through PMC.
This solution contains information about how customers on BaanERP Vb should use PMC for installing their dumps and a weblink to the initial PMC solution dump.

1) In the document "Quick Guide Product Maintanance and Control for BaanERP" (document code U7275AUS), customers can read how to use PMC.
You can find this document on the SP5 cd or you can download it using the following web-link:

Create an update VRC for sp8+9.
Derive update VRC sp8+sp9 from sp7 for all packages, register the VRC as a PMC update VRC and copy solution registry with session ttpmc2290m000 from sp7 to the new VRC. You can now install sp8+9 in the new VRC. It is necessary that the base VRC is the same as the one you copy the registry from.

Please note that you have to add VRC's for localizations derived from sp8+9. Follow the previous steps to create a PMC update VRC for the localizations. Secondly you have to copy the locs to the new VRC. If you copy the solution registry for the locs, be sure that the new VRC is derived from the old one. Otherwise the session won't work. Next you have tio change the derivation so the locs are in the right place of the derivation structure.

Good luck!
