You can also see the individual file versions in your BSE folder:

e.g. d:\BaanIV\bin

Find a file such as audit_srv.exe

Right click, select the version tab then select the File Version item name.

This would show you e.g. 6.1c.07.01

It is possible to have different porting versions of different files but they will generally be from the same porting set/service pack.

Not sure how much help this is but just thought I'd say.
