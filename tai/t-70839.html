KB 1626966
Information about the Installation and Configuration of LN Studio for working in ERP LN and information for (the set-up of) debugging of ERP LN in LN Studio for Infor (Support) Staff.
Resolution:
LN Studio is the Integrated Development Environment (IDE) which replaces the LN classic tools for developing and maintaining LN related software components, as such it is primarily a development tool.

LN Studio can help support analysts in their day to day work as well. You might have noticed that the LN UI interface does not allow to use certain classic tools session, for example Program Scripts / Libraries (ttadv2530m000). For those kind of programs you can use LN Studio. Within this article detailed information is given about several topics of LN Studio interesting for support analysts.

In the attachments of this article you will find several word documents and briefings (movies) which will help you with the installation, the configuration and the usage of LN Studio.

We advise you to go through the documents and movies in sequential order. It's up to you if you read the document, watch the briefing, or combine them both. We have named them as 1A and 1B and 2A and 2B etc. as indicator how they relate to each other. An exception is topic 4. The word document describes the set-up for debugging using the LN UI, whereas the briefing gives instructions about the debug process itself.
