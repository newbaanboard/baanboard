Hi Milind,

Within DEM you can model so called "file based activities". These activities can point to a Word document, an Excel sheet, etc. If you want to run the Word or Excel application instead of the document, just use "winword" or "excel" as the file to run (same command as you would enter in the "Run" option from the Start menu in Windows itself) .

Within the Desktop Manager it is not possible to model external client applications, only Baan sessions and server commands.

If you are running 5.0c you can use Worktop as interface instead. This is a new interface replacing the Desktop Manager, DEM browser, Process Viewer, Menu browser and more. Within this interface you have a Outlook-alike Shortcutbar in which you can create shortcuts to Baan sessions, DEM activities, Help texts, URLs and also Word, Excel, etc. But unfortunately it is not available on 5b yet.
