The way we have it set is to define printer names in Maintain Device Data.
For example:
ACCTS_A4_LTRHEAD
ACCTS_A4_PLAIN

then on each workstation, you create windows printers that have the same name but have preferences for the correct tray etc.
example:
ACCTS_A4_LTRHEAD ip address 192.168.20.1 tray 1 portrait
ACCTS_A4_PLAIN ip address 192.168.20.1 tray 2 portrait

A bit cumbersome as you need to set on each workstation but it works.

Ed

just to clarify:

The Device name in Baan Device Data could be anything memorable, it is the Device Queue field that maps to the local workstation printer name.

that is: Device Queue name must be the same as windows workstation printer name.
In my previous example:
Device could be called DEPT1_A4_LTRHEAD, but Device queue name and windows printer name must be the same e.g. ACCTS_A4_LTRHEAD

Hope that helps
