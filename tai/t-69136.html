Hi,

To check if audit is on for a table use aud.audit.is.on.for.table(const string table.code(), long company) Refer this (http://www.baanboard.com/programmers_manual_baanerp_help_functions_audtr_aud_audit_is_on_for_table)

To set an audit for table use Audit Management module. You can check this thread (http://aridhol.kudos.be/baanboard/showthread.php?p=96142) for a sample document.
