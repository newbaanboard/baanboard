Hardei, you need to be carefull with installing servicepacks and portingset. You can't just install any portingset on any servicepack. Servicepacks are related to the version of you portingset and the portingset is related to the version of your operating system.

First make the analyses below,

To which level of servicepack do you want to upgrade ( f.i. SP20 )
Which portingset is needed at least for this servicepack,
Find out which current portingset you are running and if it is suitable
In case you need to upgrade your portingset, find out the current level of your O.S. and check if this is suitable to run the needed portingset.

Good luck again!
