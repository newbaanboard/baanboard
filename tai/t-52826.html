Initially we did face some problems with certain laserjets, cant recall which one. For a permanent solution, what we did then was to use different printer drivers. We created two printer setups in Windows.
First one, the default, based on the correct drivers and will be used by other applications.
Second one, based on HP Laserjet 6L drivers, would be installed by the name "Baan Printer" (for e.g). This same name will be given in the [Device Queue] field of "Maintain Device Data" session in Baan.

This did the job for us at that time...
