Hello,

In maintain user data (ttaad2100m000) change from SuperUser to NormalUser and assign default company to that user.

Maintain company authorisation (ttaad2121m000) for that user.

------
If there's an user with authorisation like the above user (as Normal) you can copy the authorisations between them in "ttaad2100m000".

Don't forget CRDD !

There is no way to revert it back exept to copy the authorisation from an other user to user "A".
