You should have a document in the italian localisation CD explaining all steps and what should be a propper VRC/PC chain, depending on your needs (if italy will use or not cnt for example). Italian VRCs will be B40L_c4_itl0 and update B40U_c4_itl0.

From a tools point of view you need to:
- create the italian language so that you can have all labels, domains, etc. in italian
- create the VRC's and PC's (and corresponding update ones) and import the italian dumps (there's a tt dump that contains the specific italian tools langage components)
- setup PMC to cope with the italian layers
- start applying [all] the SP's for italy - you also need to apply the langage SP's for each corresponding SP after each SP: RTDD needs to be run for the italian layers only so, online for other countries.

Look for the doc in the italian CD's for more details.
