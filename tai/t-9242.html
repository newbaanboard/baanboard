Dear Spartacus,

I think the correct sequence is supposed to be: create new package combination -> rebuild DD -> restart bshell -> create new company connected to the new package combo -> create table for the new company

Since you created new company at the time when its new package combo's DD has not been rebuilt yet, Tools probably can't create the table then. But if you tried creating tables now, it will work (because by now the DD for the package combo is already rebuilt)

Rgds,

Session ttaad1222m00 is create PC in one step, and (on BaanIV at least) also builds the RTDD for you.

Check the RTDD at OS level to see the permissions and if the files are there, if not, re-run a CRTDD full for that package combination.
