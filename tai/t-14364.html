You most certainly can create indexes in Oracle outside of Baan. If you have access to Baan Tools development then you can add other indexes through Baan too. We have third party applications that interact with Baan and Oracle and these applications have created indexes in Oracle. We found that these indexes get used in the case of running Crystal Reports or other independent SQL queries.
