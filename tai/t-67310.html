Vishal,

It is not possible to use the old license daemon in a VM. The license daemon has to run on a physical server. If you have a physical server running the license daemon, you can point the VM environment to that for the licensing.

The better solution is to switch the licensing to the SLM. The SLM server can be run on a VM (same as the BaanIV environment) with the limitation that it will only provide licenses to other hosts in the same domain, e.g., slm1.mydomain.com.
