Yep, the same problem here.

I do the following :

copy BSE environment from old server to new server
copy the Baan-Installation from c:\baan\winnt\baan (you must look, where it is installed) to the new server c:\windows\baan.

Now the Problem is, that you had no services for baan.

I had created the services with srvinstw.exec from MS Resource Kit on the new Server exactly as they are installed in old server.

Start services.

Update Portingset.

The next Problem is, that you had no Program Group Baan and the old copied BECS won't work. --> Install the prefered Client for the new Portingset on the new server, then it works.

hope this helps

martin

the registry setting i had exported from registry of old server, modified and import in registry in new server
