For those who did not receive an answer:Thomas311:
1 : process 4 - Error : Read row error 506 on ttaad321000 (error 0)
2 : process 4 - Fatal error : Error 506 (Table does not exist) on SELECThere a non existing table ttaad321 in company 000 is being read. The table should be created as explained in the solution text of the solution that delivered the table definition. This is also a post-installation action that is defined in PMC.
