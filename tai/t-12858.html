You can import into a new VRC in the other environment, so I don't understand the problem that you have to restore to the same environment. As far as I know, there are no differences in the dump files between C2 and C4. As for the modifications based on the standard C2, these will have to be redeveloped in the C4 environment, since there might be table changes to the tables used.

To export only the sessions and/or code, excluding the data dictionary, uncheck the option to base the export on sessions, and, on the 2nd, 3rd and 4th form, specify exactly the components you want to export.

Another way of exporting/importing? None that I know of! And creating the patch file yourself? If you can get the layout of these files I presume it is possible...
