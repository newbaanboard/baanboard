Hi Alex,

I've been doing this only 2 times for Baan IV and 4-5 times for E-Procurement. The first step is: goto Baan Support (http://www.support.baan.com). On the left side, you see "Validation Forms", that is where you order a new license. You need to know a lot such as Number of Licenses. The answer might take some time (2 hours was best, 2 weeks the worst case).

Good Luck!

This would be considered a brand new license. So you need to go through
the doc "How to validate a single host".

Before that, you should be able to run ttiex3228m000 (Patch objects after system crash) to allow you to run sessions.
