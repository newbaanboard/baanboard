Another solution is to put:

0 and [existing print layout condition]

in the first part of the print layout condition. This makes the layout not printable.
Is a better solution, because if you make some error and delete a layout by mistake, the layout is already present and you can return easily to the starting point.
Make attention, if the layout is in the report script (obviously if exists), all the logic in it are already executed even if not print the layout.
