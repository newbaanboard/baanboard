Hi,

According to the SSA global BAAN site:

Revalidation of existing installations is required in case of the following events:

Hardware changes
Updates/changes in versions
Hardware or operating system upgrades
Changes between one database or another (e.g., Informix to Oracle)
Increase in numbers of users
Catastrophic failure of system requiring a complete reload
