Hi nowaki,

It has been discussed many times on this forum (please use the search-option):

Only older versions of the license server can be used in a virtual machine. So I guess you're trying to run a version of the license server which detects a virtual environment and stops working.

Anyway. Maybe it simpy doesn't work because the MAC-address of your (virtual) network device has changed. Everytime the MAC-address is being changed, also the "host id" changes and as the activation key depends on how the host-id looks like, you have to request a new activation key from Infor. I think only changing the MAC-address of the (virtual) network device to the proper one will finally solve the problem...
