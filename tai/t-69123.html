Hi,

You can use Dynamic SQL in that case.

Check this similar thread (http://www.baanboard.com/baanboard/showthread.php?t=26096) and this specific answer (http://www.baanboard.com/baanboard/showpost.php?p=94346&postcount=4)

Or, use wild-cards -

domain tcmcs.str10 find

find = "Fixed"

find = ".*" & find & ".*"
select tcibd001.*
from tcibd001
where tcibd001.dsca LIKE :find
selectdo
tcibd001.item = ""
endselect
