Hi there,

Unfortunately I do not know of any other way of installing help. You have to install webtop and then you can install the help files from Webtop as given in the documentation. You need to do this because the help content ultimately opens in internet explorer and webtop is the component that configures ln 6.1 for use with internet explorer. You can install webtop even if you do not want to use it.

Webtop and Worktop are 2 different GUI options - one is browser based and the other is client software based.
