Hi Bastien,

is the solution 122976 installed? If not do so.

The difference of the editors is if you use the Baan internal editor tttxt2230 or word as the external editor. If you use word incl. RTF then the asci data of the text is stored in tttxt010 and the RTF data is stored in tttxt015. The RTF data seems not to be used by Baan standard but can be used for own programming.
