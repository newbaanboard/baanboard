As per BaaN Support People (Xeon/Itanium) processer are not supportable.
Xeon and Itanium use quite different IA's, where the first one uses the same IA as the existing Pentium's, so Xeon should run Baan like any other Pentium processor. Itanium needs specific compilation and Baan has not delivered a portingset for it (I also doubt that they willl ever release one...).
Most of the Windows servers are running on Xeon, and for a long time now, maybe they are speaking about the IA-64 Xeon processor instead and not the "usual" Xeon?

I want to Port BaaN Application on Xeon Based Server, and Database server to port on Itanium based (64 Bit) server with Oracle 10G database.
I see this feasible since Baan does not care on which OS/platform the DB is running with these types of setup.
