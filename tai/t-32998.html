What I mean is in Windows 2003. I did find the tabledef6.2 in D:\baan, but the syntax is similar to the previous system.

First: Please update your profile from AIX to Windows.

Second:
1. Please check the informix instance name you want to connect to.
2. Make sure you Bring up the Informix instance (onstat - OR oninit).
3. Make changes in "Database Definitions" session OR $BSE\lib\defaults\all file to point to the correct instance.
4. Make changes in $BSE\lib\informix files for baan password, etc.
5. Restart baan.
