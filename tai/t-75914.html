the file $BSE/lib/systemmess is displayed after login. This has been discussed several times. But all discussions mentioned it as text file.

I just played with the cf$() and fg$() functions to create that file an watched the effects. The good news: you can easily add some colours to the display, even with a simple text editor:

ESC 0 = black
ESC 1 = read
ESC 2 = green
ESC 3 = yellow
ESC 5 = magenta
ESC 4 = blue
ESC 6 = cyan
ESC 7 = white

Note that ESC is the character ESC; if you are on a UNIX system and using the vi editor it's Ctrl-V folled by the Esc-Key, and it's displayed as ^[.
