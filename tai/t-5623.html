With bdbreconfig:

In case of interrupted reconfiguration while the R.<table> file still
exists, the following command:

bdbreconfig6.1 -N ttadv100222+

will use the dump R.ttadv100222 to rebuild the table.
(Warning: make copy of dump before doing this !!)
