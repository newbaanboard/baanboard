Hi NPR,

startprogram_gui: ottdskprogman -d<desktopname>
This indicates which Desktop to execute (when the user logs in).
If you check your Desktop table (ttdsk200), u'll have a Desktop = cfgreene

startprogram_gui: ttdskbbrowser -n<searchdepth> -m<menuname>
This indicates which MenuBrowser to execute (when the user logs in).
