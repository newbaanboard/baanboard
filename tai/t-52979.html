If you do a print preview with BWPrint and right-click over white space on the preview page, you can choose Change Position from the context menu. On the Change position window, there is a checkbox that says Legacy Mode. If you select that, all of the reports should print like they did on your old client. Unfortunately, this does have to be done on every client. There is a corresponding registry key, but I don't remember what it is.
