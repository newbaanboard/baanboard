More Specifically. .


1. Go to Tools -> Application Development -> Packages and Modules -> Modules i.e Session ttadv1101m000.

2. From the tool bar select New Group. This will let you select the package in which you need to create your module.

3. After selecting your package, it will display the modules present in the package.

4. From the tool bar, select New and name your module and save it.

You are done!
