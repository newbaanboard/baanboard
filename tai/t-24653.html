(1) Setup OS user accounts, and authorize them
(2) Add user account in session of Maintain user data
(3) In the session ttaad2500m000, select the user and from Specific select the option Convert SSA ERP user to Database User.
(4) In that new subsession (ttdba0915m000) you can select 'Use System Logon for Database User' and then click Convert. System will ask you the password for database admin (that is user sa). Provide that and the user will be created in SQL Server.
(5) Convert to RDD
