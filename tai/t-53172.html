Hi,

Have you installed Webtop and the web help contents? Once you installed its content, you need to specify your webtop server which contains your help files.

Try go to Session 'ttams1110m000' and under the Help Server field, insert your webtop server IP address or hostname eg:"<hostname>:portnumber/webtop".
Once inserted, convert runtime for your user and relogin. See whether it helps
