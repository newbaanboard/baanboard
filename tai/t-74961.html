Point is that PMC does just that: it makes sure that all changes are delivered in solutions, so it keeps record of the components in the delivery project and those component registrations will block delivery of later versions.

You should be able to find the delivery activity UUID (tlprj120, type of activity Delivery, note the UUID). Removing the components from ttscm115 and ttscm117 will allow you to not deliver the previous PMC solution...
