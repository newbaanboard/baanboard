I am not fond of these built in Baan Reports. However, many of these reports can be reproduced using simple SQL at the database level. I like creating Excel Query Files (.dqy) where you can go into Excel, from the Data menu, Get External Data. This will bring the data right into Excel and if you give Column Names in your query, the column names you specify will be set as well.

You can get real fancy here by defining variables so that if you hit the "Refresh" button, you will be prompted for the variable data based on the prompts you defined in your .dqy file. My users love this!

If you need some examples, let me know.
