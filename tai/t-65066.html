Try this:-

ret = run.baan.prog("compress" ," -c", RP_NOWAIT, "C:\test\test.txt", "D:\test\test_1.Z", "C:\test\err.txt")

Also, check the permissions to the folder.

And, you are trying to print the return value.

If the command is successfully executed, you will get message = 0.

If you want to put the file contents in the message, you will need to read the file using seq.* options and then put it in some variable and show it using a message() command.
