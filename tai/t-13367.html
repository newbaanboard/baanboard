Francesco,

Here are my observations-

I found 2 diff's on our Unix system. One from my unix profile path and another from the baan ksh path. I verified that they were both same file size, located in /usr/bin/diff and /bin/diff. You can check using - $ which diff
What do I set BAAN_SCM_GRP to? (or what is common there?)
You do not need to set it to anything. Bshell automatically picks it based on the current login unless you specify it to something or override when using it in the BW configuration file.
In ttadv1115m000 (Directories of Software Components), only the Forms path was changed.
Looks like a bug. It has to be set for all the other components.
Do I have to convert to runtime after this?
Yes from the specific menu. Usually this is done during the VRC/package combination creations.
