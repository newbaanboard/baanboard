ttaad4226m000 for export table - select [field seperator & directory for table files] check box
ttaad4227m000 for import table - select [delete existing table/disable domain constraints/ignore referential integrity contraints / field seperator/ file to store & name of error file] check box

both can be used for backup & restore single as well as range of tables.
