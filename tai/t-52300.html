From the Tools Technical Manual:

sh_server6.2
Shell server used to execute system-dependent commands.

I assume when a client takes login, this is one of the process started at the server side, but its taking one license...
In our LN Tools Version:
The bshell consumes a license and not sh_server.
Bshell starts a sh_server if required.
sh_server is not started when a user logs in and does nothing.
