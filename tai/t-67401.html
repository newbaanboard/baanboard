Hi,

I guess if you are upgrading, you do not need to install/stage ES again. You will need to upgrade Porting set and install PMC solutions.You didn't mentioned about porting set upgrade. You will need that. Also, 10.3 comes with additional connectors for Webservice which requires the add-on PMC solutions and this comes with da, nt, ta and tm packages.

Note -

Mandatory

• Infor Staging Wizard.
• Infor Installation Wizard.
• Infor ES Porting Set.
• Infor Support Assistant.
• Infor ES PMC Solutions.
• Infor ES AddOn PMC Solutions.

Do not stage these IUs while upgrading.
• Infor LN
• Infor Enterprise Server

As said, please follow U9497I US document carefully. Everything is clearly mentioned.
