Hi,

There can be many reasons for Runtime Error in job.

If the session is interacting with the directories, you need to check the write permissions. Check if there are any end() or exit() kinds of statements in job. Check the date initialization. Check if the session doesn't go to any infinite loops.

Try to use statement like:-

if job.process then
message("Process Completed")
job.process.error = false
endif

Check the log files too.
