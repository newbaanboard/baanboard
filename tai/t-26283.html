Hi Han,

there seems to be some confusion here about the terms "SLM" and "SLM Management Console". SLM itself is a server process which can run on a Windows or a *NIX machine (not neccessarily the Baan server!!). "SLM Management Console" which can only run on Windows, is on the other hand only the client application for managing the licence data which reside on an SLM server. The SLM Managment Consle can actually run on any PC in a network. I for exxample have installed it on my notebook and can manage any SLM-Server by changing the IP address pointing to the server in the XML file "servers.xml".
