Basicly:
- read the manual!
- upgrade portingset;
- upgrade tools;
- install newer version in separate VRC;
- check latest information upon your new version;
- create other own VRC upon;
- install help files;
- upgrade custom software by 3rd party;
- copy own customisations to new own VRC;
- copy production company to new company in old version, mind the renumbering;
- move test company to new version;
- run Data Upgrade Engine
- Test

Sometimes it's advisable to update your old version to the latest solutions!

Please check the manual which was mentioned bhushanchanda because that contains ALL information.
