A cloning needs further steps to ensure that:
- or you run for 30 days unlicensed (by patching objects after system crash)
- that you validate the cloned server with its own license
- or use the originating Baan installation licenses (license6.1 must point to the originating server)

From that message it seems that none of the above was done, so your Baan cloned installation will not work. Search the tools threads for the above options for more information.
