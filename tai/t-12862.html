If running on ASCII then just use the OS.
If running on GUI then connect with a
different bshell eg. bshellbw
now add the following to $BSE/lib/ipc_info
bshellbw s 0 0 p ${BSE}/bin/bshell6.1bw
You will now need to create the script
${BSE}/bin/bshell6.1bw in this script test if the
USER's password needs to change if so
make him change it. At end of script
start bshell as follows:
${BSE}/bin/bshell6.1 $*
