Don't have an easy manual, but found some issues unless you do the following.
SQL2000 Win2003, fresh install of LN
1. Log into the server as local user Baan with password baan
2. After installing SQL run vcredist_x86.exe. The runtime files are delivered in a file, called 'vcredist_x86.exe'. This executable is delivered in ..\InstallableUnits\SSA ES Porting Set_INTEL-WINDOWS2003 8.3a directory of the LN DVD. You have to run this vcredist_x86 executable to get the required Microsoft runtime files installed on your system.
