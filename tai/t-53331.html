It is possible that a date in one of the records related to the production order [example, Planning Data [tisfc010], Hour Accounting data [tihra100], etc.] is a date that is beyond what is defined in the calendars.

I would check the Company Calendar to see if it has enough date in the future. For example, we have enough dates upto 2014.

Then, since completion involves financial postings, also check if you have enough future dates in:

Period Table (fiscal, reporting, tax), tfgld0105m000

Period Status (fiscal, reporting, tax), tfgld0107m000

Period Table, tihra3101m000

There may be other calendars to check if you use BAAN Projects, as follows:

Cost Control Periods, tpppc0110m000

Hours Control Periods, tpppc2121m000

Periods by Quarter, tphrs3100m000


Make sure you refer to the session help of each session and consult with concerned people in Planning/ Finance/ etc. before making changes, as things may be different for you.
