JaapJD
17th February 2006, 13:14
Installing SP3 - 18 is a quick way to install an environment upto the last SP in one go. In this case all necessary data is added to PMC to be able to continue with installation of individual solutions and subsequent service packs. You will not see history or installation runs.
To see whether your installation is successful is difficult, because you had to do a lot of manual imports. But if you executed all steps in the installation guide and those steps were successful, you can be sure that the installation was successful.
From now on every time you install a solution or range of solutions, an installation run will be created, which you can see in ttpmc2503m000.

21st February 2006, 09:36
I didn't mean that you have to do additional manual imports to check your installation. I meant that if you did all steps from the installation manual, your installation will be correct.

However, I'm a little bit concerned about
cause this is the first time i upgrade the service pack
because the batch install solution (SP3-SP18 in one go) is not to be used for upgrades, but only for fresh installations, where the VRC structure follows the conventions described in the installation guide.
So, if you upgraded an existing environment, you might have problems. In that case I would recommend you to contact SSA Support.

You will be able to install solutions. But of course none of the solutions which are included in SP3-SP18 (if you imported the PMC registry correctly!). All solutions SSA makes on top of SP18 (and which will be included in SP19 or newer) can be installed.
