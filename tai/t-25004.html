hello

Try to create a new brandfile via the brand utility as described in solution 82511 / 102338.

Probebly your virusscanner corrupted the brandfile in the Windows directory during the upgrade.

Backup the tables ttiex300 and ttiex301 in company 000 via an MSSQL backup or Baan dump.
This will enable you to re-create the brandfile via solution 70604 in case it is corrupted again.

Note:
1. Not enable Microsoft Automatic Updates on Live server but do it when no users are on the system.
2. Maintain a backup or a disk image before you do any updates so you can revert back.
(At least backup the ttiex300 and 301 tables for licensing issues)
3. Disable the file [Windows directory line c:\winnt or c:\windows]\baanlic6.2 (brandfile) from any virusscanner actions.
