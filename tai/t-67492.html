Hi,

You can add a form command for the session

Activates a - Session
Session Code - Your session code to be started on double clicking


Now double click on your main session in Session's (ttadv2500m000) and under form tab, Select Default Button - >

Command Type - > Form command
Default Button - > created above

Done.

In the child program you need to code this-

before.program:

refresh.parent(-1)

to find the right record on double clicking.
