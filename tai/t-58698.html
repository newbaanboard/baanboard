Hi,

You can always select the printer in printer setup as specified by reeyah .

In case of Baan 4c4 , you can build Device type specifying the share name of the print server to spool directly.


example
\\10.10.10.1\Accounts - where 10.10.10.1 is IP of print server and Accounts is the share name of the printer.

Given that the client and print server were both in Domain or mutual hand shake is done

Steps .

1.Create a New print Device - (with appropriate name)
2.Specify 4GL as ttstpconv (similar 4gl program in LN)
3.specify the share name in path ( eg: \\10.10.10.1\Accounts)

uncheck change allowed

Note: It is always recommended to route through print server not directly to Printer IP.
