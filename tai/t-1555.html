Dear Mihamax,

You need to perform steps of soft crash.

But I would recommend to contact to BaaN support immediately.

1 Restore back-up (optional)
It may be necessary to restore a back-up on the system.
2 Remove the /.brand/.brand6.1 file
To prevent the license daemon from starting up, the /.brand/.brand6.1 file must be removed.
3 Run the rc.start shell script
Next, the rc.start shell script must be run to initialize both the printer daemon and the shared memory.
4 Validate Tools Set (ttiex3220m000)
Next, the Validate Tools Set (ttiex3220m000) session is run to create a new /.brand/.brand6.1 file.
5 Configure Application (ttiex3225m000)
Finally, all objects are patched in the Configure Application (ttiex3225m000) session.
