Hi,

The splash screen of Worktop is stored under the name "WTsplash.bmp" in the \resources directory of the Worktop installation. If you rename or remove this file, Worktop will not be able to find it and thus not show it (Wortktop won't crash on it).

If you use BW only, there is no way to remove the splash screen. It is part of the language specific resource dll located in the \res directory of the BW client installation.
There might be a workaround however, depending on your situation. Why do you exactly want to remove this splash screen?

regards, Ivo

Soem different splash screens are described here. You will get a splash screen when you start BW or Worktop but also when you start a session from BW or Worktop.
For Worktop and BW client on Baan 5, it is as i described before. For the BW client on Baan 4 there is indeed a bwlogo.bmp file in the \lib directroy, however the splash screen will still appear if you remove this file, it only won't show this icon in the splash screen.
For the splash screen appearing when you run a session, the icon is stored on the Baan server and called by the 4GL engine. You can customize the icon but don't know what happens if you remove it. I guess that BW will not be too "happy" if you do this...

Regards, Ivo
