Hi,

a few weeks ago I followed these steps:

1-> Install Operating System
2-> Install SQL Server (same path for log and database)
3-> Install Baan Master Cd (same path for BSE folders)
4-> Stop Baan services
5-> Dettach existing baandb (created in step 3)
6-> Restore baandb from backup
7-> Restore BSE folders from backup
8-> start Baan services
9-> Log on Baan -> if logged with no problems, validate your system and that's all.

If you cannot log into baan, you may have to use SQL management studio to force user role membership for database baandb users.
In that case, please post the board.
