Updates on Porting Set

The Release notes for Porting set 7.3a.02 for the UNIX and NT platforms:
MaintReger: # 15327 (BDUX8872): Oracle 9.2
Date: Mon, 9 Sep 2002 10:18:57 +0200
Created on: MaintReger
Type: porting specific change
Problem Description (Customer terms)
Oracle 9.2 is handled and reported as Oracle 9.0.
Problem Description (Technical terms)
The Baan/Oracle drivers using the shared Oracle libraries (7.3x porting set and higher) do not recognize Oracle 9.2 yet.
More specific, Oracle 9 .0.1.3.x only.
