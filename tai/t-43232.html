sukesh75
23rd October 2007, 09:34
It seems like you have all the pre-requisite settings needed for installing Ln. However as a last attempt, i would look in to the Local Security Policy of the server(you can find this in Administrative Tools). In there, go to Local Policies and inside that expand User Rights Assignment. In the expanded right hand side, check the following policies.
1) Act as Part of the Operating System
2) Log on as a service
3) Log on locally
In the above three policies, add the user "baan". Apart from that, in Computer Management>>Local Users and Groups, remove the user "baan" from the "User" group and let the username share the exact privilige the user "Administrator" shares in the section "Member of"..
