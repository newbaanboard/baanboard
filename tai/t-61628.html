HP Color LaserJet CP3525/CM3530 Multifunction Printer Series - 41.3 Error Using LEGAL and/or OFICIO (8.5 x 13 in) Paper Size
Issue
The error 41.3 Load Tray : [Type] [Size] is related to the specified tray is loaded with media that is longer or shorter in the feed direction than the size adjusted for the tray.
NOTE: This message also appears if two or more sheets of media stick together in the product or if the tray is not adjusted correctly. If using glossy paper, ensure that it has been acclimated.
In some country from Latin America, the most common paper used for some user is the named OFICIO paper size. The OFICIO paper is badly named as Legal paper, but there exist differences between those papers. The LEGAL paper size is 216 x 356 mm (8.5 x 14 in) and the OFICIO paper size is 216 mm x 330 mm (8.5 x 13 in).
Solution

For normal 41.3 error :
Touch OK button to use another tray.
Reconfigure the size in a tray, so that the product will use a tray that is loaded with the correct media size.
If the message does not clear automatically from the control-panel display, turn the product off and then on.
For 41.3 error using LEGAL and/or OFICIO (8.5 x 13 in) Paper Size:
The solution to prevent the 41.3 errors is to set the printer as 8.5 x 13 in as Default paper size when the user is using OFICIO paper size 216 mm x 330 mm (8.5 x 13 in).
