DSchneider
8th September 2009, 20:51
You cannot run a vmware in a virtualbox and vice versa. When try to start a virtual machine in a virtual machine, the needed acces to the hardware layer is not given, so the startup hangs.
You have to start vmware and virtualbox side by side and connect them to eachother via network. That will run well.
