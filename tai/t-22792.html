If you have a list of users, you can create a tttaad231000.S file with an editor and then load that file.
After that, we use a query to tag all users to be converted:

db.retry.point()
ttaad400.rkey = ttaad200.user
ttaad400.modt = "u"
ttaad400.dsc1 = ""
ttaad400.dsc2 = ""
db.insert(tttaad400, db.retry, db.skip.dupl)
ttaad400.modt = "p"
db.insert(tttaad400, db.retry, db.skip.dupl)
commit.transaction()
