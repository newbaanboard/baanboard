Did you identify the barcode directory in the driver file? I'm using MK which is very similar to Baan4 and assuming the barcode mechanism is similar, you should have a line in your print driver file that reads:
barcode_dir=htplj4
Obviously, your printer needs to have a suitable barcode font installed and the barcode script may need to be modified to call it up, for example:
echo "\033(0Y\033(s1p8.00v11.54h0s0b16384T\c"
calls my Morovia code 39s 8 point font.
