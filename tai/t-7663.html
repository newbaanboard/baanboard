Dear Grzegorz,

You can issue command right at database level. For example on SQL Server, you can issue this at its Query Analyzer:
UPDATE table_name
SET table_name.field=value
WHERE conditions
The command might differ from one database to another. As in case of modifying through GTM, pay very serious attention on what you are modifying. Be very very careful. Test the command. Back up associated tables first if you're not absolutely sure of what you're doing
