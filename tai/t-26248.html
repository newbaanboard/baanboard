SOL 207169 - ODM setup data dump for B52a
VERSION(S):
B52_a

SITUATION IDENTIFIED IN:
"Import System Data" (dmsys2210m000)

SITUATION DESCRIPTION:
ODM master set up data dump is not available; hence it becomes difficult setting up ODM in a newly created company.


SOLUTION DESCRIPTION:
ODM setup data dump is made available. Users can download this dump and import it in a newly created company using "Import System Data" (dmsys2210m000) Session.


SPECIAL INSTRUCTIONS:
Before you start importing the dump, unzip the downloaded solution file i.e. 207169.zip on your system. Copy the contents of the unzipped file somewhere on the ERP server. Start "Import System Data" (dmsys2210m000) Session and specify the path of the unzipped file on the server in the "Directory" field and click on the "Continue" button.
After you get a message "Process completed." message, see if the data has been imported using corresponding ODM Sessions.

Please note: The user who is running "Import System Data" Session for importing the ODM data dump, should have execute permissions for bdbpre6.2 and bdbpost6.2 programs located under $BSE/bin directory on the ERP server.


PATH NAME IN DIAL-UP DIRECTORY/WEB SITE:
