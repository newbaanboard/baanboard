Yep...you need to first define your package VRC for Development purposes (not runtime purposes, they can be different). You do this in "Change Current Package VRC of User" (ttadv0140m000 in Baan 4c) under the Tools / User Management / Developers Data / Maintenance menu.

After you define this for the first time, you can use the pull down options "Change Current PVRC of User" and "Make PVRC Current" available in most Tools sessions to change it.
