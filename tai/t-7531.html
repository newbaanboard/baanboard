Dear Sat,

Pls read the following thread, you have to set it up in the similar way, making the client's computer effectively a printer server for the printer attached to it.
Printing to network printer (http://www.baanboard.com/baanboard/showthread.php?s=&threadid=5353)

If, however, what you want is the instant ability to print from Baan to any printer coming with a new client (say a person comes with a new laptop and portable printer), then after installing B5 there, print to any Windows printer defined in B5 sys. Because B5 wouldn't be able to find that particular printer set up in the new client, it would automatically ask if you'd like to print to your local printer. Answer yes and it will print to your local printer. But this way, you'll always have to go through this annoying msg. It's still better if you fix it up as mentioned in previous paragraph
