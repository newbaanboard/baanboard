If this is happening when in maintain sales order lines or maintain purchase order lines (tddlev domain), there are some patches for it.

A Create to Runtime Data Dictionary is necessary. Tables, Domains and Reconfigure tables must be set to yes.
All users must be off the system when Create to Runtime Data Dictionary is running.
