For reports PMC uses the data that is shown in session "Display Objects by Report" (ttadv3536m000). So, if the report is there in an Object VRC that is derived from the PMC Update VRC, it will be reported as a customization. You can try to remove it with session "Delete Unchanged Objects by Report" (ttadv3236m000). But, this session will not delete it if the report uses components that are in customization VRC. In that case you can get rid of the report by copying it to your customization VRC and delete it again. In this way the object will be removed in the dictionary as well.
