Portingsets are downward compatible, although there are functions built in that may require software updates in the tools as well to be fully operable. My advice would be: select the 6.1c.06.05 portingset instead, also update the $BSE/include6.1 directory using solution 114812.

Is your system c2 out of the box or did you ever install software updates? Do not forget to upgrade your system with the previous already installed solutions...
