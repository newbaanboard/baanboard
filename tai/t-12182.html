You have not specified, what purpose you want to make a jobs. If you want to make job for running batch file, you can use either AT command on DOS prompt or you can use GUI Scheduled Tasks. Type AT/? on DOS prompt for more info. and install GUI Schedular through :
a. Start
b. Settings
c. Control Panel
d. Add/Remove programs
e. Click on Internet Explorer
f. Click on Add/Remove button
g. Select Add a component
h. Choose Offline Browsing Pack and press OK.

After installation you can see online help. That's it.
