Refer to -

Infor LN Installation Guide
Infor LN Release: 10.4
Publication Date: December 4, 2014
Document code: U9498IUS

Database settings
This section provides general guidelines for a correct database configuration for LN. The database instance must be created manually. The database can be created through the Installation Wizard.
The installation wizard does not create directories for database files. Ensure these directories already exist.
Disk space
For performance reasons we recommend to allocate the full size of the database tablespaces according the sizing recommendation during the creation of the tablespace. For example, if the sizing requires 500 GB disk space for the database for the coming two years, allocate that size during the database creation. In order to decrease the disk space, the compress database table on functionality can be considered.
Character set
Before you start the installation you must have decided what Character Set you are using for LN. We recommend that you use Unicode. When using Unicode, integrations with other applications are possible and extra languages can be added to your environment. Changing from Single Byte to Unicode after the installation is a very time consuming conversion. Compared with Single byte, a Unicode character set has impact on performance and requires more disk space. For more specific sizing
information and the impact on your hardware, contact Infor.
