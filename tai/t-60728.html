We to just print sessions by VRC you could use ttadv2400m000. Or if looking to compare sessions you could run ttadv6450m000 Compare Package VRC's. This will show you were you have a session in your production VRC and one of the Baan VRC's. You can read the help on it. This is the session we use after we install a SP. We run this and check our components against the baan components.
