The domain should be existing in the Tools domain's definition file:

${BSE}/tools/dd/dtt.pd/dtt.pd

Sometimes, between the tools patches & service/feature-packs some objects can be expired, so its a good idea to regenerate the shared memory file from session ttaad4150m000.
