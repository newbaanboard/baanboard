Problem
When a user tries to access text from any session, instead of pulling up Notepad, nothing happens. This started happening when his desktop operatingsystem was upgraded to Windows XP Professional. It looks like Baan can't find the Notepad executable.

Sol
We fixed it. Our PC tech figured out that with XP, he had to explicitlygrant modify permissions for the user to the Baan folder containing the executable.
