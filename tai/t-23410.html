The problem is a bit confusing but let me take a tab at translating =)

You've linked Baan user "A" to Database user "bsp" and when you convert
to runtime you get that error message?

That is becasue the convert to runtime does 2 things
1 it creates the user in the DB
2 it updates the ora_users file

since "bsp" already exists in Oracle you need to tell that to Baan
you can do this via session ttdba0515m000 select the user and
check the status. I think in your case it is set to "Not Created in RDBMS"
you can then edit the table (maybe ttdba015?) and switch this to created
in rdbms. That would fix the problem.

Hope this is close to your problem!
