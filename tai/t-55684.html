The procedure to enable license validation by means of Infor Solution
License Manager consists of the following steps:
1 Prepare the Solution License Manager:
a Install SLM Server.
b Install SLM Client.
c Configure and activate Solution License Manager.
2 Prepare the ERP Baan environment for SLM licensing:
a Install Porting Set 7.1d.12a or later.
b Install PMC solutions with security files.
3 Enable Solution License Manager.
4 Test the license validation process.
