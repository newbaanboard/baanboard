On my Server in VMware, there is the string "<HOSTNAME>:8312/webtop" set.
There is often a installation problem after uploading the helpfiles. Please check your path under the 'ese'-Installation folder.
on my side it is found at: "c:\infor\erpln\ese\webtop\web\help\baanerp"
In that folder, you will find some subfolders like "76a", "B61a" and "B61Ua5stnd".
For the normal application (not tools), the system is searching for subfolder named "B61Uastnd". So if you rename "B61Ua5stnd" to "B61Uastnd", it should work. An other way to solve the problem is to change the search order of help files in "Infor Webtop Admin"-Application.
