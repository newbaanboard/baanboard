The porting set 6.1c.06.03 will be available at the end of January 2002 per Baan. This resolves compatibility (memory) issues that existed with the .02 version and True 64. The solition in the previous tread is where you;ll get the 02 version. With 6.1c06.02 true 64 5.1 does not allow the bshell to go beyond 93M of memory. This has been resolved in 03. Beware the you may get predicate errors with this new version since its is stricter in parsing the SQL statements and you will be forced to download the latest version of some sessions where the sql statments would have been corrected.
