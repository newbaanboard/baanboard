After creating package you have to create package VRC for this package (session ttadv1111m000), the next step is Maintain Directories of Software Components (ttadv1115m000) and next step is Add package VRC to the desired package combination (ttaad1121m000). Probably you also need to run session Maintain Modules (ttadv1101m000).
Session codes are from Baan IV c4.
