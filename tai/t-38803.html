Hi Vijai,

I have been using a Linux based open source package called smssend for some years now and in combination with a stable SMS provider, it works like a charm. The homepage of the creator has been unavailable for a while but you can download the required sources also from this page: http://systembash.com/content/smssend-zabbix-clickatell/

Personally I use the UK based provider SMS-Gold (http://www.sms-gold.com) as SMS gateway. I have found them to be very reliable.

Note that the most recent smssend package does not ship the sms-gold provider file anymore. If you are interested I can post it here. However, it does require you to use smssend v3.4 instead of the latest v3.5. The latter one keeps throwing errors back at me when using the SMS-Gold service. So I stick with version 3.4.
