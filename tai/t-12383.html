Veyant,

You just need to make a session with no form and a 3-GL script attached to it.

The code is quite simple -

function main()
{
e = shell("passwd", SHELL_MWINDOW)
}

I've done the same but implemented it differently which does not involved writing a session. Above suggestion works too.

Use a menu option and select type "Shell Program".
Then write a small shell script and place it in the
/baan/bse/bin directory.
Put the name of the shell script in the form's Sess/Menu/Pr/.... field and then dump the menu.

#!/bin/sh
cd /usr/sbin
passwd


is all thats needed in the shell program.
