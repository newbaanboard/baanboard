Hi Sarvjeetb

For creating users in server you need to do the following things:

1. Create a user in server through Computer Manaement
2. Define user in oradata
3. Now add user in Baan through session ttaad2100m000
4. Define Roles and then generate the user through session tgbrg4250s000, and it will take some time to generate.

Hope it works
