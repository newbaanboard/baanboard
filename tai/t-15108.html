Hi there,

the question is what you understand as 'faxing from Baan'.
I guess that your users will use the GUI on Microsoft workstations, so one way to fax 'from Baan' would be to use a printer under Windows that in fact is a Client for a fax server. Hylafax for instance is a faxserver on linux with a neat and free client for Windows. But of course there are a lot more products available.
If you intend to send faxes directly from a 'printer' on the Baan Server, this can also be achieved. Is your AS400 capable of sending faxes already? Then it should be possible to create a printer in Baan which actually is a script and redirects the print data towards your fax application.
