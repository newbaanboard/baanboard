Hi,

How did you move it? Did you completely copied $BSE folder to new Server? If yes then it is a problem. You need to have some of the older files on the system. The problem might be that, your SLM is still pointing towards the older Server IP by reading the license6.1 file. So, it might need a change. Please follow the Documentation as it describes and it won't be an issue.

In case you have moved everything, you will need to re license again. Again, I am not sure about your version, but its always a best deal to follow standard documentation.
