Here are the steps to install a solution with PMC:

1. Download the solution from the Infor site and put it on your server
2. Scan Solution (ttpmc2200s000).
(Those steps can be done in one go with the Multi-level download tool (ttpmc2210s000), but then you have to configure some settings first with ttpmc2211s000.)
3. Start Process Solutions (ttpmc2101m000).
4. Select the solution and choose for "check to install" in the application menu.
5. Read the pre- and post-installation instructions, if any.
6. Perform pre-installation actions, if any.
7. Install the solution.
8. Perform post-installation actions, if any.

Note that you may not be able to install the solution because you need a number of pre-requisite solutions. Then you have to repeat the steps 1 and 2. This is the case that it is very handy to setup de Multi-level download tool.
