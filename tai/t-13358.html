I am sure examples of scripts were posted on the board before.
Try using the quick forum search.

The general idea for these scripts though, is that you first point the bshell to it in ipc_info:

bshell s 0 0 p ${BSE}/name.and.loc.of.your.script.ksh

That way, every user to log on to Baan can be scrutinized in your name.and.loc.of.your.script.ksh script.

The last line in that script should invoke the actual bshell6.2, optionally under conditions. In your case the condition would be that the user does not already have an active bshell.
