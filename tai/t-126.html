All that text:

The description for Event ID ( 1000 ) in Source ( Baan ) cannot be found. The local computer may not have the necessary registry information or message DLL files to display messages from a remote computer. The following information is part of the event: Env baan60 (c:\program files\baan60)

Is common on the event viewer when the application sends an event log into the OS and the OS does not get all the information fields. Baan is sending the information but not in a complete event form structure. In this case, just ignore this part and concentrate on the "real message" at the bottom line:

Message 1 : Free error - error log in file "c:\program files\baan60/log/m_help.log"

Have you checked that m_help.log file?
