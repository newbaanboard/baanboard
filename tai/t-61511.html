If you have only one command in the line the syntax should be
-- -set BSE_LANG=2 for english following connamd should start with -set

Example

-- -set BSE_LANG=2 -set PACKAGE_COMB=oper_us2 -set BSE_COMPNR=150 -set BAAN_WIN_TITLE="TEST USA: %h ( %c )( %s : %S ) %u
