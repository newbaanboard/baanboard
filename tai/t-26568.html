If you don't have much experience with Oracle I suggest that you perform a cold backup.
1. Bring the database down
2. Backup your server (everything)
3. Bring the database up again

I assume that you have storied your installation software (Oracle) in a save place? In case you need to restore it on another server you need to install that software first. (registry, or is your backup software taking care of that?)
How about Oracle patches, do you apply them?
You can get Baan up and running on another server by just copy the BSE folder and install the portingset.
Please check the procedures how to get a new license.

Maybe others can respond in case I forgot something.
