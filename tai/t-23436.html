Hi

Major Functional/technical changes in IS to IS5 are

1. Localise session for Supplier. Separate session is provided in Menu Browser.
2. Localise session for Item Master. Separate session is provided in Menu
Browser.
3. Session tdind0270s000 (Maintain localised data for Header) will not pop-up
automatically, it will be attached in SPECIAL.
4. Session tdind0271s000 (Maintain localised data for Lines) will not pop-up
automatically, it will be attached in SPECIAL.
5. Similarly, there will be two session for Print order acknowledgement
1. Print Order Acknowledgement
2. Print Order Acknowledgement (Localised)
6. In IN3/IS3, the field tdsls045.amnt is gross amount which is equal to
tdsls901.amta but in IN5/IS5, the field tdsls045.amnt will be without sales
tax, excise etc.
7. Separate batch for TDS has to be perpared & posted.

some new tables will be added and in some table new field are added.

A new field of Service Tax or VAT will be added in localise session of item, supplier.

For more detail, you may contact to baan.
