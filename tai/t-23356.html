Vishal

You have not supplied your environment and location details. My answer applies to baaniv4c4 on unix platform.

licmon[6.1] -w gives list of connected users (across the network if multiple baan instances and one licence server)

from unix prompt "ps -ef|grep bshell|grep -v grep" gives list of active logons to baan to a particular machine. If you only have one baan system running on the machine and you can identify the user by PC name then you can get your list (I use this method to kill users if necessary / requested)
