Hello Jerrod,

This is one of the standard reports (I guess but check the number of character please) that don't fit.
Calculate the number of character times cpi and you will see that there can't be more than about 190 characters on one line.

These reports were originally developed for the old matrix printers with broad paper.

THe only thing you can do about this is customize them or use a smaller font.

With bw it a bit easier to print them since a specific driver option makes the font fit (sorry but I don't know the solution right away).
