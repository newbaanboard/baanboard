ostype()

--------------------------------------------------------------------------------

Syntax
long ostype()

Description
This returns a long value that indicates the operating system environment. The possible values are:

OS_WINDOWS_NT Windows/NT
OS_WINDOWS_95 Windows 95
OS_UNIX All UNIX variants
OS_OS400 AS400

Context
Bshell function.
