Hi,

The field might be of the type text having domain like tctxtn. If you want to display directly on the report, add the field to the report layout, go to field's properties - Display tab - Uncheck Link with Domain - Now provide the length of the string you want to be displayed from the text you enter.

Now in the display length, you can mention something like 80.

The information is stored in the table tttxt010. You don't need to retrieve the text. Baan report will do that for you. But if you want to write the text in some file, then you need text.to.buf() function to do that.

You can check these 3 threads for that-

Thread 1 (http://www.baanboard.com/baanboard/showthread.php?t=60004)

Thread 2 (http://www.baanboard.com/baanboard/showthread.php?t=66758)

Thread 3 (http://www.baanboard.com/baanboard/showthread.php?t=63734)

Note - You need to add #include <bic_text> in your script to use text.to.buf()
