I have successfully changed that icon by using one of these two programs:
* XN Resource editor
* resedit
Probably there are other programs that can be used to change the resources that are part of the bw.exe program.

I'm using three different coloured bw.exe programs on my development pc and all works fine.

But one of the above programs made some trouble with mouse cursor positioniong after that. So be sure to have a backup copy of your bw.exe before experimenting...

Have luck!
