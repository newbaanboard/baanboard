you should already have a customer number and licence code for your enviroment (get this from licmon -b) now from the command prompt go to bse\bin and in this folder is the brand.exe this is the program that is run when you use ttiex3400m00 print requested system configuration.

it is possible to run this from the command promt passing it the organisation name,customer,licence code and the security code,validation tools key from Baan. The last two are generated based on the macaddress of the windows machine (more complicated if it is a unix box also checks the cpu name.


with this info brand can now be run again using the tools validation key which will greate the brand.lic file (this is the same as running the validate tools tools session ttiex3220m000)
brand.exe -o "organisation name" -c "customer number" -l "licence code" -b "security code" -a "validation tools key"
the licence service on the windows machine will now work (unless the nic has been changed).
