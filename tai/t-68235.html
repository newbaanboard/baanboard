Hello,
RMAN is an Oracle product.

Run

LIST BACKUP OF THE FILE

to see if you have any prior backups. If you do have a backup of the datafile, you can then
take the datafile OFFLINE
RESTORE the datafile
RECOVER the datafile.

If you want to "ignore" that datafile try taking it Offline
ALTER DATABASE DATAFILE xx OFFLINE

and then use "SKIP OFFLINE" in your RMAN BACKUP DATABASE command.
