I'm assuming that you are using an external scheduler like the Windows Scheduler. If so, here's what I do to schedule an LN Job in FP3 and FP6:

1. Add the job using LN Job Management and specify External Scheduler and Run Periodically. Let's say your Job Name is TEST.

2. Run the job from Job Management to see if it runs OK.

3. Setup a TEST.bat file that has the following line: call startjob.bat TEST
(startjob.bat is provided by Infor and is located in %BSE%\bin)

4. Manually run TEST.bat to see if the job runs OK.

5. Schedule the job using Windows Scheduler.
