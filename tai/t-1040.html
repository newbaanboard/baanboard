I believe it is mfc42.dll and not mfc420.dll. Am I right?

I suggest you should reinstall everything but in this order:

NT Server, NT SP3, IE4.01, NT SP4, NT SP5, NT SP6a,
IE 4.01 SP1 (it is as a prerequisite on the CD with SQL Server 7)
SQL Server 7.0
Baan Application

Be sure you have a good Master CD !!! (a newer one) It is also important not to install directly NT SP6a. Microsoft says it should be OK but it is not !!!

It is not a good idea to have IE5, since if brings a new mfc42.dll and Baan Installer does not "like" new things... (for example Oracle 8i...)
