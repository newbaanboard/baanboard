Obviously, you need to get your LN CD's from your SSA Sales Office or representative. All the documenation you need is available on SSA's support web site.
The installation is different from installing Baan IV or V. With LN, SSA introduced the concept of a Staging Area from which individual components will be installed. The basic installation concept is as follows:
1. Install OS (Unix, Linux, Windows Server)
2. Install DB (Oracle, Informix, SQL Server, DB2)
3. Install LN Staging Area
4. Install LN components
5. Install Worktop
6. Install Webtop (not needed)

With the staging area comes an installation wizard. This wizard makes the whole installation process quite easy. When you start the installation wizard, you will see a list of components to install. You can install one by one or all at once, but I won't recommend the latter. There is minimum set of components you need to run LN. If I remember correctly, the components are:
SSA Porting Set
SSA Enterprise Server
SSA ERP LN
SSA License Server
SSA ES Application Server
Basic Data Set

Again, all this is explained in the installation guides (U8072, U8204, U8074, U7434)

Good luck.
