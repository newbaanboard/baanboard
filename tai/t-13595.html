Hi,

your old machine stays validated until you throw it away, basically. The new and old validation can coextist without a problem. Actually they will not know of each other.

If you want to delay the validation of the new server, then you have2 options:

a) perform "patch objects after system crash" on the new server.
After this you can work on the new server for 30 days without a licence. For this you should have the hostname of the new server written in licence6.1

b) Just keep the licd running on the old server and kep the hostname of the old server in the licence6.1 file on the new server. In this case the new server will take the licence from the old and everything worls as usual.
As soon as you validate, you need to change the entry in licence6.1, of course.
