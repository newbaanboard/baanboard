I'm not sure that's a question that can be answered in this format.

There are some pretty big functional and table changes between those versions, especially if you're looking at site functionality. I would expect that a lot of your customizations will need to be revisited.

In-place should be possible with the data upgrade engine, although I've not done from 10.3 to 10.8. I have done FP5 to 10.4 and 10.4 to 10.7 with minimal problems.

There are too many variables to blanketly say whether it could be done in 24 hours. I believe we did a > 1TB environment in less than 36 hours.
