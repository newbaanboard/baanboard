Just start a ttstpshell (or the "Start Shell" option in the Option Dialog box that comes with bw) and enter the command "ps" to see all running processes in the current bshell. Here you can see subsession processes and report processes etc... Also some standard processes.

If you need to stop Baan bshell completely, you can try the "killall" command here, there are several other commands that you can use. The "kill" command for hanging processes can be usefull, also the "ps" command to determine what sessions do still run (still get ticks).
