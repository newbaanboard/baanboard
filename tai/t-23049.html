We do set up users like this in our OS/400.(we implemented password aging)
1) Create a user in OS with known password
2) Create user in Baan Convert runtime & Convert runtime to Database user
3) After that user can log in with known password.
4) Change the password expiration interval in OS to less than 7 days.
5) Next time whenever user tried to log in, he/she will prompt to change the password in 7 days message.
6) Once User change the password, we are going to change the password interval back to 60 days (as of our company requirement)
