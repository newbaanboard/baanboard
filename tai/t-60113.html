Hi Walter,

Maybe a bit late but LSM stands for:
SSA License Manager
It is a server/client system which Baan/SSA/Infor is using to validate the used applications.

Example:
SLM Server holds the licenses which a company holds.
SLM Client is installed on every system where a Baan/SSA/Infor application is running.
BaanERP calls SLM Client to verify if a valid license is available.
SLM Client will call the SLM Server which in turn will chack it's license 'database' for a valid license.

SLM Server can hold various license types:
- Server licenses
- Concurrent user licenses

and two more which I can not remember at the moment.
