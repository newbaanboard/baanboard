Hi,

You can use session Convert Recover tables (ttaad4228m000). Using this session, you don't need to first take sequential dump and then restore. In the first form of this session, specify the source company and table range you want to copy, and on second form, tick-mark the option "Create table for different company" and specify target company.

Also, tick-mark Delete existing tables option if tables already exists in target company and if you don't require existing data.
