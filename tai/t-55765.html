This is the immediate solution that I have found.

In BAAN:

(1) Print the report to "word"

In Microsoft Word:

(2)Change the Font size to 8
(3) File > Page Setup > Margins > Landscape
(4) Delete the extra page at the bottom
(5) Print the document

I then realized that if I had a document-formatting program, I could export the file to a directory, and then let that program automatically format it for me. [We do this for some outputs, using Formscape.]
