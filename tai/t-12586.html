I am not sure if there is any procedure document or standard BaaN tool to do it.

I made a tool for our purposes. It does the following steps -

1. Remove user from the user data.

2. Remove user from the remote user data.

3. Remove user defaults.

4. Remove current package VRC settings for the user.

5. Remove databases users and links.

6. Remove the Oracle user info for each database/Oracle SID.

7. Remove DEM user info in different companies.

8. Remove user files in $BSE/lib/u<user>

9. We do not remove the User History for tracking information.
