KB 1397917 Cannot open Package Security File
Description:
Certain messages can appear in relation with "license security files".
For instance:
"No license to run object 'd:/infor_ln/application/tfB61U_a_stnd/otfint/ointdlltfgld' (productId 0). Cannot open Package Security File."

Note: Each security file is present in the folder i<package>bic.
