You can leave 'da' and 'tt' packages as one update VRC(da is kind of tools).
You can derive seperate update VRC's for the application (B50U_c_sp15, B50U_c_sp14 etc), but the Base VRC for these update VRC's should be B50_c.
You can use session ttpmc2290m000 to copy the PMC registry to update VRC.
