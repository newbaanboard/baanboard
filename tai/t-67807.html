Hi,

You can redirect the output to a text file from a bat like this -

tasklist /FI "USERNAME eq " /FI "IMAGENAME eq ntbshell.exe" /NH /FO table > D:\new1\testing.txt

You can refer this thread (http://www.baanboard.com/baanboard/showthread.php?t=64216). There was one instance where I created another bat file to run the schedule task.
