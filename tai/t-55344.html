Hi,

Asuming the clients are on Windows Xp - you can try this.

Use the client - where fonts are working fine.

1.goto control panel->fonts , then select all the fonts - press CTRL+C (i.e Copy)
(note:you would not have right click copy option for #1)
2.Paste it in a seperate folder(CTRL+V) - all your ttf files gets copied.
3.Goto run->type regedit goto "HKLM\software\baan\bwprint" -right click ->export to a file.

after collecting fonts and reg export.

Goto the client where you face the font issue - login as admin

1.select all the fonts (that you copied) - just drag and drop in control panel->fonts folder.
2.then douldle click on reg file that you have exported .(it will restore the keys)
