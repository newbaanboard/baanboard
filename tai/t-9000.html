tjl7666
24th February 2003, 19:08
Only validate on one server. Only run the licence daemon on one server. Update the file licence6.1 in the lib directory on all servers with the host name of the server running the licence daemon before you validate. Baan support needs to understand that you will be running Baan on more then one server or they may give you a key that will drop the number of users allowed by a half or a third after a few hours.
