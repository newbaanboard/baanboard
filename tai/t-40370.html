Hi,

Open session "ttaad4111m000" and check what is the Database for parameter tables. It will be either of the 2 Databases maintained in the session "ttaad4110m000".

Then, Please check your tabledef6.1 file in "$BSE/lib" path. Ideally, all the parameter tables will have last character as "Y", which means the tables have audit enabled.

It could be possible that the "check box" for audit trail has been removed accidentally from one of the databases in this session and you will have to reset it. BE CAREFUL.....and take backup of the "tabledef6.1" file before any activity.
