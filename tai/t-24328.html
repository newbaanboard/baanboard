Run session ttiex4100m000, add a patch code with a name you want, the description as per the session, The path name where you want the dmp to be created and the VRC from which you need the sessions dmp

When you continue ttiex4110m000 will open, add all the components you need in the dmp file.


Click Special and click create patch



This will create a .dmp for the desired set of components at a desired location.

Hope this helps you.
