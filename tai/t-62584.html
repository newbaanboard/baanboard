Infor releases porting sets so the same ERP software (ex. ERP-LN) can run on different operating systems (AIX Unix, HP Unix, Windows, etc) and databases (Oracle, MS-SQL, etc).

Running on Linux would require a different porting set. See Infor KB ID 22923520.

The baan application software that is typically stored within $BSE/application can be copied because these "programs" use an abstraction layer and are machine/os independent.

The baan programs (you mention /opt/baan) cannot be copied from HP-UX to Linux. You have to install these components from Baan.
