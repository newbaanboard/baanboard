Hi,

to be able to delete a device, you need to delete all request in the queue.
Do a query in the maintain device queue session to see all seq. no still linked to the device
Then go in the Miscell menu of the Device manager, select the purge device queue session and delete the range of seq. no

You will be able to delete the device after that.
