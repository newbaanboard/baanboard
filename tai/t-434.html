Although I work with HP-UX I once installed Baan on W2K Advanced Server at home and made changes in the pmc tables as a workaround to this problem. However, as mentioned above the problem has now been solved by Baan....see solution number 107262.

Quote:
"This problem is solved in the bshell, and therefore the fix will be delivered with next Porting Set.
For BaanIVc: Refer to solution 18505
For BaanERP: Refer to solution 74687"
"WORK-AROUND:
Take care pathnames of directories do NOT end with a "/" or with a "\"
For example: Change f:/baan/pmc/uninstall/ to f:/baan/pmc/uninstall"

There is also a temporary ntbshell.exe available, containing the fix as temporary workaround. See Baan solution for download details.

Quote:
"Note: This bshell is the 7.1b.01 bshell, which can contains this fix.
This bshell is not officially supported, but only mentioned to use as temporary workaround for this problem!
Unzip this file using WinZip, so you get ntbshell.exe (size = 1.05 MB)
Replace your current ntbshell.exe in %BSE%\bin for this one. (To be save make a copy of your original ntbshell.exe)"
