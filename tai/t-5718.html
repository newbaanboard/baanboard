Hi There,

We have similar configurations in Unix all the time and we do the following:

1) Just install the test server normally

2) After installation disable the licence daemon/service

3) Point the content of the ascii file $BSE/lib/licence6.1 to
the licence server of your production system

4) Validate your installation with the same keys settings as the production system.

Should work!

If you have problems with the validation, just copy the tools and application objects from your life system to your test system. They are already validated.

As your test-server should be an image of your production system, you might want to do this anyway.
