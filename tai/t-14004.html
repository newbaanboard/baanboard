Well,

I can, but maybe it is a tricky solution, so any other board user can improve the solution:

When you print to a windows printer, select print preview.

Inside BWP application menu select File, Send...


Well you will have attached a file with bpf extension in a new mail to send. Save this file to any location and that is all.

Almost all, because you have to change to read-only the atribute of the file. If not, it will be deleted by BWP the first time you open it.

After this close the message prepared because it was only a bridge to obtain the file.
