Rather than pushing data from Baan, we actually pull data from Baan using OpenWorld to our e-Applications. Using NetList functionality in OpenWorld (which in turn uses table auditing), we can pull only the changed records and don't have to pull in all the records all the time. Of course this means that we did one initial bulk upload and then we do our subsequent synchronization using NetList.
