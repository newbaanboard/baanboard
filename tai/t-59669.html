Hello Abuibra,


correct me if i'm wrong,

you want to install the ERP on the physical windows server 2008 and then enable the Hyperv and share the virtual environment on 3 other app?

or

do you install the Microsoft Hyper-V Server and then put all the env on the same server?

if you choose the first, then you have to carefully count the processor and memory, as the first solution is hypervisor type 2, which run under OS, so you have to keep the OS running well.

for the RAID, i recommend 1+0 as it provide performance and mirroring.

Tower or rack doesn't matter actually, unless you're planning a datacenter then maybe you can use rack with networked storage.

not to be a fanboy, but maybe you can try VMware, their ESXI are similar with Microsoft Hyper-V Server, but with added HA and DR, and more 3rd party enabled.

hope that helps.
