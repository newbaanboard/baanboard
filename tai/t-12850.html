BaaN does not provide migration tools for Roles/devices/exchange schemes, SQL Queries etc.

You have to build some tools yourself or use Oracle SQL to do a direct copy from one database instance to another or use the bdbpre/bdbpost/Create Sequential dumps of Table(ttaad4226m00)/Create Table from Sequential Dumps (ttaad4227m000).
