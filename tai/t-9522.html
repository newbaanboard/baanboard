I just did installation Baan 5b on Oracle 9i.

First, I have contacted Baan and I have been told it's not supported :)

The update to Oracle 8 is required ONLY for Baan Tools and instead of running PMC, you can replace 3 files on Tools directory

- ostp_stddll
- ostpstandard
- ostpstdlib

After that you are set to GO :)
