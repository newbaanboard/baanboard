Hi nansat,

You can't be SSAs instructions for this - Start with Solution #105618
Read it completely.

Now read each solution for every porting set between your current porting set and your new porting set. Look for solutions to install, product advisories, etc.

In the porting set 7.1d.01+ there is a technical notes pdf file - that gives you detailed instructions on how to install.

That should do the trick, it's really not difficult to do the install, the key is not getting bit in the A$$ by solutions or products that are de-supported.

Good luck!
