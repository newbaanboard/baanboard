You are trying to install a language pack, in stead of the standard service pack 12. To be able to install a language pack, you need to set the B-Language parameters (ttpmc3100m000) and to specify language 5 for your update VRC (ttpmc3101m000). See also the solutions 123464 and 127841 and the documents linked to these solutions.
