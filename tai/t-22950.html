manojsharma
24th January 2005, 13:19
yes, you can do. For this, you have to create a file in $BSE/lib/defaults with the name ALL and type the following commands in the said file

DS_TIMEOUT=10
SESSION_TIMEOUT=5

which means whenever a login is idle for more than 5 minutes, it will kill automatically.
