Hi VishalMistry,

here is my step-by-step description:

1. tttxt1120m000 "Maintain Text Windows"; you may create a new Text Window here, e.g. Window Type "Item Text". The other fields should be set to reasonable values.

2. tttxt1110m000 "Maintain Text Groups"; you should create a new Text Group here, e.g. TextGroup "ITEMTEXT", Default Window "Item Text" (from step 1).

3. tttxt1112m000 "Maintain Default Text Group by Table Field"; you should set the default text group for User "" (All Users) and the appropriate text field (e.g. tiitm001.txta) to your new Text Group "ITEMTEXT".

4. tttxt1116m000 "Maintain Text Groups with External Editor"; here you simply add your new Text Group "ITEMTEXT".
