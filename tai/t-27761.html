Hi Sandy,

Maybe you could use the PMC Distributor module.

- Tools - Application Development - PMC Distributor

In this case u use PMC to create your own PMC dumps on the testing server and PMC to import the created dumps to your production environment.
In the PMC module a standard session / report is available to generate an overview of exported dumps in the given period.
