When you invoke the following sessions:

1) Maintain Database Users

2) Maintain Links between Baan Users and Database Users

just make sure you are on the Correct database when you convert to runtime - In case you have one database with Audit and one without Audit. Sometimes Baan does not default to the database you want to use.
