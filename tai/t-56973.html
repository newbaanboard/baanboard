Hi,

This is a log file in XML provided to check the changes or modifications made using GTM. You can view it like this-

Go to Run Program - > Run ttstperrlog -> Select log.aad4100.sox -> Select a device for the report -> And run it.

You will get a report showing you the table changed, fields changed, user who changed it, old value, new value, company, kind of change (new record, modified, deleted), date and time in utc etc.

This log is useful in case if you need to track the GTM changes made.
