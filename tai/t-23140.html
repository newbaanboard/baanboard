As u r installing a localization patch .... you will install it in a localization update vrc using session Import Patch...

Then make sure following things
There is know records available in ttadv501,502,503,504&505 tables
There is no file present with name as 'R.<table name>' in $BSE/tmp. If so then rename it.

Import the localizatin patch.
Perform Create Runtime for Domains only
Relogin
Perform Create Runtime for Domains, Tables & Sessions with option 'Reconfigure tables' set to YES.
Relogin
Start sesstion Create Tables to create new table introduced in the patch.

Following above steps would resolve the problem
