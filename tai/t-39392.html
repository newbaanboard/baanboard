Dear friend,
Just take the sequential Dump and Rename the last three digits of company no (Source) as company no. to which you want to import.
e.g. If you have taken a seq dump from co. 500, the file ttdpur045500.S will be created, now if you want to import this table into company 600 then just rename the dump file as ttdpur045600.S and import it into a new company i.e. 600

First take the sequential dump of source company (ttaad4226m000), after that you have to change company number in data (Session Copy Company Data - tccom0214m000) and then import data using session ttaad4227m000.
