What we've done in the past is change the Baan form icon colours. If you want to change the blue save and exit arrow to red. Go to %BSE%\gif\gfd0002. Edit the files end1.gif and end2.gif with mspaint. Change the colours of the arrows. Save and exit. The arrows will now be red on the form. You can do the same for the save icon.
