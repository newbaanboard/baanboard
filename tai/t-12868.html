You can use the following syntax:


cd /baandata/baanc4/bkup
echo job started
/baanappl4c/baan4/bse/bin/bdbpre6.1 -s -t"|" -It01 -Eerrfile -C556
echo job ended


The file -It01 which is text file containing the tables (i.e. tccom010
tccom020)

You can also use:

cd /baandata/baanc4/bkup
/baanappl4c/baan4/bse/bin/bdbpre6.1 -s -t"|" -Ntccom010 -C556
/baanappl4c/baan4/bse/bin/bdbpre6.1 -s -t"|" -Ntccom020 -C556
