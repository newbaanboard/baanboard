8th February 2005, 17:04
If you could find some way to make the order type (stored on tdpur040) to be equivalent to a field on the lines tdpur041 e.g. warehouse, then you could use table data authorisations ttaad4134m000 to only give insert/modify/read permissions by user to delete the lines where the "equivalent" field, such as the warehouse, was in a certain range.

Alternatively you could put the table data authorisation on tdpur040 by order type, which could prevent the user deleting the header - but they could still delete the lines. Better than nothing but not ideal.

Ed
