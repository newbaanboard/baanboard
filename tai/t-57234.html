Hi!

Check KB 22881488: "From Worktop version 2.5.15 onwards, Windows 8 is supported."

Worktop 2.5.15 is the latest versions (released on 5. Aug 2013).

You'll need to access ftp.support.baan.com or use the link given in KB 22881488:

"DOWNLOAD PATH Worktop 2.5:

Browse to: /download/updates/port/Worktop
download Infor_Worktop_2.5.15.zip
(size: 60 Mb)"
