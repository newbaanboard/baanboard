You can also build a small table using the tools functions to determine the real table row size x number of rows (per company) and generate the statistics before and after table reorgs.

Refer to -

rdi.table() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_runtime_dictionary_information_rdi_table), db.nr.rows() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_db_operations_db_nr_rows)
