You can change the packageVRC of the solutions, and also use filters to see the different status of the solutions. Check the help file of the session.
And please, try to get any document that explains how PMC works and how to use it if you will use it to manage patches.

Also, Baan usually requests the <b>ttstpsessinfo</b> run for a particular session so that they are able to see your VRC structure and which patches are installed for that session - this is more usefull for a Baan non-Tools-admin. that playing around with PMC sessions, make yourself aquainted with that session instead.
