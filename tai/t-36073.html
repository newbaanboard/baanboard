I don't know for sure about LN, but in previous Baan versions you can just create a device, call it any name, for instance 'Excel'.
- Printer type: rewrite file
- 4GL-program: ttstpconv
- Argument: ASCII:excel.exe %s
- path: file.csv
(- page length: 66)

This will send in csv-format and open Excel with the report. If you need more sophisticated stuff, you might want to write your own macro's, or own Baan-3GL conversion program... Search the board for lots of suggestions/methods...
