Sekhar,
You should have posted this in the Tools Administration & Installation section.
As for your problem, have you tried recreating the Staging Area again and then restarting the installation? Apart from that, check if the user login with which you have logged onto the server has necessary administrative rights. The login needs to be part of the administrators group. Do check the user rights in "Local Security Settings>>Local Policies>>User Rights Assignment" under Administrative Tools. The keys to check are
1) Allow log on locally
2) Act as a part of operating system
3) Log on as a batch job
4) Log on as a service
Either the login has to be mentioned in all the above keys or else, make the login part of the Administrators group and then add the group in all the above keys...

Hope this helps..
