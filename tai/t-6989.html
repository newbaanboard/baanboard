Product tt7.3a Last Modified / Sol. Proposed: 19 Jul 2002 /06 Jun 2002

Case description:
When we are copying text from a windows application to a BaaN text windows we are getting some error messages.

Is there any problems with the way BaaN handles text conversions or limitations.

please clarify.

Thanks!

Solution description:

VERSION(S): all

SITUATION IDENTIFIED IN: text conversion

ATTACHMENTS: n/a

SITUATION DESCRIPTION:
Error in TSSwc_width: Can't convert TSS character

SOLUTION DESCRIPTION:
When copying from a Word application into the 'Baan Editor' or Notepad editor inside of Baan, you may experience this error. There are some special characters created in Word that when pasted to a different editor cannot be converted. One suggestion is to setup and use Word as the external baan editor, which should allow the pasting of Word text. The other alternative is to save your text externally in Notepad and then copy it to whatever Baan editor you are using.

Steps:

1. Install porting set 6.1c.06.07 (biggest step)
2. Create file $BSE/lib/defaults/all in which has to be one raw;
use_mbtextconv:1
3. In Maintain user data change locale to ISO_BIN2
(not so small step for every user)
4. Do the same change (like step 3.) in Maintain device data.
