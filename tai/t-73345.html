as far as I know, the correct license is a server license named SSA ERP LN Application Source product id 7026 --for LN 10.5/10.6/10.7

With regards LN Studio:

To use LN Studio for customizations, such as changing reports and forms, you do not need an additional license. However, if you want to use LN Studio for development activities, such as creating new tables, editing UI scripts, functions, or libraries, you need a development license, and if necessary
source codes.
