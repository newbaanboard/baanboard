Could you be more specific regarding the variables?
I don't know if you are refering to all the environment variables for Baan, but the variables that must be set for Baan are:

BSE = path to you installation directory ending in the bse (eg, /opt/baan4/bse)
BSE_TMP = path to the temporary filesystem (eg, usually = $BSE/tmp)
PATH=$PATH:$BSE/bib (include the path to Baan binaries into the PATH)
Also, if in UNIX, TERM = vt100 for ascii session access
