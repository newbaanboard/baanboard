If you are on MS SQL Server, there is a document,

Installing Baan and Microsoft SQL Server with a Three-Tier Client/Server Configuration

Tools Reference Document No. TFH-006

This is available at Baan Support Site.

Three files you need to check:

${BSE}\lib\licence6.1 - contains the name of your app server
${BSE}\lib\defaults\db_resource - name of your db server (option)
${BSE}\lib\tabledef6.1 - name of your db server

It would be prudent to upgrade your porting set too.
