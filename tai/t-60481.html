Exactly, if you edit the file "all" that is in
$BSE/lib/defaults and put the line session_timeout:30

that will disconnect all users after 30 minutes.

Or you can edit the user file in $BSE/lib/user (ex: ulogin)
and insert the same line at the beginning of the file.
