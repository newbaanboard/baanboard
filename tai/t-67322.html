Hi,

There are 2 ways -

A. Create a role which will have No Authorization to the companies you want and attach that role to the users. Then, you can use table data authorization for ttaad100 table.

B. Remove the companies.

1. Delete the tables for the specific company in session 'Delete tables' (ttaad4231m000)
2. If you use Database Bisam, delete the directories for the specific company in session 'Maintain table directories' (ttaad4115m000)
3. Delete the tables for the specific company in session ttaad4111m000.
4. When the specific company uses a specific database definition, delete the database definition for the specific company in session ttaad4110m000 for Baan IV and ttaad4510m000 for ERP LN. Warning: You will have to make sure that no other company uses this database definition.
5. Remove all the references to the old company in User Data in session ttaad2100m000 for Baan IV and session ttaad2500m000 for ERP LN
6. Delete the specific company in session 'Maintain companies' (ttaad1100m000)
7. In the session mentioned previously, select Specific and then Convert to runtime.

You can follow this Thread (http://www.baanboard.com/baanboard/showthread.php?t=27212&highlight=remove+company)
