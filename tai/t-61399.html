Rexec is installed with the porting set. If you went to Windows 2008 and additionally you went to a x64 version, you want to make sure that you did not copy or moved files. Re-running the porting set installation should correct the issue.
In services there should be one named Infor Logic Service. That should be started. Also, if you go to the Infor Management console you should be able to view these services from that console (formerly known as the baanman.msc). If you have trouble pulling up that console most likely the portingset is not installed properly. Normally, reinstalling it should correct issues like this.
