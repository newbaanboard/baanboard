Hi,
Following are the sessions you would need to use for the purpose of downloading and installing solutions from Infor...

1) Generate FTP Script {ttpmc2211m000}
2) Download, Scan and Connect Solution (Multi-Level) {ttpmc2210m000}
3) Process Solutions {ttpmc2101m000}
subsequently
4) Check and Install Solutions {ttpmc2204s000} from the special menu of the above session.
5) Following which you would have to run sessions like Create Runtime Data Dictionary for Tools {ttadv5213m000} & Create Runtime Data Dictionary {ttadv5210m000}

I hope that the session Maintain Parameters {ttpmc0100m000} has already been set...
