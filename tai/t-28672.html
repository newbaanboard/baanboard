Installing slm doesn't need to influence the license. For some time we did use SLM together with licd.
SLM was used for Webtop license, Baan itself did still use licd.
However from 7.1d.12a on (don't know exactly which portingset but around this version) you can move your licenses from licd to slm. Baan now is moving to one portingset for 5.0 and LN? This is e.g. the 8.2b version. Within this version you need to use SLM instead of licd.
