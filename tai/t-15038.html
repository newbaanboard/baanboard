Max, I think Harley is looking to copy the roles of BaaN and not Oracle.

Harley, we asked for that enhancement 2 years ago and meanwhile we wrote a tool ourselves.

And BaaN has now released the solution in one of the latest patches, so you might have to check if that solution is delivered for your BaaN version.

Solution description
TOOLS VERSION(S): 7.3_a

SITUATION IDENTIFIED IN:
"Role Data" (ttams2100m000)

SITUATION DESCRIPTION:
Duplicating a role in session 'Role Data' (ttams2100m000) only copies the role but not the linked subroles and authorization data.

SOLUTION DESCRIPTION:
Duplicating a role in session 'Role Data' (ttams2100m000) will not only only copy the role but also the linked subroles and authorization data.

SPECIAL INSTRUCTIONS: None.

PATH NAME IN DIAL-UP DIRECTORY/WEB SITE:
Below are the supported VRC's based on the Baan standard derivation structures.

If your Current Tools VRC is: 7.3_a:
/ftpdownload/updates/7.3_a_tt/139903tt

COMMENTS:
Please read solution 100017 for installation instructions.
