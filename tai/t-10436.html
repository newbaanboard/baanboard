The only thing I can think of is to check the ${BSE}/application folder. For example, if your report is in the tssma module, check the folder ${BSE}\application\tsB40o_c_p00\rtssma folder. If your report is there with a "-" after it, that is the old version. You could try renaming it without the "-" and see if you could get it back. This might not work if the tools tables have lost the layout data. Good luck!
