After posting this thread, i did some digging around the knowledge base at Infor and came up with a document on how licence server works..
Quoting from that document...

When must you generate a new Security Key (SK)
=====================================================
You must generate a new SK when the following occurs :
-On the machine where the LICD program runs,the hostname is changed
-On the machine where the LICD program runs,the hostid is changed
-If the NODE where the LICD program runs is to be integrated into a pool
-If the NODE where the LICD program runs is already integrated into a pool but the members of this pool are changing.

Most of this events can be planned,except for the system crash.
Obviously, you must send the new SK to your local Baan CS&S rep.and ask for the new corresponding VK.

NOTE :
Under Unix, the hostid is retrieved from the motherboard of the computer.
Under NT, it is a computation (?)of the MAC addresses of all installed network adapters.
