Please confirm the following:

1. You have created a host with your Server's IP Address or Name with
Port: 4001
2. You have created two areas.

a. Vault Area with host as your new host and Area Type= Vault
b. Work Area which is dynamic with a correct path.

3. Check if your dmsys1511m000 is not empty.

4. In session dmsys0540m000, you have created a library with family profile as FLIB_TREE.

5. To the library you have linked two object,

a. DOCS which will be by default
b. VAULT SERVER which has you host attached to it.
