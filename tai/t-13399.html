If you do not need to do this regularly, but only one off, then I would use the following trick:

a) Print from Baan something similar on the same printer
b) Check with general Table Maint. in table ttaad320 which temporary file was used.
c) Replace it with your file
d) Reprint from Baan the print job in (a)

finished.

Not nice, but works,
