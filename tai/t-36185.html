BaanInOhio
25th May 2007, 19:23
Install the BW client in compatibility mode. I added the installer to my vista PC, changed the compatibility mode of the setup .exe to 'winxp' (thru the 'properties' option on the file), ran the setup as 'administrator'. Once created, I changed the compatibility mode of the BW client .exe to 'winxp' as well.

I ran into a few issues using vista with Baan:

You must ensure that the .tmp folder created during setup has open read/write/execute security. Mine was created as execute/read-only which didn't let me do anything with text (like maintain program scripts).

If also running office 2007, the link from Baan to excel isn't working correctly -- providing an error (from Baan) that hangs the report. The data is correct in the spreadsheet, though. I haven't investigated office to see if a default of "98-2003 spreadsheet" can be set when opening new documents.
