See solution 159902:

SOLUTION DESCRIPTION:

There are many situations that can cause this problem. We will discuss the major one:

The supplied Oracle Libraries do not match up with the driver:
------------------------------------------------------------------------------------------------
BaaN Delivers a version of these libraries via the ora_update program located in each Unix
porting set. After each porting set install it is necessary to run the ora_update program. This
creates a directory called $BSE/lib/ora/oracle_home and updates your DB resource to add/modify
the line: oracle_client_home: ${BSE}/lib/ora/oracle_home

This error (13705) comes when the version of the libraries in ${BSE}/lib/ora/oracle_home do not match up with
that of the BaaN Oracle 8 driver. To resolve this problem please run ora_update located in ${BSE}/bin

EXAMPLE(S):
$ cat $BSE/lib/ora/oracle_home/Version
8.0.6.3.0

$ $BSE/bin/ora_update

$ cat $BSE/lib/ora/oracle_home/Version
8.1.7.2.0


Hope this helps,
