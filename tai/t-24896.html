Ganesh,

Refer to the BaaN support site for more info -

The installation procedure can be downloaded via:
/ftpdownload/updates/port/NT/installnotes71ps_WINNT.txt

installation notes regarding cluster environment:
/ftpdownload/updates/port/NT/installnotes71ps_cluster_WINNT.txt

The complete portingset contains of 5 dumps namely:
The portingset binaries:
.."OS"/"OS_version"/bin/bin.dmp.Z

The API libraries:
.."OS"/"OS_version"/api/apilib.Z

The tss and oracle sql files:
.."OS"/"OS_version"/lib/lib.dmp.Z

The java files:
.."OS"/"OS_version"/java/java.dmp.Z

The shared libraries:
.."OS"/"OS_version"/shlib/shlib.dmp.Z

Sanjay, you don't need # 3, or (Take backup of Database at data level) and not all of #1, 2.

The following should be done -

Before installing the new porting set, make backup copies of the following $BSE directories:

api
java
shlib
lib
bin
