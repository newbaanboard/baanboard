harijono
4th June 2003, 13:59
- Run create runtime help session, specify full range if you are not sure which package to choose. Select all the fields: domains, tables, etc.
- Next, if you currently use windows style help there should be one windows help tied to the user that you use. You have to clear this entry in maintain user data session, then convert runtime.
- Logout and log back in, the help should be available for that user.
