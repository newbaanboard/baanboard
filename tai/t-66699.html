In Maintain Reports ttadv3130s000, you may be able to see them there already. The first field is language number, the "default" report is in say language 2, then other language versions of the report are created (e.g. language 1) where the 5th field down "link to language" has the value 2.

So for example for sales order acknowledgements, your default report would be:

2, td, sls, 440101000, blank
and the NL language version would be

1, td, sls, 440101000, 2

I think you create them from ttadv6200m000 using Copy.

Hi,

Baan IV has a great module for this, not only for reports but for all language dependent software components.

Search the web with "baan iv language guide" in google, second match is a pdf manual from baansupport, can be downloaded without login data.

Baan IV Translation Manual U7062A US.

For the main lanuages you will find ready translations from support.

You can also create own languages (we did it for hungary, copy e.g. the english texts and translate them.

Please be aware to use the correct language code or unused for specific languages to avoid countereffects.

For Reports: You develop the report in one main language e.g. english and just link the language dependents layouts to this report (linked to language).

If you use the copy programs please always check for correct ranges, otherwise you copy to much components to other languages.

Kozure
