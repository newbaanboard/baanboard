I've been able to produce this error the following way:
1. Select a user with user type "Normal user"
2. Enter some data in the session ttaad2110m000 "Maintain User Authorizations by Terminal", e.g. Terminal = "/dev/tty" and the selected user
3. Select Options and then "Convert to Runtime Dictionary ..."
3. Now, when I switch to this user (su - ...) and try to run ba6.1, I get:
