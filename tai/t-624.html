Printing on AS/400 :

There is an excelent red book made by IBM on all baan administration.

The most easiest way is to setup a baan device of type printer and device IBMSCS. You have to create and output queue that accep IBMSCS spoolfile and us a writer for your printer.

the redbook can be found here : http://www.redbooks.ibm.com/pubs/pdfs/redbooks/sg245188.pdf

And there is an update here : http://www.redbooks.ibm.com/redpapers/pdfs/redp0506.pdf.

If you want you have not to read the first one that is some big. The update contain detailed information about printing.

The is only one limitation. If you use direct printing (with a script than run at IFS level) you must have an IBMSCS spoolfile. Other type of spoolfile (USERASCII) that is used for hp printers cannot been printed by this method. There is a defect created on that and baan PEG is working on it. But with standard print method (with pdaemon) everything goes right!
