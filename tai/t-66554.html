Hi,

Check your CMF settings.

In "Service Providers" (ttcmf0110m000), there must be a provider OUTLOOK with 4GL Connector ttcmfoutlook.
In "Services" (ttcmf0130m000), there must be a service OUTLOOK with provider OUTLOOK and enabled checked.
In "Address Types by Service" (ttcmf0140m000), there must be an address type SMTP with UI Required checked and Service Name OUTLOOK.

Also check the Email Device:-
Device EMAIL
Description Outlook
Device Type Rewrite File
Locale ISO-8859-1


4GL Program ttstpconv
Argument RTF:outlook.exe
Path ${BSE}\tmp\output
Check box for Change Output file allowed
