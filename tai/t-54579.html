Solution Description: English What SLM license keys do I need for ERP LN to run ?
First SLM needs to be configured with the proper license keys before BaanERP LN 6.1 can run.

You need to have at least the following Product ID's as valid keys:

10056 Infor ERP LN 6.1 (for Application - General)
10996 Infor Enterprise Server (for Tools)
10365 Infor365 maintenance contract (as 1 concurrent user)

Optional:
10896 Infor ERP LN Service (for Application - Service)
10146 Infor Enterprise Studio (for Development Tools)
7026 Infor ERP LN Application Sources (at least as 1 Server license)
