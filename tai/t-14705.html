Hi

This is very much possible to save your .bpf extension file in c:\baan\tmp folder. We are doing same way in our organisation.

Make a new device (we have name of this device bpf), Device Type : Windows printer, rest as default.

During printing any report, pur bpf in device and select print preview. A file will be place in your c:\baan\tmp folder with tmp????????.dbf extension.
