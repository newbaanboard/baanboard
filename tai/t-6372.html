Hej,

To create Baan Logins over Citrix you need to install the BECS Application onto the Citrix Server. Using this Application you set up a default BW connection. Saving the configuration will end in a [filename].bwc file.

Further you need to create an icon (for the users) accessing the *.bwc file.

That´s how we are running it.
//Marcel
