Hi Caner,
I think Iam too late in answering your question regarding the export of Data dictionary. But its only this day that saw ur post. Anyway I would like to share a few points which I know regarding "Export of data dictionary". May be this would help you in future.

1. In your case if u want sessions to be included in dump.l file then "Delivery session from status" should be developing. If its finished then most probably you may not find any session inlcuded in dump.l This is because most of the sessions which we use will be in developing status.

2. If export based on sessions is checked, then only those components which are attached to the session are exported and not the rest of the components.

Hope above points will help you.
