Maximus : the doc you give is for Baan5. It is related to the "BaanInternet" - BI user interface



For BaanIV : the "Internet Applets" that can be found in $BSE/internet (refer to doc U7067AUS, "Internet
Applets Installation Guide) are not working fine and are unsupported by baan (and removed on some installation media) for years.


For Baan5 : in $BSE/internet you'll find the files for BI which work fine (as far I remerber) and is still supported by Baan (last version = 7.1d.03.155 released 19/09/2003 on ftp.support.baan.com/updates/port/BI)
