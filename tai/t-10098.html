Hello Caroline,

I don´t understand why you didn´t do an export data dictionary (ttiex1280m000) from your old package combination and import it in the new one (ttiex1285), but you will have your reasons for that....

The Error-File says that there´s a problem with the data dictionary. You should rename the dict-directory of your new package combination to <package_combination_new>.OLD.
After that create an empty directory with <package_combination_new> and copy all the contents of the dict-directory of the package combination of your source company in it.
If you use a windows system you might have to reboot the server before you start the bdbpost again.

Hope that helps.

Mario
