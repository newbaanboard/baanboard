Hi there

normally there should be no need to restart the license daemon.
In the past I had a windows system using a HP system's licenses.
Nevertheless the porting set for linux will NOT run this way, it checks whether the system was licensed for linux.
With other platforms it should be sufficent to point the $BSE/lib/license6.1 to the production server as you already described - this should work.
Make sure not to start a second licd6.1 from the start scripts in $BSE/etc of the test environment.
