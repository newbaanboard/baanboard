We use a standard HP laserjet 5m printer that has postscript fuctionality installed (the printer is about 8 years old now) and we use the baan provided HP4 laserjet printdriver (hp_lj4) that we use for all laserjet printers. I think this is a sort of standard driver much like epsom mx/lx100 is to dotmatrix printers.

We recently submitted a case to baan for a driver for the latest Sharp multifuction (print/copy/scan/fax) unit that we bought but their advice was still the same: Ie use hp_lj4 driver.
