If you are looking for ready made solution - Lisa's suggestions would work good.

Also refer to the posting - Client Info (http://www.baanboard.com/baanboard/showthread.php?t=10001&highlight=client)

We do have those internal tools and have them setup to run after 15/30 days to collect statistics. But if the user does not log in from a particular desktop for few days/weeks/months, you cannot track the version info. We came up with a new solution with our Desktop Infrastructure Team, who use the Microsoft SMS tools to query every desktop to check for the BaaN Windows, Worktop and DEM versions and generate a report. This report has more real time information, more accurate and get information from the Windows Registry.
