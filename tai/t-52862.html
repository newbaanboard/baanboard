audit procedure

1. Assign tables to database ttaad4111m000. Assign database with audit on to the tables to be audited . Convert to runtime.
2. Use baan login anf goto audit mgmt in tools
3. Maintain audit info file
4. assign audit security to users
5. print range of audit files

On masters table like tccom010 audit should not be problem. It is not advisable to keep transactions tables like tfgld102 , tfgld106 , tiitm001 (because of stock figures in tiitm001) in audit mainly from performance point of view.

In com010 also there are orderbalance/ invoice balance (which i think are not used and are shifted to com011 ) and turnover figures are updated from tdsst0201m000 . So test this session if u use tccom010 for audit !
