An ORA-1022 error is : database operation not supported.

i had seen this error sometime when the baan-client libraries dont had the correct version as the database server.

So, upgrade the database to 8.1.7 (8.1.5 ist not supported anymore) and update the portingset to 7.1d (which was linked with Oracle 817 libraries).
