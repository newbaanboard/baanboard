Purge Audit Files (ttaad4161m000)

SESSION OBJECTIVE
To clear the old audit files. They can be reused by the audit server to write new audit information.

HOW TO USE THE SESSION
Enter the range of table names, company numbers and sequences and press <Start Process> to remove the sequence files.
The audit files can only be purged when the security level of the concerning tables is "Purge" and the user has been authorized to purge audit files.
