in the command field form the BW configuration options:
-- -set UI=ascii
Save the bwc file (ie in c:\program files\Baan\lib\user\ascii.bwc)

Make a batch file with this command (1 line):

c:\Progra~1\baan\bin\bw.exe c:\Progra~1\Baan\lib\user\ascii.bwc mtccom000000001

When you start a batch file this will give you the "old" ascii interface (this works for BaaN IV, i hope that's also your BaaN version)
