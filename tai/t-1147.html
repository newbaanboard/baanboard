No, because you cannot export the PMC data reffering only to the modules you want, meaning that you would also export your tools patches PMC data, and this ones are not sync'ed between your two machines.

Unless, you are using the same level of tools patches on both but then, another problem would be with the solution history (time stamps and installation order), meaning that a PMC export from one system to the other can mess them all up, so, not advised at all.

If you need a proper test system, then it must me an exact copy )with extra stuff on it, not vice-versa) of the live. Depending on the level of stuff you already have on the test system, probably the best would be export what you have, copy the prod. to test and then import the former... all depends on what you have already setup!
