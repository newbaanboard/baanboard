Hello,

First check if there are any *.New files in the dict. If there are any, that means the Convert to RDD was not done properly. Also check table ttadv501 - 505.

A tip:- Take a backup of the folder named with your Package Combination with DB and BSE Backup. Now, rename that folder.

Now, run ttadv5210m000 (Create Runtime) for your package combination with full range and all selection. Relogin baan. It will create the folder again. Now, check if there are any .New files. If there are, do a convert to run time for the package combination. Relogin Baan.

Thats it. But, please take the necessary backup's. And, as jclju1 said, you can try restarting the Shared Memory as well. Again, this problem has a lot of reasons and every reason has different solutions. So, All the Best
