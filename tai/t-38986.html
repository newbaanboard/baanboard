I have to correct myself. There is a way to merge them... . I've looked at the tools online help. Look for the aud* commands. With this commands you can write a program that reads the audit files (run the program at every server so you read the local files) and write the data to a table. From this table you can re-create the files with the data of both servers.

It's not an easy way but it is possible. I think it's easier to save/archive the data for further use. Or just read the data from one server and save it in a table for reporting.
