Used with Binary Logging and Tracing (BLAT), see knowledge base of Baan for more information. Also check the Technical Tools Manuals under Baan Documentation on the support site of Baan.

BLAT was made to be a replacement of TT_SQL_TRACE, DBSLOG, BDBDEBUG etc. Does create binary logfiles, much smaller and different information can be extracted from the files, see also $BSE/api/blat for examples to create your own tools for reading the logfiles.
