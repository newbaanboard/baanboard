It seems that you manually changed the Base VRC of the Update VRC to B61_a1 (or 7.6_a1_tt in case of tools). After initial installation it was B61_a and 7.6_a_tt. You were allowed only to install solutions for these Base VRCs. By installing new Service Pack 1, PMC will automatically change the Base VRC to B61_a1 and 7.6_a1_tt. After that you will be allowed to install solutions only for the Base VRCs B61_a1 and 7.6_a1_tt. This is the mechanism PMC uses to prevent that you install the wrong solutions for the active Service Pack.
