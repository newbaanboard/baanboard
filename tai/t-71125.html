- click right : check to uninstall.

It may be useful to check session ttpmc2503m000 also (Solution Runs). LN will install prerequired solutions. Those solutions are bundled in a solution run on installation. So if you want to uninstall a specific solution together with all prerequired solutions, tipmc2503 will be able to do just this.
