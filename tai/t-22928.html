Hello,
you need a SoftwareTool like WIN-lpr (line printer remote) if using UNIX platform running on your W98 Box.
On UNIX create a queue (remote), in BaaN create a printer "direct" directed to the remote queue.
WIN-lpr is listening to this queue.
