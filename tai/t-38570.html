Hi darpan,

Which login you are using for accessing Tool's session ? Because while installation if you have given same code for user name and database group then this message will come. Let's assume that you had given "baan" login name and if you have made database group as baan in oracle or sql, system will take it while installation, but it will not allow to open any tool's session. When you link user to database group in ttdba0515m000 manually sysetm will not allow to make pair of user and database group with same name.

So what you need to do is to create the demo or other user. Make a database group other than the user name. link that user that database group.

For baan user system will not allow to access any tool's session, so you need to create user other than baan.
