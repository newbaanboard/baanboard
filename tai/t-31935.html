In the subsession 'Copy forms by sessions' (ttadv2201s000), you have to enter a form-code/name for the new form in the field 'To form'. When not, the field 'To form' is defaulted to the same form code as the old one, which is not allowed, therefor you receive the message 'form code not allowed' which is totally justified..

Normally you give the same name you gave your NEW session, with a '1' added at the end. You say that this doesn't work, this must work? Check your package vrc, and the entered (new) session name. Go to the new session in ttadv2500m000 and via 'select' choose 'forms' to see which form is linked...

Good luck!!
