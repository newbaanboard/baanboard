Check your Baan logs they may offer more info on the problem. Also read
Quick Guide QG1819 - Licensing and Validations: Frequently Asked Questions.

The Baan license can change the max users if your using more then 1 server.

What is Network Management?
Network Management allows a number of servers networked together to run 100% of the users for 72 hours when one or more servers fail. For example, suppose you have a network with two servers and 100 users. You can run all 100 users on the one remaining server for 72 hours, allowing you time to repair the failed server. After 72 hours, if the failed server is still not running, only 1/2 of the users, or 50, can be accessed from the second server.

(We say 1/2 in this example because there are only two servers. The denominator of the fraction will be the number of total servers, and the numerator is the number of servers still running. So, if you have a four server network, the number of licenses would be divided by 3/4 if one server went down.)

Network Management is automatically activated for every network request.

What is High Availability?
HA allows you to run 100% of users indefinitely when one or more servers fail. This option is restricted. Please contact Baan Company for details.
