Hi,

Beware while doing the Convert to runtime for the shared tables. You might lose your data. Remove the table sharing, take sequential backups and then do a convert to runtime.

There is no need to shutdown anything. Just logout and log in back. And then create the table with the sequential dump that you had taken (only if required).

Please make sure you have all the backup's i.e. DB & Application before going for this activity.

Also, its always better to test it on a Test Environment first.
