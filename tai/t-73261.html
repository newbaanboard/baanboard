Found an other KB on Infor-Support: 1678744
I think your Version of the LNUI may be to old to support standalone mode.

Description: Problem Description:-Is there Logout Option on LNUI ?

Resolution:
The LNUI does not have any Logout option. The only option is to use the LNUI with fslogin in Mingle, then there is option for Logout from Mingle, that will also logout the user from LNUI.

With LNUI 11.2.4 , LNUI support Standalone mode, once the configuration are done, one can use the standalone url for LNUI, With this standalone url LNUI supports SignOut from LNUI.
