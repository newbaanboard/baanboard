Well, setting up LN table sharing is exactly what I'm doing today!

As far as I tested, if you don't share the logistic tables, the "create table" session will create them for each and every of your finance companies. So, you end up with a lot of tables in the database that will never be used but eat up space (twothousandsomething tables for each company!).

Another thing I would recommend is to use the table sharing modeller. It's a little effort to get used to it, but afterwards new company numbers are easily integrated and it' not so error-prone as doing everything by hand. There you should also find some predefinied table sets telling you which of the finance(!) tables you will have to share

Hope that helps
