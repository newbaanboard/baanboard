Hi,

Create a session with main table tisfc001 as Session type "maintain" and Window type as "Multi Main table".After creating the session go to the session specfic options and select "MMT Satellite Sessions". You can add the session with main table tisfc010 in this to show the tisfc010 related session as lower part to the session with main table tisfc001.
