Hello,
There is a manual in the document CD of BaaN IV that describes the step by step procedure of installing MS-SQL and BaaN. Anyway, the steps are -
1. Install MS-SQL.
2. Install BaaN (Master CD).
3. Install Porting Set.
4. If there is an extension, then install that too (Update CD).
5. Install Service Packs.
6. Create tables and import the data tables and models.
7. Create VRCs.
Thats all.

Hope this helps.
