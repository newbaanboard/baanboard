In Baan IV the session to link Baan user and database user is ttdba0110m000 (Tools>Database Management>RDBMS Admin>Maintenance). So create the user > convert to runtime > link with database user > dump users.

I follow the below steps
1. Run ttdba0110m000 > Go to the second form
2. Click insert > select baan user name in the "BAAN User"
3. "Database user" should be same for everyone, for us it is "baan" which I think is the default. You can also copy from existing users.
4. Save and dump users.

Note: Need to do this for all existing databases (maintained in ttaad4110m000). You can use find to select the databases.
