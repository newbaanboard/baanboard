In Roles you need to setup an ADMIN role and assign the following Objects and Actions
ATTRS ATTRS-GROUP
CH CH-GROUP
CP_VER CP-GROUP
CR CR-GROUP
DOCS DOCS-GROUP
DOC_REV DOC_REV-GROUP
FILE FILE-GROUP
FLDR FLDR-GROUP
HARD COPY HARD_COPY-GROUP
LIBR LIBR-GROUP
LINK_ATTRS LINK_ATTRS-GROUP
LINK_ATTRS_PROP LNK_AT_PRP-GROUP
OBJECT_FAMILY OF-GROUP
ODM_PARAM OP-MODIFY
OBJECT_TYPE OT-GROUP
QUERY_RUN QRUN_DELETE
QUERY QUERY-GROUP
RELATED_OBJECTS RO-GROUP
RULE RULE-GROUP
SYSTABLE SYS_TBL-GROUP
TASK TASK-GROUP
TG TG-GROUP

We are implementing at the moment as well and thought that ticking High authorization role meant that you didn't need to add specific objects and actions, but until we did we couldn't add or alter any thing either.
Don't forget to add the ADMIN role to your employee.
