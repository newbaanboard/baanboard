Hi Shahad,

Assuming that you have had imported defaults.edi (from edi.baan.com -> Ln latest update) and have BEM organization up n running...... If not then you need this to get std. setup data for EDI.... Well, here is the answer to your question about coy 812:

I did a research on this and found following info
--------------------------------------------------------------------------------------
Since ERP LN we have an internal Baan standard called "Bemis", We do not have a separate "X12" defaults.edi for ERP LN.
You can find the downloadable defaults.edi and the documentation on the following website:
http://edi.baan.com/
--------------------------------------------------------------------------------------

Therefore, it does look correct that “BEM” is the organization used for ERP LN (X12 looks like to be valid for Baan4 but not ERP LN 6.1)
Please try to access URL link http://edi.baan.com/ which should provide you the download for the corresponding defaults.edi and required info.

Hope this will help...
