The Menu is used when you are using the Workflow facility of Baan: You model business processes within DEM, and the sessions within those processes are presented to the users in the right order by some sort of scheduling system.

So, the Menu gets activated only if the session is called from the workflow client.

The whole functionality is more or less alpha software in Baan IVc2. I do not know if in any higher versions the workflow facility of Baan software is mature enough to be used in a production system.
