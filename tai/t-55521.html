Hi,

You were right - BW does not work in CE (not even Portable version).
but , their is an alternative.

You can use RDP in Windows CE 5.0 to connect to baan from Terminal Server.

http://support.microsoft.com/kb/312102

Note : In case if you need to make a Terminal Server - you can try trial version for 180 days and then reset the trial in 2 min.

The best part is - How to reset Trial is available at below MS link.
http://support.microsoft.com/?id=839878
