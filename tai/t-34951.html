gcharles, see my first post, I gave you the session codes: ttams3133m000and ttams3144m000. Via the Menu Browser you go to Tools -> User Management -> Authorization Management System -> Role Data. Then in Role Data (session ttams2100m000) you select a role, and via 'specific' Session Authorizations -> Session Authorizations By Company, and Table Authorizations -> Table Authorizations By Company

Hope this helps...
