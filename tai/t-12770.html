Vish,
2) all NORMAL users are not able to place the files created using seq.puts() on to UNC names directories eg: \\new-baan\output\test.txt.
Check the return value for seq.puts() if its -13 then its a permission issue.
13 EACCES Permission denied
This occurs when an attempt is made to access a file in a manner forbidden by the protection system
