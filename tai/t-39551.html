24th July 2007, 14:30
You can take up one license from your live server by making the necessary changes in your license6.1. Replace the existing entry with
Live Server Name, Test Server Name

Stop your baan license service on the test server(if its running in demo then this service will be turned off by default). Optionally add the live server address and name to the host file of the test server(i.e if they are on different subnets)

If it doesnt work after the above steps then you could do the following step too..
In the session Maintain Security code/validation keys {ttiex3102m000} enter the same customer code, license code and organization name as your live server and change the seq. number to 2..

I hope it works..

sk

P.S: The above setting is for a windows platform..
