Well you can use the setenv command

setenv("BSE_HOSTNAME","Your_value")

But you need to be cautious in this operation as the impact may be huge.From the
ERP Guide


setenv()
Syntax:
function long setenv (string env_var, string env_value)

Description

This sets the value of a specified environment variable of the operating system. It returns non-zero in case of error.


Arguments
string env_var
string env_value

Context
This function is implemented in the porting set and can be used in all script types.

Example
long ret

ret = setenv("MY_ENV_VAR", "this_is_the_value")
