Hello,

FYI, I finally used a functionnality introduced from WebUI8.7.
- In LNUI Admin page, go to “Infor ERP Environments”>“System message”.
- Set “ Poll for changed system message every (in seconds)”
- Activate "System Message"
- Add your customized message in $BSE/lib/systemmess
