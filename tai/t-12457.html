Hi learner

The build-up of the error codes stays the same, thus, if you have an error > 1000, you have to subtract 1000 for the database error. With Oracle, you can then use the "oerr ora <error>" command. Actually, since you're working on NT, you should be able to get the database errors (with some description) from the event log.
