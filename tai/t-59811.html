At this point, I concluded the problem resided between the printer and HPUX.

I have been corresponding in HP forum trying to resolve this problem and have been advised the printer will not work on UNIX system because the printer is host based. Bottom line, we selected a printer that won't work in our environment.

Up till now, we have successfully added remote printer to HPUX for all printers with either a print server installed in the printer or externally attached for network connection. Since the HP LJ P1606dn had a print server installed, we thought it would work.

Given the technology changes, will anyone provide advise about what to avoid regarding our future selection of printers?
