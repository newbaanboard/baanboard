Execute SQLServer2005_BC.msi.
This file can be found on the SQL installation CD/DVD:
.\x86\setup\x86\SQLServer2005_BC.msi for 32-bits system or .\x64\setup\x64\SQLServer2005_BC.msi for 64-bits system.

This tool installs some dll's that are needed to make Baan-IV or ERP_LN work on SQL2008.
