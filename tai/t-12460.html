Is the problem solved ?,

Suhas is right, this happens normally when you have External Text editor defined, and Notepad gets infected with a Virus/ Corrupted.

You will find 2 entries in Windows directory of client.
Note.com and
Notepad.exe

You will need to start client in DOS mode to delete Both Files ( You may get access denied error from windows) and then copy Notepad.exe from a clean machine.

Another common reason is user deletes baan tmp folder on client machine by mistake. In such Cases create tmp folder in BaaN directory.
