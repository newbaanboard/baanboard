Refer to:

SSA® License Manager 2.2 - Installation and Configuration Guide

for more information. There is also a SLM-Windows Help file -

Client Configuration
Configure to which SLM Server you want to connect.
SLM Client uses the servers.xml file to connect to a SLM Server. This dialog box helps you to edit the servers.xml file. The servers that are in the SLM Servers to be used list are written to the servers.xml file. You can add servers to that list by adding them from the SLM Servers available list, or you can add servers manually.
SLM Servers available: The installation wizard did a broadcast to find out which SLM Servers are running. The results are displayed in this list. When a SLM Server is running in Slave mode, also the Master server is displayed. To add a server to the SLM Servers to be used list, select it, and click Add.
SLM Server: You can add a SLM Server manually. Enter the name of the SLM Server.
Port: Enter the port number of the SLM Server. To add the server to the SLM Servers to be used list, click Add.
SLM Servers to be used: The SLM Servers in this list will be written to the servers.xml file when you click Next. You can add a maximum of 4 SLM Servers. To remove a server from this list, select it and click Remove.
