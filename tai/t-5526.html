Yes there are.
See the support site on SESSION_TIMEOUT.
Licenses that become inactive (user does do nothing but have menubrowser open; or even open sessions but no updates etc), will be closed down.

There is also one for the cases when the server loses connection with the front end (like when windows station is closed down without shutting down baan windows client or closing the telnet terminal emulation program). This one is the DS_SERVER_TIMEOUT.
