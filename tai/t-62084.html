This may be caused by a installation-path-change of the latest bw-clients. The default pathname changed from c:\programdata\baan to c:\programdata\infor in the latest clients (programdata may differ in other languages i.e. "programme" in a german XP-installation).

We're installing an old version of bwc and run an immediately update on version 101 (or newer) afterwards. This will use the old pathname on Windows 7. (The old pathname is needed because of some addons we're still using.)
