Hi KY Chuang,

As far a I know this cannot be arranged in Baan itself as the licence meganism is just checking the number of users and not which module of Baan is used. The only solution to arrange this is with (login) scripting. Therefor you could create a file with the users and the deparments they are working. When the user log's in to Baan you need to check the file with logins and departments, you can compare the users in the table with the users currently logged in into Baan ( licmon61. -w ) or create counters per department. Based on the counted users per deparment and the maximum number of users you allow per department the user can login to Baan.
