As far as I know the licence is tied to the ethernet address of the first network card, regardless of the setup of this card, and regardless of the assigned IP address. Thus, you should not have licence problems when configuring the card.

As for the speed, it will only make a significant difference if the current network is congested and has a lot of collisions. If this is not true, I don't think there will be much difference.
