This something to do with the access rights to execute the program. I am not able to explain you the technical reasons but the following solution should work if the earlier one doesn't work.

1) Uninstall BaaN client from the local client machine
2) remove the registry key HKEY_LOCAL_MACHINE\SOFTWARE\Baan (after a registry backup)
3) give administrator rights to the user
4) install bwclient from this users login
5) remove the administrator rights of the user
