but can I then modify the obg4 and mex4 VRC's in Maintain Package VRC's ttadv1111m000 and set these two VRC's to be derived from my new one? Are there any other sessions/steps to complete this process successfully?
Yes you can. After changing derivation structure you have to rebuild both package combination (obg4, mex4) in Session Packages by Package Combination. At the end of the process you must have new fd files on $BSE/lib for package combinations ong4 and mex4. Before start with changes be sure that you have good backup.
