Check / compare settings Protocol and Portnumber:
BaanLoging defaults to port 7150
Rexec defaults to port 512
In the Bshell Environment tab under Command only use:
-set NO_WORKTOP=1 -set BAAN_WIN_TITLE="%h '%S' [%s] (%p:%u:%c) %f"

Compare settings with users who can login.
Also check $BSE/log dir
Check user settings (user is logging in with OS name set in ttaad2500m000 under System login (not the User account under User)
