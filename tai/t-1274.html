This utility can be used to simply send a message to all your current logged on baan users. Users that will log in will also see the message. If the file is updated the users will also see the updated message.
The message must be written in de file ${BSE}/lib/systemmess and the ttstppollmessage must be loaded during the startup of the bshell.
