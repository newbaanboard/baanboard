Unix has a default Sytem Administration Manager (SAM), where you can configure user password settings, though I'm not positive about the level of complexity your require.

To access the utility, simply login as root and type [sam] at the command line. You can navigate your way to users and groups and you should find options to configure password requirements and aging.
