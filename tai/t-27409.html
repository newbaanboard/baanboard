Hi folks,
maybe i can put some light here although i'm not on Baan IV but on Baan V.

In our Version the paperclip-button is part of functionality of the ODM Package. If ODM is activated then the button becomes active for some predefined sessions.
ODM works only with bw/Worktop

The sendto.desktop is a function introduced with the cmf module in Tools.
You can send a so called Baan-Shortcut (a bwl-File) to your Desktop or via E-Mail to a recipient or to a task (by default only in combination with MS-Outlook for which a client-connector is provided by Baan).
This works with Worktop/bw.

When using Webtop the sendto (to desktop or mail) functionality works in a different way. You get an URL that calls the Webtop and starts the appropriate session.

Hope this helps,
Andreas
