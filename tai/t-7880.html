Baan created a new method for new installations, called 'Batch Installer'. After the installation of the mastertape, all service packs up to 8 can be installed in one go. This installation is not done with PMC, but all PMC necessary tables are filled to have a consistent registry and to be able to install solutions and service packs on top op SP8. Also for BaanERP 5.0 c a batch installer is available; this one goes to SP7.
