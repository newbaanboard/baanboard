I took this approach and it worked. For those having the same problem try this out.

I noticed that the software did not make any registry entries on the machine where it was succesfully working. So all I had to do was copy the ibaanPDMcore folder into the WinXP SP2 machine and then Put in the required environment variables ( BPDM NETNODE, etc etc...) and it worked.
THere was a problem with report generator. To tackle that, I just installed only the report generator from the install kit. THis seemed to work okay.
