Hi,

If on Windows, check your hosts file under C:\Windows\System32\Drivers\etc\

You will see something like this:-

# localhost name resolution is handled within DNS itself.
# 127.0.0.1 localhost
# ::1 localhost

Under this:- add a line with the IP address of this server and its fully qualified domain name.

e.g 192.168.100.20 server_name.domain_name.com

Now, do a slmconfig -c and try to add a new entry again.

Restart the slm service. Do this for both the servers.
