We have a similar configuration.

1. I would use the webserver because Baan IV does't use SLM
2. ?? request a validation key from SSA support and install it on the SLM
The SLM doesn't need much CPU power.
3. If you use the webserver, it doesn't make sense because if the server is down you also don't need the SLM... .

Only keep in mind that you don't need a licence for the WebTop itself but for every Baan Server where the Webtop is connecting to.
I.e. we have 8 (7 and developing) Baan Servers and so need 8 WebTop licences.
