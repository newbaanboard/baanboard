You should be able to access Triton or (I think) Baan IV ASCII using terminal emulation on a Mac. Certainly we used Mac for both network and dialup access on a Triton 3.1 system.

For GUI, the answer has been "no". That is, there is no Baan client for Mac-OS. In theory, you should be able to install the windows client on a Mac running Virtual PC, but I haven't heard of anyone who has tried it.
