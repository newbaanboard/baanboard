If you want to view Help pages via the web, you can use Webtop / eBOKS (acronym for Baan Online Knowledge Solution) to view the help pages. The Help displayed is always first exported to XML. eBOKS reads the XML files and converts them on the fly to a displayable format (HTML).

In this way, it is possible to view help, get updates of the help, and/or write your own (XML-based) help in addition, without downloading the actual help files to your client each time. Access to a central Help webserver running eBOKS is sufficient.
