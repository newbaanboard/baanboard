Hi,

You need to go to ftp.support.baan.com

You can find the solution list in excel files for every month. Now, you need to check in your solutions session(ttpmc2101m000) whether the solution is installed or not.

If not present, just download it using session ttpmc2210s000(Download, Scan, Connect Solution) Once downloaded, you can install it.

The Username and Password for ftp are present in $BSE\lib with file name as ftpbaan.
