Those are not sessions, they're programs. You'll have to start them via a telnet session. As for manuals, consult your help. They're under Technical Information/Database Tools and, if you click on relations in this screen, it brings up the links to the help of these commands. For syntax on how to use these commands, use the -U command line option.

To start, create either fixed length or seperated dumps of the tables where you are encountering problems.
