Hi

all users in Baan on a W2003 Server must have the permissions to log on locally and also that they are allowed to start a local service. You will finde these settings in ADS Default Domain Policy. The easist way to configure it it create a group for all Baan Users and give these group the needed permissions.

Second:
All users need read and write acces to you database directorie.
