This only works for Windows. You cannot use the Worktop with Linux because no Linux compatible binaries are available. Of course you can try Wine or a Windows based virtual machine. For the end users I'd recommend using the Webtop which is web based and compatible with the most operating systems and web browsers. For developing and administration you'll still need the Worktop because many of the tools sessions don't work in Webtop.
