Hi,
Both the servers need to be revalidated.
Execute the following sessions:
1. ttiex3100m000 - Maintain Requested System Configuration
2. ttiex3100m001 - Maintain Requested System Configuration(tools)
3. ttiex3400m000 - Print Requested System Configuration
Send the System Configuration to Baan. and Update the Security Code / Validation Key once recieved from Baan in session
4. ttiex3102m000 - Maintain Security Code / Validation Key
5. ttiex3220m000 - Validate Tools Set
6. ttiex3225m000 - Configure Application
