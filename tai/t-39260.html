Please double check that you have got right RSC. Maintaining incorrect RSC brings your BaaN into 30 days demo version and then you need to prayer to Support.

If you have already got new RSC then the procedure is very simple.

Open session Maintain security code ttiex3102m000 and enter as is validation code. Please veryfy it fully.

Run session ttiex3220m000 Validate tools set

And lastly ttiex3225m000. Logout and login. Please note your server should be single user mode.

Thats it.
