The path in the parameter session is the path, where the system put the solutions by scanning/processing the solution.

Read carefully the quick guide regarding the PMC setup, make sure that you understand well how PMC works.
The paths you entered there is for internal PMC handling of the solutions; do not use them yourself.

Setup a directory locally (usually within bsp home directory or something like that) and then use that directory to store your downloaded dumps and when scanning them, point Baan to that directory.
