This is what's located in "Maintain device" under "driver".
So if : Driver : lu_lj4_s2quer is enabled
(Don't use, it's my own!)

(First character)

To create a new printer file you will need to request the technical guide from HP. You may find it on their website.

The file is a reserved list of commands that Baan looks for to control the printer. They are a set of Escape characters or control characters that will change font size, select paper bin etc....

Infor does not normally create new printer.inf files. They now have a windows server printer in the device manager that prints to the Infor Print Server client. You must have porting set 8.5 or higher for that to work. It will use the normal windows drivers.
