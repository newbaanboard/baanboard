You will need a import exchange change for the importing of data to the new company using the sames ASCII output file layout as
the export. Then change the Maintain table relations form 2 to oerverwrite yes (rest of the fields no). And you will also need to change the fields relations on form 2 to overwrite 'yes' on the fields you wish to change, and 'no' (like item number), on the fields to leave as is.

Hope this helps some too!
