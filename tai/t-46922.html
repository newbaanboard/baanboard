Also refer to the Solution ID 200450
Solution Description What SLM license keys do I need for ERP LN to run ?

When starting SSA ERP LN we get errors:

Fatal Error: No license to run object '[$BSE/%BSE%]/application/...........' productid (10056)
Fatal Error: No license to run object '[$BSE/%BSE%]/application/...........' productid (10896)
Fatal Error: No licence to run object '[$BSE/%BSE%]/tools/tt7.6a_a/ottstp/ostpoledaemon' productid (10996)
Fatal Error: No licence to run object '[$BSE/%BSE%]/tools/tt7.6a_a/ottdsk/odskbwbrowser' productid (10996)

SLM is not configured with appropiate keys.

SOLUTION DESCRIPTION:

First SLM needs to be configured with the proper license keys before BaanERP LN 6.1 can run.

You need to have at least the following Product ID's as valid keys:

10056 SSA ERP LN 6.1 (for Application - General)
10996 SSA Enterprise Server (for Tools)

Optional:
10896 SSA ERP LN Service (for Application - Service)
10146 SSA Enterprise Studio (for Development Tools)

For other Product ID's we refer to internet at http://onepoint.infor.com where you can select "Request a Software license Key".
Here you can select "SSA License Management (SLM) Key Request Form " and press the "More information" button.
At the bottom you can select the group "SSA ERP LN" and press the "Information" button again.

Make sure to enter your SSA ERP LN Server with the appropiate "Rank" and "Server ID" below the license key
