Hi,

Please mention your Version so that one can provide you with the session codes as well.

Basic steps are -

1. Create a package (ttadv1100m000)
2. Create the modules required in the package (ttadv1101m000)
3. Maintain Package VRC against the package (ttadv1511m000)
4. Add the Package VRC against the existing Package Combination(ttaad1121m000) and do a Rebuild DD.
5. Logout and log back in and check if the directories at the Server level are updated.

The session mentioned are for LN.
