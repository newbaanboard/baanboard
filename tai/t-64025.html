Just to let you know the outcome.

The new porting set creates new rc.x files in $BSE/etc with .nw at the end of the filename. However, it does not copy these over the existing rc.x files so when the installer wizard tries to start Baan after the porting set upgrade it is trying to use old x6.1 binaries and not the new x6.2 versions.
