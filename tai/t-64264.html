BaanERP Programmers Guide


getenv$()

--------------------------------------------------------------------------
Syntax
string getenv$( string env_var(256) )

Description
This returns the value of a specified environment variable of the operating system. It returns an empty string if the variable is not available.

Context
Bshell function.

Example
string name(15)
name = getenv$("LOGNAME")
