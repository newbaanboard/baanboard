hello, yes there is :)

You can remove the application locks by running the session 'Remove application locks' [ttstpdellocks - for Baan IV and ttadv9598m000 - for ERP LN all versions] (Tools - database management - miscell). If you are sure that the application that has set the lock is not running anymore then remove the application locks only! The session displays the locks that have been set. With the option [D] you can delete a lock. By using the session 'Remove range of application locks' [ttstpdellock - for Baan IV and ttadv9298m000 - for ERP LN all versions] you are able to remove a range of application locks. You can use this for instance in a night job after all the users have left the system and when no other programs are running. When running this program you must ensure that no locks are set. They may cause problems when running other programs in a night job.
