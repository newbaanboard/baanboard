The basic steps are:

a) Install a fat client on the new server to get the registry settings and service configuration of Baan

b) Copy the old installation to the new server

c) Adopt configuration

The tricky part is step (c) obviously and depends on your old configuration, database used, licensing setup etc.

I suggest you give it a try and the community here tries to help you with whatever problem you run into.
