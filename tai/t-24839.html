Did you check your authorizations in the sessions ttadv0142m000 "Maintain General Developer Authorizations" and/or ttadv0141m000 "Maintain Developer Authorizations by Package VRC"?

With Session tadv0142m000 you authorize a user to develop software of all packages VRC's and with session ttadv0141m00 you authorize users to develop applications only by package VRC. In this session the authorizations can be defined at the level of modules and languages.
