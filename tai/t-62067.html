It's probably caused by the contents of $BSE/log/install. One of that files contains the current config. Ask support to fix it for you.

As a work around you could pick up the pmc files from the Enterprise Server 8.7 cd and install them directly by using process patches after you have scanned them.

Don't forget to update your tools Addons as well.
