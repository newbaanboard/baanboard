Hi,

ba6.2: This is the Baan ASCII Client Program. It is a binary executable that is platform dependent. Thats is, for each type of Operating System, Baan release a seperate version (Porting Set) of this program.

bshell6.2: This is the Baan Shell Interface. This program communicates with ba6.2 and the database drivers - and hence the interface for providing all Baan information to your client program ba6.2.

There's not much technical information around - limited info in the Triton Technical Manuals.

I've used BAMODE - when using Job Daemons. Cannot remember using BAAN_EXTJOB.
