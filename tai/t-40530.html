There are no dependencies defined in PMC from tools services packs to application service packs. The only exception is SP8, for which application SP6 is a pre-requisite. The reason for that is that upto SP6 tools were included in the application service pack. (SP7 didn't contain tools fixes.)
Of course there can be a note somewhere in the installation guides or in the knowledge base about the compatibility. I did not check that.
