Hi,

If on LN, you can use the CMF functionality provided by Baan. It has all the functions to enable you to send the mails with all the required configurations.

Refer this : - CMF (http://www.baanboard.com/programmers_manual_baanerp_help_emessage_connector_overview)

Else, if you still want to use BLAT, you can write a 3GL script which will run the blat commands on the server with the required parameters. And then you can attach the script to your session from where you want to send the mails.

Following things are required:-

1. Blat utility on Client/Server
2. smtp server with relay authentication if there is any

Just install blat using following command:-

blat -install smtp_server_address email_address

To send mails :-

blat c:\test.txt -to recipient_mail -subject "Hel
lo" -server smtp_server_address -ti 10 -log C:\blat.log -u your_email -pw
your_pass

The above will send a test.txt file to the recipent. If any error occurs, they will be logged.

Hi,

What you can do is, create the bat file at the run time. i.e. When you are printing some report or sending some notification, you can create the bat file using seq.* functions and then simple run the bat file using run.prog command. In this way, you can change the subject line and the recipients at the run time either by passing the parameters or by hard coding based on some conditions.

I guess you are trying to send the mails from the server side, so this is the method you can use. I use, SQL Mail to do this.
