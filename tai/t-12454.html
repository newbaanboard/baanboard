Status of the job should be 'In queue'. Is the user jobd also configured correctly? It should start in the company in which the job is defined and the password should be stored in the bwc file.

Make sure you store the password if you are loggon on as that person since the password is crypted with an algorithm that uses the account information. Check it by logon as he job users and start baan by using the bwc file. It should start the main menu without asking a password.
