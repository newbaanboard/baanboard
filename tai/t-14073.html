If you look at the help info of the Sequential File checkbox you will get an answer to your question. The following information is there:

"If you select this option, the components in the specified file are
connected to the given solution.

The file should have the following format:

<comp.type>|<package>|<comp.code>|<languages>

comp.type must be a numeric value, according the domain ttiex.comp.

The selections on the second form also apply to this option."
