Description:
Fatal Error : Error 523 (Database full, please contact your system administrator)

SITUATION DESCRIPTION:
When creating tables, or adding records to a table, error 523 is returned.

SOLUTION DESCRIPTION:
Error 523 means: Database full.
This error is introduced in the newest porting sets.
It is normally logged in the file ${BSE}/log/log.oracle

When the database driver receives an message from Oracle like 'tablespace full' or 'Maximumm number of extends reached', it translates the message to a general error 523.
To see what the exact problem is, you should look in the Oracle log file: alert_.ora.
Here it is exactly explained what went wrong.
Probably you need to add a new datafile to a tablespace.




As mentioned above, Please check your Oracle/DB Logs to see what exact problem was present..
