For those who want to replace MS-Office by Open Office.... for Baan no problem either. I've got it work now.
The setup in Baan ( Devices ) is identically like setting up an Excel or Word device. For Open Office Calc just replace excel.exe by scalc.exe and for Open Office Writer replace winword.exe by swriter.exe.

On the client side you'll need to check the Windows path environment variables. The path for Open Office should reside in the windows PATH environment variable. If it not available just add it via My Computer - Properties - Advanced - Environment variables.
