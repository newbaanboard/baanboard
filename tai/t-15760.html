JaapJD
25th June 2004, 14:42
What NPRao suggested is exporting and importing the data of the customization parts itself, and not the real components attached to the customization parts.

If you want to copy the components, you can use PMC to create a solution with the components (session ttpmc1221m00), export and release it, and install it in another VRC. Of course you have to do some PMC setup.
