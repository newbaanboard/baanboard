It seems that for generic text, there is nothing EXCEPT the Microsoft UNIDRV driver.

Also, it seems that this is a known problem that Microsoft reports that started with Win2k SP3. They only offer the perverbial "Load the latest updates" as a solution, but an up-to-date SP4 as of today's date till does not fix it.

Workaround: In the Printer Properties for the device, change the Printer Properties - Unprintable margins - Left from the default of ZERO to ONE.
