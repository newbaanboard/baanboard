Ivo Ladage
29th November 2002, 10:53
The hint above should work. ottstppollmess is intended to let the system administrator broadcast messages to all Baan users. Each 5 minutes it polls the file $BSE/lib/systemmess and displays the text contained by it in case of a new message.
This object is usualy put in the BMS mask data to let it run automatically on startup. So if you disable it as described above, the error should disappear.
