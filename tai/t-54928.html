toolswizard
5th February 2009, 19:01
A patch is usually a single file with a few other files consisting of the component list, error file.

When you have the listing you supplied this is done through Export Data Dictionary. You need to use Import Data Dictionary and not import patch to get this in.

Data Dictionary lets you import and export based on a package, and often is used for a range of components. If you have components across different packages, you need to create a dump for each package.

A Patch allows multiple pataches and individual components to be combined into a single distribution.
