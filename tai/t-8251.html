instead of 116210.

3 - Install the latest update for object ottstprepdll :
Use at least solution 116099, but latest is preferable (see list below).

4 - Install latest standard programs (General Solution 10923)

5 - Install latest spoolopen/spoolclose objects (General Solution 13758)

6 - If Porting Set is older than 6.c.05.02, update at least
to this release, but the latest is preferable.
See General Solution 15219(Unix) or 18505(NT)


If the problem is still not solve at this stage, install additional solutions
below :

7 - Install latest BW/BWPRINT client (General Solution 13743)
NOTE: there is a new functionnality in BWPRINT.
Refer to solution 117758.

8 - Install latest locale information (General Solution 101830)
