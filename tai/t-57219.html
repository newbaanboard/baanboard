Sergio,

The solution is quite simple. Use PMC. You can use it like Infor uses it to distribute solutions. It's really a very strong tool in order to keep your software consistent.

Start on the development server the session ttpmc1100m000. Create a solution. Add components, generate the dependencies, export the solution.
Before doing this you probably have to define your base vrc on both development and production

Scan the solution on production, install it with the save components option. This is important because only than you will be able to roll back the installation.

Regards,
