Is there any way to load these files into MS SQL tables of different database?
If these are clones in MS-SQL database, then yes, you can use the Import Data options in the MS-SQL Server Studio.

If you mean importing the Baan/LN tables from one environment to another the you can use the ttaad4227m000 - Create Table from Sequential Dump session.
