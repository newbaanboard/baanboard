Hi,

Option Dialogue is a technical tool that provides control over some various Baan Client/Shell functions.

Of course, it possible to "work without it", but its not possible to start a Baan Windows Client without the Option Dialog - not that I know of anyway.

BaanV also has this Option Dialogue, but the functionality is slightly different.
