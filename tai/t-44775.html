Refer to the Worktop release notes -

Language selection

When installing Worktop 2.2, all languages present on the media are automatically installed. In Worktop 2.2, the user can select what languages should be installed during the installation of Worktop.

The -set language option is for setting the language inside the Baan sessions/screens.
