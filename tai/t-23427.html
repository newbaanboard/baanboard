The problem is solved now and it works fine!

a) Download solution 142197 and unzip the files.

b) Copy this new folder "baanerp" into this directory of your locally installed web help:
C:\...\<your Web Help folder>\tomcat\webapps\baandocs\help\

C:\...\<your Web Help folder>\tomcat\webapps\baandocs\help\baanerp
beside of dir "documentation" and "webtop".

c) Restart your SSA Web Help
