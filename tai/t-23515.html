Hello

Create a batch file like export.bat
Content:

bdbpre -Itablelist -t"\0" -C[CompanyNumber]

where after the -I option is the file wich contains the names of tables to be dumped like

tiitm001
tiitm004
...

where the -t option is the separator between the records
where the -C option is the company you like to use for export

The dumpfiles are stored in the directory where you start the batch job

The file must run on the baan server. Be sure to set the BSE envrionment.

Then connect the export.bat to the scheduler...
