Alick:

Yes the "Default Password" is the database group password. Only select "Use system logon for the database user" if you want to map the
Baan users system logon to the database user. If you have the documentation CD that came with the software, check out the Baan Administrator's guide (U7189BUS), has a section on the RDBMS that should answer most of your questions.
