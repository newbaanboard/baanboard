Rdheeraj,

I guess for NT it would be similar to UNIX. So the next thing you should do is:

Setup Audit File Directories (ttaad4116m000): this setting determines where Baan will store your audit information. The easiest is to use 1 subdirectory for each company under $BSE/bin:

Table Selection Table All Company Path Name
Comp

All (other) tabl Yes ${BSE}/audit/#

Then:

Assign Tables to Databases: in this step you will assign tables you wish to be audited to your 'audit trail' database. Note that parameter tables are mandatory in auditing.

Don't forget to Convert to Runtime!
