Hi Francisco,

When you receive this error, you did install the latest version of ttdllreadmenu but not the latest porting set needed for this. bms.init is a bshell function only supported in the latest porting set.
I would recommend to uninstall the solution with which you upgraded ttdllreadmenu to version 6, which is 130014. The fix in 130014 is only to initialize the BMS Mask list, as specified in ttadv4188m000.
You can check the version of ttdllreadmenu installed from within Worktop; open the File->Properties dialog. It should state version 5, version 6 calls bms.init which requires the newest porting set.
