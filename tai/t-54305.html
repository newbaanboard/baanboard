Hi,

45 EDEADLK - Deadlock situation detected/avoided
Description:
This error can indicate that:

* The maximum file size limit has been exceeded
* A deadlock situation occurred

Solution:
If the maximum file size limit has been exceeded, ask the system administrator to increase the maximum file sizw Baan wide. If a deadlock occurred, contact Baan Company.
