Is it possible to have different date formats for Reports/Forms in 2 separate companies in the same BSE? I am referring to Baan V.

If I have a company (say 100) that follows the US date format (mm/dd/yy) and want to have the European date format (dd/mm/yyyy) for a different company (say 200), is it possible?

This is the only way I know in which date formats can be changed in Baan:

· You can specify date formats either by Language, or
· You can specify date formats by domains

If you go to session ttaad1510m000 (Languages) you will see that date formats can be included by language. However, if we change the date format here, it will affect all companies that use the language  hence it is not company specific.

If you go to session ttadv4580m000 (Date Formats), you can specify a date format code which can then be linked to a particular domain (for e.g., tc.date). This will also change the format of the date in all places where this domain occurs and can be used for report specific changes too. However, this again is not Company Specific in any way.

Any suggestions/ideas?
