I found my file back via an ex-collegue. I will post it so that others will be able to use it. It is very sipmle to create / maintain, and as far as I remember it works on Baan IV as well, although right now I tested it only on Baan 5.0b

All you have to do is create a (text) file and define the shortcuts. An example is:


f2:process(ttaad4100) | GTM
f3:process(bapgw0502m000) | Activity log
f4:process(baxml0510m000) | Maintain Gateway
f5:process(ttadv2500m000) | Sessions
f6:process(ttadv2530m000) | Scripts and Dll
f7:process(ttadv3530m000) | Reports
f8:process(ttadv2560m000) | Functions
f9:process(ttaad4500) | GTM
f10:process(ttadv4551m000) | Messages
f11:process(ttaad2500m000) | User Data
f12:process(tcibd0501m000) | Item Data

Name the file 'k<baan-username>' (so for user 'baan' it will be 'kbaan'), and place it in the $BSE/lib/user directory.

Enjoy!!

En.
