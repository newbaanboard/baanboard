In ERP LN you can add field level authorizations where you could have the following scenario... You want two groups of users to open the same table the same company but you want one group of users to only see regular POs and another group of users only see EDI orders. It is possible. Think of the same scenario where you let one group of users open as session and only be aware of certain jobs they can run and another group of users open the same session and only see records that match an opposite or specific criteria, and so on... If you open an incident in infor365 support can give you a solution with detailed steps on how to do that. I can tell you it can be done with table field authorizations in the authorizations module.
