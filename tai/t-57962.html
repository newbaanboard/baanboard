"The result was that I have access to the configuration but when I tried to configure the worktop the login box appears and i can access to baan( the baan option dialog come up and i can start the baan shell via this dialog and the ps-command shows me that the ottstoledaemon was startet ), but the worktop says me that he is waiting for the connection."
You can solve it using 'worktop_client.exe' instead of 'worktop.exe' at worktop\bin folder.
This application works fine to create the .wtd file. After that, you can keep using worktop.exe to open the configuration file.
