Newer portingset gives you bug fixes, performance improvements and eventual also support for newer versions of the operating system and database.

If you don't upgrade the os and portingset and don't have run into bugs you could keep the environment like it is right now.

At a certain moment you will need a newer portingset if your rdbms is no longer supported or if you migrate to a new server with a newer operating system.

However don't give up your support contract. If you don't have support you will not being given a new license key which you need if you are going to migrate to a new server!
