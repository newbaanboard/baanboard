Hi

As far as i know, below are few common useful shortcuts of GTM.

c = to change the fields (pkg, mod...)
y = to see the table fields
e = to exit from y
* = to see the domain properties
* and 0 = to see the domain name
* and 0 and enter = next table fields with domain name (you have to enter till last form)
0 = to see the number of records
k = to see the indexes
space = to see the menu bar
n = next records
p = previous records
b = last record
t = first record

Please let me know if you get new ones.
