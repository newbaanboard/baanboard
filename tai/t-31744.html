SITUATION DESCRIPTION:
A porting set is a set of binaries, which are operating system dependent. It is the layer between the operating system and the application software. Baan regularly releases a new porting set. A bugfix number recognizes every modification made in the porting set.

Each porting set has a porting number. Initially, the binaries have the same porting number. If a fix has been made on a binary before the next porting set has been delivered, that binary will have another porting number. For example, a porting number for HP binaries can be PA.1021, but the Informix driver inf_srv6.2 which has been modified after the release date could have port number PA.1123. Release notes will be available for each fix.

NOTE1:
For currently supported versions of the OS see the technical notes of the last released portingset.
