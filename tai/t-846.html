You have to extend you Baan license to Oracle8.
Installing an Oracle8 Client on your Baan Server, setting up SQL-NET and changing the Database definition in Baan to ORACLE8; creating database user for ORACLE and linking existing or new BaaN user to the ORACLE-Database user; setting up ora_storage, creating Baan-Database user in Oracle, setting up Tablespaces .......
