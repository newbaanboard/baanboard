couple of things to look at.
1. You MUST be logged into the server as local user Baan with password baan, when installing LN.
2. You must ensure that SQL security is set to both Windows and SQL authentication
3. Ensure that password complexity and password size is set so that the password baan can be used. SQL 2008 uses your windows password security settings when setting passwords.
