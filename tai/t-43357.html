I found two sessions: copy VRC combination (ttaad1220m000) and Copy software components to a new VRC (ttadv6265m000). What is the difference between them?

ttaad1220m000 will create a new "Package Combination".
ttadv6265m00 will copy VRC components into the target VRC you mention.

As my understand if I use the first one, the second will be redundant, right?

The 2 sessions are independant. There is no relation between the 2.

Of course I can do a export data dictionary and import it to the new VRC...
Yes, this can be done. But its time and resource consuming process.

Go for this:
1. Create a new package VRC manually for your purpose.
2. Use session ttadv6265m00 to copy components.
