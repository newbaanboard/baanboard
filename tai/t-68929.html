Hi,

You can use Table Data Authorization to restrict the approval of PO.

Steps -

1. Create a new role if you dont have one
2. Go to session authorization by company and add Full Authorization to all companies
3. Go to table authorization by Package and add all the packages with Full Authorization except "td" package
4. Go to table authorization by Module and add all the modules of "td" package with Full Authorization except "pur" module
5. Go to table authrozation by Table and add all the tables of "td" package and "pur" module with Full Authorization except "tdpur400" table
6. Go to Table Data Authorization and add 3 records as show in the attachment

That is all.
