GaryEd
3rd June 2013, 16:10
You need to add two new database definitions in baan (ttaad4110m000). One for the mandatory audited tables and one for everything else. In the parameter field you specify the path to the "secondary" db instance. I believe for this to work correctly if the db instance is on the same server as the primary db the secondary instance has to be a named instance.
In the assign tables to database session (ttaad4111m000) you specify the company number for the tables you want to use the new db.
Ultimately it all ends up in the tabledef6.1 file in {bse}/lib
