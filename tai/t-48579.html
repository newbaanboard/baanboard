The session for Additional File is ttadv2570m000. You have to check if the particular additional file exists or expired based on the package-afile-vrc combination. Additional files follow the same VRC derivation rules as sessions, scripts etc. They are typically located in $BSE\application\package-vrc\aVRC\a-filename.extension for application packages and for Tools you have to check under $BSE\tools
