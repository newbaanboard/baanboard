You can search www.support.baan.com in Documention for U7432B US "iBaan ERP 5.2a Administrator's Guide" and review the section on VRCs. Basically after you have the standard, you put your own customizations on top, and then add a developer layer. This allows you to run companies with your own customizations while allowing a place for developers to write code.

To create a new Package Combination and VRCs, see the ttaad1222m000 "Create New Package Combination / VRCs (OneStep)". This session greatly facilitates the process of creating a new layer based on an existing one. If you wonder what it does, see the help which explains all the steps this session does if you prefer to do them manually, step-by-step.
