Hello Roy,

You may use User History feature to see how many times the report was printed. To enable the User History, select History check-box in the User Data Template and then perform CRDD for all the users. Once this feature is enabled, you may print the user history with session ttaad2402m000.

As far as the modification to the report data goes, you will have to enable audit on the tables used in the report.

Hope this helps.
