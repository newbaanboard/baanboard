It depends on what server type you're speaking about:

- If it's UNIX box, you'll need to connect using HyperTerminal or something similar, then you can use ba6.1.
- If it's a Windows box, you need to setup a RAS server on your side and then connect using Dial-Up. Then you can use the BW.

Search the web for this, you'll find plenty of information (mainly Dial-Up).
