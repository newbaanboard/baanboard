Hi Ravin,

Because you have full authorization at form command level but the user is not having full authorization for that session, that user is not able to see that form command. The authorization group at form command is the minimum authorization that user should have to be able to see/use that form command.

Make the authorization group to Display and the other user should be able to see it.
