Hi here my little Input to the story:

If you like to plan ahead, then you approach is most likely anyway the following:

a) inform Baan of your plans
b) Do a test (!) migration to the new system
c) Promise Baan (in writing) to switch of the old system soon
d) Validate the new system
e) test the new system
f) Repeat step b and go live witha validated system
g) Switch of the old


This way you never have the issue to start with.
