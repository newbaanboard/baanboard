Lets try to clarify things:
1. you have machine A running Baan licensed
2. you have machine B running Baan licensed [?]
3. each machine has its own license daemon [?]
4. you want to make a copy of A into B (full clone)

IF 2 and 3 are correct, then your license information is no longer valid (license in the objects/DB no longer matches the B:/.brand file). You must re-license B again (no need to ask for new keys, just re-license with previous ones and path objects)

IF 2 is right but 3 is not (you are sharing the license daemon, 2 sets of vaildation keys), then you need to check how you had them working before (2 sets of keys, same keys but only one server for the license, ...)
