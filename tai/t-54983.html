Yes. I know about ES 8.5a, but as we do the upgrade to get the same version of Baan environment compared to our live environment, we need the older version first.

Btw. upgrading the tools only (without installing SP/FP) works fine. You only have to download the spt/spt files (for example spa76a4.tar.gz and spt76a4.tar.gz) from Infor's FTP server. After moving both in a directory like "Infor Enterprise PMC Solutions" the Staging Wizard recognizes the upgrades as installable units. Then the upgarde can be installed as usual through the Installation Wizard (Feature Pack for Enterprise Server).

Of course it is not recommended to mix several versions of portingset, tools and feature/service packs but sometimes there is no other choice...
