With SLM you could use one of the servers as master license server, the second as slave and have it licensed for 30 users. If one of the servers fails the application can still be used on the other. However after 3 days it will support only half of the number of users unless you buy the HA option.

Question is: are both licenses the same right now? Are both slm server in the same lan? Not sure if you should use it over a wan.

I don't think you can just add the other slm server to the servers.xml since the documentation states that a client can only be connected to one SLM cluster.
