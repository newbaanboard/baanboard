Hello,
check the HiDox LOG-Files !

There are several directories with log information that can be helpful to research problems.
{DocumentServerHome}\fdds\log\fdds.log contains the DocumentServer log with start information etc..
{DocumentServerHome}\fdds\log\mail.log contains the mail/fax log when mailed/faxed through the
document. It shows successful and failed mail/fax
{GUIHome}\fdgui\fdgui.log contains the GUI log
{DocumentRoot}\log contains the fd_html2doc.log / fd_html2pdf.log / fd_xml2
.log in case the communication method is client instead of DocumentServer.
