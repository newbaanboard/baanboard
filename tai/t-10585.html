For PMC to be enabled, you don't need RCS. In BaanIV, RCS is only needed for CMT; in BaanV it is also needed for SCM.

There are several sites, which have a link to a solaris 8 RCS, e.g. http://www.filibeto.org/sun/solaris_8.html

I have a Unix script available, which checks whether RCS is correctly installed to be used by CMT (and SCM).
