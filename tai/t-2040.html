Hi Carol,

The menus are related to package combinations
The authorizations are related to users
The devices are global

A user when is created is linked to a company(ies) and package combination. Authorizations are then created for a user (done by you).
When you delete a user, you unlink it from the package combination and company(ies) (done by Baan) and the authorizations are deleted (done by Baan also). Everything that is global or related to the package combination is not affected.
