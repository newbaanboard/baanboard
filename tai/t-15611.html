You'll need to relicense the cluster, especally if you plan to move to a network license.

The .brand file stores information about the system you are on, in Unix, it stores things like CPU serial number.. in Windows it stores things like the MAC address of your primary network connection. So the files are completly incompatible between systems.

I think, the brand files stores it's own inode number on unix. This is the reason, why you can not recover a brand file from backup. After the recovery the brand file will just not be correct anymore.

Anyway, if you move from unix to windows you need to relicence whatever.
