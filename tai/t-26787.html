Hi Agoes,

Question 1: On Unix you would just copy your BSE folder and be done with it. Because you are on windows, you should first make a duzmmy installation (to get the registry settings) and then copy your old installation on top.

Question 2: Keeping the old folder structure is the most easy, but you can also change it. If you do, then you need to manually change some runtime files in $BSE/lib and also in certain Baan admin tables. This will need a good deal of Baan knowledge!

As long as you run on windows, you will have to make an installation to get the required registry settings (for windows services etc). This is independent from the fact, wether you keep or change the folder structure.

I call it a dummy installation, because you do not need to bother with any configuration steps. This is all covered, when you copy your existing Baan on top. On the installation CD you will find a document describing the installation. But it will be something of an advanture, if you have never done it before or at least have seem somebody doing it. You might seriously consider getting some outside help for an activity like that.
