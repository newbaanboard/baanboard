As far as I know, if you use named users, you can only reserve the same amount of user accounts in session "Maintain users", while, if you use cocurrent users, you can reserve much more users accounts in session "Maintain users". BaaN cannot identify what type of user licence you bought technically. In other words, even if you bought named users, you can also reserve user accounts in session "Maintain users" more than your licence amounts technically. No problem, no error message will appear. However, if you really do that, you are out of the original agreement with SSA. SSA would take actions for law because of your activities.
PS: Named user licence price is about 50% of cocurrent user licences.
