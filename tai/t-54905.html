In the solution for each porting set, there should be a link to a technical notes document for that porting set. That should have the major differences between that porting set and the previous. There should also be a link to a bug fixes text file.

I am not aware of a comprehensive document that shows all differences between all porting set versions. Depending on the version, the tech notes document does sometimes have the differences/improvements for several PS versions in a single doc.
