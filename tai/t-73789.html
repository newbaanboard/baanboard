I thought that the porting set upgrade would help but unfortunately not, but when I uninstalled the latest tools solution 10.7.4.1 everything returned to normal and works.

There is a KB 2190344 which solves the problem.

"Error 39 occurs while scanning a solution file. This happens when the solution is created with PMC version 0. The number 39 is misleading, because 39 is a file size instead of an error number.

Work Around:
Create the solution again with pmc version 41. "
