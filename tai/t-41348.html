Use export DD (ttiex1280m000) to create a dump from the VRC that contains your customisations on the current production server. Dumps need to made per package (tc/td/ti/etc).

Use import DD (ttiex1285m000) to import the dump on the new server. Import the dumps into a new create "own" VRC on your new server.

After the import run Create Runtime DataDictionary (ttadv5210m000) for the current package combination.

Actions preferable exectuted by user bsp ( due prevent permission problems).

Good Luck

JP
