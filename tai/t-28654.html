You should check if you have those standard components. As a vendor, we never send out a dump containing standard components, so maybe if you have those components already, you should ask for a new delivery.

Technically, where they go when you install them depends on how you answer that check box for:
Write software components in other Package VRC
If you say 'Yes', and usually you do, it will put all components into your custom VRC. If you say 'No', you may get them updated in the standard (if it does not throw the 'not authorized to change the standard' error, not sure). But you will also then get the custom components in a VRC called 'B40Cc4itl0', which you may or may not have named exactly the same on your system.
