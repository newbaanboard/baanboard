If i have read it right, then you got to uninstall your webtop 4.2 and install 8.4...there is no upgrade path as far as i know...
What i meant by "its possible to upgrade from 4.2 to 8.4" in my previous post is that users of webtop 4.2 can use 8.4 by just replacing their existing 4.2 with 8.4..
The following should answer all your questions on Webtop 8.4...

For SSA ERP LN 6.1:
==================
Requirements to run Infor Webtop8.4 on SSA ERP LN 6.1 need to be:
porting set version: tools servicepack: application servicepack:
7.6a01 SP1 not applicable

Solution# 144972: How to add an Webtop build on an existing Tomcat enviournment
Solution# 144998: Webtop validation, SLM, SSA License Manager
Solution# 147959: General solution for SLM, version 2.2 (IMPORTENT NOTE: Make sure to use the latest SLM version availible.)
SLM2.2 documentation can also be found on the installation CD of webtop4.0 (BD20.000631)
<diskdrive>:\Utilities\SSA License Manager\documentation\SLM2.2 - Installation and Configuration Guide.pdf
Solution# 200856: Latest 4GL engine to enable change-DEM-company for WebTop.
Solution# 205880: When initializing the Webtop formcache with pre-generated XML dumps (Offline form generation) the formcache is not properly initialized
Solution# 206790: Enable change-DEM-company for WebTop.

Infor Webtop 8.4 (Application plus Installer plus Documentation):
/ftpdownload/updates/webtop/ME0135a/Infor_Webtop__8.4_buildME0135a.zip
(Size: 156970 KB)

Infor Webtop 8.4 (Application only)
