On BaanIV/ERP the only file that matter is $BSE/lib/licence6.x : this has the hostname or IP adress of the server, so you can use one or the other.
Look for this file in NT, if it's not found, then probably the entry is in the registry, probably under HKEY_LOCAL_MACHINE\SOFTWARE\Baan...

A change of server name or IP adress does not need a relicencing (only if the root disk fails or there's a change of the motherboard-UNIX or network card-NT).
