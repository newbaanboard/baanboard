Baan Solution Nr 122541

SITUATION DESCRIPTION:
When using the external editor (notepad) for texts the changes are not written to the database. No error occurs.

SOLUTION DESCRIPTION:
There was for a short time a bad version of the GUI B40c.63. This version is identified with PA.1909 in the bw.exe. Do not use this version. The actual version for GUI B40c.63 is PA.1936. This one is working.
