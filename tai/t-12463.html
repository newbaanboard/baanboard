Yes you can use the old installation CD's, but probably the old CD's were for installation to SQL Server 7.0. Baan now has installation directly for SQL Server 2000. Previoulsy, you installed on SQL Server 7.0 and then upgraded your database to SQL Server 2000 with normal SQL Server utilities.

I would get the new installation disks that work directly with SQL Server 2000, assuming you are still paying support. If you are not, you'll have to do the manual upgrade - but you will still have to relicense your new servers anyway.
