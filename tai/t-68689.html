Hello There

There is a User Management Module in Application Administration and in that there is a Sub Module of User Data present. In that there is a Print User Data session. What you can do is to Run the Session according to your range and take the output in an ASCII file, (when Baan will ask you to input a device put ASCII) and then take the file in a specified location in your system(It will ask for a file location to put after you put the device as ASCII).

After that download that file in to your local disk and open it in MS EXCEL, and split the columns in tab delimited mode and seperate out the Columns and then out of all the Columns present just select whatever columns you require. This is a simple but effective way.
