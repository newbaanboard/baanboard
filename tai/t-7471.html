The display device picks up the width it needs from the session Reports "ttadv3130s000". If you have created a report with a certain width and then increased the layout sizes inside, you will not be able to get a scroll-bar in the display device to see the complete report. Go ahead and fix your report definitions. No need for a new device (actually one cannot specify the width when creating a device of type display).
