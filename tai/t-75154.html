In Tools you can click on the session and see who made the latest change.
But in general, the tools hold no history other than first version and last change.

When a formfield is added or removed from form used in job then also the input in the job table must be changed likewise.

But again, when someone changes the job-session setting (like used form fields during job execution only) in table ttaad503 there is no history unless you set audit trail on this table. Then also note: audit trail on tools sessions require to be set on tools company...
