Wich fields?

If you miss REFCNTDU and REFCNTD, you will never see these columns in any Baan DD as these are used by Baan administration.

If you see some fields with e.g. #1 at the end this could be an split filed in Oracle because Oracle 7 can only handle columns of 255 length.

If you see fields with $1 at the end, these fields indicate that you use an array field.

If it is none of the above, please describe in more detail what you are missing.
