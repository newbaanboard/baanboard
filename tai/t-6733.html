Hi *ALL,

it is know that the install on an AS400 machine is not the fastest Baan installation way. A copy of an existing environment is possible. The time consumtion is in the way the AS400 is handling files. The creation takes a long time on that machine but it is much faster when working on it.

If you want to make a compleate installation you can use the new batchinstaller for B50_c. This will install service pack 3 till 7 in one step. I am doing such an install at the moment and to do it on Baan4 level (it was available some weeks earlier) it took only half a day for the compleate Batchinstaller (OK, that was on an Unix machine and not on the AS400).

So for B50_c I would suggest to use the Batchinstaller.

I hope this helps,

Garfield ;)
