The fonts by paper type refer to the size of paper possible that a report can print to according to the font that is used by the report.

e.g. if the paper is 8.5 x 11 and the small font is defined as being able to print 132 characters, Baan will accept the device if printed to without warnings. Note this is for DIRECT device printers. When printing to a direct device, the print driver specified for the device is used (which usually is setup in concert with the characters of the page in fonts by paper type).

If you are actually using bwprint to print to a windows printer, then the print driver for the printer in windows is used.

If things are getting cut off, the report you are sending to the printer is outside the capability of the printer for the font selected, so you'll either have to slim down the font size or choose a different paper size or change your report.
