If you use ASCII then you can press ! exclamation mark. This will print the current screen. Then you can select ASCIT to print the screendump to an ASCII file or direct it to a printer.

In BW you can press Alt-Printscreen to make a screendump from the current screen. When pasted in Wordpad you can print it.
