Yes, you can copy a range of device preferences.
With ctrl+w you can copy the range of device preferences to the new user.
In "maintain device preferences" use ctrl+w, select first record , select last record.

At the time of user creation in Maintain User Data, instead of using the "Insert" option, go to a user (whose device preference you wish to allocate to this new user) and choose the "Copy" option. Make the necessary changes in name and menu and save the record. This way the device list gets copied to the new user.
