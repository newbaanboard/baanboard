Creating a parallel env. is the right way. Next step would be to bring your Live and test env. at par (i.e install upto SP4 and other individual patches/solutions which you may have loaded) in the Test Patch env.

Than install SP5 and SP6 in Test Patch env; test for the functionality/tools change and if things are fine; install it in live enviornment.

I dont think there is any short-cut!!
