You have to create a user data template (ttams1100m000) and enable the options and then link the template to the user data (ttaad200m000).

I see you are on BaaN-4 series so check for the correct session codes and do a convert to DD, logout and log back in for the changes to be effective.

In 4, if you want to allow the user to run any program from the File Run, in Maintain User Data just check the "call session by code" box. The convert to runtime & restart that user's bshell session.
