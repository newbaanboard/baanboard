When a service/feature pack is being installed, the system changes the name of the default bshell file (ntbshell.exe for Windows systems). You should make sure that the file ntbshell.exe existes in BSE\bin. If it has been renamed to something else (bshell_inst I think), rename it to ntbshell.exe.
