VERSION(S):
BaanIVc4 SP9
BaanERP 5.0c (also known as Corelli): SP5 or later, or SP4 with "SP3 tools update"

This solution provides the latest update of Worktop.
This update includes new bugfixes and enhancements. Refer to the textfile ReleaseNotes.txt.
This updates does not include language files, documents and Online Help !!!
These components are delivered with the WorkTop CD which contains a full version of WorkTop.
This CD can be ordered via the normal distribution channels.
