Where is your WebUI installed? I found solution for Linux on Infor Support:

Description:

When trying to log in user gets "Error: Connection broken during login" and WebUI server is installed on a Linux OS.


Resolution:

If in the Linux log the following error appears "FAIL: exec per_source_limit from=xx.xx.xx.xx", set this resource as unlimited for rexec in Linux server and then users can connect via WebUI successfully.
