a) Between Client and Application server

Depending on your setup the initial connection is either done with the rexec protocol on Port 512 or with a propertitary baanlogin "protocol", by default on port 7150 (I think). If blogin is used, then the user can configure the port in the client. After initial connection client and application talk in a properitary protocol.

b) Between Application server and database

Depending on your database the vendor specific protocoll.
