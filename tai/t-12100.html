Open the bwprint application from every client PC. Choose Help/Fonttest. Select the font. (ex. small) Right click - Reset to default.

Now it should work fine.

It is allways a good idea to unistall completely the old BW application at the client PC, clean the registry and the directory where BAAN was installed. Then restart the PC and install the new BW application. In this way you prevent the fonts getting mixed up.
