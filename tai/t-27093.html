Just a note for the future, in ERP-LN for Feature Packs, the Installation Wizard will expect the $BSE/lib/install directory to exist to indicate you are installing into an existing environment--otherwise it will think you are doing an install into a New ERP-LN environment. This is documented in the appendix part of the Feature Pack documentation. So bottom line, in LN, you need to keep the $BSE/lib/install directory around :)
