What is a porting set?
A porting set is a set of binaries that are operating system dependent, and is the
layer between the operating system and the application software. Baan regularly
releases a new porting set. A mailfix number recognizes every modification
made in the porting set.
Every porting set has a port number. Initially, all binaries have the same port
number. If a fix is made on a binary before the next porting set is delivered, that
binary will have another, higher, port number.
