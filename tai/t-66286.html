Hi,

your profile says BaanERP (Baan V). If so, then you can use the function tccom.dll4030.read.address.time.zone to find out the time zone for a given address code say Shipment Address(whinh430.stad). The function basically returns "tccom130.tzid" from addresses table. Here the function interface:


function extern boolean tccom.dll4030.read.address.time.zone(
domain tccom.cadr i.address,
ref domain tczone o.time.zone)
{
DllUsage
Expl. : Returns the time zone of an address
Pre : Address must be filled
Post : -
Input : i.address - Address code
Output: o.time.zone - Time Zone
Return: false - record not found
true - record found
EndDllUsage
