Neha,

Refer to the document CD or the webhelp (if you set up the webtop)

Infor ERP LN 6.1 - Pre installation Guide for Windows (U8072H US) This Installation Guide describes the procedures you must carry out immediately before an Infor ERP LN 6.1 installation on a Windows 2003 machine.

Infor ERP LN 6.1 - Pre installation Guide for Unix (U8073F US) This Installation Guide describes procedures you must carry out before an Infor ERP LN installation on a UNIX computer

Infor ERP LN 6.1 - Overall installation Guide (U8204G US) This document is an Installation Guide that describes the procedure to install the entire Infor ERP LN software suite as a single installation.

Infor ERP LN 6.1 - Specific Installation Guide - Updates - Infor ERP LN 6.1 (U8744E US) This document describes the update of an INFOR ERP LN environment with Enterprise Server 8.4 and the update of an environment with Enterprise Server 8.4 and ERP LN 6.1 featurepack 3 in one run.
