Hi,

Yes you can do that. There are multiple ways.

1. Create a New Role. In that role, Provide a condition in Table Data Authorization. Condition will be, No Authorization for table tdpur400 if tdpur400.hdst = Approved. It will not be disabled, but the user will receive a message saying not Authorized.

2. Go to Session -> Specific - > Form Commands. For Approve function, try setting the Authorization Group to Full Authorization. Now, for the users whom you want to restrict the button, just remove their full Authorization for this session.

3. Clone the session and create a new one without the button. Now, provide access to this session to the restricted users.

4. Use Session Wrapper.

5. Use UE DLL.
Hope it helps.
