Please be aware, that starting from Portingset 07.07, Baan does not support the setting ora_collumn_format any more. So anybody still running in the oracle 7 format, must convert some tables by using bdbpre/bdbpost to the oracle 8 format. A hint how this can be done is in R$BSE/lib/ora in the script conv7to8.sh.
