Refer to:

SSA® Enterprise Server - Technical Notes for Porting Set 8.2b

Important note: installation of security files
This note is not applicable for customers on SSA® ERP LN 6.1 SP2.
If you are using SSA® Enterprise Server 8.2 (tt_7.6_a2, tools version that came with SSA® ERP LN 6.1 FP2) and your application version is:
- SSA® ERP LN 6.1 SP0, or
- SSA® ERP LN 6.1 SP1, or
- SSA® BaanERP 5.2a
you need to have installed solution 206321. If you do not have this solution installed you will get licensing problems with the application sessions. If you are using an older version than ssa® Enterpirse Server 8.2 (tt_7.6_a or tt_7.6_a1) and your application version is:
- SSA® ERP LN 6.1 SP0, or
- SSA® ERP LN 6.1 SP1, or
- SSA® BaanERP 5.2a
you need to have installed solution 208211 and run the correction program mentioned before installing porting set 8.2. If you do not have this solution installed there is a change you won’t be able to access the environment again. Solution 208211 also describes a work-around for that situation.
