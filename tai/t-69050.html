Updating to Infor LN 10.4
This section describes points of attention about an upgrade to Infor LN 10.4.
For the complete update procedure, see Infor LN - Specific Installation Guide - Updates (U9497).
The Enterprise Server ISO image contains several solutions on top of the Enterprise Server feature
pack. These solutions are mandatory and installed automatically, and should not be removed.
