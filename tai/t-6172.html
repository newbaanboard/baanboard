Hi,

shortly the session name meaning changed a bit from Baan4 to Baan5. In Baan4 the meaning od a "m" in the session code was "main session" and the "s" was "sub session". For Baan5 it is "m" as multi occ session and "s" as single occ session.

But for the other error (parameter sessions) please check if the mandatory tables are in the $BSE/lib/tabledef6.2 file. So make sure that the convert was successfull.
