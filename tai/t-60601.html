Learner,

There is a solution in Infor365 about upgrading porting sets, please review it: 719596

Basically, you need to look at the porting set technical notes, between your current porting set and your target porting set. You can always find them at the ftp site. For instance, for 7.1d.16 you would go to ftp://ftp.support.baan.com/updates/port/7.1d.16 ; for porting set 8.7a.01 you would go to ftp://ftp.support.baan.com/updates/port/8.7a.01 and download the technical notes pdf.

For 8.6a.03 patch HP aC++ A03.85 is required. IN HPUX, you would check for that requirement with the following command:

swlist -l product 'PH??_*' | grep aC++*

In a test system I have the output was...

PHSS_38140 1.0 aC++ Runtime (IA: A.06.20, PA: A.03.85)

So I should be able to install that porting set.

If you are in doubt check with Infor support.

Regards,

Nestor.
