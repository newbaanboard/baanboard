Yes that is possible.
Make sure you have the latest patches for ttadv6250m000. In our versions, it would not remove a component which is currently checked-out. We got a enhancement request, which was fixed in recent version it generates a output report. You would also need the correct Package VRC authorization etc to make it work properly.

Also be careful who can run this utility, they can wipe out some/all components :(
