Hi,

Use ttiex1280m000 for export
And ttiex1285m000 for import

You can specify the desired reports and sql-queries in ttiex1280, and by importing via ttiex1285m000, you just specify the export file location.

It is recommended to check the content of the export files, by going to the specified folder, and checking the dump.l file; this file contains all the selected components, where
r = report
f = form
p = program script
o = object etc


Good luck!!

En.
