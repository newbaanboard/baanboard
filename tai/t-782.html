I think that the BW uses the default functionality that Windows provides, meaning that the cursors used are the ones provided by Windows.
Try to set at least the blinking interval to the fastest possible to get a more visible one.
Even if you can change the Text cursor, when yo uclick on any insert box/field, Windows uses an internal one, that it's not user configurable, so we are out of luck regarding changing it!
