Hello,

You can make use of pstat().

for example,

procid = -1
procid = pstat( procid, progname, info )

while procid >= 0
procid = pstat( procid, progname, info )
if progname = "tdpur4100m900" then
kill(ids)
endif
ids = procid
endwhile


In above example, you can make use of predefined variable 'parent'. it will return id of your current session.
