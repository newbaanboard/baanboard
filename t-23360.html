Hi

setting up of SMTP Setting can be tricky but this works. It helps when you are 100% sure ablout your connection data. Use an eMail client say "Outlook Express" (Not the Outlook) on your Crystal Server and set up a your Email account so that you can send a test Mail. When you get this to work then you can go ahead and enter the information in CE.

When you have a limited number of licences then you can create generic users in CE: for Example create users "Sales" or "Accounts" instead of users "John Smith" or "Mary Webber". then all sales staff for example can login using the same user name.

Thumbnails: You must also select save thumbnails when publishing (saving) your report to CE.
