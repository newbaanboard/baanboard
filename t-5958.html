If you're using Baan IVc:

If the item is a standard item, and not a custom item, then you can get there by linking TFGLD410.item to TIITM001.item (Item Master). From the there you can use the TIITM001.citg (Item Group) field which can be linked to TCMCS023.citg (Item Groups) for more info if needed.
If it is a custom item then the only way you could tell, from this table, is if the project field (TFGLD410.cprj) is populated.

Hope this helps.
