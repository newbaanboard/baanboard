One of Crystal Enerprise's partner organisations - APOS Systems Inc - is offering a solution called "View Time Security".

It takes advantage of Crystal's processing extensions capability to format activity log transactions to an external database. Using this solution you can obtain statistics on report usage, system activty performance, User ID activty, date/time stamping, etc. Their website is www.apos.on.ca.

I obtained this information from the crystaldecisions knowledge database, so it isn't a personal recommendation as such. There may be other vendors in the UK offering similar solutions.

If you are reporting out of the Crystal Enterprise APS Database, your best bet is to stream out the data (via XML) into another database for reporting. There is an administrative crystal "Plug-in" that will allow you to fully expose the APS database (which has many fields that are all combined into 1 field) and parse out the data.

Once upladed into a new database, there are many vital stats that can be retreived.

Note: Crystal Decisions does not reccomend or support reporting directly off the APS.
