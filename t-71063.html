There is a registry Path in ION Services where there is standard Documents as well as Custom Documents. There you need to register in Custom Documents...

You need to make a Folder with xsd File as well as an xml File where you mention which all verbs are applicable for your BOD.Like Sync,Process and Acknowledge. Then you need to zip the Folder and then import the Zipped File. The name of the Zipped File must be the same as your BOD name. For example, if your BOD Name is MySalesOrder. The Name of the file is MySalesOrder.zip file.
