Applications Analyst

The Applications Analyst position involves reviewing, analyzing, and modifying programming systems including testing, debugging and installing to support an organization's application systems.  This role consults with users to identify current operating procedures and to clarify program objectives. Position may be expected to write documentation to describe program development, logic, coding, and corrections. Also, writes manuals for users to describe installation and operating procedures.   This role requires working knowledge of relational databases and client-server concepts. Relies on experience and judgment to plan and accomplish goals. May lead and direct the work of others. Responsibilities include:• Analysis of systems software and applications, databases and web-based applications• Work with functional areas to build user requirements• Assist in testing of new programs and enhancements to existing applications• Write documentation to describe business purpose and functionality• Participate in user testing, user training and documentation of new applications, programs, software and databases• Develop full project plans including requirements, timelines, milestones and test plans• Advise users on appropriate action and follow-up• Perform some administrative duties to help support applicationsRequirements:• B.S. in Information Systems or B.S.</p>
