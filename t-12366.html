You can use also,

VB
Set BaaNObj = GetObject("", "BaaN4.Application")


So you can launch VB from BaaN, and then use this in VB code with no logging form.

You need some piece of code in your BaaN session like this:

BaaN

long d
d=activate("ttstpoledaem")
set.pgrp(d,d)
|here lines to execute vb code with getobject statement. No
|question for user and password
kill (d)
