I posted earlier about our work with RFID. Just to recap: We have incorporated writing to RFID tags into our existing bar code label printing software within Baan. We have then created applications to run on fixed and handheld RFID readers to interface with Baan and our RMC3 middleware, respectively, to bring data back into Baan. There should be some sort of press release coming in the next few weeks.
