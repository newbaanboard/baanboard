Refer to your other posting -

Error Messages (http://www.baanboard.com/baanboard/showthread.php?p=59300#post59300)

Get Messages from session
SYNTAX
string stpapi.get.mess.code(string session [, ref string err.mesg])
ARGUMENTS
session Name of the session this command is executed on.
err.mesg Text of the message.
DESCRIPTION
This retrieves the messages generated by the indicated session as a result
of an stpapi.* function call. The messages are returned in the opposite
order as generated in the session. Messages generated by the AFS or
4GL-Engine are also returned. These messages are generated when the AFS
is not used correctly by the programmer (for example, wrong order of
function calls).
RETURN VALUES
String that contains code of error message or the empty string if no
error message is found or only the error text is filled.

The 2nd parameter contains the error message.
In your piece of code, it should be contained in - errorMessage
