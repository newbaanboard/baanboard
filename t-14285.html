I saw this problem once before, can not remember where. Some of the session scripts set the default answer for a question in the session code and then stpapi.enum.answer may not work. You can try changing the default answer to that question in Maintain questions to see if that solves your problem. If not I do not know of another solution that does not involve modifying source. Maybe someone else has another solution.
