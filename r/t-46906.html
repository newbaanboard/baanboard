Hi all ,

This is basically a file handling utility developed using BaaN Tools , which makes all common file operations easy through an easy to use Graphical User Interface .
Though only few operations are included in the utility , you may add some more functionality to the existing one . Possible that there might be few bugs but its working perfectly on NT environment . Slight modification can be made for it to work on Unix or possible can make it platform independent .

Features included at present are :

1) Color Indicator to show the file exists or not on entering the filepath
2) Open Remote/local file with your desired application
3) copy option - client2server , server2client etc
4) Delete a remote/local file
