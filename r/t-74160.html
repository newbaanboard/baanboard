I think you're are dumping (printing) the text in the internal coding of LN. I never did access the text-tables for printing.
When writing a text to a file I used text.to.buf() to load the text into an array. The array has to be a array of multibyte strings to hold the codepages of languages like German, Russian and Chinese.

Prior to writing the text into a file it has to converted into a normal (not multibyte string). See the following discussion:

http://www.baanboard.com/baanboard/showthread.php?t=71770&highlight=utf
