For PCS - Projects the integration with Open Plan Professional from Deltek is now available. The link does enable Planners to integrate Activities with Open Plan to set up Multi Project Planning.
Supported either by the Multi Project Function of Open Plan or by the Interface, Planning for all PCS - Project can be done in on Tool. To support the Resource Leveling in OPP all Resources are loaded from Infor and grouped to a department based Resource Structure. Based upon individual Calendars a very detailed, automated scheduling process can be set up.

Similar integration with SFC and Purchase orders can be set up supporting the quality of PCS - Scheduling.

Open Plan is a very sophisticated scheduling solution used in several Infor Client Projects. It is userfriendly and a a very powerful tool.
