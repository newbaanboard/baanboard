First of all, if you are a current Baan customer you can contact support at crm.support@baan.com or at the USA toll free number 800-678-3448.

Also, this should be covered by the standard documentation.
Anyway, username, system id and authkey are unique values given to each user. Each organization sets this up . These values are used to create unique rows when you add rows in the detached database. When attached we use the system id to create unique rows. The empcode and the system id must be unique. These values are also used during dbsync to see if the employee is valid to get rows from the server.
If you create a empcode/systemid/authkey as djones/123/123st for a user Dave Jones while installing then these values should be entered on the server database thru the admin screens -Employee and System Users . In order that this user gets the data he should also be put in a sales territory.

Hope this helps. Again you can contact support if you are a current Baan customer and need more help.
