As far as I know the stpapi is not implemented in the Baan IV b2 tools, however you can easily check if the object for ttstpapihand is present (normally in $BSE/tools/<version>/ottstp). If the object ostpapihand is not present it will not work.

Another possibility would be running the Baan IV c4 tools on your Baan IV b environment. As far as I know this should be possible. We have a customer with Baan IV b doing this.

Hope this helps...
