Hi Henry,

It would be better if you set the page size (depending on the paper you want to print on) at the client m/c using the printer properties of the printer which you want to use. There in the paper size you can specify length and width and this works perfectly. We used to print single label also and printer would simply roll up to that level only. At BaaN report level you can keep -1 in the paper size.
