Open Plan - one of the leading Project Management & Scheduling Tools -
is now linked to several Modules of LN, Baan V & Baan IV.

Open Plan provides Project Planning Data for Moduls such as TP, PCS and SFC.

Recource Scheduling can be performed for Projects or Orders against Skills, Resource - Groups or Single Recources. Using Availabilities of Recources is a vital function in the Scheduling Process.

The Interface is used by several well know Baan Clients

Open Plan provides Multi- Project and Multi- User Functionality.
