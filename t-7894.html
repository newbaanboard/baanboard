I have already found the answer to its question.
it is necessary to use the function S.EXPR$
in this case
val_1 = "10"
val_2 = "val_1"
result of S.EXPR$(expr.compile(val_2) is "10"

A faster alternative:

get.var (http://www.baanboard.com/programmers_manual_baanerp_help_functions_variables_interprocess_transfer_get_var)(pid, val_2, x)
