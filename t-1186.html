Hi Ken,

Baan does look for the next available number.

Unfortunately though, BaaN always checks 25 numbers, then brings up this message and if you press "yes" it checks the next 25 numbers and so on.

So, if you keep pressing yes, it will eventually find the first available number.

Karin

P.S.: That only applies to manually entered SFC orders. PRP and MRP seeks the next available number without bringing up messages. I heard about problems with transferring MRP though and I believe there is a patch available for that.
