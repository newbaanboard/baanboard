If I'm not mistaken, the db_resource will be mostly empty, maybe with the level1/2 entry only (can't reme,ber anymore) but this is inserted by Baan when you select your DB type when install it.

After the installation you should optimise it since the DB driver will be using the default values (if db_resource is empty, it will use the defaults - and not the best ones usually).
