Hi Frank,

It doesn't seem to be a problem of Baan. This message is coming out of NT, and not Baan.

If you type the following at an NT command prompt: net helpmsg 1330
you will get the message that you're receiving when trying to log into Baan. The problem, then, lies in the NT authentication and not in Baan. Please have your network adminstrator check to see if this user can log directly onto the Baan NT server.
