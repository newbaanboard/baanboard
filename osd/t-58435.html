There is no specifics from INFOR on how to do this.

An in place upgrade should resolve should convert your Baan data. If you detach your baan DB from a SQL 2000 system and attach to a SQL 2005 DB this will automatically upgrade your Baan DB to SQL 2005. Of course this method you will also run into permission problems that will need to be manually correct and possible orphaned schemas.
