Baan Tools gives you the option to change password before it is expired.
Once it is expired, you cannot get into Baan to change it. You have to do it manually on the OS. This is standard behavior.

There are options to tweak things around by changing ipc_info to invoke custom shell scripts before bshell is called. You can use the search option to find the already discussed threads.
