If you want to perform some actions connected with baan while using directly AS400, write:
GO BAANLIB/BAANMAIN - Baan menu will appear then you can choose options like STRBAAN, ENDBAAN to start or end baan server, there are more of them there;
There are a lot of CL commands that will allow you to manage users, jobs etc. like
WRKACTJOB - work with active jobs
WRKDSKSTS - work with disk status
WRKSYSSTS - work with system status
GO SAVE - it will display menu with all SAVE options
GO ASSIST - options to manage your system, users and devices
etc.
I will send you a file with CL commands, but be carefull while using them.
You can install Operation Navigator on Win2000 to manage most of the things mentioned above in a graphical way.
