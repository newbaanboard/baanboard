You could use the standard vt100 emulator supplied with CE and connect via a middleware application such as...
www.arcgateway.com (http://www.arcgateway.com)
in my experience, you wouldn't really want to run a full Baan client on a handheld - it becomes too cumbersome especially in a shop floor environment. Best is a streamlined version of the standard session with the minumal inputs required.

Symbol do a nice rugged CE device with a built-in barcode scanner as well as options for GSM/WiFi connections. Other hardware exists where you don't even need CE, simply running vt100 alone. It depends on where you intend to use the device.
