Kereni,

Win2K is supported by Baan at SP2 and above. For an Informix environment, I would not recommend any version below 7.31 (latest TC version). The porting set should be at least 6.1c.05.03 and I would suggest that you try and get on the 6.1c.06.03 or latest version.

You could TRY the Win2K upgrade without the Informix and porting set upgrades, but I would discourage the combination myself and would bite the bullet and do it all at once (hopefully you have a new server to test this on in advance).

Hope this helps,

-- Jim
