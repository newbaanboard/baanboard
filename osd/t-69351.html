Problem:
ORA-1045: user lacks CREATE SESSION privilege; log on denied

Solution:
Do not grant create session using role, grant directly to user:

grant create session to myuser;

Refer to -
https://www.techonthenet.com/oracle/errors/ora01045.php
http://support.esri.com/technical-article/000002254
