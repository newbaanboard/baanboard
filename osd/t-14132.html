No it will not . Because brand file resides in root . That has to be present in the root of new host . Just copying that to another server will not work. You need to get new validation keys from Baan.

Further u need to check whether old porting set is compatible new hardware / OS . If not get a new porting set from Baan.

You also need to check whether underlying database will initialize or not(including it's validation) . If it's Bisam u need to register all Baan tables again.

So check and confirm all these details.
