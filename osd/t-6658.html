Hi Tim,

I am running as a test Baan IVc4 an 50c with ora8_srv6.1 under Oracle 9.0.1/ HP11 with the actual portingset. It seems to work ok, but I had some problems while loading parallel with several bdbpost. Oracle got stuck with 100 % CPU. Only shutting down and starting Oracle solved this problem. Loading with several bdbposts one after the other worked.
My system is a test system with only a few users, so I am not sure if it will work ok with heavy load of an live environment.

I think you should wait for the oficially supported Portingsets to run baan4 on it.
