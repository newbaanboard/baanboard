I don't know the complete procedure.
But there are some new things especially for upgrading to newer service packs:
- Service Pack Upgrade Programs (ttspt1500m000)
- Data Upgrade Engine (ttspt1200m000)
In the 1st session all upgrade scripts are stored that are needed to run before the new SP can be used.
With the 2nd session you can do the upgrade fully automatically.
