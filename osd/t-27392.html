Hi,

i know that we were going to migrate from a PA_RISC platform to an Itanium platform but we're using QKEY and it's not compatible with Itanium. So my experience on this is make sure that all applications running on the PA-RIasc system is compatible with the Itanium.

As for Baan, i was told that i needed to install a patch on my PA-Risc server first, then transfer everything on the Itanium server (except the bin directory). Upgrade my porting set and validate Baan.

Oracle told me that they had no issues with Itanium.
