You need to import the Batch install patches( solution dumps) for tools, da and application into respective package VRC using ttiex1284m000.
Then you need to import the PMC registry.

Once the import is completed, you will see all the solution in "process solutions" and service packs in " Process patches".

Because you are using import patch (ttiex1284m000) to import the batch installer, there won't be any PMC history.
