Hello,

Just in the process of upgrading from IDS 7.31 to 9.40.

IDS 9.40 has been put on a new server running Solaris 9 with Baan 4c4 SP15 the porting set is 6.1.c.07.04

The one thing we did notice about the newer version is that we had to do some work on the extent sizing due to the fact that the indexes were seperated from the table extents. However I suspect that this is a IDS 7 > 9 issue but in the event of IDS performance issues it's always worth checking your extents.

Immediately after we'd imported the Baan data into the new db we too had some horrible performance issues but a UPDATE STATISTICS MEDIUM seemed to clear these up.
