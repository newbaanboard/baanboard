You would have to use something called as a Baan Sizing Tool. This would tell you what configuration best suit your needs. Each Baan module has a weightage when it comes to cpu utilization. Since you are insisting on HP, HP Proliant 370 G5 dual core processor with min 4gb of RAM and ample hard disk storage would be a start...
