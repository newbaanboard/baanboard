You need to verify the service name with the entry in the sqlhosts file. Being a unix guy myself, I assume the $informixdir would be something like c:\informix on windows. You must have a sqlhosts file for the server to work.

sqlhosts example
ol_testbaan my_machine_name turbo

services example
turbo 1526/tcp #ol_testbaan

You really need to think about upgrading. IDS 2000 reached end of support on 30-Jun-2004.
