Hi lindar.

My first idea comes from your hint "after many years ...". Couldn't it be that your printer is really getting old? Sometime the mechanic needs some oil and so on.

My second idea is to save the spool file (unix: disable the printer, copy the d* file from /usr/spool/lp/requests/<your printer>/, enable the printer) to check if it is really(!) reproducible -- turn the printer off for half an hour or so and then print that file again. You could also print that file to another (identical) printer and you could have a look at that file ...
