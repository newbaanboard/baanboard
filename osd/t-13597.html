Baan does support most of the actual 64 bit operating systems including HP-UX 11i. But as there is no need to go to 64 bit for the Baan binaries itself, because they will never need to adress so much memory, that it cannot be done with 32 bit, they will stay 63 bit. But it is no problem to connect from a 32 bit Baan to a 64 bit database.
