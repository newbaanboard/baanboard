You can do a native sql2000 installation.
Immediately after launching setup, (so before any question) go to C:\winnt\temp\xxx and look for msql7 driver file. Replace this file with a more recent driver ( for example msql7_drv.dll from 6.1c07) and setup will recognize sql2000 database as a sql7 database.
Hope this help.
