I was finally able to solve the problem after few weeks of frustration by restarting server and the database. It seems that db2cli.ini which resides in the /$DBINSTHOME/sqllib/cfg need to add some line under [common]. Add the following line

[common]
MultiConnect=3
IgnoreWarnings=1
CloseOpen=1
UseLightOutputSQLDA=1

Save the file and login back to LN. Now i'm able to browse thru the session without that error.
