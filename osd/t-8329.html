All porting sets that support Oracle 8.1 can handle Oracle 9i too.
So 6.1c.05 / 7.1b and higher should work, but I would choose a later one.

Please note that Baan does explicitely NOT support 9.0.1.0 - 9.0.1.2

All current Baan installers for Windows can not handle 9i (as they are based on svrmgr). When new full install media become available this will be solved.
