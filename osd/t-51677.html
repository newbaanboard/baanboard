That would be true if you validated your license as a cluster. you'd have a .brand file on each server and validate each server separately.

licmon6.2 -B should give you a different sequence for each server. and you should see each server listed.

If that's the case don't touch the brand's, if you want to copy one $BSE to another server, just do that and leave the original server's root partition alone.
