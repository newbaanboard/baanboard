This was moved from Tools Admin & Installation since this needs DB knowledge and limited Baan knowledge.

Baan recommends open_cursors = 5000 and db_files = 50 only to be setup (all depends on the ammount of users) and use the medium size for 25 > nr. users <75 and large for nr. users > 75.
Check the $ORACLE_HOME/dbs/init.ora file for other small/medium/large values.

In any case, these are starting values since there are no good values initially, and optimization is an ongoing process for your init<INSTANCE>.ora file...
