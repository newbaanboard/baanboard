Baan has not yet released DB drivers for Oracle 8i or 9i yet. However, customers are using Oracle 8 drivers for Oracle 8i installations. I did a good amount of testing for Baan IVc4 and Baan 5.0c with Oracle 9i using Oracle 8 driver and the latest porting sets. It looks like, Baan is behaving better with Oracle 9i than Oracle 8i. But I have a very controlled environment. I will be soon testing this (Oracle 8 driver with Oracle 9i DB) on a customer site with about 300 G DB on a Unix environment. As soon as I am done, I will post the results.
