Ramiro,

Level2 is available only for SQL Server 2000. It has not been tested with SQL Server 7. (An enthusiastic attempt might show u that level2 works with SQL Server 7 too, but never do that.... it hasn't been tested... means don't use it).

Level2 will be available for BaanERP5.0c and BaanERP5.2a onwards. BaanERP5.2a is GA now. So level2 driver is available already.

More Details:

(level2 is not yet released for BaanERP5.0c. It is under a beta customer program and can be released any time soon!)

BaanERP5.0c - SQL Server 2000 - level1 and level2
SQL Server 7 - level1 only

BaanERP5.2a - SQL Server 2000 - level2 only
