Hi,

Guideline:
1. Install the baan in new server and install latest service porting set.
2. Create the PMC and install the latest service pack in updated std.
3. Now create the VRC structure same as old server.
4. Export data Dictionary of customised object from old sever and import in new server in customises VRC.
5. Now import dump of data through seuential dump from old sever.
6. Now create table from sequential dump in new server.

some problem can be created in some table like Item master due to change in table dict so you can resolve through piping in sequentail dump files.
