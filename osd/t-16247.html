EdHubbard and malutz are both correct about validation going against the MAC address of the NIC on a Windows server. As long as the NIC remains the same, no revalidation will be required.

You will merely need to run Validate Tools Set (ttiex320m000) to recreate the baan.6.x.lic file in the event that you swap in the hot discs. Once that is done, you will be able to start the license service.
