Mahadevan,

If you want to take backup of registery, you can run "regedit" and select Export Registery File from Registery menu.

If you want to make ERD (Emergency Repair Disk), you can run "rdisk/s" by inserting floppy. The repair info. is used to recover a bootable system in case of failure. This utility should not be used as a backup tool.

If you want to make ERD (renamed in 2k & 2003), you have to use Administrative Tools and Backup option.

Hope this helps.
