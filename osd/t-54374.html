In our setup, we use the Informix version in the installation directory.
example: /informix/9.40.fc9

Install the new Informix version /informix/11.10.fc3

Install new csdk in $BSE/informix/11.10.fc3

Shut down old version. Start new version. In Baan database setup, change $INFORMIXDIR to new Informix installation.

Pretty high level overview, but gets you started. If you have more specific questions, let me know.
