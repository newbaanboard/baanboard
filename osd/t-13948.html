When you request a validation key the Baan application checks some data of your hardware like MAC adress of network cards and so on.

Out of this data Baan is generating a unique key for your machine and use this for the generation of the licence key.

I am not 100 % sure but the IP might be part of this and therefore your daemon is not able to start any more. In this case you might have to request a new licence key.
