Has anyone used portingset 6.1c.07.10 with Oracle 8i?

We just received this from Baan support on one of our problems:

Because Oracle has desupported all versions 8i and earlier, SSA no longer develops or tests software for the desupported versions.

With that being said, the fact that Oracle 8i is not listed in the release notes does not necessarily mean that the current porting set (6.1c.07.10) will not work with Oracle 8i. It might work. The fact that Oracle 8i is not listed does mean that the current Baan Oracle database driver has been developed for Oracle 9i and later and may not work with 8i in an optimal way (we do not know for sure, though, since the new drivers are no longer tested against Oracle 8i).

The fact that Oracle 8i is not listed also means there will be no support for any problems encountered in using the latest porting set with an Oracle 8i database.

The last Baan IV porting set to offer full compatibility with Oracle 8i is 6.1c.07.05. That porting set was released in October 2004. Porting sets beginning with 6.1c.07.06 no longer support Oracle 8i.


Just wondering if anyone has already tested this porting set with oracle 8.
