I'm assuming you inserted that NULL outside of 3GL / 4GL, since to my knowledge 3GL / 4GL does not allow working with NULL values in the database.

If that assumption is correct, then you've confused the Porting Set by storing unexpected data in this table, and it gives you a confusing answer. In other words, basically you introduced data corruption...
