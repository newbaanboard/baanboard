Here is a handy script with a short example for Informix Table Activity. this is from an HP-UX, Informix 7.3x environment but I have used it at a number of customer sites.


select tabname,
(isreads + bufreads+ pagreads)as reads,
(iswrites + bufwrites +pagwrites) as writes,lockreqs,
(isreads + bufreads + pagreads
+ iswrites + bufwrites +pagwrites) as tops
from sysmaster:sysptprof
where tabname not like '%abaan'
order by tops desc;
