JamesV
21st February 2002, 04:42
I would wait a little while before doing the upgrade.

On HP-UX the results have been mixed. Some sites upgrade without problems and others when going to 8.1.7 have inconsistent execution times for queries. We are working with 4-6 accounts and Baan support (US) on attempting to isolate what causes these sites to have performance issues.

We have some sites running successfully on 8.1.7 -- just make sure that you test your batch processes extensively before going live to make sure optimizer related code issues are resolved in advance.

-- Jim
