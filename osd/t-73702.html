You should be able to use ttadv4420m000 to get information on the 2 tables for menus. Keep in mind the menu descriptions by package VRC so if you modified the menus you could have the same one in multiple VRC's.

As far ttaad200 and the menu tables there is no relationship that I can think of that would be easy to get. I guess you could say this use has permission to this session and it is on this menu. Depending on how your security was setup each user record could have the menu they see when they log in - so there is that on the user record. What few users we have that use the menu all point to the main menu tccom00000000.

If you are looking to check which menus are used by different users, then you can use the following SQL query -


select distinct a.t_user, a.t_cpac, a.t_cmod, a.t_cmen
from tttaad200000 a, tttadv360000 b
where a.t_cpac = b.t_cpac
and a.t_cmod = b.t_cmod
and a.t_cmen = b.t_cmen
and t_csbn = 1
