Type /usr/baan/shared/bin/BclmServer -k to kill running server and /usr/baan/shared/bin/BclmServer to start this server again (with default settings).

You can see all options with command /usr/baan/shared/bin/BclmServer -h.

/usr/baan/shared/bin is default path and could be different in your case.
