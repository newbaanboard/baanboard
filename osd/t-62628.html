we have migrated from unix to windows before. your question is part of the migration issue. whatever you did through unix script can be converted to windows totally, no problem. you will still have plenty of ammunition for your disposal. solutions includes:
- windows batch script
- vbscript
- blat for emailing, also from windows 2008 you have emailing capability from the server
- task scheduler
- visual studio (SSIS/SSRS/SSAS), comes with SQL server disk already. you just need to learn the technology. also comes with c++, c#, ajax, vb,asp.net ....totally limitless :)

i even converted our faxing,emailing and texting to cellphones to windows. worked like a charm :) good luck.
