Zain,
Backing up LN folders is not a bad idea. However, backing up the db file may not necessarily give you a hassle free restore. You should probably back up the DB2 Database Backup file and not the Database file by itself. If there is a window(timeframe) where in you have users offline, then you could even take a sequential dump..
