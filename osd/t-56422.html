BaanIV_a1 is an extremely old version and it would be recommended to upgrade to BaanIVc4, but this could be a lengthy and painful task.

Yes there are larger organizations using BaanIV with Oracle 11G, but many have migrated to LN already.

You will want to review the technical notes for Porting Set 6.1c.08 which is the first porting set that supports oracle 11G. Also make sure your OS is supported and is up to date as well.
