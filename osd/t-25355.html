Hi
When bshell tries to start a database server and
tbase(_tp)6.1 has not been started up it will return error
510 which means: "Database not on" or "Cannot connect to
the database". In this case, the reason is that it cannot
activate a new user process, and therefore cannot connect
to the database.

Hope this help.
