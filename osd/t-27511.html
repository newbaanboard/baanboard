hi baan98,

I am not sure about the Baan IV, but I did migration of Baan V from windows NT to windows 2000 and now planning to move to windows 2003. I think it will not be so difficult, you just install the basic software and restore application objects and database from the backup. If the directory structure on the current server and on the new server is same then it will as such not require changes in the files.

Yes, you may require to change the server name for database and license services, as it could be different then the current installation. You may use the same Server name for the new intallation, and it will not require to change the host name for database and license service as the current server name, but in that case only one server can be connected to the network at same time.
