Just migrated a customer from HP-UX to linux (redhat 5). However that was with Oracle.

If the kernel is tuned according to the recommendations of Oracle than there are no real issues apart from that we moved to the baanlogin protocol instead of rexec.

For installing a portingset with the installation wizard you should have rexec enabled and have a ftp server installed.
