NOTE3:
13-05-2009
New version of the portingset released, New PA numbers are used, check PA number.
This was needed for a problem in bdbreconfig where only one table was handled even if there are more tables in the list.
18-05-2009
Also new techincal notes made available for this portingset.

NOTE4:
18-05-2009
SQL Server 2008 SP1 has been successfully validated with porting set 6.1c.08. As a consequence, SQL Server 2008 SP1 is supported now with porting set 6.1c.08.

The Installation Wizard for ERP LN uses a Microsoft addon package called DMO (Distributed Management Objects) for an initial installation on SQL Server 2008 SP1. DMO is not part of the SQL Server 2008 SP1 installation, but can be found at the SQL Server 2008 SP1 Installation medium. Search for the .msi file "SQLServer2005_BC.msi". There are two flavours, 32 and 64 bits. To be installed after SQL Server 2008 SP1. Package does not install if the wrong version is taken.
