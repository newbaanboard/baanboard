Can anyone provide details about converting MS Access database into ERP LN format? I have heard about using SCA engine to dump data in LN format and then using LN engine to update data files.

Yes, you have heared the truth. Some remarks:

1. The Tool is called "ERP LN Dataload" and/or "Any2LN" and is used to speed up the migration process of basic data from old (non Infor) ERP Systems to ERP LN.
2. The Access DB has a predefined format. The customer has to fill in his old data into this format.
3. The dump is made via the SCA Engine. The SCA Engine is a Windows program that need to be licensed by Infor.
4. The import to ERP LN is done via the LN Engine.
