As mentioned several times, errors > 1000 are database errors, you get the database error by taking of 1000 from the error number: in this case, Baan error 1006 translates into MSSQL error 6.
I'm not a SQL server expert but from google I managed to find out that it means "Error 6: Specified SQL Sever not found". Maybe it is not running?!
