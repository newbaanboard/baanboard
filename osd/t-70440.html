Hi jeffersyuan.

Running a Baan session from unix/linux command line would require ba6.1. So I guess that will not work at your situation.

But: ttadv5213m000 "Create Runtime Data Dictionary for Tools" will simply (re)write files in your $BSE/tools directory. Maybe you can save that directory (e.g. to a tar file) and thane take a previous version from your backups?

And as a last option, you could check differences and modify some ot the files with an Editor.
