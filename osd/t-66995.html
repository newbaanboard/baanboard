Hi, Ronaldo!

We have run successfully our LN FP7 both with Linux/Oracle and Windows/SQL Server with Vsphere 5.0, with no hiccups at all.
That given, I must say that with the same VM configuration and some tuning, the couple Linux/Oracle runs about 50% faster than Windows with SQL.
But both were very satisfactory for our 60-concurrent user environment, and I'd not hesitate to grow up to 200 or more users in virtual environment.
As Thiago said, pay attention in the portingset and SLM, as the newest versions are required to go virtual.
