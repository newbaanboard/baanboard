Hi Jericho,

Please keep in mind that 7.3.4 is the only Oracle 7 version that is still supported by Baan.

Installing on Oracle 7.1 might succeed using old porting sets (linked with Oracle 7.1 libraries), but more up to date ones (linked with 7.3) might result in errors and data corruption, especially when you do not use US7ASCII. (Has to do with incompatibilities in the Oracle Required Support files).

Using 8.0.3 will result in runtime errors unless you install patch set 2 and several additional patches (fixing bugs in session switching and transaction handling).
Baan support started with 8.0.4 onwards, but some of these old versions are not supported anymore.
