other space consuming parts on file level:

$BSE/lib/TIME.HIS file (could grow to gigabytes)

$BSE/edi/.../store_sent and store_recv files

$BSE/tmp

$BSE/log

check files created by jobs with file printers (option append) (wich always appends new lines to existing files)

check the baanuser directories for space consuming files.
