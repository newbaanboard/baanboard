James
30th April 2002, 17:34
As far as I know, there are no Baan Client versions for PDA's. Having a Baan Client on a PDA would not really be feasible anyway. The program would take up a lot of the limited space on the PDA, the network connection would be v slow for GUI, and the menu/sessions perhaps cumbersome to draw/read.

If you want Baan access from your PDA, a far better option is to use a Terminal program to connect to your Baan server via Telnet, then use the Baan ASCII client ba6.1. You'll get a better idea then of how a Baan GUI client would fair, in terms of response times - if one were ever to be available.
