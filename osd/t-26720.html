The major challenge may not be the change of database, but the change of application server platform. Switching from Triton to SQL Server implies that you are switching from Unix based system to Windows system.

You get to install the Baan system again on the Windows system. Remember to make sure that all the data dict, patches and customization has been ported to the new server, before you migrating the data.
