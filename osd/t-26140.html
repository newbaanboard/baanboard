19th January 2006, 18:19
Ok, there you got the answer; I guess you have not yet created runtime dd. The difference between CREATING runtime dd, and CONVERTING CHANGES, is that creating creates (renews) the runtime dd for the entire selection, and convert changes, only updates the objects that have been changed within Baan, and thus written into the ttadv500,501,502 tables.

One tip though: if you don't know what you're busy with, maybe it is worthwhile hiring a consultant for setting up your Baan environment...
