It would not be recommended to migrate to a windows NT system. You would want to migrate to Windows 2003.

If you do migrate from baan5c to LN you will want to use the SCA engine to migrate the data and any customizations over. To use that though you will need to contact INFOR consultants or a 3rd party contractor to do the migration.
