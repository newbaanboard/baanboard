Hi,

You need to do port forwarding for the port over which LN is deployed.

e.g. 8312

You will need to assign the Static IP to your server and then map your Static IP to your Local IP from your router and its done.

Additionally, for ION, you might need to configure your hosts file to add FQDN against the IP address to make sure you can ping and access the IP address using FQDN.
