Do you want to retrieve data from Baan without using any software?

You may want to look at qptool for dos. This utility allows you run an SQL from a dos command.

It is very handy, and can be automated to easily and quickly fill Excel. You can do much more with this tool to automate processes and create interfaces in almost no time.

qptool for dos works for Baan 4b, 4c, 5b and 5c where Baan is installed on a Unix Server.

Download a trail version. Enjoy
