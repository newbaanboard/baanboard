We are glad to announce a new product

BaanMate

The easy way to get data from the Baan database.
Allows you to extract data from the Baan database using dynamic queries that are built easily with a graphical user-interface.


- It can be configured to work in minutes

- Allows you to Browse Baan tables and fields and make links between tables

- It uses the Baan Data Dictionary. No need to remember table names like tisfc012550 and field names like qcmo and qbfd. System also recognizes links between tables.

To download a fully operational demo, visit our web site at www.nazdaq-it.com
