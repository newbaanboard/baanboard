Some people on this new board already mentioned Croko Reporter. A lot of people will not be familiar with it!
Let me explain very short what it does, and how you can test it yourself:

Croko Reporter is a program that creates a SQL query for the baan/triton database which sends the output to Crystal, Brioquery, Access or Excel. Very easy to use because Croko reads the baan datadictionary and tells you which tablerelations make sense. You drag and drop, and croko builds the sqlstatement. Very quick because it does not use odbc or so. Users are authorized for some reports and can only press a button to see crystal preview screen, printout, an excel screen, or access database or report.
