just a hint. In fp7 you have 3 options, both are creating external data in XML structure. One is to use create "IAR" file in BSE tmp folder, you can open it in CR and filter/change data as you wish. this data file, once created, should be opened manually in CR
Other option is that you implement Microsoft SRSS interface SAOP interface and "route" calls to CR.
third option is I think very old. On extreme portal, you can find some DLLs that were implemented as connectors to CR. AFAIK they are not supported anymore, maybe they still work for your version.
