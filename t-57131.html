There's not yet any integration between Baan (Erp LN) and Infor PM 10. Questions is also what's possible to integrate. In most models the Infor PM 10 structure will be different (more aggregated) than the structures in the underlying ERP's. Infor PM 10 customers will quite often buy and sell companies faster than they are able to integrate into a common ERP.

Data integration is typically solved by using Infor PM 10's strong WEB based dataloader. This dataload is often run by end user's, so it's often preferred to to have an SQL extract producing a CSV file from the ERP, which the local users then grab with the WEB data loader. The local user's are able to maintain their own local mapping from local structure to the corporate structures.
