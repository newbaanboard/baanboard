NAZDAQ launches a new version of “Baan Query Tool”

http://www.nazdaq-it.com/imgs/bqt-27mix.jpg (http://www.nazdaq-it.com/utilities.php?prod=querytool)NAZDAQ launches a new version of “Baan Query Tool”



 
The new version of this Ad-hoc query tool for Baan and ERP Ln is now available for download (Version 2.7). It includes the possibility of adding formulas and formatting output. With Baan Query Tool (http://www.nazdaq-it.com/utilities.php?prod=querytool) you can bypass the Baan limitation of 255 chars in the width of Baan reports and make queries with unlimited number of characters and unlimited number of lines.
 
Baan Query Tool comes with an elegant user interface – and easily handles special Baan data types like text, array and enum.You can download a fully operational version from here (http://www.nazdaq-it.com/utilities.php?prod=querytool).
