The NAZDAQ product is also tied to an outside fax server software - an ASP Fax Server of some sort. With NAZDAQ though you can have a much smaller fax server product, something like FaxMaker and have the NAZDAQ perform the Baan work, just use the fax server to send the faxes and track the status. With Merkur I think they can perform all of the Baan integration on their own, and probably more robust fax services, but for more $$.
