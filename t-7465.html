I don't know if it is exactly the same thing, but my solution was this:

Go on your form(s) for this session, in maintain forms, then check if there is a Short Description, if there is none, put one.

For an unknown reason, when I could not create a dll with ttstpcreatdll, it was because one or nore of the form(s) for the given session had no short description (This solution was given to me by Baan support).

If this does not solve your problem, then it is something else.

Dear all,

It's solved! Turns out that the way to fill the form is slightly different in B5. This is how it should've been filled

Package wh
Module ltc
Session 1500m000
Name of DLL f1500m000 <- B5, won't see "whltc" here

We have B4 sys as well and in B4 we can see the package & module code in the line above, hence more informative.
Thanks gous99 and others for looking into this.
