Probably the biggest change you will notice is in the appearance of the home page. Following changes and additions were done:

the welcome panel
All user and stats information is now displayed in a clear and concise manner in the top block of the forum. The following items will be displayed:
your current avatar (or a default if none selected)
new threads and new posts indicators
all Private Messaging information including the link to your Private Messaging area
time information
the 'new posts' link
some statistics on members, threads, posts and views
the 'newest member' link
the links to 'log out', 'mark all forums read' and 'forum leaders'
the current online users and link to 'who's online?'
categories in separate blocks
The main parts of the forum have been put into separate HTML tables to ease the viewing of them.

moderators/forum leaders display
all moderators/forum leaders are displayed in a neat drop down box instead of flat text. Each entry in the drop down list is a click-through link to the moderator's profile.
The moderator/forum leader drop-down box will also appear on the second level forum display:
e.g.: http://www.baanboard.com/baanboard/forumdisplay.php?s=&forumid=1

forum icons display
each forum now has its own icon to make identification of the forum easier and to enhance the general look of the forum home page.

rounded corners
all forum pages are now displayed within a rounded corner box. The colour of the box changes with your style choice (see below)

change of links layout
the bottom list of links to our other pages etc have been put into a proper HTML table. Also a link to our sponsors page has been added.

When reading threads you will notice a '+' or '-' icon next to name of the poster. Clicking this icon makes it possible to either contract or expand the contents of the post, an operation similar to opening and closing directory trees in file managers.

When a post is collapsed, it will only display either the post title or the first line of the post.

Additionally, there are two more options available at the top of each thread page: expand all and contract all. These options will either expand or contract *all* posts of the thread in one go.

And finally an option is available to view old posts in collapsed state by default. This feature is user selectable, meaning that each user can toggle this option in their User Control Panel:

User CP > Edit Options > Contract posts you already read (Yes/No)

The default is setting is 'No' indicating that all posts will be shown in expanded or normal view.
