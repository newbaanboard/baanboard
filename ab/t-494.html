Hi all,

We installed v1.0 of the Community Bulletin Generator which provides most of you with a report on the threads and posts at the start of each month.

The most important changes are:
choice of HTML or text output: this is a user configurable option. The default will be HTML so please check your User CP setting (http://www.baanboard.com/baanboard/member.php?s=&action=editoptions) before the next bulletin will arrive! If you do not want to receive the bulletin in HTML format then make sure you do:

User CP > Edit Options > Receive Comuunity Bulletin (type) > select 'text'

Password information is not added to the bulletin anymore. Instead you will be presented with a link which enables you to reset your password in case you have forgotten it.
For those who can receive emails in HTML output, I recommended to have the bulletin emailed to you in HTML format as the layout of the email is far better than the normal text one.
