Hi all,

Following the recent release and upgrade to v2.2.4, a security leak has been published that allows potential hackers to retrieve a user's password hash value and consequently gain access to the bulletin board using that user's credentials.

A manual fix has been applied to secure Baanboard.com but if you suspect that your account might have been comprised then please either change your password or alert the administrators.

Thanks to toolswizard for alerting us of the bug
