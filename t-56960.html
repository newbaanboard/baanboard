Hii Sumesh,

Few days before I've done the same things what u want....

U've to write DAL on the maintable.
use the follwing in the DAL script:

while add New record - before.new.object(), after.new.object() (DAL2 only)
while Change any record - before.change.object(), after.change.object() (DAL2 only)
while delete record - before.destroy.object(), after.destroy.object()
while save record - before.save.object(), after.save.object()
After commit transaction hook - after.commit.transaction()
