utilise la commande CHR$()

Example:
CHR$(65) | Represents "A"
CHR$( ASC("A") ) | Represents "A"
CHR$(27) | Represents the escape character

Dans ton cas le caractere '+' a la valeur ASCII 42 => donc CHR$(42)
