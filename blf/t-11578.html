Hallo Anke,da ich Deinen Aufbau bezüglich Adressformate nicht richtig kenne, will ich erstmal ganz allgemein bleiben.
1. Anlegen eines allgemeinen Formatcode
Hier sollte ein Code "Blank" angelegt werden, der eine Standardvorlage ist.
So gilt dieser Code, wenn das System keinen speziellen findet.
2. Länderformatcode
Dieser Formatcode wird in der Länderdatei hinterlegt (Common-Tabellen-Log.)
3. Adressformatcode
Dieser Formatcode wird in der Adresse hinterlegt.
Hier sollte nur ein vom Länderformatcode abweichender Code hinterlegt werden.
(Achtung die Adressen und somit auch Adressformtcodes werden für die jeweils
gültige Rolle ausgelesen - z.Bsp. Rechnungsempfänger bei Fakturierung.)
Falls Du immer noch Probleme hast, kannst Du mir Deine Definition des Format-
codes als Hardcopy schicken. Vielleicht gibt es in der Definition einen kleinen

Fehler.Tschau Heiko
