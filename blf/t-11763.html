Hallöle,
dies (von Jutta erwähnte) Baan Dimensions basiert auf der alte Proloq-Lösung, die via einer eigenen VRC voll in Baan integriert ist. Dabei muß über die Artikelgruppe bestimmt werden, ob die entsprechenden Artikel über "Dimensions" laufen. Im Dimensions können "n" Dimensionen verwaltet werden, es können also Längen, Flächen und Raum-Größen hinterlegt werden. Genau so kann man aber auch z.B. Oberflächengüte, Korrosionsfestigkeit, Farbe etc. als Dimension hinterlegen. Gemanagt werden diese Dimension-Artikel dann in einer Art Chargenverwaltung. Zusätzlich können zur "Überführung" eines Dimension-Artikels in einen anderen Zustand (z.B. 2 neue Breiten durch zerschneiden einer Blehtafel) "Schneideaufträge" verwaltet werden, die wie kleine Produktionsaufträge aussehen.
