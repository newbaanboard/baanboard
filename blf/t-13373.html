Desupport Dates for Oracle Database Versions

Solution description
The following list was compiled from Oracle Metalink and includes the desupport dates for the versions of Oracle as of 06 March 2003:

ECS = Error Correction Support -- EAS = Extended Assistance Support -- EMS = Extended Maintenance Support (Please see Oracle Metalink for definitions of these three types of support)


Oracle 7.3.4 (the terminal release of Oracle 7 series)
ECS - end date 31 Dec 2000
EAS - end date 31 Dec 2003
EMS - end date 31 Dec 2002

Oracle 8.0.4
ECS - end date 31 Mar 2000
EAS - ends 31 Mar 2003
EMS - end date not given, but it is safe to assumed that it has passed

Oracle 8.0.5
ECS - end date 30 June 2000
EAS - end date 30 June 2003
EMS - end date 30 June 2002

Oracle 8.0.6 (8.0.6.3 is terminal release of Oracle 8.0 series) ECS - end date 30 Sep 2001 EAS - end date 30 Sep 2004 EMS - end date 30 sep 2003

Oracle 8.1.7 (8.1.7.4 is the terminal release of the Oracle 8.1 series) ECS - end date 31 Dec 2003 EAS - end date 31 Dec 2006 EMS - end date 31 Dec 2005

Oracle 9.0.1
ECS - end date 31 Dec 2003
EAS - end date 31 Dec 2006
EMS - end date not given

Oracle 9.2
ECS - end date To Be Determined
EAS - end date To Be Determined
EMS - end date To Be Determined

btw.
new installer supports only 8.1.7 or 9.2.0
