>Gibt es eine Möglichkeit nach dem Inhalt von einmaligen Post- oder Lieferadressen zu suchen? Ein Query funktioniert leider nicht.
>Besten Dank im voraus.

>MartinEinmalige VK Liefer- und Rechnungsadressen stehen in der Tabelle tdsls042

mit dem Index auf Auftrag, Position, DatensatzartDatensatzart (ckor) = 0 -> Lieferadresse des Auftrags, der Position

Datensatzart (ckor) = 1 -> Rechnungsadresse des Auftrags, der Position
Query (z.B.):select * (oder definierte Felder)
from tdsls042
where tdsls042.orno >= tdsls042.orno.f and
tdsls042.orno <= tdsls042.orno.t and
tdsls042.ckor >= tdsls042.ckor.f and

tdsls042.ckor
