Nosotros tenemos un producto llamado adaptador de transacciones que te permit conectar a Baan con sistemas externos.

Este producto funciona muy bien en Baan IV o Baan V.

Supongamos que tienes una aplicacion java o web y a travez de ella quieres hacer un llamado a una session Baan para que grabe un registro.

Entonces tu aplicacion llama al adaprtador de transacciones este se comunica con tu aplicaccion via un servidor de funcion utilizando ya sea el DDC o bien con AFS.

Esta transaccion te envia mensajes de error o de okay o bine tambien se puede utilizar para obtener inofrmacion de Baan como consultas.
