Mira, si hay una main table los datos serán cargados. Pero puedes hacer una query extension (vea las funcciones query.extend.*) con una condición que nunca sea true (como una fecha muy antigua o otra cosa). Después puedes crear un button para definir los datos que quieres, cambiar la query extension e usar rebuild.query para el 'refresh'

No conosco otra manera, si alguien conoce puedes compartir aqui, gracias!
