Hallo,

Hierbei ein Beispiel aus einer Session für Texts:

if nbtext = 1 then
id.texte = seq.open(fich.texte, "w") | write
else
id.texte = seq.open(fich.texte, "a") | append
endif
if id.texte < 1 then
message("erreur fichier texte '%s', %",fich.texte,id.texte)
end()
endif
if seq.puts(lutext,id.texte) then
message("erreur fichier texte '%s'",fich.texte)
end()
endif
seq.close(id.texte)
