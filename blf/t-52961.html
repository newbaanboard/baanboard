Hallo Muckimaxe,

das deutet auf ein Problem in der Namesauflösung Eures Netzwerkes hin. Evt. ist der Server NTSRIK00 nicht im DNS eingetragen und der Ping hat sich die IP über einen WINS Server geholt. Oder aber der W2000 Rechner hat gar keinen DNS-Server Eintrag in den Netzwerkeinstellungen.

Probiere mal dieses Tool aus http://support.microsoft.com/kb/830578/de

Und mache das gleiche mit NSLOOKUP. Evt. siehst Du dann schon wo das Problem liegt.

Ich hatte mal einen Fall bei dem DNS und WINS unterschiedliche IP´s für einen Server zurückgemeldet haben. :mad: Mal ging es und mal nicht.
