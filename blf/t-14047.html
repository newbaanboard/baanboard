Bonjour Sylvain.

Utiliise le Menu Tools/Gestionnaire de texte/parametres texte/gestion.

Voici les instructions principales :

1. tttxt1120m000 = Création d'un nouveau type de fenetre.
ex : type fenetre = fenetre_ntp ; type box = ligne; 1ere col = 2 ; 1ere ligne = 3 ; nb col = 80 , nb lig = 23.
2. tttxt1110m000 = Insertion d'un nouveau groupe.
ex : groupe = fenetre_ntp ; fenetre defaut = text (fenetre existante du systeme)
3. tttxt1116m000 = Insertion d'un groupe de texte avec editeur externe
ex : groupe = fenetre_ntp ; Description = Texte notepad
4. tttxt1113m000 = Gestion des groupes de texte par defaut par utilisateur
insertion du groupe 'fenetre_ntp' pour un utilisateur
5. Quitte BaaN et teste.

Bons tests.
