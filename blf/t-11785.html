Hallo,Den Effekt kenn ich auch.

Als Workaround:
Starte die Installation des Service Pack von einem seperaten Rechner, an dem nicht gearbeitet wird.
Meine Beobachtung ist, wenn man an dem Rechner weiter arbeitet, an dem die Installation (import) läuft, so tritt dein Problem auf.
Wenn an dem Rechner während der installation nicht gearbeitet wird, so läuft die Installation problemlos durch.
Zusätzlich hilfreich ist die Solution 80627stnd für den Tools bereich.
