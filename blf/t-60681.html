Hola,

Puedes crear un vrc paralelo, ya que baan te lo permite, lo único que debes tener claro es tu estructura de VRC, es decir tienes un vrc std, luego puedes derivado de este un vrc de actualización y derivado de este un vrc localizacion venezuela y luego de este derivar un vrc para venezuela producción y luego de este un vrc de test, lo único que debes hacer es crear un vrd derivado de las actualizaciones que se llame vrc localizacion colombia y luego derivar un vrc colombia producción y luego de este derivar un vrc colombia test, así tienes dos ambientes totalmente difrentes con vrc para localizaciones para cada país.

Tienes la sesión Copiar VRC ttaad1222m000 (Create New Package combination one step), que te permite realizar lo que necesitas.
