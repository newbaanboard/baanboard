Hola

Una tabla Oracle de un sistema externo a Baan la puedes ver como una tabla de Baan definiendola como una tabla, pero sin crear la tabla.

Luego via procedimientos de Oracle la defines como virtual y la puedes accessar, actualizar, etc, como si fuera una tabla de Baan.
