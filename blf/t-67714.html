Infor ได้มีการ Release Infor ERP LN 10.4 ใน January 2015 มีการ update feature และ functionsหลายส่วนครับ และ มีการ upgrade webtop เป็น webui 10.4 ซึ่งสามารถใช้งานรวมกับ Mingle (standard ได้) แต่ Mingle ตัวเต็ม version จะมีการขายแยกครับ Mingle เป็น software ด้าน Social คล้ายๆกับ facebook แต่สามารถเชื่อมต่อเข้าหน้าจอการทำงานบน Infor LN ได้เป็นอย่างดี

และยังมีส่วนเพิ่มเติมสำหรับลูกค้าที่ต้องการใช้ พวก workflow และ Incontext-BI
Infor ได้ lunch Infor ION ซึ่งจะสามารถเชื่อมต่อกับ Infor ERP LN และทำให้การทำงาน
เป็น workflow ได้ง่ายขึ้น เช่นการ approve purchase order ถ้าใช้ ION จะสามารถ set workflow ได้ และยังทำงานร่วมกัน Mingle ต้ว full version ได้เป็นอย่างดีครับ

หากท่านใดต้องการข้อมูลเพิ่มเติมในส่วนของ Infor Products, Infor ERP, Infor BI.
