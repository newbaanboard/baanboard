Das einfachste wird sein ein eindimensionales array zu nutzen

domain tcmcs.long test(3*13)

und den index einfach zu berechnen

test( 2*13+5 )

Mehrdimensinale Arrays lassen sich so immer auf eindimensionale array umrechnen. Der Compiler macht, glaub ich, auch nichts anderes.
