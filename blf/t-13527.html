Un porting set es un conjunto de archivos/programas (binarios), que son dependientes del Sistema Operativo. Es la capa de software que esta entre el Sistema Operativo y la Aplicacion Baan (sesiones, programas,etc..) . Periodicamente Baan saca nuevas versiones de porting set, arreglando errores y/o optimizando procesos. (siempre adjuntan un archivo indicando las mejoras o arreglos a errores)

Cada porting set tiene un número de porting. Son especificos para un Sistema Operativo determinado.

Pueden ser bajados desde el www.support.baan.com
