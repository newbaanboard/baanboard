Hey,

richtig, man muss die Liefermenge auf 0 setzen. Dann kann man in der Session "Sendungen" die Sendungsposition markieren und anschließend über die Zusatzoptionen "Nicht versendete Positionen zurückgeben". Dadurch wird ein Umlagerungsauftrag erzeugt (Vorsicht, man kann auch mehrere Umlagerungsaufträge generieren). Das System erzeugt einen Umlagerungsauftrag vom Lagerplatz KOM zurück in das Lager. Dann noch den Lagerauftrag entsprechend freigeben und an die Backorder des Verkaufsauftrags denken (entweder bestätigen oder auf 0 setzen).

So haben wir das Problem auch schon einmal gelöst.
