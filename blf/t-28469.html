there are 2 ways:

(1) BWC File:
Create a new bwc-File with the following Command:

- -- -set USER=<username> tiitm0101m000

(2) Worktop
Use Worktop. there you can make hyperlinks or call session from a history.

(3) Desktop:

Use Desktop with all necessary session.
