Primero el rollo en Ingles
"Function Servers (FS) are Baan program objects used for transactional data. Baan introduced them along with Distributed Data Collection (DDC) protocol as a means to access Baan sessions using a function call. DDC protocol passes session arguments to Baan and returns a result code and, if required, error message back to the caller; providing both result and error reporting. Function Servers are fast and reliable, making them an ideal base for customizing Baan functionality and integrating additional technologies into Baan enterprise systems."


Basicamente el DDC es un program script que realiza las mismas funciones que la session natural de Baan pero sin pantalla como medio de captura.

Se pueden tener 2 tipos uno como no DDC que es el que tu requieres y otro como DDC real que es el que se utiliza para comunicar Baan con equipos como pantallas conectadas en forma inalambrica con codigo de barras y Baan, o bien conecar robots con baan o basculas o lo por medio de paginas web o pantallas en Visual Basic hablando a Baan.

Si quieres mas informacion echame un telefonazo y te platico a detalle.
