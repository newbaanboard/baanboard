viel möglich, doch wissen die wenigsten, wie man es nutzt.>Kannst Du mir bitte mitteilen, wie Word als externer Texteditor

>verbunden werden kann Das ist ziemlich einfach: Als User bist Du in den User Data einem
Text Group Template zugewiesen. Diesem Template sind ein oder
mehrere Textgruppen zugeordnet (Default Text Group Template). Sollten
mehrere Textgruppen zugeordnet sein, so wird vor der Eingabe des

Textes nachgefragt mit welcher Textgruppe geschrieben werden soll.Dieses kann unterdrückt werden, in dem einem Text (z. B. für Angebote)
eine bestimmte Textgruppe zugeordnet wird (Default Text Groups by

Text Field Template). Dann wird nur diese Textgruppe herangezogen.In einer Textgruppe kann der Editor für die Textgruppe ausgewählt
werden. Es gibt 4 Möglichkeiten: BaaN-Editor, Multi-Line (Mischung aus
BaaN-Editor und Notepad), Word mit RTF (für Bilderchen), Word ohne

RTF (Nur ASCII-Modus).Solltest Du kein Super-User sein, so brauchst Du übrigens noch eine
Berechtigung für die jeweilige Textgruppe (Text Group Authorization

Template)>...und ob Du schon Erfahrungen damit, bezüglich der

> Speicherproblematik hast? Die Daten werden zentral auf dem Server in der Textdatenbank
abgespeichert, darum sollte man sich wirklich überlegen, ob

man alles als Word-Dokument abspeichern muss.Gruss
