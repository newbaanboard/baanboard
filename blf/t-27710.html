También desde baan con una select se hace bien, p.e.
select tdpur045.*
from tdpur045
where tdpur045._index1 ........
as set with 1 row -----------> sólo toma una fila
order by tdpur045.orno, tdpur045.pono, tdpur045.srnb desc ---> ordena descendente, así sólo encuentra 1 fila de cada posicion y la srnb más alta
