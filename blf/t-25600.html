Dear Matthias,

i have the following information (we use Baan IV)

(1) Change the first icon of the form masks (save&exit) e.g. make a red boarder instead of the blue one by edition the $BSE/gif/browser.gif file.

(2) Use the tool of nazdaqe (baan color changer)
http://www.nazdaq-it.com/de/utilities.php?prod=colorchanger

(3) Baan IV allows you to modify the colour only in the desktop management
e.g. the boarder color of the desktop menu appears in the specified color, but the form itself is still standard.
(ttdskprogman -> define the desktops)
(ttdsk2105m000 -> data per user)

(4) change the folder icon of the menu browser e.g. make a country flag.
Problem: the form itself appears the same.
(ttdsk6220d000 -> Symbol menu.select)

(5) Modify the company description shown in the status line of the browser
(ttaad1100m000 & Convert User Data to RDD (stored in $BSE/libs/user/u<username>)

(4) Baan V allows you to modify the title information.
see this thread: http://www.baanboard.com/baanboard/showthread.php?t=23303&highlight=title+baan

Hope this is helpful for you.
