You cannot get the sources for the "tt" package, its BaaN's propriety code. Yes there are internal tools in BaaN R & D to do that and they are not released to the customers.

Alternatively, you can create new package combination and VRCs. Export all the software components for all packages in one environment and import to the new VRCs/system/environment and then use the ttadv6450m000 to compare those 2 VRCs.
