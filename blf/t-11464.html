VERSION(S):B40_cSITUATION IDENTIFIED IN:
"Reconciliation Payments/Receipts" ( tfcmg2103s000)SITUATION DESCRIPTION:
This session was creating a realized currency loss entry when no currency rate
changes had occurred between processing payment and reconciliation.
SOLUTION DESCRIPTION:
Modified function source tfcmg2000 (Reconciliation Procedure) so that a
rounding action will be performed on the currency difference field
(tfacp200.cdam and tfacr200.cdam) in order to clear out currency difference
amounts that should not be reported due to procedures setup for rounding
amounts. Modifications are marked with defect number 36432-1.
