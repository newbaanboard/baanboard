cheque los siguientes puntos

En el script , debe colocar el usuario que ejecuta la tarea, y ese usuario debe existir en el VRC donde esta la cia desde donde se ejecuta la tarea

Contenido del rc.startjobdm

# If you want to use several companies you can create several users for e.g.
# root who probably is running this script. If so, copy this script and use
# the next line to define the USER variable (<root100> is just an example).
#USER=<root100> ;export USER
USER=bsp ;export USER

Reemplaza la linea USER=<usuario>; export USER

usuario es el nombre del usuario que ejecuta la tarea


Otra cosa puede ser que los parametros en la sesion ttaad5100m000 (Mantener Tareas) el parametro Periodico debe ser SI

Eso son los primeros puntos de chequeo
