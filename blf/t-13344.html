Session:
Starten einer Session bei Anmeldung unter Baan:
in Command Zeile im BWClient "-- Sessioncode" eingeben

TimeOut:
in Command Zeile im BWClient "-- - set session_timeout=10" eingeben.

TimeOut:
Wenn Du möchstest, dass Timeout auf alle User zugreift dann:
erstelle ein File "all" in $BSE/lib/defaults. Der Inhalt sollte sein:
session_timeout:10
