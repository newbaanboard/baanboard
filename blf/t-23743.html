hallo one4you

wie csecgn bereits ausführt ist dies Druckerabhängig. Für Laserdrucker (HP) benötigst Du eine Barcodemodul, welches dann den Befehlssatz zum drehen beinhaltet.
Für ZEBRA Drucker kannst Du die Druckersprache ZPL einsetzen.
Für einen Barcode128 der gedreht angedruckt werden soll kann die folgende Funktion aufgerufen werden:


function string zabra.barcode128( string direction(1),
long hight,
string interpretation.line(1))
{
| direction: N = normal
| R = 90 Grad im Uhrzeigersinn gedreht
| I = 180 Grad gespiegelt
| B = von unten nach oben 270 Grad gedreht
| inter...line N = Keine Klartext unter Barcode
| Y = Klartext unter Barcode
string outputbarcode(16)
outputbarcode = BC &
concat$(",",
direction,
hight,
interpretation.line,
"N",
"N",
"N")
return(outputbarcode)
}

Hallo zusammen,

wir verwenden die Software "Nice Label" in Verbindung mit Zebra-Z4M-Druckern zum Drucken von Barcode-Etiketten direkt aus BaanIV.

Wir erstellen die Etiketten-Vorlagen in grafischer Form in Nice Label. Diese Software stellt einen Trigger zur Verfügung, welcher immer auf Text-Dateien in einem bestimmten Verzeichniss prüft. Aus Baan erstellen wir Textdateien, in denen auch ein Verweis auf die Etikettenvorlage vorhanden ist. NiceLabel fügt die Informationen zusammen und druckt die Etiketten aus.

Der Baan-Anwender gibt in einer Baan-Session also nur den Baan-Auftrag (bzw. die Laufnr.) ein und am Drucker kommen die fertigen Etiketten raus.

Lt. Auskunft unseres Nice-Label-Spezialisten können auch Barcodes vertikal gedruckt werden. Wir verwenden die Lösung in einer reinen Windows-Umgebung, aber es müsste auch unter Unix funktionieren.
