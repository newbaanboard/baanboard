Pour voir quel est le problème :
- il faut ouvrir l'option Dialog qui se trouve dans la barre de tâche
- ensuite dans l'option dialog ==> il faut cliquer sur le bouton Show BW messages windows
à l'interieur de cette fenêtre tu auras un message qui pourra d'éclairer sur l'erreurs (manque un paramètre, .....)

Rmq : en effet cette fenêtre ne s'affiche pas automatiquement.

Par ailleurs, il me semble qu'en fct de la version du client BW, celui ci n'acccepte par forcement les noms longs,
il faudrait peut être essayer
1 - c:\progra~1\crimso~1\cedt.exe dans la session ttadv0110m000
2 - ou bien de mettre ton editeur dans un autre répertoire (ex c:\crimsom)
3 - mettre "c:\program file\crimsom editeur" dans le path pour que cedt.exe soit automatiquement appelable sans passer par le chemin complet

Voilà
