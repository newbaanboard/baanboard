Dear Zechasso,

one technical solution:

Tools -> Database Management -> Print -> Print reference to parent

Enter the table with the "Arbeitsgänge" and the key values (Arbeitsgang).

The report should contain all referenced tables (like Arbeitspläne) where the task is still used.
