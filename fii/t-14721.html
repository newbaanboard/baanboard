Dear Friends u r right? here is the solution from BaaN Support.

SITUATION IDENTIFIED IN:
Maintain Inventory and WIP Transaction Accounts (tdinv8150m000)

SITUATION DESCRIPTION:
How to activate field warehouse for transaction origin production SFC and financial transactions production result,surcharge & operation cost? The field warehouse is activated for other financial transactions except these three.

SOLUTION DESCRIPTION:
It is standard functionality that you cannot use warehouse for production SFC result,surcharge & operation cost.
For these transactions there is no direct link to a warehouse and therefore it is not possible to use warehouse.

For example it is not possible to use warehouse as criteria for operation cost due to the following reason:
The warehouse on the order is where the finished product will be placed. The operations within a production order may not be linked to the same warehouse as the manufactured item, they may be linked to the components. There is not a clear link between the operation costs and the warehouse either. The operation costs are written from hours accounting and they are not linked to a warehouse.

For the combination of Production and Production result the following key elements are available:
- production order data (contains the key element project)

For the combination of Production and Surcharges the following key elements are available:
- production order data (contains the key element project)

For the combination of Production and Operation Costs the following key elements are available:
- production order data (contains the key element project)
- project data (contains the key element project and financial customer group)
- service package data (contains the key element package part and service type)
- employee data (contains the key element employee)
- work center data (contains the key element main work center and work center)
- machine data (contains the key element machine)

For other transactions warehouse can be used as there is a direct link to a warehouse. For example the issue is linked to a specific warehouse (the goods are picked from this warehouse).
