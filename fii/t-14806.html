If distribution module "td" SLS and/or PUR modules are used for sales orders and purchase orders then the sls and pur tables need to be interrogated also.

The definition of open transactions probably needs to be clarified. The previous note from baandude will be entirely correct if the definition is: sales delivered and purchases received but payment not yet received/payed. If it is: sales and purchases in the book but not yet delivered / received then the td tables will need to be considered.
