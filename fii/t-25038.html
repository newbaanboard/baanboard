I have used the Baan 4c4 budget system for many of my clients. You will need to create a budget and decide what accounts and what dimensions you want to have this for. We always used an exchange scheme to upload it. When it is in the system you will see ledger/dimension balances with the name of the budget you select when running the session and it will show both by amount vs budget and % vs budget. There are many reports that use this. I don't have a document but would be happy to do some remote consulting for you in getting this setup.
