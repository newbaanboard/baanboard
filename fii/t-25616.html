Please check in your transactions integration setup that

firstly in session tfgld4126m000 Maintain Transaction types by transaction Origin The transaction that is occuring should have been setup with field Project transaction as YES

and then secondly integration setup is being done in session tipcs8150m000 Maintain Project Transaction Accounts for the transactions occuring.

That should resolve your problem.

We found it:
Maintain Projects (tipcs2101m000) Form 2
- Book Item Costprice Sales > Yes
