Hi there,

Because of reconciliation purposes "integration accounts", so accounts used in the mapping scheme, can no longer be used for manual postings.

More customers complain that this is too strict and a change request is accepted, but will take a while to realize.

In the meantime you can go to session tfgld4595m000 (reconciliation) and click on the transaction which you think is wrong and needs adjustment and through specific you can create a reconciliation correction. You can process then these corrections, which will result in postings.
