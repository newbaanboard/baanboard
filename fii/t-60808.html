After I posted this question I investigated some more
and this what I found:

a. it is not possible to reverse a finalized reconciliation document in a standard way.
b. I checked the option of using a negative Tran, however this option is not accessible at transaction header if the transaction category is payment reconciliation, and reconciled invoices cannot be shown in the sub session.
c. The solution is using a correction program tfcor9919m000 and to recalculate BP balances (tfacp2245m000/tfacr2245m000) after the correction.
The disadvantage is that this session has a warning: [COLOR="Red"]"This program should only be run by an employee of INFOR. If you are running this correction program at own initiative, you accept all responsibility for any damage that might occur. Run tfacp2245m000 / tfacr2245m000 after the updates"
