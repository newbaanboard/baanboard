Hi there,

In FP3 with the new integration studio (ECLIPSE based) an integration was built with E-procurement, but this integration is available for anyone.
For cost invoices it has been tested, but for invoices related to orders not.

I do not know how you can order the .bi files which you can import in ECLIPSE, but I guess you can contact baan support for that. Here an example file for the standard create method on the BDE PurchaseInvoice.
