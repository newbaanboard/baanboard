If you are using a Sales/Revenue Integration the account set up for the debit and credit is in session tipcs8150m000. You can set a Debit/Credit account per Cost Price Component (or no CPC which will apply to all CPC's). This can be done for a combination of Financial Customer Group, Project and/or Financial Supplier Group.
