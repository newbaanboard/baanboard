hi,

You actually can approve the PO even though mapping scheme is not initialize. It is control by the Purchase Parameter, Log Financial Economic transaction. But this can only allow you to approve PO. You need the mapping scheme to be active in order to confirm Warehouse Receipt Order.

Besides that, mapping scheme can't be activated without details. I guess you can try to create a new mapping scheme by using the default account setup to insert a default ledger account code for each of the integration documents.
