Hi
What version of Baan?
Are you running mcr ( multi-currency extension )?
If you are running multi-currency, perhaps there is some 'garbage' in one of the amount fields. This happens to us ( BaanIVc3mcr), we get 'garbage' in one of the unsued amount fields on the tfgld106 table. We only use 2 currencies, and the amount field allows for three currency values. The third currency value will sometimes contain 'garbage' that does not translate with any of our currency rates. We have to clear these values out before we begin any year end closing processes. Maybe this is what is happening for you?
Hope this helps!
