The query extend can give you the reason why an order is not shown.
If you zoom to that order line session, then press CTRL+SHIFT+8. Then go to the tab of the query and see what the query is which is running on that session. By looking at the restrictions in the where clause and comparing that with the data in the tables should give you the reason why.
