Here is a copy of a Baan support case of mine on the subject. The problem is because handles are not released, and if Baan is not completely shut and restarted, eventually the O/s gives up:

"Sort error 6: No more files can be created in this moment - OK"

After clicking on OK the document prints.

The error occurs only once for each user.

The TMP file in {BSE} that stores print files in not even approaching being full.

Help please!

Thanks

VERSION(S):
BaanIV

SITUATION IDENTIFIED IN:
Rexec.exe

SOLUTION DESCRIPTION:
As per discussion and comments the problem with the release of the handles for the rexec.exe will be solved in the next portingset 6.1c.07.06. Release date due is end of January 2005
