If you are running baan5c servicepack 6, you can install the office integration (software and documents are at the servicepack6 cd).

this integrations provide you the possibility to export baaninfo to excel or word and import/modify records into baan from excel.

This integration will work properly if your information is from one baan table.

For example if you want to change the invoicemethod for several relations, you can import the records from baan into excel, make some changes and import the modified records into baan.
Baan will check in the same way when you enter the information manually. You can also add new records.


Maybe you can use this functionality.
