Hi

Dimension is posted based in the integration element setup. In case the in the integration element if you would not have done the setup for the data/logistic element that was used on the trnsaction, system would have posted it without a dimension. For e.g. let us say in the options by integration element you have set for integarion option=warehouse, and then for warehouse 1 you mapped dimension XYZ. The transaction if done on warehouse 2 will be posted but without dimensions. To check this, what you can do is before posting the integration transactions check table tfgld415 and see if the warehouse is captured or not.

Hope this will answer your query. Contact me if you have any doubts
