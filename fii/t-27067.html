One thing you want to consider when you 're-use' an existing transaction type is how it will effect your payment reporting. Payments are associated with a transaction type / bank relation combination. If there are only 2 existing transactions agains the transaction type, it might be OK to re-use. Especially if there has been some time since the last transaction date. (such as a new fiscal period or a new fiscal year since the last transaction date). You could also set up a new bank relation and use the transaction type in a new transaction type / bank relation combination.
