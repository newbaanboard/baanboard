Integration lines are getting created based on the Cost Components linked with the items. Please note that it is cost components and not cost center. In costing module you may please check surcharges by item or item group. For some of the items more surcharges might have defined and for some items less.
It is the standard design of Baan and there is no such set up that reduce the number of lines in integration.
