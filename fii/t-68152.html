1. Set up Integration Elements in session tfgld4520m000, Integration elements are tagged to the Integration Mapping Scheme's Debit and credit Transactions.
2. Set up Transaction Types in session tfgld0111s000, e.g. for Production Orders description would be production integration/ Transaction Category-Journal Voucher
3. Define series for Transaction Type in session tfgld0114m000, same is also tagged to integration mapping scheme.
4. Finally, You have to define mapping (tfgld4500m000)related to 'Transaction Origin - Production' and Financial Transactions stated in the post above.
a. Add. Calc. Office Variance
b. Completion
c. Efficiency Variance
d. Issue
e. Operation Costs
f. Price Variance
g. Receipt
h. Subcontracting
i. WIP Transfer (Issue)
j. WIP Transfer (Receipt)
5. With every credit and debit transaction, you have to associate ledger account in session tfgld4501m000.
