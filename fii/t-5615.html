26th June 2002, 21:58
This message only appears in tfgld4200 script.
If you have this script in U stnd, then:

If your message is this "Invalid dimension", this message only appears in these two scenarios:

1.- The dimension is not defined in the range of dimensions per ledger account. You checked and it is all right.
2.- The integration comes from external package, field tfgld410.tror = 21 (tctror.extern) If this is the case, BaaN checks if this dimension exists and if not writes this error.

No other case for this error message in B40Ustnd
