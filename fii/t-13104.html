Hi,

I have done a Baan5c FST implementation and for sure the customer was not very thrilled.

You will need the crystal reports and Baan native driver 2.0 inorder to access the reports.

THe FST tables in Baan5 are modified drastically and generate the values into the FST tables which again have to be accessed from crystal and the report generated. The only advantage the specific account/dimension mapping is done at the FST level.

The values still have to be selected from crystal like you would have created any other report.

Try to look for the FST documentation in the support website under documentation. They have something in pdf. As as you say it does not open, try to go to some other location server for support. You could also request, open a case for the documentation.


Good luck.
