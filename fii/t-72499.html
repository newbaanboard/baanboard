Hello,

all you need is to create a single file (e.g. CSV) for all 3 mandatory tables. Once you have uploaded/filled the 3 tables you can use LN standard Session tfgld2215m000 which creates a Batch once it is run. You ougth to use the option "Check" upon running this session as it will give you valuable information if anything is wrong (e.g. Account blocked, Dimension not available, etc.).
