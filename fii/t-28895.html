Additional from the previous reply, before start Rebuild transactions, you should process currency revaluation by:
1. Write off currency differences (A/P) - tfacp2240m000
2. Write off currency differences (A/R) - tfacr2250m000
3. Calculate Currency Differences - tfgld5201m000
and then finalized these transactions.
