Hi atyeo5,
this error is typically forced by the following situation:
The ledger account xxx.xxxx has set Dimension 2 as mandatory in the chart of accounts. In Integration the system is not able to match a value for Dimension 2 depending on your setup in options by integration element.
So check the integration option in the Sales Invoice against the setup in options by integration element and Integration Relations by Elements.

Hope this helps,
Andreas
