Hello,
Baan, by default, can copy financial statements between companies at the same server.
If you want to copy a financial statements between companies on diferent servers, you can do a dump from all tffsst tables from one server to other.

The problem with this procedure, is that you need to correct the company number on the other server, and, if the account plan is not the same, you will also need to correct the account plan as wel as dimensions range.

Hope this helps
