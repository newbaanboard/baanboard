Hi,

There are several ways to do it. Like you can use order by clause for date and BP. And use combination of on.change.check() and if changed() to check if month or BP is changed. Now, you can use 2 variables like one for monthly total and other for BP grand total.

When month is same, add the amounts to the monthly total variable. When month changes, make it zero. And same for BP. If BP is same add amounts to grand total. If BP changes, make the grand total zero.

You can either do this way or you can play with the report scripts & layouts. Depends on what suits you.
