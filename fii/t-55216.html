Hi mitesh,
The session you have mentioned (tfgld4201m000) is the Post Integration Transactions to Finance Transactions. If your requirement is only to post the sales related integration transaction, then use the following setting
Transaction Origin (From - To): Sales - Sales
Financial Transaction (From - To): Inventory on order - Receipt from Project

w.r.t the financial transaction, i have chosen the A to Z of options in it as that would guarantee all the possible combination related to sales will get posted. The usual combination (depending on integration setup) would be Sales - Issue, Sales - Revaluation, Sales - Inventory Transfer, Sales - Revenue, Sales - Material Costs & Sales - Sales Invoice.

hth
sk

P.S: Prior to running this i hope you have executed the Process Delivered Sales Orders {tdsls4223m000}

sukesh75
14th March 2009, 13:43
Hi mitesh,
Since you already have a timeout of 10 min in place, i reckon that you already have the setting "session_timeout:10" in either the "BSE\lib\Defaults\all" or the "BSE\lib\user\u<username>" file. Just change this value from 10 to 30 and it should do. If you put this setting in the former file, you set it for all users while in the later file, you control it by user (Ideally suitable to take out the restrictions for admin users by entering a large value).
