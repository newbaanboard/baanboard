Use a dummy bank transaction type.

For the unallocted document a new unallocated document with negative amount and same rate need to be created and directly assigned to the existing una. payment. This will create a balance on the dummy bank interim account.

Now in the same dummy bank transaction type create a normal payment in tfcmg2100s000. This will clear the interim account exactly down to zero. When you check the non finalized trans. you see a small currency difference posting which is fine.

As well I suggest to create an inquiry at Infor Support. It is not a good way of working what i explained but i see no other way.

Let me know your comments.
