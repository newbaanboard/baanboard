SOL 104943, Peter Plooij, 2000-09-06
The sessions tfacp1450m000 and tfacp2421m000 don't print cost invoices if the selection field "Invoices to be printed" is set to "Registered", "Reg./Matched", "Matched", "Approved". This is different from the sessions tfacp2420m000 and tfacp2424m000. In the aging analysis (tfacp2420m000) the cost invoices should always be included, therefore in order to keep the functionality the same in all sessions, cost invoices have to included for the above selections.
