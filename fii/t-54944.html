If a batch is not appearing for finalization and also in the Print non finalized transactions report, if means that it is already selected for finalization in another run number. To know what is the run number associcated to this batch, use ttaad4500 and go to tfgld100 and here find the batch number.At the bottom you will get the run number that is linked to this. Note it. Then go to tfgld109 and find this run number. It should have the status as Free.

Then finalize it.
