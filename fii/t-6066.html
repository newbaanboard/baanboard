hi,

as Steve told thats the right approach. But if you dont find any records in those two sessions, the only way is to resort to the GTM and change the status in table tfgld101 to "Ready" so that the batch would be available for Select Batches for Finalization (tfgld1110m000).

By this you can restart the Finalization process using the above session.
