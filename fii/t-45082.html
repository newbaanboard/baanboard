Mario,

The next billing request number is calculated in the Billing Request session (cisli2100m000) by incrementing the last used number as stored in SLI Parameters (cisli100, field cisli100.brid).

The last used billing request number is displayed on the Last Used Numbers tab of the SLI Parameters session (cisli1500m000) but it is not user maintainable.
