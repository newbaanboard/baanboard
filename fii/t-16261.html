This error occurs when two users are trying to post the integration transactions for the same transaction origin. When first user is posting the second user can not post the same transaction becuase the batch is being used by the first user.

Generally this log is not a problem because the transaction would be posted by the first user. Please check that the transaction has been posted and delete the log from GTM
