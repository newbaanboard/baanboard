Hi Srilal,

This message is returned from the program tfgld1101 when the user does not have authorization to use the first session in the transaction type definition. (tfgld011.ses1).

It is checked using function tt.session.permission which retrieves, from data dictionary, the start and end time that the user has permission to a specified session.

I suppose it is best you show the problem and my answer to a system administrator / programmer.
