As per the error message it means that there is already a posting for the invoice and another line (duplicate) with the same invoice number is trying to post.

Run correction program otdcor0341 in the non update mode to check if the transactions are duplicate. If yes, then run the correction program again with the update mode set to Yes. This will remove the lines and the error will no longer appear.

If its not duplicate then please let me know.
