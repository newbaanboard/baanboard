Dear pfsilva,

Ramsundar is right, but I want to add that there is only 1 code for depreciation available for posting to finance. What to do then?

Create one code as Ramsundar said. Go to tffas0107m000 and create a specific combination of the desired asset in combination with the depreciation code for posting. U can change defaults as you see fit. Look at the field help for a good example. U probably want to depreciate on purchase value.
