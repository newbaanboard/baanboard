Here's a quick overview of the Integration Tables in Baan IVc4:

tfgld410  Integration Transactions
oContains most key finance details
oLedger account and dimensions for all debit lines
tfgld411  Tax Distribution for Integration Transactions

tfgld412  Invoice Details for Integration Transactions
oInvoice details (Payment terms, due date & misc) for invoices from sales orders
tfgld413  Receipts Schedule for Integration Transactions
oACR receipts schedule if applicable for invoices from sales orders
tfgld414  One Time Addresses for Integration Transactions
oOne-time address details for sales and purchases
tfgld415  Additions to Integration Transactions
oFor each transaction, the Item Group and various other Logistics elements that are available for integration mapping to ledger/dimensions.
oThe record in tfgld415 is deleted when the integration transaction is posted.
tfgld416  Non-posted Integration Transactions
oCompany, Trans Origin, Financial Trans, Log Date & Time, seq number and unique reference key for transactions that have not yet been posted.
oRecord is deleted when the transaction is successfully posted.
tfgld417  Posted Integration Transactions  Credit Lines
oThe Ledger account and dimensions, batch number and line number for the credit lines of integration transactions.
oThe table name is "Posted..." but the entry is made when the transaction is logged.
tfgld418  Integration References
oSelected reference information: Trans Origin, Financial Trans, Trans Type, Doc number, order number and position and the integration transaction status.

I don't remember at which version Baan separated the credit lines from tfgld410 and posted them separately to tfgld417.
