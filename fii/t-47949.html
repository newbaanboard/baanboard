GTM can be done for this.

First note down the ANP document number and delete it from tfgld102 batch. Now for the batch status to get corrected, just open the batch and save and exit. The batch will have the correct status - Ready

Now for this ANP, firstly check in the cmg tables if any line is present in tfcmg101,103,110,111,112,113 and delete from there. Next would be tfacp200, if present delete from here also.
