I would agree, it's a big risk to try and duplicate a session such as this.

There are other alternatives. The 'wrapper' tools can allow you to generate code that calls the standard code, allows you to add your own code to it, without modifying or owning the source code.

Baan programmers use concept and call the results a 'Bolt On'. RMCis (my company) puts out the tool called 'RMCgen' (Link to RMCgen page (http://www.rmcis.com/rmcgen2.html)), and Full Scope puts out the tool called 'QKEY', which do similar things, these can be used by developers without source code to add their own logic to sessions (and other things). There is also a free ware version without support, you can find this under the code and utilties forum on these boards, search for the keyword 'rush'.

Good luck whichever method you choose.
