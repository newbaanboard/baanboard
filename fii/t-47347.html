Hi,

Reconciliation group:
Consist of a reconciliation Area (an enumerate) and a reconciliation sub area (a number).
Group of integration transactions which are mapped to the same kind of ledger account. (For example all integration transactions which are mapped to an inventory ledger account belong to reconciliation group Inventory 1)

Reconciliation element:
Logistic information belonging to the financial transaction (For example element code warehouse which is logged during an issue or receipt) Per reconciliation group you can define 5 reconciliation elements.
