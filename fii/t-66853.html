Please try the following steps:
1. find which Integration Document Types (IDT) and Business Objects (BO) have the mapping errors;
2. check your mapping scheme to find which elements you are using to mapping the above IDT;
3. Go to the integration transactions with the mapping errors and find the logged elements;
4. Modify you mapping scheme according to the comparasion of the logged elements via the elements in you mapping scheme.
5. Map the integration transactions again.
