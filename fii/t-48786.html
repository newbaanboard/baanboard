I would use ttaad4134m000 to restrict the transaction type (or range of transaction types that can be selected by user).

example - to prevent transaction type JNL being used:

User abc99
All companies selected
Table tfgld101
Data authorization Read

Cond 1, Sern 1, field ttyp, operator Equal, Value 1 "JNL", Value 2 empty, end of expression.

Don't forget to Special, Convert to Runtime for each One you enter.

Get user to log out and back into Baan. They should get an error "not authorised to insert this record".
