Your currency should exist in both the session Maintain Currencies {tcmcs0102m000} & Maintain Generic Units {ttaad1106m000}.

The Format Code 095 should have your currency listed in session Maintain Formats by Generic Unit {ttaad1107m000}..

The Format Code 095 should be present in both the session Maintain Default Formats by Format Code {ttaad1108m000} & Maintain Formats {ttadv4185m000}.

Check it out..
