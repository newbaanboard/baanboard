First you have to make sure that tcfin0210m000 has been run - check again if that helped. If not, you will have to check if the ledger mapping element for IDT 10001074 is being used for another IDT which concists of different integration elements. If so, you will have to create a new ledger mapping element for IDT 10001074.

Last but not least run tfgld4558m000 and look up IDT 10001074 and see if the WH, vendor etc. are ticked.
