I do not know if you still have problems, but you should check tfgld452 as well for unposted integration transactions as this table is introduced to speed up the closing of periods process.
Maybe using print session tfgld4482m000 could show you the unposted integration transaction.

There are no background processes or calculations done by the system when closing a period, only BOD publishing actions for ION.
