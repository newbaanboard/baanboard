When the Global Confirmation is selected, it will open up session tfacp1237m000. Here you can use the following settings:

Order related Yes No: Only picks up Order related transactions. In combination with "Only orders where D/C in Balance" only 0 balance orders will be "accepted"
Manual/Opening/Reversal/Undefined: Every transaction of this kind withing the given range will be "accepted"
