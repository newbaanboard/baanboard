Jaime,
You must have created a Sales Integration Element in session Maintain Integration Elements {tfgld4120m000). Drill into this Sales Element and check out if you have defined the dimensions properly for each Integration Option you have defined in session Maintain Options By Integration Element {tfgld4121m000}. Chances are you might not have done a mapping (defined a dimension code) for the new line of product you have created.
