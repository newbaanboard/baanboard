If you are using Baan IV, you need to have more then one logistic company, since you explain that there are different item structures needed. If thsi implies different costprices for the same item, you must have multiple logistic companies, since in Baan IV an item can only have one costprice (in f.e. ERP LN one can have multipe costprices).
The number of financial companies will depend on whether you must have multiple balance sheets.
Sales orders, purchase orders, pcs projects, TP projects, warehouses, workcenters can all be linked to a finance company.
You need to give a little more detail on the company req in order to give you the optimal advice
