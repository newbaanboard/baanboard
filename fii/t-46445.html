When someone opens tfacr5210m000 application lock will be set to avoid concurrent usage of the session. The application lock will be released after closing the session. Due to some reason (like unexpected power failure) if the application lock is not deleted, nobody else can use that session.

Remove the application lock set using session "Remove Application Locks" (ttstpdellocks).
Run the selection Again.
