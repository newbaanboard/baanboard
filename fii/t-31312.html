That session looks the following:
1. Integration transactions check status in tfgld418 (in tfgld410 is not important), it doesnÂ’t have to exist transactions for that period in tfgld415 and tfgld416. If exists post them.
2. All Fixed Asset transactions for that period have to be posted, check that in tffas020 table.
3. Check batch status in tfgld100 and tfgld101, check batch with open or ready status
4. Check not finalized transactions in table tfgld102
5. If you are closing the last period in the year ( 12 or 13) all profit & loss accounts should be in zero.
