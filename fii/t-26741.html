Hi Han_Solo,
it also takes me some time to understand the concepts of fam-tables.
The Codes and Descriptions of segment codes are maintained in tffam5510m000. The matching to the max. 8 location segments is maintained in tffam5210m000. The data is in table tffam510.
The Codes and Descriptions of the location codes assigned to the Segment Codes is maintained in tffam5100m000 and the data is hold in tffam500.
Table tffam530 holds all used combinations of location codes, but only by id's.
Last values by key-type is hold by tffam050.
So if you want to know where the descriptions of your location codes are defined try this:
Take lkey from tffam120. Search with that value on tffam530 with index 1.
With the values from loc1 to loc8 you can search on tffam500 for code and description of location code.

Hope this helps,
