As per my understanding, you want to create one cheque using Manual Anticipated procedure. In your post you mentioned about supplier transaction. I hope you are using the correct transaction type for creation of manual anticipated payment. If you are using the correct transaction type, then the supplier transaction can be entered only in the details level. But you can give the cheque number on the header. In session Anticipated Payments (tfcmg2110s000), you can find the field "Check Number" After the field Supplier and Before the field Date of Creation.
