This error is given when the dimension of the original transaction is empty where the dimension option of the ledger account is Mandatory.

This may happen if the dimension option was set to Mandatory, but transactions with empty dimension where still present in the fiscal year.

In order to be able to proceed, the dimension need to be filled via GTM in tfgld106 table for the transactions with empty dimension. Also, it will be required to rebuild the ledger history for the account and dimensions (tfgld3203m000) for the affected fiscal periods (note: if the fiscal periods are finally closed, it will be also required to open them via GTM, and closed them afterwards)
