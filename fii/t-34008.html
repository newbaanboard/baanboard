The field amount cost item is not always used. I assume that you are using Baan 4.

The field was use in this situation. The item on the purchase order that was approved was a cost item. The cost group of the item was set up in the ACP Parameter. In this situation, the user have to add cost for the cost item before he can approve the invoice. The 'add cost' will populate the field you mentioned.
