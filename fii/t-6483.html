good morning sir,

i could understand the scenario. to track the itemgroup on the basis of projects in an PCS environment define integration element options tfgld4121m000 for the particular Integration Element with Dimension 1(say Project) priority 0 or 1 or 2, and for Integration Option 1 "Project (general)" can be defined and for Integration Option 2 "Item group" can be defined and for method field select 'ByElements'. Then by selecting this by clicking(tick) go to special and define the relation for which project,which itemgroup u want to map it to the concerned dimension.

also plz see to it the Maintain Transaction Types by Transaction Origin session tfgld4126m000 for this Integration Element is maintained as debit for the combination PRP Warehouse/Material Cost, Production (SFC)/ Operation Cost, Purchase/Material Cost, PCS/Surcharges, PCS/Actual Costs of Goods Sold.
