There is a Baan Quick Guide (QG1027 - tfgld900Xm000 Series Programs) that can be obtained from the Baan Support web site.

The purpose of these sessions is to correct the company numbers after the tables in one company have been copied to another. Example: If you copy your production company 100 to a test company 900 then you would run the tfgld900x sessions to change the company 100 to 900 in the test company.

This session change the company number in then referentials fields.
In the target company you can change the old company number to the target company number (in referencials fields). If you run tfgld9004m000 you are running 9001, 9002 and 9003.
Example: You run 9001, 9002 and 9003 or run 9004 only.
