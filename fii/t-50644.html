The session found in open entry reports is tfacr2415m000. Normally you run it with Print Differences Only and Print Invoices set to yes (default). The session should be run during a quiet time as a) it may take some time b) you don't want transactions occuring that might impact the results. Also be sure you have the latest solutions installed for the session as there have been some updates.
