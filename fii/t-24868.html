Hi Mialim,

In short, Intergration Element is to capture dimension in the integration transaction. Integration Element need to be specified in Options by Integration Element and we need to assign detail dimension in Integration Relations by Range.

For example:
1. Integration Element :
Create Element: EXP (Expense)
2. Option by Integration Element:
Option: Dimension 1 (Cost Center)
Option: by Work Center
Method:by Range
3. Integration Relations by Range:
Range: Work Center A - Z
Dimension Code: 1 - 99

Dimension define to which work center the transaction is posted to finance.
