Hello,

once you have initialized mapping data you can disable the IDT in the Integration Mapping Scheme (tfgld4573m000). Go to the tab "Document Numbering / Compression" and remove the tag in "Log integration document type". When you first start the session the column my be hidden so you have to make it visible.

Please be aware that this action has to be done with care. You will have to check if removing that IDT will affect the contra reconcilation element (e.g. interims). Provided that you know what you are doing there is no problem with that what I am aware of.
