Hi Thomas,

not sure if you will *need* this. There are a lot of Baan users in Germany that don't have it, and some more who have it but don't use it and some more who have it but use only a part of it. There is some functionality added that some old-fashioned german bookkeepers like to see (like "Journaldruck mit Gegenkonten", oder "Auszifferungen") because they are used to it. But these are implemented "on top" of the Baan way of doing financials and those concepts don't go together nicely. Anyway, there are some helpful features for thing like "Umsatzsteuerverprobung" - even if badly documented. And again some other sessions for debitor/creditor analysis, but they are as unreliable as the standard sessions in some cases.

Hope that helps a bit
Uli
