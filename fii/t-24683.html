Hi there,

not sure whether I fully understood what you are looking for, but if you are interested in watching which updates have been performed in the baan tables, I would advice you to set the environment variable DBS_LOG=0000010.
This makes the bshell log all DBS-actions (of this user) in the file dbs.log (in the home directory of the Baan user). As you will be using the GUI, you must enter the following in the bottom input field:
-- -set DBSLOG=0000010
Be careful to watch your disk space, the dbs.log may get REALLY huge fast.

hth
