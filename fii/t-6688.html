Not sure I understood the question properly, but overheads are recovered in product cost just the same as direct labour -
Set up a new cost component (Cost Type = Operations Cost)
Assign it to the operation via the operation rate code $/ht or whatever.
This is a variable i.e. overhead total depends on processing time etc - you could also consider using surcharges if you wanted to assign $ per unit.
