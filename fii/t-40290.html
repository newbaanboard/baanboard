use these steps .....
1.Post all purchase related integration till a particular period.
2.finalize all purchase batches till that period.
3.Run session "Select values for GRNI Reconciliation(Interim Table) tfacp1235m000. Put the ledger as you told. It seems these sites are new, so put the period /year well covering all transactions pooibly took place. Use "Only Finalized Transactions" and also put flag on for "Print error report"
4. Run session Global Acceptance of GRNI Reconciliaton tfacp1237m000. put year/period covering from the beginning. use order type=purchase. select "all posting" and also put on the flag "only orders with debit and Credit balanced". Put very minimal tolerance, if u think.
then "Accept".
5. Run Report "Print GRNI Reconciliation-Balances tfacp1435m000. this will show u the details of reconciliation with the Interim GL and you will be able to identify the orders which are participating in the reconciliation.
