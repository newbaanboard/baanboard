All US banks have a Routing Number as well as a SWIFT number. They are used as unique identifiers for the bank. Routing numbers are used for transactions within the US while the SWIFT numbers are used for international wire transfers.

In LN, the session 'Bank Branches', allows you to maintain both International Bank ID (which can be used for SWIFT Code) and Domestic Bank ID (which can be used for Routing Number).

You will need to check how Cotel software integrates with LN and if it uses the Domestic Bank ID field of Bank Branches to capture routing number.

If you want to transfer money to US bank then you need routing (or 'ABA') number If you want to transfer money to European bank then you need SWIFT number.
