Hi!

I could locate the problem and is now fixed.

It was in the initial set up. We hve to give the acc.dep figure upto 31/12/01 for the asset in the preceding fiscal year of intial set up(i.e.2002). For the year 02 till the period of intial set up acc.dep is entered. Then the system is calculating correctly for the period 12 in the year 02. In the last period of each year, the system will look at how much should be the total dep amt for the particular asset for the year and deduct the amount that is already depreciated, which would include any accumulated depreciation data recorded in the current year. This is subtracted from the amount and the result is the depreciation for the last period.

Steps:

1. Change the intial set up to 'No" in tffas001 thru GTM (form3).
2. In session tffas0118m000 go to special and select Maintain
Periodical Depreciation and Revaluation results.
3. Enter the dep code and asset and year as 2001.
4. For the period 12 enter the accumulated depreciation amount
which will be as on 31st Dec 01. Do this for all assets whose
date of use is before year 2002.
5. For the year 2002 enter the acc.dep amount till the period of
intialization. Do this for all assets.
6. Change the intial set up to "Yes" in session tffas0118m000.
7. Run session tffas2200m000.
