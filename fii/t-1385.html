Hello,

have a look in tfgld4510m000, search your production order and see on form 4 if all bookings are posted.
If not then you have to set up the integration scheme for it, tfgld4126m000, tdinv8150m000 and if necessary tfgld4121m000.

These session codes are all for Baan IV, in BaanERP it is a little bit different. But follows the same logic.
