There is a workaround . Each financial statement is a Baan report tffst145011??? - last 3 numbers being the financial statement layout code. You can create /edit a report script in this report to get the decription the way u wish.

The problem with the workaround is that whenever u regenerate the layout , ur changes in the report will get lost unless u save the same separately.
