Cost-to-be-specified functionality is not implemented for landed costs. Landed costs related to cost/service items, eventhough the item group has the cost-to-be-specified flag set to yes, are handled in the same way as if this flag was set to no, so normal cost/service items. The only way to get those landed costs to your fixed asset is to add those cost yourself manually in FAM.
You can ask support to create an enhancement request for this.

Regards,
Michiel
