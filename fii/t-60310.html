Apologies for not getting back sooner.
The report writer is not user friendly and there are no menu buttons or help screens.

If I remember well, the report changes you seek are not overly difficult to do, but you do need to be aware of a few things such as the values being called up in the report script and how these are being handled in the background.

If your experience with the report writer is limited I would suggest getting outside help, particularly if this is the only report change you are likely to make.

If you think there are more report changes to come it might be worth the investment in getting some training on report writing. - It will also strengthen your skill base for future employment opportunites.
