A user documentation document can be found in the solution text of solution number 225123, which explains the switching off of IDT's (not recommended).

In short it comes down to this:
The field Log Integration Document Type is initially hidden on the Document/Numbering Compression tab in session Mapping Schemes (tfgld4573m000) and can be put on the form via customize grid and can then be switched off.
