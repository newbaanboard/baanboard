Hello,

I have seen your conversion data setup. First position is used for incoming receipt. We already reserved that position for record type. Date format should be setup properly. In your file as there "/" symbols we need to setup properly. Like Year 1 to 4, Month 6 to 7 etc (5th position is "/", exclude this). for the incoming receipt/outgoing payment you need to enter some text like "i" for incoming receipt and "o" for outgoing payment in conversion data setup. As per my knowledge date, amount and incoming receipt/outgoing payment are mandatory fields for one record. Yes, you are correct one batch will be created for one file we convert.
