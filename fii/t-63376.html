Dear TeddyBear,

I am not sure if this session is available in your version or if this session is the one your looking for:

Period End (tffam8205m000)
Use this session to:
Purge historical data on the basis of the historical retention rules as specified in the FAM Parameter (tffam0100s000) session.
Add a new record to the FAM Parameter (tffam0100s000) session by incrementing the current period by one. If the current period is the last period of the year, the current year is incremented by one and the current period is set to one.
Set the year-to-date amount (YTD) depreciation to zero for all asset books with asset status entered and if the current period is the last period of the year.

Note:
For assets subject to ADR or MACRS group tax reporting, ERP LN does not purge data from the ADR books, even when the retention periods have been exceeded. ERP LN will purge the ADR books for these assets when all the assets in the vintage/group account have been disposed.

Above are the help files from the session itself. You can test in the demo server.
