There is a lock in table ticpr100 Cost Price Calculation Codes. If you have authority to maintain tables get to the command line on a Baan screen and type "ttaad4100" when the tables come up change the Package to "ti", Module to "c", Table to "100", and the company number to whatever company you are working in; you will need to search for the calculation code that is locked (i.e. STD) then change the Cost Price Calculation Active parameter to 2 (No).
