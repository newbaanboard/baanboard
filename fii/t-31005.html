If you change the purchase value with 1 month run and you execute the session tffas2200m000 with the option Recalculate. An reversal transaction with the wrong value and a new transaction with the correct will appear for the period 1.
If you run that session without that option the first register will stay the same but the second period will have a register with the correct value.
