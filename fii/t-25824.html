a) use a standard functionality JVIMPRT Exchange provided by Baan.

Using Excahnge export or your own programs create the 4 files required by JVIMPRT to load the information, (the last file contains text may not be required)

Export the info from company WWW and then load it with The Exchange Import in company YYY.

Attached you will find a document that I wrote like 2 years ago, Sorry it is in Spanish but the screen shots will give you the idea.



b) Order a customized session that will do the following:

This will be a customization that you could create or order someone to develop for you.

This will populate records in 3 new processing tables from company WWW into company YYY.

Use the new tables create the transactions in GL and report error at document label regarding GL accounts, and Dimension or not balanced document.

This new session will simulate that you are typing the info manually and use the same Baan logic.
