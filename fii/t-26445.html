In a dependent MC situation, it is sufficient to have the same reference currency - from the reference currency the transactions are then calculated correctly to all the HC in use - because you only can define exchange rates between reference currency and transaction currencies!

In an indepdent MC situation, all 3 Home currencies of all companies must be exactely the same, because rates must be defined from transaction currencies to ALL the home currencies in use
