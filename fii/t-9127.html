When you use tfgld4121 to assign values to dimensions you have to treat each dimension, with its priorities, independently from the other dimensions !!!
In tfgld4122 and 4123 you can assign values regarding only the Dimension / Priority that you checked in tfgld4121.

Hi Narayan

You can not use the same element twice for the same dimension code. This is because Baan will not be able to identify what amount is posted to each dimension. In the example given by you if diesel is to be debited to Farm, Plant & Admin how can BaaN determine what amount is to be posted to each dimension. If you have a situation that you must do it, I suggest that you include such item in the 'Cost' group under your ACP paramenters.
