bbsmanian,

The c/a checklist goes through the open entries (tfacx200 table) and checks for each record if the corresponding data in gld can be found and is for the same amount. If that returns no errors you may still have transactions on the gld accounts that are not linked to anything in acp/acr, such as journals posted to control accounts (shoot the users and the consultants who set the system up if this is the case).
