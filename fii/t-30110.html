2nd March 2007, 08:30
Just change the account type from P&L to balance sheet and your job is done. When you will do a close year you will see that the closing balance will be brought forward for this account as well.
One point to be noted here is that if you have not closed the previous financial year then for this account Baan will get forward the closing balance. So if you do not want an opening balance for the current year then i suggest that you first close the previous year and then change the account type.

I have tried in my system "BIVc4 with SP21" and the software allows me to change the account type from P&L to balance sheet. However if u happen to change from BS to P&L it will not allow and throw up a message that "account has opening balance can not change"

Also you need not do any rebuild for this kind of a change. Rebuild is necessary only if you change the parent child relation or sublevel. That too running rebuild ledger history itself will help. You need not run session rebuild opening balance history thats 2 time consuming.

Speaking about rebuilding from transactions, there is a new session released quite some time back tfgld3206m000 in comparison to tfgld3203 it is really very fast.
