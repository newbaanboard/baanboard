We had this problem as well. The answer is the settings that you run the report with. By default it will select all dimensions and all accounts, when in normal operation you are only interested in a small subset. You have to go to the selections that you are 'not' interested in and make sure they are excluded from the report (crazy interface - really helpful to the user ... grrrr) by setting them to go from ZZZZZZ - ZZZZZZ.
