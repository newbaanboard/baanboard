I appreciate you are using LN but this might still apply.
If you change the structure of COA in Baan 4c you have to run the session "rebuild ledger history" [session: tfgld3201m000] not only will this recalculate the balances but it will remap the structure.
I've never come across the error when only changing a description but it may be because every time you change something now, it is recognising the structure has been modified but not rebuilt.
