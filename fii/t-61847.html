Please check Infor knowledge base item #1000565, there you will find T-account documentation for the subcontracting with material flow

However, please be warned that if you are (re)using the same item for the subassembly level of multiple "real" items, the finance integration is crap. While logistically there is no issue doing so, the finance integration will either average the WIP of completely different orders (when using MAUC on that subassembly item) or set the WIP to zero (when using FTP).

The whole concept of subcontracting with material flow is something that is only implemented 80%. If you don't care for your finance accounts, you can use it, otherwise it's pretty much useless, unfortunately.
