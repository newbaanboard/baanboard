Try this session which i think is hidden...

tfgld1116m000 - Change Interrupted Batch Status.

Apart from this you could also check for the session
ttstpdellocks - Remove Application Locks
or
tfgld1108m000 - Change Access Mode by Batch

If none of the above works you might want to use General Table Maintenance (ttaad4100) and update the table tfgld100 and tfgld101. But this should be the last option you try...
