You did not specify the version, my reply is regarding B40_c, I do not know whether this is applicable for B40_b. I suppose it does.

If you do not receive any data in tfgld419, verify if parameter "Store Service/Project/Transport Datas" (tfgld450.sstp) in session "Maintain Integration Parameters" (tfgld4150m000) is set on 'yes'.

This parameter is only used to fill the tfgld419 table.
