There are 2 options .

1. Take the output of tgld3402m000 without headers into an ascii file and use the same in ur external program.
2. Use table tfgld201 and tfgld203 as data source in ur external application and maniuplate those tables to get the trial balance figures .

Think 2nd option is more suitable .
