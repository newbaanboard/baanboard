Hi there,

If you are using the dependent currency system, then you are by definition only allowed to change the rate manually between the transaction currency and the reference currency. If the reference currency is not the first home currency, then you first need to rotate to the reference currency and then you will be able to change the direct rate between the transaction currency and the reference currency.
The calculation from the reference currency to the other home currencies is always done with the internal rate type.
