Please find my comments under your steps
1. Create a new warehouse with warehouse type as "consignment (not owned) "
yes, but you should link the this WH with the specific Business Partner (Supplier).
2. Create Purchase Order type for both "consign replenish" and "consign payment"
Yes, this step is correct.
3. Create a Purchase order and receipt stock to consign not owned warehouse
Yes, but use the 'consignment replenish' order type you created in Step 2.
4. Transfer inventory from consign not owned warehouse to own warehouse
no, this step is totally wrong.
5. A consignment payment Purchase order is created and receipt to owned
warehouse
no, this step is totally wrong. It is not necessary to do this step.
