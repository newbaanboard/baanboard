Hi,

Stop maintaining Receipts.
Post all integrationtransaction type Purchase and all integrations Production SFC / Subcontracting.
Finalize the batches.

Select the values using tfacp1235m000.
Then run tfacp1435m000 and tfacp1432m000
Compare the amount on orders in tfacp1435mooo with the total of tfacp1432m000. Compare the total of tfacp1435m000 with the amount on the GRINYA account on the trial balance. If there is a difference take the opening balance of the first year of use into account.
If differences occur between tfacp1435m000 (amount on orders) and tfacp1432m000 run tfacp1436m000. That list is recalculating the order logistically and compare the amount with the selected finance transactions.
If you have differences run otfcor0641 to see what happened. Don't update records with that correctionprogram because it's not stable.
If diffences occur please explain the variables, I think I can help you.
