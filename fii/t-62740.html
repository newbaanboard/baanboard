Hi,

Search the records in tfgld416 table for all type of transaction and log date lying in period range, check Integration Transation Status of found records in tfgld418 table for following cases


Status 'Not for Finance' - Delete record from tfgld416 through GTM.
Status 'Unposted' - Transaction need to be posted to finance through session tfgld4201m000
Status 'Erroneous' - Check error log in tfgld4532m00 and resolve it before posting the trasaction to finance.
Status 'Posted'- Delete record from tfgld416 through GTM.
