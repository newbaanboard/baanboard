Hi,
There is no function "parent supplier" in Baan IVc4 but you can use the factoring company for the same purpose.
The purpose of parent customers is to group (relate) transactions with customers that belong to the same external company (for instance customers belonging to the same MacDonald's head office). Also it can be used to collect from the Parent Customer all invoices that were issued to the child customers, in CMG.
Hope this will help.
