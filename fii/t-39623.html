Anticipated cash documents are not allowed to be reversed - if your transaction is anticipated perhaps the message you are getting is related to this?

You might want to try making sure that the reversing transaction type allows negatives - this is necessary for cash reversals - go to tfgld0511m000

Also there are number of patches released for this session in LN (221764, 221101, 220385, 220396, 219775, 219207, 219266, 219230, 219197, 217918, 216346, 213636, 213837, 213164, 206985) so maybe some of these need to be back ported to Baan 5.2
