follow the following steps for resolving background process error :

1. run the session tfgld1214m000 & check the same.
2. if problem is still, check tfgld102 table thru GTM (ttaad4500). check backround process no. 0 or 99.
3. if background process no. is 99 then delete the error recs thru GTM (ttaad4100).
3. change document status as "Deleted" in tfgld018 table for field "status of document".
4. run session tfgld3206m000 for rebuild ledger/dimesion history for period tfgld3206m000) with op. Bal. = "No" and fiscal / reporting = "Yes".
5. Finalize the batch and check the balances.

This is steps for clear / resolve your problem.
