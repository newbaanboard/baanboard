As other member mention it, make sure that your integration setting is correct. Then you need to follow this process for getting the accurate information. (I am assuming that you are not using "Real time" transaction posting)

1. Run the integration posting at midnight (create a batch job to run at 12.00)
2. Run the inventory valuation report at midnight (create a batch job to run at 12.00)
3. Next day morning clear and post only the error integration records
4. Finalize the entire integration batches.
5. Now compare the GL balance with Logistic report.

If your integration is perfect then the above step will give you a minor difference
