nick.akre
30th October 2015, 13:54
1. Set up Warehousing Order Type for Internal Warehouse Transfer in session whinh0501m000.
2. For transaction Type 'transfer' in order type tag Inbound/outbound procedures to the Order Type.
3. For Inbound Procedures like Receipt and Inspection and outbound procedures like Outbound Advice and shipment, Respective activities has to be defined, if you want you can keep them automatic.
4. Create A transfer Manual Order from Warehousing Order and tag the order type created in step 1,2 and 3.
5. In Shipping data give information from location to location, keeping the warehouse same and Process the order.
