Hello,

The exchange scheme for "jvimport" can be found in solution 236825 for FP5. Release after FP5 have this scheme already.
This Exchange Schem is delivered as an additional file ("tfgldjvimprt.xch") which can be viewed in session "Additional Files" (ttadv2570m000).
This Exchange Scheme can be imported via session "Import Delivered Exchange Schemes" (daxch5204m000):
- Specify Package "tf";the path and name of the XML file that contains the exchange schemes to be imported.
- Specify Module "gld";
- Specity Additional File "jvimprt.xch";
- Specify whether existing exchange schemes must be overwritten.
- Click Import Schemes.
Now, the Exchange Scheme is present in the session "Exchange Schemes" (daxch0501m000) with name "jvimprt".
