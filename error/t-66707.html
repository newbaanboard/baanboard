Hello,

DB2 SQL error: SQLCODE: -1585, SQLSTATE: 54048 (SCI94737)

Technote (troubleshooting)

Problem(Abstract)

DB2 SQL error: SQLCODE: -1585, SQLSTATE: 54048 (SCI94737)

Symptom

com.sterlingcommerce.woodstock.profile.ProfileFinderException: listAllByEntityName: Finder Exception occurred:
SQLState: 54048
Message:? DB2 SQL error: SQLCODE: -1585, SQLSTATE: 54048, SQLERRMC: null
Vendor:? ? -1585

Cause

The DB2 instance did not have all 4 sizes of Temp tablespace defined.

Resolving the problem

To resolve the issue, the DB2 instance must have all 4 sizes of Temp tablespace created: 4k, 8k, 16k, and 32k. In addition, these must be set as System Temp tablespaces, and not as User Temp tablespaces.
