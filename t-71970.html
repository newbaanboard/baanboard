Hi,


If your 3rd party app can host and consume a webservice, LN has the capability of calling that webservice using SOAP methods. Else, you can use BDE's too. Other option could be near real time solution where your application is calling LN DB Stored Procedure in some interval to check the latest values from that specific table you require. Multiple ways, you need to decide which one to use.
