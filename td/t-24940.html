Hi,

use something like the following



function select.marked.occs()
{
long occ.no

for occ.no = 1 to filled.occ |filled.occ is a predefined variable!
if mark.table(occ.no) then
do.occ.without.update(occ.no, do.something)
endif
endfor
}

function do.something()
{
|here you have access to the field values of the
|marked occurence!
....
}


You can also use the function do.occ() which lock the occurence for the duration of the call to our "do.something()"-function above.
