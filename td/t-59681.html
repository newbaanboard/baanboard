As I understood:

Form design is in language 2.

You just want to show italian language data (e.g. label).

2<-6 : Form in language 6 needs entry in field "link to language": 2

(1) Please use the "Translation" module of BaaN IV:

Here you can use "Copy language dependent Software components", source language is "2" (english), target language is "6" (italian), select option "Forms", on form 2 you have to specify the correct form number (from/to), please be carefull otherwise you copy the whole stuff to language 6.

Option "..link to source language ..." shoud be selected. (don't remember the exact label ... it is the first or second option.

The Tranlsation module is a complete Infrastructure to maintain your software in an multilanguage environment. Interesting to see which components are languagedependent.

There is very good documentation (at least for this feature) for the Translation topic with about 100 pages from infor.

(2) Easier and safer way:

Just create the form with the formnumber from language 2 in language 6, save it and then enter "2" in the field "link to language", dump it -> finished. (similar to what mark wrote)
