For creating array, you need to define the array like this -

domain domain_name arr_name(size)

e.g. To create a Date array of size 10 here goes the code -

domain tcdate dates.arr(10)
long i

i = 1
select tdsls400.*
from tdsls400
as set with 10 rows
selectdo
dates.arr(i) = tdsls400.odat
i = i + 1
endselect
