Dear Baanboard,

had the same error.

First thougt was the forms ... changed the forms to others, fitting nearly from other session which worked, error still appears.

The reason was:

"Change key", Session "Maintain available Indices by Session".

(1) deactivating the additional indices
(2) running the session, no error appears
(3) activating the additional indices (like before)
(4) running the session, no error appears anymore.

Hi,

it can also be a programming error. Example:

long array.size
long array.data(1) based
...
alloc.mem(array.data, array.size)

If array.size is 0, you get that error.

Btw. If you compile your script with debugger, you get "... for sym (array.size)" instead of ".. for sym ()".
