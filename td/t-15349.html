Hi,

It is possible to display data from 2 tables in a display session.

In the form use the fields from table 1 only.

In the script, for the section "init.form". select data from table 1 and in selectempty write another select from table2 and then in the selectdo of the sub query assign each field in table1 (that is defined in the form) the value that is selected from table2.

Write the same for "choice.next.set" and for "choice.prev.set" and this should serve the purpose.

Hope this is of help to you. In case of any clarification let me know.

Regards,

To use a Multi Main Table session
A Multi Main Table (MMT) session is a combination of multiple sessions,
each of which has a unique main table.
MMT sessions are especially designed to present data with a typical parentchild
structure, for example:
.. Sales order and the related sales order lines
.. Financial business partner group and the related ledger accounts and
control accounts
.. Work order and the related work order activity lines, work order material
resources, work order other resources, work order measurements, and
work order hours accounting
Although this data is distributed among several main-tables in the system,
the data can all be presented and edited from a single parent/child session.
This enables users to enter data quickly and to maintain a good overview in
their work.
A typical parent-child session has two areas:
.. The upper part contains information and functionality about the parent
.. The lower part contains one or more lists that show data about the various
types of children
1-12 | To get started
Both parent data and child data can be edited inside the session, which
makes the sessions extremely powerful for data entry. In addition, a
parent/child session also offers the user a complete overview of all the
important data related to the object. The following figure shows an example
of a parent/child session:
Figure 1-3 Sample parent-child MMT: Service Order - Lines (tssoc2100m100)
For detailed information on MMT sessions, refer to Multi Main Table
sessions in the SSA Enterprise Server online help.
