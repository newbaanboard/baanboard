As Wiggum said, whenever you are running the convert to runtime, run it for all packages and modules. It is infact recommended to run for entire range, as there can be some references to other tables.

If we don't give the full range, the system cannot convert the chagnes properly.

Also when you have the components are distributed more than one VRC. If you modified something in lower VRC tables, you must run the conver runtime in all higher VRC's too.
