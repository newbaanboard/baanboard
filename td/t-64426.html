Hi Ken,

Some thoughts/remarks for your project:

As you are going to switch hardware, you will very likely have to license the environment again. While the 6.1c.xx release path of the porting set still has the license daemon, the newer releases also support the SLM licensing. SLM can be installed on another server and allows for more flexibility.

About the porting set upgrade, Infor is going to retire the 6.1c.xx release path of the porting set by the end of December 2014 and no fixes will be provided afterwards anymore. Customers are being asked to upgrade to the 8.x release path. As changing hardware and switching to a different release path are both more time consuming tasks, you might want to consider doing this together.

About Informix, I recall hearing that Infor is not bundling the Informix drivers with the newer porting set releases anymore. It is still supported, but you need to ask Infor support for it.

Stephan
