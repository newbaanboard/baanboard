Hi teddybear,

void set.enum.values.for.field( const string field.name,
[ALL_ENUMS_EXCEPT], [ enum_value, ... ] )

ALL_ENUMS_EXCEPT
This optional macro specifies that the succeeding list of enumerate values
represents those values that are to be excluded from the enumerate set.
Otherwise, the list of enumerate values represents the values that are to be
included.

Try giving value for second argument as empty.

Regards,
--Milind
