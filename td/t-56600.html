Hi Rahul,

you have to generate two labels with the same name:


One with Context "Session, Table or Report Description" for prorgam Tab. This is usually a plural name. e.g. "Lines"
One with Context "General Use" for Menu Items. This is usually a singular name. e.g. "Line"


Compile the labels and enter the name of the labels in maintain Sessions last Tab in field "Label of Object". Compile the Session. Now you should be able to have correct menu entries and tab name.
