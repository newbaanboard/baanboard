mark_h
11th October 2006, 16:19
Is this a session or report you developed? You should be able to modify the script so that you know the last line and using something like 2 detail layouts and with the data you send the variable used in the layout output expression. Example detail.1 is the non last line layout and its output expression is type=1, detail.2 is the last line and its output expression is type=2. So what you do in the program script is determine the last line to be printed - lets say you were printing SFC planning (Order, operation). For each new order (using on.change.check) go locate the last operation. Set type to 1 until the operation equals the last operation and then set the type to 2. In the layout for detail.2 you can add your line and bold the layout. There are probably other ways to solve this, but this is what popped into my mind.
