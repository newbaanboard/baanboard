Depending on what you are trying to do on what steps you would take. Are you changing the source script for the report program to pull the new field? If yes, then just add to input fields on the report, put it in a layout and compile it.

If you are going to populate the new field in the report script. Do not add it as an input field. Just declare it at the top of the report script. In the appropriate section (before.field, after.field, etc.) write the query to pull the field. Then add it to the correct layout and compile.

So what exactly is the problem you are having?
