One way that it could be done would be with the Open World (or what ever the name is this month) Connector to Java. There is a java sample engine that comes with the package. You build the XML message in Baan/LN and send it to the bus component that has the Java engine outside of baan to pass it to MQ series. We did this with Baan IV a few years ago.
