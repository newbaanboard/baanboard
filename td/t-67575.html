Hi,

Blank date should have 0 value.

So, you can do this

long cnt
cnt = 0
select tdsls100.*
from tdsls100
where tdsls100.prdt = 0
selectdo
cnt = cnt + 1
endselect

| Not cnt should have total no of records where date is not entered

You can also use count() function in your select.
