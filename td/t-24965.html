okay, here is my brief description.

1. Run ttadv4120m000 Maintain table definitions to give your new table a name (but be careful when using official baan modules such as td, ti, ...) within the correct VRC.

2. Use the Fields and Indices buttons to add these definitions as well.

3. Use the option "Convert to runtime dictionary"!

4. Log out.

5. Once logged in, run GTM (ttaad4100) and open your new table. When you are asked about non existing tables, you can create it here.

-- okay. Now you have a new table and its up to you to use it in baan session, reports and so on.
