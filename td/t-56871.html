Hi,

The default report font is stored in the maintain reports -form 2 (field ttadv330.font)
Every layout can have a different font - maintain report layouts - form 2 (field ttadv331.font)
Also every field can have a font - maintain report layout fields - form 3 (ttadv334.pf)

I hope it helps,
