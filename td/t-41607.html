Hello,

if i understand your request the right way, you'll print the purchase reminders for a specific buyer.

These tables are user in tdpur4403m000 .
table ttccom020
table ttcmcs007
table ttcmcs046
table ttdpur021
table ttdpur040
table ttdpur041
table ttdpur045
table ttipcs020 |#72007-1.n

Add item table in declaration :

table ttiitm001

built a reference to itm001 and declare input fields for buyer:

extern domain tcemno buyer.t, buyer.f

extend your select statement with :

and tiitm001.buyr >= :buyer.f
and tiitm001.buyr <= :buyer.t
