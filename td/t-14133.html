0 means successful completion as per Baan help. And I think even 72 means successful completion and it is equal to the value of argument mode (2nd) . 72 means (64 +8 ) . Just check the interpretation 64 & 8 in Baan help

8 SHELL_MAXWINSIZE The terminal emulator will size its main
window to the maximum number of lines. A new
main window will be created.

64 SHELL_NO_OUTPUT The terminal emulator will not produce any
output and implies the SHELL_NO_MWINDOW
option.

Instead of using shell command u can also use run.prog function . Please check this board for run.prog for complete information on this.
