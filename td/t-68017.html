When printing you need to setup the printer settings by changing the background color to Black and text color to white and save the printer. I guess its not possible to handle it through Baan/LN.

Also, not all printers are able to do white printing. If not then use a little gray.

Also check - this (http://word.tips.net/T000848_Printing_in_White.html) for more info.
