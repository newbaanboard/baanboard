Dear all,
is it possible to set conditions on receiving users/e-mail addreses with DOM device? For example: I need to send a document to specific receivers if it is coming from a specific department. Both information is available on my document.

Best regards,
Alex
JaapJD
22nd August 2017, 09:11
Yes, you can do that. Depending on the exact requirements you can either implement a custom receiver type 'department', or specify a receiver independent destination in your document type. In the condition for the receiver independent destination you can check whether an email has to be sent for the department and the address can be specified as a variable (e.g. #email#), which must be mapped in the report rule.

Using a custom receiver type you can add the operating user in cc. Using a receiver independent destination it will always be a separate email (with the operating user in "to").
