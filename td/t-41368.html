Hi,

You need to do three steps to write to a file.

1) open the file for write using seq.open() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_directory_file_operations_seq_open)
2) write any string to the file using seq.puts() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_directory_file_operations_seq_puts)
3) close the file using seq.close() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_directory_file_operations_seq_close) when you finish writing

Here is a simple example:

long file.ptr
string str.var

str.var = "My line"

file.ptr= seq.open("c:\myfile.txt", "w") | change the path to what ever you want.
seq.puts(str.var, file.ptr) |str.var is a string variable that contains a line that want to write to the file.
seq.close(file.ptr)

You can find more advance function in the following link:
http://www.baanboard.com/programmers_manual_baanerp_help_functions_directory_file_operations_synopsis

I hope it helps,
