there is no need to use db.lock.table() [or for that matter db.bind()! :D ].

use something like this:

table ttdroe750
long ret
...
function refill.tdroe750()
{
| first clear content
db.retry.point()
ret = db.clear.table(ttdroe750, 0)
commit.transaction()
| now fill table with new data!
....
}
