I don't know whether this will work in Baan IV, but in 5.0 I have used the following in a graphical session where I am using an already-declared window called main.win:

| Set window-cursor to hourglass...........
change.object( main.win,
DsNtitle, " ",
DsNsuperiorPointerCursor, DSCAPPSTARTING)
update.object(main.win)



| Do some processing........



| Update the window Caption with new name and reset cursor......
change.object( main.win,
DsNtitle, " Personalkalender f¬År " & a.per.names(1, list.item.1),
DsNsuperiorPointerCursor, DSCDEFAULT)
update.object(main.win)
