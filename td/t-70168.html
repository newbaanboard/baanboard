Hello,
i suggest to add a new table to your system.
(requirement development license)

tccom010.namf is multibyte 30 characters (City 2) with domain tcnamf

Add a new table with field xxyyy999.cuno (tccuno ) and xxyyy.999.mail (domain tcmcs.str50) or (domain tcmcs.str100) - enough characters.
Create a maintain session for this table.
Add this session as choice.user to tccom0110m000.

In all sessionscripts / reportscripts you'll be able to use the fields from the new table.

Avoid to change a domain. Check where domain tcnamf is used.
If tcnamf is used in other tables, avoid to change. There will be side effects in DLL or other functions.

Regards
