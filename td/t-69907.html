Is it possible to login inforLN 10.5 with Worktop ?
- Worktop was an additional tool used for logging in. Now, you don't need that anymore.

- Is it possible to develop customizations wthout LN Studio?. If this question is true, Do we have any problem later if taking this option?
- As already said, you just need to install bw and use BECS to log in and do the development. You need to set the bshell command as -set NO_WORKTOP = 1 for that. But again, its better to learn and start development using LN Studio. It has some good features and greater capabilities.
