I am creating a library in Baan IVc4. Is it possible to have that function return multiple values?

For example:

function extern long getvalue(domain tcmcs.long f.number)
{
value_1 = f.number * 10
value_2 = f.number * 100
return(???)
}

I would like to return both value_1 and value_2. I tried:
return(value_1, value_2)
and
return(value_1)
return(value_2)
but without success. Is this even possible?

It is not possible to return more than one variable. But you can use arguments to pass values back to the calling function. These arguments must be passed by reference.

Example:


function getvalue( domain tcmcs.long f.number,
ref domain tcmcs.long value_1,
ref domain tcmcs.long value_2)
{
value_1 = f.number * 10
value_2 = f.number * 100
}

As explained by JaapJD it is possible to return multiple values from a function .

This is the way i have used it to get data from main table os a particular session.

Example ::
on.main.table(week.qty,week1,smop.week1,pmop.week1)

functions:
function week.qty(domain tcweek tmp.week ,ref domain tiqep1 qty, ref domain tiqep1 qty1 )
{
select tikcl061.smop,tikcl061.pmop
from tikcl061
where tikcl061._index1 = tikcl061.ncmp,:tikcl061.year,:tmp.week,:tikcl061.citg}
selectdo
qty = tikcl061.smop
qty1 = tikcl061.pmop
selectempty
qty = 0
qty1 = 0
endselect
}

No Need to use "return( )"
