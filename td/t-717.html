OK, I never really tried ttyeno. I also show tgyenox as yes/no with a check box. I would think that a tg domain should allow integration.

If that fails, you just might want to add your own tc domain (e.g., tcyn.chk), add it in the session I mentioned, and then run Convert Runtime Data Dictionary. Then once you exited and re-entered Baan, you should be able to use it at will. Of course you could pick an existing domain like tcyesno, but that might have unwanted consequences -- I would not suggest that.
