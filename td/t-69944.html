Hi Kush93,

When we use aggregation function in SSRS with XML type DataSource, it is necessary to convert the data into relevant datatype.

Because by default, the rdl file assumes the binding xml data as string. Thatâ€™s the reason why it shows error when you do the subtotal. So we need to use the expression:

=SUM(CDEC(Fields!Inv_Amnt_USD.Value))

The function CDEC() will convert the xml binding data to decimal type. Then you should be able to get the results you want.

Hope this helps.
