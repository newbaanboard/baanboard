in the program script you can write the following code to restrict the user to do so.

field.fieldname:
before.input:
fieldname = ""

on.input:
select tcmcs002.* from tcmcs002
where tcmcs002.ccur = :ccur
selectdo

your code
selectempty
message("Currency Not Found in System" )
input.again()
endselect
