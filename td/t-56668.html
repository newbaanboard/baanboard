Since some years we are using a GPL tool for this. I'm not 100% sure (its working very well and I just implemented the BaaN part) but I think that it is from artifex (http://artifex.com/index.htm).

Short process information:
We print a Baan report as a normal report for Laserjets (PCL) to file and copy the file to a folder. A collegue of mine created a shell/perl script thats picks up the file, passes them to pcl2ghost and so creates the PDF from a normal PCL datastream. The result is send as an email to the requesting user (or other recipients, depends of some parameters). All components of our solution are free/GPL tools.
