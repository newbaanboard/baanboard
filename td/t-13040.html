I have done a similar thing using the following method:-

1. Print Baan report to file
2. Use windows scripting host (vb script) to automate the generation of a PDF using PDFlib (from www.pdflib.com).
3. Copy the pdf to the client pc using bw client
4. Generate a vb script that will send the email using "outlook redemption" (search google)
5. Copy vb script to client pc using bw client
6. Start vb script on client (using bw client), the vb script creates an outlook mail message and attaches the PDF and emails it.

Quite fiddley to setup but very flexible. It can also be made platform neutral since most of the grunt work is done on the server and PDFlib is available on many platforms.
