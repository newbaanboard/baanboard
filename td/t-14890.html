Ttstp error messages are defined in 'Maintain messages' only .
Only that u can not see it . U can check the same in GTM for
table ttadv450.

These messages are not defined for any particular table or
session. Normally in application programs these messages are
executed for invalid domain or references data entry (domain
error or reference error).U may check the 'maintain domain'
or 'maintain table fields' sessions to see the error message
codes.
