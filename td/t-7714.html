ksthakur,

It is strongly advised to avoid the datatype set. It is not supported in the GUI and sql operations also fail on them. It is better to split it up into separate boolean domains (true/false).

If you really want to use them and find out the values, use the functions bit.or bit.and bit.set (http://www.baanboard.com/programmers_manual_baanerp_help_functions_bit_bit_and_exor_in_inv_or)
