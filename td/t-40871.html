I don't know why it would come accompanied by an audit message but error numbers greater than 999 are DB errors. You need to subtract 1000 to get the actual DB error code.

Get your DBA involved. See if the error below might be the reason.

Oracle error 25153:
25153, 00000, "Temporary Tablespace is Empty"
// *Cause: An attempt was made to use space in a temporary tablespace with no files.
// *Action: Add files to the tablespace using ADD TEMPFILE command.
