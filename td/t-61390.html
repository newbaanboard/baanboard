Easy Way:
There are color pickers that may help you.
Google top pick was: http://www.iconico.com/download.aspx?app=ColorPic&type=free

Not so easy:
With tools I had
- Take a screenshot of the Baan session
- Paste into Paint.NET (http://www.getpaint.net/)
- Use color picker tool and pick the gray color
- Press F8 to show the Colors window
- Press >>More button

Read off the values
R-236
G-233
B-216

The above values on a Baan 5c system using Infor BW 8.7a.8 (Unicode) client on a Windows XP system with Windows XP Theme.
