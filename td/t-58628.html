Hi,

You can use the function tt.session.desc (http://www.baanboard.com/programmers_manual_baanerp_help_functions_runtime_dictionary_information_tt_session_desc) function to get the session description.

If you want to get description of session by querying the tools tables then, by querying ttadv200 you will get value of label code for session description in field ttadv200.zf_clab. This label code does not have package pre concatinated to it. It by default uses the package in which session is present.
