JaapJD
8th May 2017, 10:09
It seems that you need to set a filter on the list. Multiple rows are written by the print session to the XML, you can check that by printing to the designer device and opening $BSE/tmp/ier_<userid>_<report>.xml. By default Infor Reporting suppresses identical rows in a list. Because you changed the data of one row by adding the unit to it, they are not identical anymore, so both are printed.
