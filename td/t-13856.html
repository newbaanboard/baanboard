Please update your profile with Baan version (ie 4 or 5).

If the delete field should always be set based on update setting then you can set delete value and also not allow the user to change the value.


field.delete:
before.input:
if update = tcyesno.yes then
delete = tcyesno.no
attr.input = false | Block input
else
delete = tcyesno.yes
attr.input = false | Block input
endif
