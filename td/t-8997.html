This is how I would do 'Session Help' in BAAN IVc (I would think the same should apply for BAAN ERP):

- Copy the session to the current VRC.
- Click <Text>
- You are asked which type of help you wish to create (Session Help, Technical Note, etc.) Choose Session Help
- Assuming that no help already exists, it will ask if you wish to create text (or copy or select, etc.) Chose <Create>.
- It then asks for a text group. Assuming you have declared a text group 'text' for session and other such help, type in the group 'text'
- It then opens up the text editor and off you go describing what you need.

After you are done saving the help, run the session to create Run Time for Help for just the session. Exit BSHELL, restart and start your session and you should be able to bring up the help.


Hope this helps.
