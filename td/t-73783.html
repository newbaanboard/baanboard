The direct database functions (db.update, db.insert, etc) will bypass DAL/DAL2 functions. You shouldn't use both (and should move to using DAL2 wherever possible). The direct database calls are still there because there are some very specific instances where you may want to use them, but db.x and DAL2 functions are NOT interchangeable.
