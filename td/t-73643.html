Tables
maximum record length is 3072 bytes (for multi-byte columns take a factor 2 into account.
For example, a multi-byte string of 256 characters will be 512 bytes long in a multi-byte environment.)
maximum number of fields is 1024
maximum field length is 3072 bytes
maximum number of indices per table is 20
