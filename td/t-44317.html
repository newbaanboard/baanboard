in your baan client >> click on configuration and then
enter the following text in your command field.

$ ba6.1 -edebug -- -keeplog dbgbdbact

and then login to baan
and do the action what ever you want . once finish go to latest file generated under $BSE/tmp folder and check the latest log file
$BSE/tmp/bshell.xxxxx
and open that file using cat or more command .
you will see the db level operations
