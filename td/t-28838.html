I came across some useful documents published by BaaN-

Document code : U8952A US
Release : SSA® ERP LN 6.1
Document title : Software Component Converter Installation Guide
Publication date : December 05

This document is an Installation Guide that describes the various steps a user must follow to install the SCC tool, which you can use to migrate the
customization built on SSA Baan IVc4 to SSA ERPLN. This Installation Guide contains the following chapters:
Chapter 1, “Introduction,” provides an overview of the SCC tool installation.
Chapter 2, “Installation of SCC,” describes the step-by-step procedure to install the bc components of the SCC modules.
Chapter 3, “Post-Installation Steps,” describes the steps you must follow after you install SCC and the licensing part of SCC tool.

This chapter provides information about the purpose, scope, and intended audience of this Installation Guide. This document describes the process to
install the SCC tool, which you can use to migrate the customizations on SSA Baan IVc4 onwards to an SSA ERPLN environment. This document also
guides you through each step of the installation process.
Purpose
The purpose of this document is to provide the task-oriented methodology of a Software Component Converter (SCC) installation.
Target audience
This document is intended for SSA Baan IVc experts with technical knowledge of the SSA Baan software architecture, who intend to use the
SCC tool solution to migrate the customizations on SSA Baan IVc4 onwards to SSA ERPLN.

The Installation Guide is also intended for the following individuals:
Developers
Architect
Consultant
User profile
For a smooth installation, extensive knowledge of SSA Baan IVc4 and SSA ERPLN environments is required. This knowledge must span both the
technical area of the SSA Baan and SSA ERPLN applications.
Scope
This document highlights the SCC installation process in the SSA ERPLN environment. For more information on the actual conversion process, refer to
the Software Component Converter User’s Guide (U8948A US).
Components
The software is delivered as the “bc” package, in the “SCC” module, on the SSA ERPLN environment.
