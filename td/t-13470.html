I don't know about Baan V but in Baan IV u can use sql.arse, sql.exec etc functions along with 'like' clause to achieve this objective. Check dynamic SQL in this forum,u will get lot of threads relating to this.

So effectively your SQL looks like


select empno,
from emp
where data like "abc"



You can also use shiftl$ and strip$ functions also to trim the data from both side. Baan technical help is quite comrehensive and illustrative for this functions.
