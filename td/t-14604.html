When u do brp.close in the program script , report program sorts the data u sent from the program script. This way the first sent record could be somewhere else when u check in before.layout of detail line .

The report sorts as per the sort fields given by you in the report. Just check to find out the correct detail line where your computed figure is available .

So value calculated in the program script is present in the report but not necessarily the first line .
