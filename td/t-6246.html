Sarah,

You could use the text.to.buf() function to move the text into a two-dimensional array ( http://www.baanboard.com/programmers_manual_baanerp_help_functions_text_fields_text_to_buf ). Then, using a loop and the pos() function you can find where your "FLX" text resides.

I hope this helps you get started.
