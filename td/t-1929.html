Hi Gurmeet
In Fact it is basic to do it as if you don't know P.L.
Firstly Select Layouts from maintain report section.
Secondly, press add layout button and select after.report layout
after that select all layout and edit all.
In part of after report layout, add the same field you want to sum.And double click this field and Change aggregate Function value to TOTAL .
Lastly Compile you report . That's all !
