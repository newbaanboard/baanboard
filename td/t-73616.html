Hi,

as Dittmar says the problem ist that you assign empty string"" to an enum variable and this is not allowed as eneums are basically numeric!

I suggest you use the predefined value empty which is basically zero as follows:

declaration:
table twhinh200
domain whinh.hsta status

detail.1:
before.layout:
select whinh200.*
from whinh200
where whinh200.orno = :tcitr300.oboi
selectdo
status = whinh200.hsta
selectempty
status = empty
endselect
