Hi,


Use len() function to calculate total length and then taken last 3 using something like this -



long ls

string seqstr(100)

seqstr = "LONDON"

ls = len(seqstr)

seqstr = seqstr(ls-2;3) | seqstr now stores the last 3 characters of the string which is "DON"
