I build simular correction programs in my time at Baan Support. These were build only to run as an object. They were created as 3GL program without standard program. So no session or form object.

Therefor they can only be runned realtime and not in a job.

However often the scripts of these correctionprograms are present on your system. If you know how to modify them you could turn them into a real Baan session with session and form object. Then of course you could run it in a job.

Hopefully this helps,
