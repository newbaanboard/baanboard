So..you want to see the order it calls everything, printed out? I don't think directly the C option can be sent to output.

One suggestion to see the straight line version of your 4GL program, if you have the source, is to run it through the DDC Function Server Generator (tuddc9201m000). This will put the program in straight line, and create a function for the insert, modify and delete, and sometimes other actions the session. There is some ddc overhead to weed through, but it's handy as a printout for the straight line logic.

Hope that helps.
