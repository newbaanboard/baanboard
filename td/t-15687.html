Hello Ralf,

I had the same problem.
I guess your Source code looks like this:

#include <bic_audlib>
#include <bic_audhdr.h>

Then you get all these errors.


The only thing you have to do is change the order of the includes:

#include <bic_audhdr.h>
#include <bic_audlib>
