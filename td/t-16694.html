HI,

I am talking about the session "Customizing forms" "ttadv3500m000". In this session, select your form and open as "Edit/View Form". Then in menu "Edit" select "Special Field". Then Zoom to define a command, and insert a new command in session "Form Command" (ttadv3518m000). Here you can define a label for your button.

Yes, in session Form sepcial field (ttadv3103s000) command length should reamains 0 for the default lenght.

Try this, if any question then ask.

In ttadv3111s000(Spec. Options) is where you describe the choice.user options. The descriptions are used to determine what is on the screen for the button. Then use ttadv3114s000 to place the button on the form. Or if you want you can edit the form and insert a button that way - in this case the button can be anywhere on the screen.
