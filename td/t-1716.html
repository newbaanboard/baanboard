Hi,

It is possible to extract text using exchange. You need to set the text up as a type of multiline text. Your output will look something like this:

N010000006
{\{txta 2
This is the text that will be entered in to the tables
}\}

where txta is the text field name and 2 is the language. You can read it in the same way.

This was done on Baan 5.
