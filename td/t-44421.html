You probably mean to display a total line in a multi-occ display session.

You set the predefined variable fattr.total.line to true in the before.form or before.program section. Then you calculate the total fields yourself in the program, and you use function display.total.fields() to display the total fields.

Syntax for the function:
display.total.fields( string fieldname1, long value1 [, string fieldname2, long value2 ] ... )
