I am not sure what the real question is? So I will give this a shot - You could use dir commands(dir.open,dir.entry, etc.) to read through the directory and put file names onto a form. I guess you could put as many file names on a form as possible. Or if you are looking for some like a file browser for the server then see this post (http://www.baanboard.com/baanboard/showthread.php?s=&threadid=1844&highlight=file+browser).

Another way to handle this is to create a temporary table to hold the file names. Then at the start of the program you could just use the dir commands to populate the table. Then a normal Baan display session could be used to display the file names. Lots of possibilities.

Good Luck!
