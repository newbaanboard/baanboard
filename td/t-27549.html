Hi,

First you will have to open the file using seq.open() for write:
http://www.baanboard.com/programmers_manual_baanerp_help_functions_directory_file_operations_seq_open

Then write what you want on the file using seq.puts():
http://www.baanboard.com/programmers_manual_baanerp_help_functions_directory_file_operations_seq_puts

After that close the file using seq.close():
http://www.baanboard.com/programmers_manual_baanerp_help_functions_directory_file_operations_seq_close

Finally copy the file to the local using server2client():
http://www.baanboard.com/programmers_manual_baanerp_help_functions_client_file_access_server2client

There are other functions that open a file on the local for writing and write on it, but I don't trust them!

I hope it helps ;)
