I assume you mean Start Debugger from the Extensions session (ttext1500m000). To do this, select the report extension and Start Debugger. Set a breakpoint where you want the extension to suspend and start the session to print the report.
Few notes:
1. Ensure the print session is started after you started the Extensions session, because it must run in the Activity context of the extension.
2. Ensure that at least KB https://www.inforxtreme.com/espublic/EN/AnswerLinkDotNet/SoHo/Solutions/SoHoViewSolution.aspx?SolutionID=1897326 has been applied.
