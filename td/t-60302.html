There is no mechanism to checkout/checkin in Baan4 b2. For that you need at least BaanERP 5.0 or ERP LN. However, you can achieve this in the session "Maintain Developer Authorizations by Package VRC" (ttadv0141m000). If you uncheck the option "Authorization for Components where no Designer", you can prevent other users to work on your components in the specific VRC. Note that this does not work for all components, because not all components have the designer attribute.
