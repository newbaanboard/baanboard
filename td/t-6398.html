Because the parallel menus in ASCII interface have to show up as child menus in the GUI menu browser, some clever(?) software engineer of Baan exploited the fact that there are labels on those menus which give the user the effect that the "active" one of the parallel menu lables at the bottom appears highlighted.

Now the submenu in the GUI is named with the label that is marked in the editor as INVERSE, BOLD, and YELLOW (or something like this, I always forget the exact combination - you will have to check that within a Baan standard menu).

That's really an amazing feature...
