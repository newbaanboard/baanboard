The field header.dsca comes from either the Service Order Header description (cisli250.desc) or the Sales and W/H Order Invoice Header description (cisli240.dcsa), as far as I can tell. Does that help?

To reset a variable to an empty string before it's print, try this; find the layout that it's printed in, then in the report script use:

layout.number:
before.layout:
header.dsca = ""
