Just an idea, seems a little bit weird, but it works...

Store the PDF file as a baan text, obviously do not use the standard baan text editor but you have to develop a DLL to do this.
Basically the DLL must contains:
- A function to 'pack' the pdf file as a text.
- A function to 'unpack' the text in a temporary file with extension .pdf and a call to winapi to display it.

Years ago, I've written an application to store and display word files with this method, so the same method can work with pdf files.
