Here are the basic set required to define a new paper size. I recommend that you read the help for each session before you actually do the setup.

1. Set up new paper type (Maintain Paper Type - ttaad3110m000)
2. Set up fonts for new paper type (Maintain Fonts by Paper Type  ttaad3111m000)
3. Set up defaults for report (Maintain Report Defaults  ttaad3105m000). This is optional but if used will insure that report is printed on correct paper.
4. Set up device (Maintain Device Data  ttaad3100m000). This is optional but if used will insure that report is printed on device with correct paper type.
