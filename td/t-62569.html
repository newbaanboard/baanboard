You couldn't get the 2nd screenshot if you didn't solve the first one. So I assume the problem of the first screenshot is solved.
When you preview the report in Business Intelligence Development Studio, you need to have a file ier_<userid>_<report code> in $BSE/tmp on server. Do you have that one?
Running the report from the report server (3rd screenshot) is not supported. You should run the print session in LN and select a device that sends the data to the MS server.
