See if you have Sensitivity Labeling module which you can use for setting colors to all sessions/reports company wise. You can enable it from Maintain Parameters(ttaad0100m000). That is one way to do what you are looking for.

You can find this module under - Tools -> Application Configuration -> Sensitivity Labeling

I hope its available on your version of LN.

In case you use BW, just add -set BAAN_WIN_TITLE="%h '%S' [%s] (%p:%u:%c) %f" to your config line, it will tell you hostname, session name and code, package combination, user, company and active filter in the caption of each session.
