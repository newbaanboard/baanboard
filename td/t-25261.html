I think that there is no IF-IN syntax in IF-THEN-ELSE statement. It is undefined in baan. Instead of it , there is ON-CASE statement like that :

ON CASE expression
CASE expression_1:
-------------------------
CASE expression_2:
----------------------
ENDCASE

You can also use:


if pos(",1,3,5,7,8,", "," & str$(month.v) & ",") > 0 then
....
endif

The extra commas are not really needed in this case, but if you use strings like "8,11,12" you would also accept month 1.
