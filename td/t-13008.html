There are a number of ways, but the two easiest, or more common options are;

ODBC - to connect directly to your Database (you will need ODBC drivers for the database you are using). But be warned: do NOT insert records, or update key fields using ODBC. The hash columns get severly annoyed when you do that. The advantage with this way, is the tables can be linked to the MS-Access database.

OLE - you will need to use VBA to do this, but you can create OLE objects and use code to retrieve the data you want (there are a lot of threads, relating to this, if you search through the Baanboard).
