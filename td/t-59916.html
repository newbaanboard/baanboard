Hi Amit, Here I think you need to make changes in your customize font and you are not able to find the exact location where you should double click to change the font properties. If this the requirement then Here are the steps you can follow:

1) Uninstall the font from Local System with admin rights (Before that take a back up of the font).
2) Run the report, You will find Text display instead of your customized font ( Here in your case it is "JC" and "500").
3) Double Click on the Text ("JC" or "500") assign it with some normal font (say "arial").
4)Close the report
5) Now Install the font with admin rights in your local system.
6) Again Run the report and you will find the same Text ("JC" or "500") and now double click on it and assign your customize font again.
