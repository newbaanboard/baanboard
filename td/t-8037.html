Go to the option dialog and select "Start Shell" (ASCII: F6 - Start Process) - you will start ttstpshell. In this shell you can start any Baan session as wel as menu's.
mtccom000000002
[ m for menu ]
[ 2 for language 2 ]
[ tccom00000000 main menu (ASCII) ] or
ttdskmbrowser
[ for the GUI menubrowser ]
or your DEM browser etc...
