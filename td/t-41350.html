I have a value called tot_length = 15.5000

I want to combine this into a string and add an 'm' onto the end to display on a report.

Syntax used is : field_cut_length2(1,1) = str$(tot_length) & "m "

When I get the result, it is "15.5m" not 15.5000m

How can I preserve the decimal places?

Thx !
Hitesh Shah
6th September 2007, 07:05
field_cut_length2(1,1) = edit$(tot_length,"ZZ9VD9999") & "m "

should accomplish the result for u. Use shiftl$ to remove leading zeros if u wish so .
