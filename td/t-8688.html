Hi,

1. creating printer drivers for matrix printers is not that difficult but you need to have a programmers manual.
2. check the printers documentation if it can emulate e.g. an epson printer for which Baan does provide drivers
3. existing drivers can be found in $BSE/lib/printinf/<first character>/<drivername>
4. More information can be found in the online tools help.
