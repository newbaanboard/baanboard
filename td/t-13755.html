Hi,

You should enable the standard form command 'duplicate occurance'.

Then in the program script, you write a section choice.dupl.occur: and you call the single-occ session the same one that is being used when inserting a new record), fill "some" fields based on the selected record, and the rest remains standard functionality...

Good luck!!
