In FP7 you can use Microsoft SQL Server Reporting Services (SSRS) for that. The guides are available in the InforXtreme Documentation section (document numbers U9656 and U9657).
You still have to develop a classic report, but you don't need to bother about the layout. The SSRS design linked to that classic report can produce the graphs you need.
