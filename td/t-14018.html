Function val()
--------------------------------------------------------------------------------

Problem
Function val() consumes more CPU time than function lval().

Solution
Use function val() only for variables of double and float type.
For variables of type long, integer and byte type, use function lval().

Example
Good example

long number
stringnumber.s(10)

number.s = "3251"
