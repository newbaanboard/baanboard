Could be permissions for text handling as set in
tttxt1111m000.


POPUP-HELP: Authorisations (tttxt006.perm)

There are three types of authorization to work with the texts of a text
group:

Use
The user is authorized to read texts and link
them to a record, e.g. an order, item, help text
code etc. He cannot change the texts.

Update
The user is authorized to create, change, read and
use texts in the text group.

Read
For the specified text group the user can only
read texts. He has no authorization to link or
update them.
