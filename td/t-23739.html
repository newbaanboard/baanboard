Hi there,

simply create a new printer in Baan, which in truth is a script.
----ttaad3100m000
Printer: print2times
Descr: foobar
Prttype: direct
Locale: ISO-8859-1
driver: hp_lj4
Queue: /path/pr2.sh %s
..
.

---
In this script, save the file (printer output) to a temporary file on disk.
Then print this file to as many printers as you like in the script.
This will work as long as the printers concerned use the same driver and are accessible from your server.
---begin of file pr2.sh---
cp $1 temp.prt
lp -d printer1 tmp.prt
lp -d printer2 tmp.prt
----end of file----
(change the lp command and device parameter if you use another application to print)
Make sure to give the script execution rights for group bsp and make sure that groups bsp has write access to the directory you write the temp file to.
