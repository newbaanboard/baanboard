It sounds like you should be using date.to.utc,

Syntax:
function long date.to.utc (long yearno, long monthno, long month_dayno, long hours, long minutes, long seconds)

Description

This converts the specified local date and time to UTC long format.


Arguments
long yearno
long monthno
long month_dayno
long hours
long minutes
long seconds

Return values
The UTC long format value.
-1 Error.

Context
This function is implemented in the 4GL Engine and can be used in all script types.

Example
long utc
utc = date.to.utc( 1991, 04, 20, 23, 12, 45 )
