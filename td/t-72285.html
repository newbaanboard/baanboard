With Extensibility you can add it to the XML that is sent to SSRS. What is written in the Extension Developers Guide for Infor Reporting, applies also to SSRS. In the report design you can use the function tt_getApplicationImage. See the Infor Enterprise Server Plug-in for Microsoft SQL Server Reporting Services Development Guide.
