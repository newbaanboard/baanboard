:) Problem is solved.

My Baan (Baan ERP in NT ) was installed in directory
c:\Program Files\Baan (default in instalation wizard)
Then bdbreconfig builds a path to temporary files las
c:\Program Files\Baan\Tmp
and cannot open it because of blank character inside !

I reinstalled BaaN to c:\baan directory and bdbreconfig is OK now !
