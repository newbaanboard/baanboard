Hi Saumya,

the big difference is that if you change the code of a function you have to recompile all the scripts that use it. On the other hand if you modify a DLL you only have to recompile the DLL itself and not the scripts that use this DLL. All these scripts are automatically up to date and use the latest version of the DLL.
