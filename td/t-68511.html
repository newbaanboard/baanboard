Printers are Windows Printers?
If so:
Print to bwprint with "Preview" checked.
In BWPrint you get the file in display mode.
Here you can try to change the font characteristics (right click on a text) and see if the text is complete in the file when you lower the character width (or make CPI higher).
If the text is there, it is a problem with the printer
If the text is not there, it is a problem with the report
