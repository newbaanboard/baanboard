Rosalba,

The steps for using Maintain Idef1-x Diagram Data (ttadv4131m000) are:

1. Insert a new diagram and define the attributes.

2. Maintain Entities (ttadv4132m000) to assign the list of related tables that will be displayed on the diagram.

3. Generate the Diagram

4. Edit the Diagram (if necessary).

Note that when you insert a new diagram in step #1, the diagram will default to your current developer's package VRC. If you want to select a table from a different package (other than your current P-VRC) then the package of that table must belong to P-Combo for which the current P-VRC is defined. That is to say, the P-Combo must be defined within Maintain Package VRCs (ttadv1111m000) for your current developer's P-VRC that you are using.

Hope this helps clarify things for you.
