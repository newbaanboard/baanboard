RMCgen and QKey both allow you to modify (to some extent) the functionality of Baan program scripts, libraries, etc. It would take a while to fully explain, but let me give a program script example. In the program script, there are sections of code for all of the various form fields and actions. Using these programs, you can add new sections, add new code before or after the standard Baan code for a section, remove the Baan code for a section, etc. What you can't do without knowing the Baan code is pick and choose which lines of standard Baan code within a section you want to execute. I hope that makes sense.

With that said, these tools can be extermely powerful. For more information on RMCgen, you can go the RMCis web site (http://www.rmcis.com) . QKey is made by FullScope.
