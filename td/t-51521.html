Use the flag, but use it properly:

in subsession:


.....
before.program:
flag = FALSE
.....
main.table.io:
after.write:
flag = TRUE
.....


in main session:


....
import("flag", flag)
if not flag then
....


There is no need to add 1 to flag, just set it to true. The export is done automatically, but not the import...
