Hej,

I have no system to try for the moment, but i guess that this should work:


domain tcmcs.str11 origin
domain tcmcs.str4 alpha
domain tcmcs.str7 num

alpha = origin(1;4)
num = origin(5;7)

if isdigit(num) = false or
val(alpha) <> 0
message("input.error")
set.input.error()
endif
