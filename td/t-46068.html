Run the session ‘User Data Template’ (ttams1110m000)
Make sure the template code ‘Default’ attaches in every user data.
Go to second tab ‘Authorizations’ in the same session ‘User Data Template’
and select the option ‘Application personalization’.
Click on save button. And run ‘Convert to DD’ as shown in the session.

Re login LN with user and check the ‘view’ tab ‘customization grid’
will be enable in every application session.
