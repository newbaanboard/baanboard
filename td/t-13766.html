Hi,

You can use ask.enum() standard function for using Question in yr script.


domain ask.enum( string quescode(10), domain default_enumval [, arg ] ... )

aurguments are
Question Code that u have maintained.
Default answer eg. if domain is tcyesno then tcyesno.yes....
So that when Question is asked the cursor would be on default answer.
