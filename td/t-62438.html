4th June 2012, 22:02
Assuming you have developer id and VRC permissions.
(1) Copy report to your development VRC.
(2) Edit the report
(3) Add new input field to report.
(4) Then maintain the layouts - I usually just maintain all layouts.
(5) If necessary expand the header layout and add header for the field.
(6) If necessary expand the detail layout(or layout where field will go) and then use edit and add the new field.
(7) Save and exit report. Then compile the report.

These are some of the basic steps and depending are what you are doing it is going to be a little different.
