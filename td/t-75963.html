Below there are the instructions i googled.
According to the Infor LN Report Designer Development Guide, you can backup the report design by following these steps:

Open the Report Designer (ttstppersrep) session and select the report that you want to backup.
Click the Export button on the toolbar. This will open a dialog box where you can choose the location and name of the backup file.
The backup file will have the extension .rptdesign and will contain the report layout and style information.
To restore the report design from a backup file, click the Import button on the toolbar and select the backup file. This will overwrite the existing report design with the backup file.
