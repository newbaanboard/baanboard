Hi there,

set the environment variable DBSLOG to 0000010.
If you use the GUI, this can be done in the lowest input field via:
-- -set DBSLOG=0000010

This will create a file dbs.log in the users homedir.

Be careful - this is for testing / debugging only ... the file will grow extremely!

But to see what tables are modified by a certain session this is handy.
