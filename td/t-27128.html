The graphical lines drawn on a form using the <CTRL>[G] option are visible only when using the 'green monster' screens (VT52, VT100, etc.).

It is not very pretty, but to be able to see the graphical lines on the GUI display, use the <ESCAPE>[G] to draw the lines (actually, it always draws a box, so the 'line' looks strange on the form editor, but once placement is achieved, will look just fine on the GUI form.)
