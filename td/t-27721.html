I found this. If you want to get 'Module & Session Code' as return value, you must define the variable's name(include array) like 'cses.*'. Otherwise you fail. I think, it had already defined by Baan.
Similarly, define variable's name like 'ttstp.session.*', you can get 'Package & Module & Session Code'. No matter what you fill the 'Zoom Return field'.
