Can you check the following ?

Suppose cu is package mod is module,

if you tried to convert cumod100 after making changes to the table, then in the above said folder cumod100.new file would have got generated.

Just remove the file 'dcumod100' from the folder and rename the file 'dcumod100.new' to 'dcumod100'.
