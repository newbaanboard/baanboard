Hi,

right of the "T" (text button), there is a buton with a black arrow and a blue question mark. This button is the field help button. If you press it, the mouse cursor changes to arrow and question mark; in that state, you can press on fields an get the online help for them.

I did that for you:
tdsls040.cbrn --> tcmcs031.dsca

You say tdsls040.ssls? Normally, it's tdsls045.ssls. Unfortunately, it's a bit more tricky and normally done by a standard function to get the description.
