Hi,

your question is not clear to me, but i'll try to answer it:

if you change a formfield and you want to see the new value, so refresh the field (i.e.: datafield.f) like this:

...
datafield.f = "NEW VALUE"
display("datafield.f")
refresh()
...

If you change more than one field, then use simply "display.all()" instead of "display("fieldname")".

Hope this helps you. :rolleyes:
