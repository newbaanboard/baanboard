You mean a program script vs. report scripts. No, the functions are not all the same. The structure, the sections, the flow of the standard program, they are different...

In report scripts you program in the report layout sections, in the program script you program the field, form and action sections. Etc etc etc

Normally you will use the program script to write the code of the actual program, and in the report script (IF you write a report script) only things that could not be programmed in the program script. Common examples are initializing and updating calculation variables, and controlling whether or not a certain layout shall be printed..

This is it in a nutshell; hope it helps you...
