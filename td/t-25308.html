Ain't SOX great!

If you have Oracle as your database, you can use Oracle INSERT, UPDATE, or DELETE table triggers to populate another table with change info like userid, date, time, or any other available item you need. Using triggers does not noticably degrade system performance. If you know PL/SQL then the rest is pretty easy.
