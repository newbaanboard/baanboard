this is a Quote from Solution 210711:

TOOLS VERSION(S):
7.6_a2

SITUATION IDENTIFIED IN:
"Table Timestamp Definitions" (ttadv4136m000)

SITUATION DESCRIPTION:
It is not possible to query on tables by means of a timestamp column. This is required during export of data as off a certain UTC date.

SOLUTION DESCRIPTION:
Session ttadv4136m000 Table Timestamp Definitions can be used to specify tables that must have a UTC timestamp column. This column will have the reserved name 'rcd_utc'. This column will be automatically filled during database updates. It is possible to use this column in a query to filter out all records before or after a certain UTC date/time.


After you added a table in ttadv4136m000, you must create the RDD for this table.
