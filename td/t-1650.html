The following tables are declared

table ttdinv001 | item data by warehouse
table ttipcs021 | customises items
table ttdrpl100 | replenishment order lines
table ttdsls041 | sales order lines
table ttdsls045 | sales order deliveries
table ttdsls047

It also reads conversions from tiitm001
