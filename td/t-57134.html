Hi Saritha,

This has to do with BOD publishing (SOA enabling of BaanIV). But you should not get this message in case you are not using MyDay (or any other SOA component).
Do you have data in table tlesb300 (check via ttaad4500)? Only if this table has entries then it will try to publish BODs. Only at that moment the required setup needs to be checked.
