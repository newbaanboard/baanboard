We use Software995 to convert Baan reports to pdf format. It's pretty slick.

After installation & set up of a new device in Baan, print from any Baan session & specify pdf as the output device. A "Save As" box will open prompting the user to name the file. Once saved, a new message window opens in Outlook w/ the saved file already attached for emailing.
