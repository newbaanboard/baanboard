Hi JP

As long as you did not compile the previous version of your script is still available in the $BSE/application/<VRC>/<P> directory. If this is not the case your only escape is a restore from back-up.

Example program script location:

$BSE/application/tdB40O_c4_fawe/ptdpur/p<name>

Example report script location:

$BSE/application/tdB40O_c4_fawe/rtdpur/r<name>

Good Luck!
