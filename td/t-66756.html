Hi Sangeeta,

please check windows printer setting zebra -> "Advanced" -> "Printing Defaults" ... here you can set the label size and orientation and other dimensions. Additionally you can check "Printer Setting" and set "Always use driver settings" to "yes".

Hope this helps.
