Hi Manish,

The characters "********" displayed are nothing but no reference character.

When application lock is set the application locks related data get stored in two tables
1) ttadv997 [Application Lock Header]and
2) ttadv998 [Application Locks]

In some cases record in ttadv998 get stored with application Id (ttadv998.applid) as 0 for which system can not find any reference record in table ttadv997 hence no reference character get displayed on session.
