Well it kind of looks like a BOM report. What I typically do is just clone the report to a new report. Then I remove fields not needed, like the level. So if I wanted to sum only the sub-items I just leave in the sub-item and quantity. Then sort/sum how I want. No need to even change the actual session and everything is done in the report. And if all you want to see is a total by sub-item I would sort on sub-item, then do a after.field layout for it with a sum on quantity. Hope this helps!
