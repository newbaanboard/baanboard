Hi,
Try out this way:

In Baan under User Management -> Default Settings we can specify default session to be run on log on.

Create a BWC file with a user of limited acess, may be only the report he is supposed to see.
Store the Password in the file.
Define Session Group in ttaad2107m000.
Define Startup Session in ttaad2106m000.
Attach session group to the user in ttaad2105m000.

Convert to runtime DD the settings.

Log on to Baan, and the session defined in the Startup Sessions starts on log on
