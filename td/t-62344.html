sin()

-------------------------------------------------------------------------
Syntax
double sin( double x )

Description
This computes the sine of x, in radians. The x argument can be a long or a double.

Return values
The sine of x.

Context
Bshell function.


So try : sin((164.4/8)*PI/180) should give the sin in DEG !
