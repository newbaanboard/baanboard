Hello,

built your barcodes in a library / function like :

|* 06.06.03 [15:42]
|* Build Barcode from Variable and determine length
|* For WinBaaN the maximum characters in barcode can be 40
|* All barcodes must have a corresponding length to the number
|* of characters in the string
|* Code 128 autoswitch (Height = 3)
|******************************************************************************


| Input

extern domain tcmcs.str40 barcode.value |String 40
extern domain tcmcs.str40 barcode |String 40
long barcodelength

| Output

extern domain tcmcs.str40 print.barcode

function dhcombd005()

{

barcode = str$(barcode.value)
barcodelength = len(strip$(barcode))

if barcodelength = 1 then
print.barcode = chr$(27)&chr$(22)&"*#!"&barcode
endif

if barcodelength = 2 then
print.barcode = chr$(27)&chr$(22)&"*#"""&barcode
endif

if barcodelength = 3 then
print.barcode = chr$(27)&chr$(22)&"*##"&barcode
endif


if barcodelength = 4 then
print.barcode = chr$(27)&chr$(22)&"*#$"&barcode
endif

if barcodelength = 5 then
print.barcode = chr$(27)&chr$(22)&"*#%"&barcode
endif

if barcodelength = 6 then
print.barcode = chr$(27)&chr$(22)&"*#&"&barcode
endif

if barcodelength = 7 then
print.barcode = chr$(27)&chr$(22)&"*#'"&barcode
endif

if barcodelength = 8 then
print.barcode = chr$(27)&chr$(22)&"*#("&barcode
endif


if barcodelength = 9 then
print.barcode = chr$(27)&chr$(22)&"*#)"&barcode
endif

if barcodelength = 10 then
print.barcode = chr$(27)&chr$(22)&"*#*"&barcode
endif

if barcodelength = 11 then
print.barcode = chr$(27)&chr$(22)&"*#+"&barcode
endif

if barcodelength = 12 then
print.barcode = chr$(27)&chr$(22)&"*#,"&barcode
endif

if barcodelength = 13 then
print.barcode = chr$(27)&chr$(22)&"*#-"&barcode
endif

and so on ..................

will be more flexible.

Hello,

this is an own function to handle the correct barcode printing for "windows printing"

In reportscript :

#include "idhcombd005" |Barcodefunctions



barcode.value = strip$(tiitm001.item)
dhcombd005()


Variable barcode.value is given to the function and the variable print.barcode is given back with the correct value.
The length of the itemcode can be variable, always the correct barcode is generated.

It can be used in every report.


Hope this helps.
