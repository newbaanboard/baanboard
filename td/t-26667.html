Hi,
You can use 1 exchange scheme, however, you will need to maintain 2 ascii files and 2 batches - one for each table.

If you require the output to be in the same file then you may want to use SQL Queries - ttadv3180m000 - in Baan 4. Hopefully the session code is same for Baan 5 as well.

Hope this helps.

MG.
