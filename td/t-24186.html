Go to session ttadv2100m00
Find the session where you need to increase the report page width
Go to form 2
click on Reports button (it opens ttadv3130s000)
Find the report where you need to increase the report page width
Go to form 2
Increase the Page width here to 200
Go to Form 1 of ttadv3130s000
Click Compile button
Save and exit from ttadv3130s000
and save and exit from ttadv2100m000

Now run the report
