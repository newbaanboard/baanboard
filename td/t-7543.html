hi eddie,

in general I would say that random() returns values within some range (long); let's say 0 .. N. You like min ... max, okay. So you could do some minimal maths to convert it like value = min + (random() / (min+max)).

Btw. also have a look at rnd.i(<max>) whicht can at least return a range 0 ... max-1.
