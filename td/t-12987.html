Alternatively, if you are on BaaN-5 versions you can also use-
function process.zoom()
{
query.extend.select.in.zoom("ttaad200.* ")
query.extend.from.in.zoom("ttaad200 ")
query.extend.where.in.zoom("ttaad200.utyp = ttaad.utyp.normaluser
and ttaad200._compnr = 000")
}

Refer to -
query.extend.where.in.zoom() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_sql_query_extensions_query_extend_where)
