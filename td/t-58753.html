Hi,

use separate fields:
where b.t$orno = ttdpur045001.orno,
and b.t$pono = ttdpur045001.pono

If you are using type-2 Driver in Baan, then Oracle already has access to correct index and can use it to speed up the query execution time. If you have the old type-1 Database driver, you may have to manually create a suitable index in Oracle.
