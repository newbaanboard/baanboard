Please check if a record exists in session:

ttaad1108m000 - Format Codes

Record for 004 should exist. 004 is for cost prices.

ttaad1107m000 - Maintain Formats by generic unit

There should be an entry for your home currency (tccom0100m000).

Example:

Format Code: 004
Generic Unit: EUR ZZZZZZZVD9999 (or USD or whatever).

It is used in Forms, Reports, Scripts etc. like %A004, where %A indicates it is an amount, 004 is the format code.

Online help says:

%Axxx[,CUR]
Use for amounts. xxx is the code for an amount format defined in the data dictionary.
CUR indicates the currency code to use. It this is omitted, the default currency of the company of the user is used.
