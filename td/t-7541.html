I've been doing it the following way:

In your form, add some text fields, e.g. text(1) ... text(5) where 1..5 is the element number; type e.g. tcmcs.str60.


function fill.text()
{
long rc

rc = set.mem(text, "")
rc = text.to.buf("tiitm001.txta", language$, 5, text)
}
