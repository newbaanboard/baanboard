There have been alot of changes and bug fixes in the standard program for AFS. The standard program is tied to TT in the service packs and to the porting set. Once you can get to an update porting set you should be able to use AFS. Also, not all Baan standard programs are compliant with AFS. I know that 4c4 has been brought more current with the service packs, but I am not sure about the other versions.

Good Luck!
