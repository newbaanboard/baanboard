Hi,

I have wrote you the following function:
function resize(long Hght, long wdth)
{
long My.Win

My.Win = current.mwindow()

change.object(My.Win,DsNheight,Hght,DsNwidth,wdth)
update.object(My.Win)
}
I have tested it in Baan 4 and it works. I hope it helps,
