The best way is to test it. Since you're creating a display session, you don't have to worry about corrupting the data :). Just remember that the query extension, if it contains anything, is appended to the main query with the sequence:

"select * from <table>" & " where " & query.extension

So you can't start the query extension with the order by, but you have to add some condition first. Whether it will work, I'm not sure. The group by will probably NOT work, since you would have to change the main query to include something like sum(<fieldname>). That doesn't work!

A general remark (again :p): why not work with a temporary table, filled with the data you need to display?
