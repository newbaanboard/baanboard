Make sure you have defined the variables correctly. The following code worked for me:

string buf(100),var1(3),var2(3)

buf = "abc" & chr$(29) & "def"
string.scan(buf,"%s"&chr$(29)&"%s",var1,var2)
| after this command var1 had "abc" and var2 had "def"
