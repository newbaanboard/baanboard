You can better use rdi.table to get the number of fields in the table, use alloc.mem to allocate enough space, and use rdi.table.column to get the field codes and tt.field.desc to get the field descriptions.
Keep in mind that the rdi functions also return information about some hidden table fields (see rdi.table).
