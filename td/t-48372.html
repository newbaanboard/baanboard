Hi,
You can do this with the help of session Maintain Default Text Group by Text Table Field {tttxt1112m000}, without having to write any code. In here you enter the text field of the table (for e.g.: tdsls001.txta:Sales Header Text), that you want to set the default on, and attach the text group. The next time to click on the "T" button on the toolbar, the default text group window will automatically open rather than giving you multiple choices..
