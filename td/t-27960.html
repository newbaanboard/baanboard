Dear nansat99,

possibly this works:

create a table in baan with fields and domains matching your external table.
(you could compare a nearly matching table in baan and on oracle level, what oracle types matching the baan type.


(e.g. tc.mcs.string // tc.mcs.long -> no to special domains).

Create the table with baan. (Maintains table -> Add Fields, Index etc.)

Be aware to add 2 fields to your existing oracle table not seen in baan -> Referencecounter and Update Referenccounter (default value 0).

Delete the table with oracle methods.

Rename your existing oracle table to the technical name of the baan table e.g. baan.ttisfc991100. (Or make a oracle view).

Now try to make a simple select statement from within a baan script.

Regards,

Kozure
