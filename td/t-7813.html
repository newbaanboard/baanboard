About stp.reset.value() function all I have got is programmers manual - see below:

9.99 stp.reset.value
NAME
stp.reset.value - function to reset the current form field

SYNOPSIS
void STP.RESET.VALUE( )

DESCRIPTION
This function changes the value in the current field back to the
old value. This can only be used after input is given.

USE
This function can only be used in the subsections:
check.input
on.input
after.input

If this function is used in the subsection check.input it must be
sure this function is not called in a check before update of the
database. So in check.input you should program:

check.input:
if not before.update.check then
stp.reset.value()
endif
