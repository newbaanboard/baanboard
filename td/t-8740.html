Yes, we definately can configure the Print Server for Baan application.

Basically what we need to do is on your server (Unix) create the REMOTE printer device which will have the ip-address or host-name of the printer server followed by the device name on Print Server.

Then within Baan application, just use the create unix device.

I have done it for one client.
