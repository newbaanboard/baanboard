In this thread (http://www.baanboard.com/baanboard/showthread.php?t=7048) you find an attachement from RobertB to display labels on a form. You have to modify the code, so that you can assign a DsNgcForeground ressource to the label. I think it have to be an object type of DsCgpText. It is not possible to change a color of a standard field. There is a lot of programming to emulate a standard field.

I highlight data fields with attr.cf = 4 (inverse), try this and look if this match your things.
