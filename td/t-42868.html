It is written in the documentation the DsNstringArray should get the value of the selected cell:
A one-dimensional array that holds data values for selected cell(s). When filling multiple cells, separate the data values with null characters. The data is filled from left to right and from top to bottom.

Maybe you should select the cell first (by code using DsNrange) and then get the data. All the attributes can be found here:
