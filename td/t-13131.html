This is again Manoj. If you compile a source with the bic6.1 complier using the option -p, a special object is created with extra instruction for each function. After ending the execution of the object, for each function the measured time spent is printed. For a special function, only the spent time of that function is counted, not the time of functions called by that function.

Hope this will work for you
