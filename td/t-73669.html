Hii,

You can use below statement for your non-main table...

query.extend.select()
query.extend.from()
query.extend.where()

And for Enum condition you can directly use enum constant number because every enum has fix input so that will be a limited range...
Example : query.extend.where("whinh200.oorg = 51") | 51 = production.man
