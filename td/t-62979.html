In ERPLN only 3 SW components are language dependent: labels, messages and questions. So, report layout is not language dependent.
You don't need to copy anything. What we are doing is:

1. Export Language Dependent Data to XML - Wizard
2. Translation
3. Import Language Dependent Data from XML - Wizard
