ttadv3100m000, second form, "Foot margin" : -1 or number of lines.Foot margin (ttadv330.fmrg)

The number of lines allocated for the footer. If this value is -1, the
footer margin is calculated by the report writer. When you have a number
of different footer layouts of different sizes which are executed on
conditions at runtime, it is difficult for the report writer to
calculate the footer margin. So in such a case you have to specify a
number of lines for this margin.

Mandatory Input : Yes
