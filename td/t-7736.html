After the zoom on whatever field you call the menu on you can try using the import function. Look that up on the programmers guide in the forum. Something like:


field.xxxxx001.fld1:
after.zoom:
import("tccom010.cuno", my.cuno)
import("tccom020.suno", my.suno)
if not isspace(my.cuno) then
xxxxx001.fld1 = my.cuno
else
xxxxx001.fld1 = my.suno
endif


I am sure that can be written better - I think there is a way to know which menu choice they made so you don't need to import both and decide which.... but that's how you transfer variables from 1 session to another.

As a side note, this depends on the sub-session being written properly, to 'throw' the cuno and suno's up, but Display Customers and Display Suppliers would be written so.
