This session is easy to use. For example, you have created a help for a session in Maintain Session Help, after that you have to create its run-time to make it available to use.

Then you use the session you mentioned, Create Run-time Help, choose the correct VRC where you created your help, mark the session check box (in this example), go to the second form and fill the name of your session help. Note that you will be able to type something in the second form only if you have marked the check box in the first form ok !!
