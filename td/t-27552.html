Hi Pradip,

There seems to be no problem with your report script. I vaguely remember facing such a problem some time back. I would suggest you to follow steps below,

1. Create backup of your existing report script.(Very Important)
2. Delete the existing report script & create a new report scipt(blank).
3. Start copying small sections(10-12 lines forming logical set) of script from your backup report script to the newly created report script, Compile simultaneously till error occurs.

In my case , it was a unnoticable character which was accidentally typed in the script. The character(i think it was a single quote ') was missed by me and was caught only after this painful exercise.

Your script apparently doesnt seem to have the character but there is no harm in trying above method.

Let us know your findings and start hunting :)
