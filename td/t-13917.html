The bic_files are include files that are linked at compile time. Most functions are available via standard program and no further action needs to be taken to use the functions. In some rare cases you have to include the files explicit.

Like in:
#include <bic_text>
(see these (http://www.baanboard.com/programmers_manual_baanerp_help_functions_text_fields_synopsis) functions)
