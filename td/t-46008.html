Hi,

Assuming both reports belonging to same group, you can check & use predefined variable reportno as shown below,

choice.print.data:
on.choice:
if rprt_open() then
if reportno = 1 then
read.main.table.1()
else
read.main.table.2()
endif
rprt_close()
else
choice.again()
endif
