You may use this template -

db.set.to.default(ttable_name) | sets default values

db.retry.point() | Set retry point in case of failed transaction(optional)

dal.new.object(table_name) | start insert operation
dal.set.field(tablename.fieldname,value) |set values to fields
...
ret = dal.save.object(table_name)
if ret = 1 then
commit.transaction() | commit transaction
else
dal.get.error.message(err.msg) | get the last error message
abort.transaction() | abort transaction
message(err.msg) | display error message
endif
