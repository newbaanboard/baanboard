Depends on your O.S. ( operating system ), multiple options are available.

Step 1 ( Both Unix & Windows )

Create software dump using ttiex1280m000 ( session based ).

Step 2:

Using Windows OS

Zip the created directory from step 1 to a ZIP archive file using f.i. WinZip

Using Unix / Linux

Zip the created directory from step 1 to .tar archive file using the tar cmnd.

syntax tar cvf dump_baansession.tar <software directory>

Step 3:

send the winzip or the tar file to the receiptant.
