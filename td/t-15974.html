When you have to ability to adjust Forms through customizations, you can change a form's name.
But in this case you need to copy the original VRC to your own customized VRC.
Afterwards, installing new versions will become more difficult. This means: not only the form's name has changed, but a copy of the complete form exists.

Also note that most standard forms use the 'session description' to be shown at the top. You need to change that to the 'Form description'.

Changing the name of a session in a menu is also possible, but that does not change the form name when launching it.

I don't know if any other possibilities exist.
