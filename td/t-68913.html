Hi,

In the "Sessions-ttadv2500m000", open the sub session "ttadv2100s000", in this session go to last form named "Form", on this form you will get a field named "command Type" set it as "Form Command", and select a form command in "Default Button Field".

Using this option you will be able to get only one button above "Close" button.

Another way is to get all 3 buttons on form, i mean attach each button to a field on form.
