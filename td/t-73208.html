You can't interact directly with the form command inside a UE, but what you can do is - if the form command is updating/inserting the table record, you can put a condition to restrict that update in before.before.save section of UE for that particular table. It won't disable the form command, but if executed, it will not work if the condition is not met. Other way is to use a wrapper script. (http://www.baanboard.com/baanboard/showthread.php?p=2475#poststop)
