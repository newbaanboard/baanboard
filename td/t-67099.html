Hi Eric,

try:

Address:
function extern zoom.to.addr()
{
hold.cadr = cxjdb001.jadd
start.session(MODAL, "tccom4130s000", "hold.cadr", "")
}

Contact:
function extern zoom.to.ccnt()
{
hold.ccnt = cxjdb001.ccnt
start.session(MODAL, "tccom4140s000", "hold.ccnt", "")
}
