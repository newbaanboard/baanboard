The only way is as suggested, through changing / adding validation to the program script. The form simply will not allow you to check that box if it is a table field.

You can use a wrapper tool for Baan Programming (such as my company sells: RMCgen, or FullScope sells: QKEY, or even the freeware version on these boards, search for "Rush") to basically ADD validations to the standard code without modifying the standard code. This preserves your licence agreement with Baan, and is easy to upgrade when service packs are installed. (easier than a source code modification, anyway). There are plenty of posts about those products - search for 'wrapper'...
