I think you are on the BaaN-4 version. I am not sure if BaaN-4 has the LTS (BaaN's new Language Translation System) or not.

If LTS was enabled, you can set your language to Spanish and if the Spanish texts are not available the English would be used as the fall back language.

If LTS is not available you do not have that option and I am guessing this is your custom report so in order to work it the way you like with different language you might have to customise it to handle that way.
