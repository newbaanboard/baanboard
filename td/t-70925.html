JaapJD
21st March 2018, 14:11
Because LN UI is the only supported UI for Infor LN 10.5, you can only create those components that can be created with tools sessions that run in LN UI. Editing scripts, report layouts and forms is not possible with LN UI.
Using BW to do development may work, but is not officially supported by Infor.
I would recommend to start using LN Studio. Yes, it is a learning curve, but it has a lot of nice features in it.

For LN Studio itself there is no license needed. However, if you want to develop domains, tables or script components, the Development license (10146) is needed.
Infor LN 10.6 has Report Designer, no license is needed.
