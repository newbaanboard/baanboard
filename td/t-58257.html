Here I am assuming that table has maximum one record and you want session like maintain parameter session. If my assumption is correct; then you can achieve this as below:

Table:
Create table with sequence as index

Form:
Don't put sequence field in the form or make it display field.
Disable the Copy/Global Copy option

Script:
In add.set choice option you can check whether a record is already exists or not.
If it does not exist then assign some value to sequence field.


choice.add.set:
before.choice:
if record.exists() then
choice.again()

else |****also you can move else part to before.write subsection of main.table.io

|Assign sequence field to specific value
|tdpur000.seqn=1
endif
