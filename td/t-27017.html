I solved this problem myself, here is the solution

ttstpconv does not use spool.pg.length (in my case B5c - oct 2002)
instead, it uses ttaad320.pgln
so I just added these 2 lines

table tttaad320
import("rcd.tttaad320", rcd.tttaad320)

this solved my problem
