string GET.COMPANY( ) This function returns the name of the current company.

long GET.COMPNR( ) This function returns the identification number of the current company.

long SWITCH.TO.COMPANY( long new_compnr ) This function switch to another company and sets the internal predefined variable 'compnr' to the value of new_compnr.

COMPNR.CHECK returns TRUE (not 0) if the user has authority and FALSE (equal 0) if the user has not. The user authority for company numbers can be maintained by the Super User in the Data Dictionary.

Hope this helps
