What you can do is - add form fields, and write your own logic to fill those fields in before.field: section of those fields.

e.g. If you have tdsls401.orno, tdsls401.pono, tdsls401.sqnb and you need to add Shipment#, you can add a form field called shpm.f and write the below logic -

field.shpm.f:
before.field:
shpm.f = ""
select whinh431.shpm.f
from whinh431
where whinh431._index5 = {whinh.oorg.sales,:tdsls401.orno,:tdsls401.pono,:tdsls401.sqnb}
selectdo
endselect
