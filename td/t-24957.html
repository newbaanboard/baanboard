Hi,

tt.company should do the job. Make sure to define include <bic_tt>.

tt.company()


Syntax
long tt.company( long company, ref string comp_desc() mb, ref string deflt_curr(), ref long first_weekday )

Description
This returns information about a specified company.

Arguments
company
The company number.

comp_desc
This returns the company description.

deflt_curr
This returns a string containing the default currency of the company.

first_weekday
This returns the first day of the week defined for the company in the data dictionary.


Return values
0 error; company not found
1 success
