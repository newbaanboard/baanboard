All,

In order to set things right:

In baan IV you can put buttons (almost) anywhere on the form by using special fields of type button. This is done in the (static/character based) form editor. (steveauckley is right).
In baanERP, buttons can be placed on the right and connected to a field (field buttons). This can be done in the Dynamic Form Editor (baanprog is right too).

So, they are both right. It all depends on the version you're talking about.
