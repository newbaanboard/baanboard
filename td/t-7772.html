23rd November 2002, 09:43
It is possible, but the field from which you are auto zooming should be such that, all the mandatory fields for Maintain Lots Session are already filled before the time user comes to this field.

Before saving the record, baan will execute check.all.input() and an error will be thrown, if data is incorrect.

so the solution is:

in the maintain session:

zoom.from.<field.name>:
on.entry:
execute(update.db)
|after this import the Lot number as it will be updated in the table.


You should test all possible user actions and try to make it error proof.

Regards
