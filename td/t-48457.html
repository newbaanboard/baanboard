Hi,

To check if multiple records are selected, you can use predefined variables like mark.table(), number.of.marks :

boolean mark.table()
This array indicates which records are marked. The size of this array is set to fattr.occurnr. For example, if records 3 and 8 are marked, then:

mark.table(3) = true
mark.table(8) = true

and other elements in the array are false.

long number.of.marks
Number of selected records in the grid.


Using these variables you can see if multiple records are selected.. So write code in choice section (choice.mark.delete), if multiple records are selected give appropriate message and skip delete operation by using choice.again().

This is one way to do.. I know this can be done in multiple ways...:cool:
