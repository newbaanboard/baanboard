I have a problem with the spaces in local directory folder names. Let me explain the problem.

Consider directory C:\Baan\Process Books. In this path I have couple of files, let us say 10, that I need to process. Now, to get these 10 file names on to a file I am using the command...

local.dir = "C:\Baan\Process Books"
ret = app_start("cmd /c dir /b "&strip$(local.dir)&" > file.txt", strip$(local.dir), "", "", "").

Problem: I can get the list of 10 files only if there is no space for folder name "ProcessBooks". If there is a space viz ., "Process Books" then the above command simply retunrs a number. But the same command works fine if try it out from command prompt.

Please advise what is that I am missing here.

Thank you.

Hi,

You will have to add quotes around the path in the cmd command. Here is the working code:

local.dir = "C:\Baan\Process Books"
ret = app_start("cmd /c dir /b """&strip$(local.dir)&""" > file.txt", strip$(local.dir), "", "", "")

I hope it helps,
