You wanted to know where to look up the ORA error numbers and what they mean. You should have received a documentation disk with your Oracle installation pack that has a separate document listing all errors. If that does not tell you enough, and is usually doesn't, then go to Oracle's on-line help system called Metalink. If you pay Oracle support fees and have a valid CSI number you can register to use it at no additional cost. I find it to be an indispensable tool for resolving many questions and problems. I have provided a link to the Metalink page that will allow you to register.

http://metalink.oracle.com/
