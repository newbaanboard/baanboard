If the domain of your string is left-aligned:

string s(20)
s = "000123"
while s(1;1) = "0"
s = s(2)
endwhile


If the domain of your string is right-aligned:

#include <bic_tt>
...

domain titedm.revi s

s = "00123" | ie. "<blank>0123"

s = shiftl$(s)
while s(1;1) = "0"
s = s(2)
endwhile
tt.align.according.domain(s, s, "tiedm.revi")

Hi Ken,

To remove the leading zeroes from the field, convert the string field to a long/float field using the function lval()/val() and then again convert the field to string using the function str$().
