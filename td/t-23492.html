Here is the code sample u can use in report script.


declaration:
double runningtot
before.program:
runningtot = 0
detail.1:
before.layout:
runningtot = runningtot + value2tot | value2tot is input variable.
|use runningtot as print expression in detail layout.


If the records are presorted in program script , u can accumulate the value in program script and use the running total as input variable and print the same.
