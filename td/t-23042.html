Could you detail your requirement?

Example : say you have a print session with 2 fields on the form

order no from and to
date from and to

If you are looking to run the session only for a set of order numbers and / or dates, you can enter those during set up of job. i.e., open the session form,fill in the details and set it up as a job.

for date fields you will be prompted to verify if you need the date difference carried forward (i.e., if you have today' date in the 'to' field, by saying OK your to field is set to tomorrow when the job runs tomorrow if interval is 1 day.)

These defaults are set only for the user associated with the job run.
