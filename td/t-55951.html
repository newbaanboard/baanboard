I'm not sure what you are expecting. The BO Integration Kit and Driver is not supported for LN, and all BO support for all versions of Baan finishes with XIR2. However, you can use Crystal with Baan using standard ODBC database drivers - eg MS SQL Server Native client. Obviously there are disadvantages with enumerated fields, field descriptions, and text, but the reporting performance is excellent.
