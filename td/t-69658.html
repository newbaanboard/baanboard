in detail layout you will have to dynamically assign the value of Position Number and Item to an array, and pre calculate the value of quantity before printing the layout.

And for every detail you will have to check if it exists in the dynamic array? If it exists then do not print it and if it does not exists then you have to pre calculate the quantity and print it.

Use lattr.print = false for not printing the records.
