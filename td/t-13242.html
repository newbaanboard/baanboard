Hi,
If you mean that you want to read some existing text file sroted on the hard disk into the text editor and store it as an item text,
Yes this can be done.
1) Go to "Maintain Items"
2) On the item you want the text to attach, open the text editor for that record.
3) In the text editor, go to File -> Display/Copy File. Specify the file name.
4) Specify the block of the text which has to be copied & paste it in the relevant place as per instructions given alongside.

Thus you have text copied from a text file into the item text.

You can also use Exchange Schemes. Use the option multiline Texts.
I used it to import item text as we set up the our baan system. I could provide an excel file, where the user can enter a item number and several textlines per language and text type (item-text, purchase-text, sale-text). A macro then creates the importfile for baan.
