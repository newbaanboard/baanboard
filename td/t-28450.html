Well assuming you can create an array of warehouse and lots in the correct sorted order it is pretty simple to run through the records:

i = 1
| You use total warehouses here. Excess will be worked off in order of
| warehouses.
while i<=total.warehouses and total.short>0
if qty.warehouse(i) >= total.short then
|Send warehouse, total.short to report
total.short = 0
else
total.short = total.short - qty.warehouse(i)
| Send warehouse, qty in warehouse to report
endif
i = i + 1
endwhile

Something like this should work.
