Just set the standard variable fattr.total.line to true and the total line is created. The function display.total.fields can be used to set values for the fields.

before.program:
fattr.total.line = true

on.display.total.line:
display.total.fields( "whwmd000.amnt1", tot_1,
"whwmd000.amnt2", tot_2 )
