Set your preferences in the Tools Parameters session (ttadv0100m000) and execute the run-time option, logout & log back in for the changes to be effective.

Suppress Messages
If this check box is selected, suppression of messages is enabled. Authorized users can suppress message dialogs.

A user can only suppress messages, if the Manage Suppress Messages check box in the user's user data template is selected. See User Data Template (ttams1110m000).
Note

Suppression of messages only works in the Infor Ming.le-LN Plug-in.

Enable Remember Question Selection
If this check box is selected, suppression of questions is enabled. Authorized users can suppress question dialogs.

A user can only suppress questions, if the Manage Remember Question Selection Allowed check box in the user's user data template is selected. See User Data Template (ttams1110m000).
Note

Suppression of questions only works in the Infor Ming.le-LN Plug-in.
