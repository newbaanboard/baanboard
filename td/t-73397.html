We have a running Kardex integration with our Baan IV system. Kardex offers a wide range of interfaces which looks very easy for the first time. But when we got to the implemention, there were some details that made it necessary for us to implement workarounds because they could not configure our needs. We decided to use a simple file interface (Unix -> Windows); this works very stable.
