Hi,

Go to the Maintain Forms session (ttadv3100m000), browse for your form, click on "Edit Form" button, inside the form editor go to the "Form" menu and click on "Field Order". There you can choose if you want to use Maintain/Generate the sequence.

You can also use the to.field() function in side your program script. You can read about that function here:
http://www.baanboard.com/programmers_manual_baanerp_help_functions_form_and_form_field_operations_to_field

if you want to change of screen you have a predefined variable named autonext.form , you only have to assign true.

If you want a function for automatic change of field you can use this function.

function tabulador()
{
long event(EVTMAXSIZE)


evt.type(event) = EVTCHANGEFOCUS
evt.client.sender(event) = pid
evt.client.command(event) = 1
evt.focus.key(event) = KEY_TAB
if not send.event(get.pgrp(pid),event) then
message("proceso no responde")
endif
}
