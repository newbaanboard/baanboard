I can only think of achieving this thro' report script.
Basically you will need to code the logic of summation of total quantity in report script.
Something like as follows,

before.report.1:
before.layout:

detail.1:
after.layout:
sum.qty = sum.qty + line.quantity

after.report.1:
before.layout:
value = (sum.qty * price) * 70/100.0

You can access the total values by total..# where # is the serial number of the total value starting with 0.

total..0 is the 1st total value in the layouts of the report, total..10 the 11th

Not documented, but you can see it when you put a report in debug mode. I do not recommend using total..x - if you add another total field onto the report the total..x could change. So lets say total..1 is total qty and you add a total on price - the total..1 could now represent total price. All your formulas could change. I think you can find several threads on this forum about the total..x fields.
