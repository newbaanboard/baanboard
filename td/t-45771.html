Hello,

1.) create user B at OS level
2.) with ttaad2100m000 select user A and copy this user to user B
If different, edit the fields "system login" and "Name"
3.) <Ctrl>[E] = Copy Authorizations between Users
zoom ! --> convert to RDD

So you see, all basics can be done in one session.

Tools/User Management/General User Data/User Data (ttaad2500m000 / ttams1100s000)

If the new user already exists, make sure the Startup Menu in the User Data session -ttasm1100s000, first tab System Data - is the same for the exiting user.
