Hi, welcome to the Baan world. I'm not sure about Baan IV, but in BaanV you simply mark the session, hit the 'duplicate' button to duplicate the session. The system will then ask you if you want to copy the Forms as well. In case of report session, the system will offer to copy reports as well. Note: if you choose not to duplicate forms/reports, the old ones will be linked to the new session.

For the program script: if you want to use the existing script you don't do anything; the script will be linked to the session. If you want to duplicate that one as well, you simply duplicate the script, and in 'sessions' you assign the new script to the session.

Hope this helps you.

Kind regards,

En.

Hi Umesh,
Create the new PVRC's derived from the appr. VRC's.Maintain the
paths for the compnents of the newly created PVRC.
Create a new package combination and include the new PVRC's in the
package combination.
Rebuild the DD.
Create a new company and attach it to the package comb.
Convert to runtime. Create the tables for the new company. You can
also import data from any of the existing companies.
Login into your new company and enjoy exploring Baan.
