Hi John,

Standard BaaN dont have email send capability. You have to integrate third party software with BaaN. You can search here email send and I am sure, you will get enough info.

I have done this using sendmail and blat. That are very good sessions. We are able to send Salary Slips / IAD info. / PO / SO.

You have to maintain email IDs into table. You have to make text/CSV files using -
spool.fileout = shiftl$(shiftr$(tiitm001.csel))&".csv"
brp.id=brp.open("rtdinc040211001","FA",0)


Then using these tools you can pass eMail IDs, Text files as parameters.

Thats it.

Hope this helps.
