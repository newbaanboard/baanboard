Joop ,

Here is the part of documentation on field input . U can see the arrow keys , tab keys, escape key and zoom keys trapped Baan standard program itself to generate its own events. So not possible to trap these keys in standard 4gl programs .

---------part of documentation-------------
do.input.field()
if in.ret = ZOOM then
before.zoom
zoom.to ...
after.zoom
goto zoom.ret

endif
if choice = f.to.choice or
in.ret = arrow.up/down/left/right/tab then
goto after.field
endif
before.checks

-------------------X------------------------

Using event generating functions which I suppose u have tried , u may be able to trap these keys but u would loose the benefits of standard program pre-defined logic .
