Hello.

I have a simple script that will check a variable and upon a certain condition make a field invisible or not. When the field to be made invisible is of domain tcitem it will not work.

If I temporarily change the variable to some other random domain, say tcpono, it works as expected. Is there something extra to be done maybe with a segmented field?

Running the command is.invisible("var123") is returning true or false correctly. The form in BECS and UI however are not hiding the field.

sample code:
field.varxyz:
after.field:
if varxyz = tcyesno.yes then
inputfield.visible("var123")
else
inputfield.invisible("var123")
endif

Yes, you need to mention the specific segments in the call:

inputfield.invisible("<your item field>.segment.1","<your item field>.segment.2")
