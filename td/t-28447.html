David,

Which particular BaaN version are you using in this instance? Are you up to date on the tool patches/solutions? There are few fixes related to display browser in LN Tools.

file.display() - is this your own own custom function or a baan function?

The tools standard function is:

function extern display.file( domain ttaad.path file mb,
domain ttdesc60 file.name mb,
long rep.width, long rep.height,
domain ttaad.clan bidi,
string activated.by(13),
long rm.file)
###################################################################
Function to display output file e.g. from bic6.1
by using the display browser.
###################################################################
Pre: file : complete path of file to be displayed
: max = 60 characters
file.name : used in caption of window
rep.width : max.width of lines, used for
sizing of window
rep.height : amount of lines, used for paging of file
bidi : used language code for bidi
actived.by : calling session
rm.file : remove file after displaying.
Post:
File will be removed after exiting the browser when flag rm.file is true.
###################################################################

The vertical and horizontal scroll bars will be initialized based on the height and width and the line length and number of lines in the fille.
