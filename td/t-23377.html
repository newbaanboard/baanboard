In Baan V you find the same option under (Maintain) Sessions, 'Specific' pull down menu, 'Indices' option. Just need to check the new index, give it a Sort order (the order the indexes appear in the drop down when the session runs), and the View (for Type 3 Forms). Compile the session when done, then run, the new index should appear.

Note: the same procedure applies when you add a new text field to the main table and want it to appear with the 'T' button - use the 'Text Fields' option right below Indices.
