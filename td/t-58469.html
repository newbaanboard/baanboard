Not sure whether this answers your question, but you can design a regular expression describing valid file paths, and then use expr.compile() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_expressions_runtime_expr_compile) with the IN operator and l.expr() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_expressions_runtime_l_expr) to check whether a giving string is a valid file path.
