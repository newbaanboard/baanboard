Hi,
You can use the function inputfield.invisible(...) to hide form fields. The argument can be comma-separated field names...

Example: if you want to hide fields B & C if the input in the field A is "No"...

field.A:
after.input:
if A = tcyesno.no then
inputfield.invisible(field.B, field.C)
endif

Those functions are only available with dynamic forms, aren't they? Then in BaanIV they won't help.

As far as I know, you can not dynamically show and hide fields in BaanIV. You can, however, disable input:

field.b:
before.input:
attr.input = (a = tcyesno.yes ? true : false)

In BaanIV you can also use:

field.b:
before.display:
attr.echo = 0

Value in field will not be displayed.

It's not possible to hide the field in BaanIV
