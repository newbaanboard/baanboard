Hi,

I didn't look at that code before. However, you can use the net send (Windows) command to send messages. The messenger service (http://support.microsoft.com/kb/168893) must be activated in order to send/receive messages.


D:\Documents and Settings\georgea>net send /?
The syntax of this command is:


NET SEND
{name | * | /DOMAIN[:name] | /USERS} message

I hope it helps,
