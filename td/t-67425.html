Hi,

You need to create a 'Question" for that and not message.

For that, provide the domain as tcyesno.

Then use this -

ret = seq.open( filename, "w" )
while ( ret <= 0 )
if ( ask.enum( "tccom00001", tcyesno.yes, ret, filename )
= tcyesno.no) then
return
endif
ret = seq.open( filename, "w" )
endwhile
