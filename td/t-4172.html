If you "maintain a report fields", you can also chose a barcode type. The definition type for that has to be in your Sytem. Normaly this files are in a folder "barcode". The path to this folder has to be declared in your printerdriver. Often the last line.
Use the field help in "maintain report fields" to find out about the rules how the barcodefiles on your system have to be named. Normaly tehy are named: type01, type02, type03...... Then you have to enter for the barcode type in "maintain report fields" a "2" if you would like to print a "type02"
