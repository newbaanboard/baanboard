NAME
bell$ - Warning bell character sequence.
SYNOPSIS
string BELL$()
DESCRIPTION
The function BELL$() returns the character sequence that makes the warning
bell on the terminal ring. The bell sequence string can be send to the terminal
using PRINT. Note that the bell only rings after the function REFRESH has been
called.
EXAMPLE
print bell$()
refresh()
