It comes from tfgld set of tables (viz tfgld201-206) . It calculates opening balance from opening balance tables for a ledger / dimension for a particular year and totals upto period home currency figures for all currency in that year for period type fiscal and prints in the report . And finally it calculates closing balance = OB + debits - credits .

If non-finalized included it will use non-finalized figures in table and if it's for complete group it will total up the figures for all companies in tfgld001 .

Hope this helps.
