Yes, you can use it with multiple fields but you have to spearate the fields by comma.
E.g.
query.extend.order("field1, field2, field3 desc") <- field1 & field2 ascending order, field3 descending order
query.extend.order("field1 desc, field2 desc, field3 desc") <- all fields in descending order
