Han Brinkman
3rd June 2004, 07:11
You probably meant a library. So you have to compile the libraray in debug mode as well.

Functions in Baan are what in C is called include files. So they should be in debug mode if the script is in debug mode.

Normally one of the specific options gives you the possibility to see which libraries are linked to the script. (or pragma useddll statements are used in your script).

Just one addition to that: Sometimes it's impossible to compile a DLL with debugging (e.g. on a live system; all users would be affacted ...). In such a case, I pick pieces of the DLL code (1) or even the complete DLL code (2) and add it directly to the script and compile that (only) with debugging.

1) pieces of code -> This requires a bit more work, sometimes lots of declarations are needed as well.

2) the complete DLL code -> if that is possible and no variables appeare twice.
