Hi Linya,

Make sure that the VRC in which you have that session has Software Configuration Management (SCM) active. Because if it is active then you cannot modify that session directly as seems to be in your case. You have to check out that session first to enable those checkboxes, after which the abovesaid changes will come into affect.

Remember, check-out first & then try to modify.
