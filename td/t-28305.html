Jatin,

I checked out LN system and there is no $BSE/include6.2/bic_tiv

From LN series, some of the tools includes have been moved to $BSE/tools/tt7.6_a/ittbic/ and there is the file - ibictiv0

From your screenshot, it appears that the compiler is searching for this file in ${BSE_REM} a remote host. Are you using the MAS (Master Application Server Setup)? If not, this parameter should be empty. Check the BaaN environment variables.

Also, check in the fd6.2.<package-combination> for any BSE_REM entries and also for the following line:

itt:${BSE}/tools/tt7.6_a
