Hi,

You need to declare the variable as extern -

declaration: ;
extern domain dmpris stock.value
detail.1:
before.layout:
stock.value = dmoes001.stck * dmoes001.spri

Or, directly add the variable in Report Input Fields by selecting the report -> Specific -> Report Input Fields. Then, you don't need to declare it in your script.
