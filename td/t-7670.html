baanprog,

The function, ttseq.automess.on() is located inside a tools (tt)include. The source of the include (tt) has to be present on our systems/environments for our scripts to be compiled and BaaN does not deliver the tools sources (tt) to the customers. Hence, I think its for BaaN internal use only.

If a function is defined as an "extern" in a tools DLL we can use it even if we dont have sources or we can use standard 3-GL standard functions BaaN provides us using the #include<bic_xx> statements.

I hope this clarifies the issue.
