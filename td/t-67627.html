Hi,

This is a known bug and will happen every time for the new scripts after 8.9a.05 Portingset update. Infor has released a Tools solution for this bug. KB 1581192 has all the details. Please check and get the solution downloaded in your environment.

The old scripts might need manual update of TIV version as specified in earlier post.
