Is there equivalents sessions to ttadv3240m00 and ttadv2230m000 for forms???

Refer to -

Create Runtime Data Dictionary (Forms and Menus) (ttadv5214m000)
Use this session to convert the forms and menus, or changes in existing forms and menus, to the run-time data dictionary.
You can use the Multilevel check box for software distribution purposes to ensure that only the latest version of the forms and menus are available. This is instead of all versions in all package VRCs that are available. Define up to which package version the forms and menus must be converted to the run-time data dictionary.
Run this session, for example after changing/translating labels. Note that this session must also be run after changes in software made by the following utilities:
Copy Software Components to New Package VRC (ttadv6265m000).
Delete Software Components by Package VRC (ttadv6250m000).
Import Language-Dependent Software Components (ttadv6231m000).
