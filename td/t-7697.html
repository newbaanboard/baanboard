spartacus,

If you like to code in the choice.cont.process section you need to use spool functions -


choice.cont.process:
on.choice:
if spool.open("","",1) then
start.process()
spool.close()
else
choice.again()
endif


for more info on the spool functions refer to the link - spool.open (http://www.baanboard.com/programmers_manual_baanerp_help_functions_spooling_spool_open)
