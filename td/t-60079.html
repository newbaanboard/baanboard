Hi jcropp,

Under Windows XP the Settings of BwPrint were saved in the Registry under [HKEY_LOCAL_MACHINE\SOFTWARE\Baan\BwPrint\Fonts]. You can export these Registry-Entries into a file and change the string HKEY_LOCAL_MACHINE to HKEY_CURRENT_USER and now install that file on your Windows 7 workstation. Also you can set the fonts and spacings on one Windows 7 Workstation and export these settings from [HKEY_CURRENT_USER\SOFTWARE\Baan\BwPrint\Fonts] into a file and install these registry settings on all other workstation if it works correct.
