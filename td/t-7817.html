If I interpreted your question correctly, I would assume not.
The function message(...) can be used in these type sessions, but the user must depress the "OK" button in order to continue and the message is removed.
This is equivilent to the instruction mess( "messcode",1 )

The instruction mess( "messcode",0 ) causes message to be displayed, without an user interaction, in a status field of the form which you do not have.

So may conclusion would be no it is not possible, but you can try it to see what the result is.
