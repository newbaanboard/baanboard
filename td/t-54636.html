First a qkey script will have |#parent in the script. You must get your declarations in the correct spot. If you put then inside a pobj block they will be stripped out. One thing you can do is to use the qkey editor - I think it is qkvi. This will open the script, strip out all qkey commands. Then you can put your code in. You can see an example that I put up in the code and utilities forum.
