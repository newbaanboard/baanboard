Hi,

If you are using india localisation then you can use function tdind0003.decode to convert figures to words.

usage :

itdind0003.decode(amount,1,decode0,decode1,decode2,decode3)


Input variables:
- amount to be decoded, maximum 11+2 i.e amount
- decimals required (boolean)
Output variables:
- decode0 Decoded amount of 130 chrs.
- decode1 Same decoded amount in two strings of 65 chars
- decode2 "
- decode3 Decode of the decimals

decode0 is defines as of domain tcmcs.str130 other decodes as tcmcs.str65.


remember to include dll as
#include "itdind0003"
