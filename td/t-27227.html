U can also use ttaad4500 ( General table display )

Select te needed table ( Package / Module / table / company ). On each field are able to zoom and request the possible values.

After the needed table is selected, press 'Y' to continue. All fields of the table are visable.

Press '*" to change to programmers mode, press '0' ( zero ) to show the related domains.
