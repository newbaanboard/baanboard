The problem you are facing is that Baan Client is not able to open the editor. The reason is easy: It cannot find the path to it.

Workaround:
1. you can copy the file (probably wordpad.exe) to your baan inslalation path, for example C:\Program Files\Baan\Bin. It is only 200 KB...
2. Or, you can write the whole path to the editor (executable) in Developers parameters. Sometimes it is a problem if the path contains spaces.... This is most often used if you have a "smart editor".

Be sure that you have chosen to open the editor locally.
