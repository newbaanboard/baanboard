once upon a time this worked

Set BaanObj = CreateObject("Baan4.Application.701")
ThisWorkbook.Sheets("Performance").Activate
BaanObj.Timeout = 10
On Error GoTo BaanAutomationError

B_function = "call.session(" & Chr(34) & "tiitm0101m000" & Chr(34) & ")"
BaanObj.ParseExecFunction "otdapidllrecpt", B_function


Where call.session is a baan dll which did this:

function extern call.session(const string sessname())
{
zoom.to$(sessname,Z.SESSION, "", "", 0)
}


This was as far as I took it. Never really had a reason to go farther. You can search the AFS/DDC/OLE forum for other potential solutions.
