Maybe we can help with more info.

You want the error text diverted to a file, yes? Are you in control of the source code? If so, instead of using the 'mess' command, you can create an ascii file yourself and send the text of the error to a string then to your file. This can be done using a command like:

error.mesg = form.text$("tihra1101.01")
|* Employee not allowed

|* tihra1101.01 would be a message number defined in Maintain Messages

To fill messages that have a variable in them, you can use something like:

error.mesg = sprintf$(form.text$("tihra1101.01"), emno.in)
|* Employee %d not allowed

If not from your code, let us know what type of messaging you are trying to capture.
