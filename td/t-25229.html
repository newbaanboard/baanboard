How similiar? Is something like the first 20 fields the same? If it is something like that then you might be able to use the record buffers(rcd.taracp010 = rcd.taracp014). this would set the first x number of fields the same. Then all other fields must be set - don't skip one. If only the first couple of fields are the same then just set them manually.

I agree with mark_h . additionally If there are relations between these fields of tables.
You should select these fields in SQL and use REFERS TO . For example Table_A.field1 relates TABLE_B.field1
SELECT TABLEA.fileds
from TABLE_A,TABLE_B
where TABLE_A.field1 REFERS TO TABLE_B
