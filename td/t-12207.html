Whether you build your own solution through app_start() or use Baan Hot Link from NAZDAQ, you do not have to install AutoCAD on your client in order to view AutoCAD files.

You can install an AutoDesk product called "Volo View Express" - I think it is free. This product can be installed at any client that wishes to view AutoCAD files but does not necessarly want to maintain these files. Check www.autodesk.com under "Products" for more information about it.
