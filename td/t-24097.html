You may also need to check out the data for the report. Sometimes a non-printable character gets into the data and causes this error. Basically the report expected to read x number of input variables and instead found y. Put the report in debug mode and search for string.scan. This is probably where the error occurs. By checking what it did read in (in debug mode) it may help you find the bad data.
