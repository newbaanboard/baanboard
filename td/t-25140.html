From your code i think you want to get the immediate previous date (tcmcs008.stdt) from tcmcs008 for the field (****.stdt).

If it is so, then you can try the following code :


select tcmcs008.stdt
from tcmsc008
where tcmcs008.stdt<:****.stdt
order by tcmcs008.stdt desc
as set with 1 rows

It means you will get the max date (tcmcs008.stdt) but it will be less than the date you compare with.
