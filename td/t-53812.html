Hi,
The session that changes the package combination of a company would be {ttaad1101m000} Change Package Combination by Company. Thats the session thats available in both Baan4c and Ln and so i guess it should be the same for Baan5.
The other considerations would be, to move your customized objects (sessions, forms, reports etc) & users to the new package combination. The objects could be copied by using Export Data Dictionary {ttiex1280m000} and restored to the new customized VRC package using Import Data Dictionary {ttiex1285m000}..

Please Note: Keep atleast one super user (baan) attached to the standard VRC just in case the other users are not able to log in with the new customized VRC.
