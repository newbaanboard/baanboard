Hi Baan1612!

Follow the following steps and you may be able to print directly to EXCEL without any modifiucations to Baan reports!


Set up an EXCEL-Device if not already there:
Device type : rewrite file
4Gl Program: ttstpconv
Argument: ASCII:excel.exe %s
Path: excel.csv
Change the List delimiter in Start -> Settings -> Control Panel -> Regional Settings. Click the “Number” tab and in the “List Separator” field, replace the current default separator (usualy ";") to Pipe ("|") (In Windows XP there is an extra step involved.
When you open the Regional settings window, you have to click customize next to the language choice drop down.)
Print your report to EXCEL-Device!

If you need more then 255 columns for a baan report designed for MS-Excel or you will not use any delimiter (like '|'), you can customize your report like an XML. The effect will be that Excel opens the file and the user will have nothing do do (like: this filed ist numeric and so on). With MS-Excel Version 2003 it is possible to give out an XML (spezial designed for Excel) with a Baan report. This report contains XML tags which are needed by Excel. You can try this by yourself: Open a new Excel file, type in data and save this with extension *.xml. Then open this file with any editor and you'll have a look on the structure. Then move this file to another name with the extension *.xls and double click an it. In the attachment you'll find an example.

Good luck! Peter
