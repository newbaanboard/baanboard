Hello,

ORA-04098: trigger "string.string" is invalid and failed re-validation

Cause: A trigger was attempted to be retrieved for execution and was found to be invalid. This also means that compilation/authorization failed for the trigger.

Action: Options are to resolve the compilation/authorization errors, disable the trigger, or drop the trigger.

You can also try running the following show errors command to check for errors in the trigger:

SHOW ERRORS TRIGGER trigger_name;

When you have a syntax error in a trigger, you can issue this query to see the exact reasons why Oracle threw the ORA-04098 error:

select *
from
user_errors
where
type = 'TRIGGER'
and
name = 'TRIG_NAME';
------------------------------------------
