The idb functions cache (relatively small) database tables. This is faster than reading the data from the database over and over again. If you want more information, see the dll usage of the dll where this function is defined: tccomdll0201.
In order to make your script compile, add this library to you program script. I assume there are other unresolved functions as well, because you cannot use this function on its own; you need an initialization.
