function long before.save.object(long mode)

I guess your question refers to the mode argument.
In case of a dal.new() the value of mode is DAL_NEW (i.e. -1). In case of dal.update() the value of mode is DAL_UPDATE (i.e. -2).

Within the before.save.object() hook you can test the mode you're in like this:


on case mode
case DAL_NEW:
|* Here the code for an insert. Mode will have value -1.
break
case DAL_UPDATE:
|* Here the code for an update. Mode will have value -2.
break
endcase
