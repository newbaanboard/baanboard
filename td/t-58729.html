Hi Richard,

You can try to modify the bw.exe using the software 'Resource Hacker' (http://www.baanboard.com/baanboard/showthread.php?t=53385)

Again as Sushil told in the link mentioned by me above is that you have to try it out on your own risk.

You can get this software by googling it.

I have tried it on my machine. I followed following steps
1. Open bw.exe in Resource Hacker software.
2. Go in the 'Dialog' menu of it. There will be 2 sub menus '144' & '145'
3. Open sub menu '145'. You will find one node in it named '1033'
4. When you select this node you will find the login window at right bottom and also some script for it.
5. the first line says "145 DIALOG 0, 0, 245, 80". Change the highlighted '0' to '-1' and save it.
6. On saving, it will automatically change from '-1' to '65535'.
7. Now when you try to run this new exe it will pop up the ligin window on top left of the screen.

Hope it helps you.
