Here is some info I found from my old emails and on here -

http://www.baanboard.com/baanboard/showthread.php?s=&threadid=656&highlight=ttdskcs


the purpose of the sessions
(1)ttdskcs - Colour selector,
(2)ttdskfs - Font selector and (3)ttdskmsg?

The color and font selector were used in the dsk module (e.g. Desktop Manager and Menu Browser). These sessions are no longer used, because the Windows defaults will be used instead. Use of these sessions can still be viewed in session ttdsk3160m000.

The session ttdskmsg is used by the Menu Browser to display messages.
