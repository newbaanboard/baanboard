My guess: you don't pass a fixed local path to server2client and your file is sent to Baan tmp directory (like "Program Files\Baan\Tmp" ). It worked with XP, but Vista and W7 does not accept an aplication writing temp files in Program Files directory (unless ran as Admin) for security reasons. You may:
- obtain newer BW version, I think that new versions use "Program Data" to store tmp files
- or, rewrite your script and put a fixed path
