As far as I know there is no document that says which function is in which dll. Using bic_info on the baan server will reveal the functions in a DLL. So you can search for examples on this board or the DLL itself, plus try using the bic_info6.1 (assuming your version is 6.1) to look at files. The only one I know is ottstpapihand for the stpapi functions.
