Use the sort option by descending date, and use 'as set with 1 rows"


select tcmcs008.*
from tcmcs008
where tcmcs008._index1 = {:yourcurrency}
order by tcmcs008.stdt DESC
as set with 1 rows
selectdo
found only your record
selectempty
no records for currency
endselect

small addition. don't forget date.
select tcmcs008.*
from tcmcs008
where tcmcs008._index1 inrange {:yourcurrency,0}
and {:yourcurrency,:DateYouNeedOrToday}
order by tcmcs008.stdt DESC
as set with 1 rows

found only your record
