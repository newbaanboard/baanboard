If I have understood your problem correctly. Try this solution.

suppose there are two footers.

after.footer.1

after.footer.2

and x is contion. If x is true you want print footer.1 and if is false you want print footer.2
So write this below script it should work.

after.footer.1:
before.layout
if not x then
lattr.print = false
endif

after.footer.2:
before.layout:
if x then
lattr.print = false
endif

But specify footer margin 0 in reports second form.
