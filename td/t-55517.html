Hello,

when you dump the tables with ttaad4226m000 on the source system then check the box: "Exclude logical tables from dump". If this box is checked, then you can export all companies in one run. The logical tables will be dumped only once.

You can do the import on the target system also in one run for all companies. As recommeded earlier modify ttaad4120m000 before you import the data.

Hello gget79,

when you export all companies for the range from 117 to 617 in one run with the checkbox "Exclude logical tables from Dump" marked, then:

- All tables from company 127 will be dumped. (they are all physical tables)
- Only the physical tables from company 117 and 617 will be dumped.

The logical tables in company 117 and 617 (which are linked to the physical tables in company 127) will be only once in this whole dump (as tables for company 127).

If you have other companies in between, then you can do the whole thing also in three runs for evry single company with the checkbox marked. The result will be the same :-)

Regards

Dieter
