I have successfully emailed (thro' outlook) from BaanIVc4 using Java classes that use the Java's mail API. I don't think there is a direct way to do it from BaanIVc4 without using any 3rd party tools or programs. The java classes would obviously be like a 3rd party program but it is free and most of its source is open source that is found in the examples along with the java mail API. You would have to install java on UNIX to execute / run these classes. From Baan side; all a user has to do is press a button or hit continue and everthing happens in the background. You can execute a shell script from the Baan program which in turn will execute the java classes to email the files. Again, the java classes can email with an attachment in any format. But the file to be attached has to be converted in whatever format you want, before emailing it.
