That's the (non)beauty of release notes. They are basically useless. It's pretty much an undocumented feature in Baan, but release notes DO NOT go over when you move an object from one VRC to another.

You should be using TECHNICAL DOCUMENTATION to document your objects and stay away from release notes. Release notes are only good for an object which will stay put in the VRC. e.g. temporary notes about why the object is in the VRC or whatever.

Tech docs go over whether you copy to existing VRC or copy the object to an entirely new object.

I personally never use release notes and always use tech docs to document. That and customization admin, which not many people use, but is great bang for the buck.
