Hi,

No you can't customize any tools session. But, I will recommend you to revisit the requirement. I am assuming, the requirement is to add Invoice Date field to one of the query.

To that that, you can add Invoice date to the required query. Select the query name -> Go to Specific -> Query by Easy SQL -> Select Query table - > Go to Specific - > Query Fields.

Now, you can add the Invoice Date field and click "On Form" checkbox to get it on the form.
