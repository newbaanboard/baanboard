Using ERPLN from FP3 (with should be pretty up-to-date) and later versions (FP5) you can very easily achieve this using birt.

You can convert the standard reports to birt format, import it into birt, modify the report and export it again to ERPLN. You need to print it to the RPT device.

It's only working within Webtop.

But having seen this I can tell you that it's a piece of cake to achieve this.
