Here is an example for handling substrings for multi-byte variables.

long counter
string str(10)
string mbstr(10) MB
...
|*
|* Wrong programming
|*
for counter = 1 to len(str)
if str1(counter;1) = ";" then

endfor
|*
|* Right programming
|*

endfor
