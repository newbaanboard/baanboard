I have the following code in a report script to control the printing of a layout.

if ( itm.csig = "MLA" ) then
lattr.print = false
endif


I now want to change the above to include all itm.csig that start with ML. What's the best way? Can I use a wild card or do I say "If itm.csig = XXX or itm.csig = XX1 or itm.csig = XX2"

a simple answer from another amateur(!), you could write:

if ( itm.csig(1;2) = "ML" ) then
lattr.print = false
endif

This will test all itm.csigs starting at character 1 for 2 characters.
You may wish to shiftl$ first if you also have an option "ML" [i.e. no 3rd character] and the field is right aligned.
