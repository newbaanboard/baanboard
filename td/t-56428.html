Here is a working example from a code I wrote:

save.file = seq.open(filename, "r")
while not seq.eof(save.file)
cont = seq.gets(line, MAX_LINE, save.file)
if cont<>0 then break endif
| do what ever you want with the text...
endwhile
seq.close(save.file)
