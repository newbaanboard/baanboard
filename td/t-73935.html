Hi!
There might be any of 2 reasons your are not seeing that table in the list.
1)Either you have entered some value in selection criteria of the session, which does not match with specifications of the table(ex : VRC, Package etc.).
To get the full data, do not enter anything in the selection criteria of ttadv4420m000 and click on print.
2)or You are searching the table name literally as 'ttfacr200'. the preceding 't' stands for table in scripts. so the actual name of table is 'tfacr200'.

Hope this helps.
