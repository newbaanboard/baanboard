There are many ways to solve this simple issue in LN:

1. add code in the when field changes section for each field

if field1 = tc.yesno.yes then
field2 = tc.yesno.no
|* disable.fields("field2")
else
field2 = tc.yesno.yes
|* enable.fields("field2")
endif

2. If they is a enumerated domains you can use them in the DFE as radio buttons, list box, drop down so that only one option can be selected.

3. If they are not enumerated domains then you can still dynamically create a listbox using the tools functions - set.list.values.for.field() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_form_and_form_field_operations_set_list_values_for_field), set.enum.array.for.field() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_enumerates_set_enum_array_for_field)
