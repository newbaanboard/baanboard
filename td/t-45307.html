You need to do as follows:


choice.text.manager:
before.choice:
if tisfc001.osta = tcosta.closed and tisfc001.txta > 0
then
attr.textmode = 1 |* 1 = Read-only
endif


Somewhere else in the logic, you may need to set this back to Read and Write:

attr.textmode = 3 |* 3 = Read and Write
