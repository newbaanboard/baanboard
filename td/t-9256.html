Within Baan IV there are three ways to add link new sessions to the "Special" menu option of another session.

1. Within the form editor the "choose" option offers one "zoom-to" option. Enter a menu or session here and it will automatically activated via the "special" menu. Note: This option is not available in some of the older versions of Baan IVb.

2. Add the new session as a user-defined choice option. This option can only be used if you have access to the source code of the program script in order to add code within the "choice.user.<number>" section. If so, you can add "zoom.to$" function to launch the necessary Baan session when the user defined option is selected.

3. If the session is already calling a menu (via the "choose" or other means) then the menu can be modified to add/remove sessions. Note that this procedure is very common for standard sessions.
