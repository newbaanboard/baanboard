To activate the device you want directly, use the brp.open() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_reports_brp_open) or the spool.open() function (http://www.baanboard.com/programmers_manual_baanerp_help_functions_spooling_spool_open). Then you do not need to go to the "Select Device" session at all. If you were willing to edit your session script you can use that too.

However, we still need to cancel (kill) the old device. It is doable (you need to find the PID of the device session and kill it), but maybe there is a better way after all. I will let you know if I thought of something better.

- George
