Sometimes the way the layouts are arranged in Maintain Layouts is not the way they will actually print when you run the report. This is because the index on the ttadv331 table is by the field name, not by the sort value defined in the "Input Fields" session. This can be especially confusing (and irritating) when using the "Maintain All Layouts" option because the layouts won't appear in the order you desire (but will, of course, print correctly later).
