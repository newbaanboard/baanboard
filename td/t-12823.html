By super level, do you mean a menu with the four (now five ;)) entries? Having that would make the menu tree deeper. Also, if you log on using ASCII (ba6.1) you will have the following:

Finance
-->Cach Management
---->Supplier Payments

And this last would be four screens "next" to each other, activated by using the cursor keys (ASCII!!). That's why they call them parallel menus! They co-exist "next" to each other. Since the introduction of the BW, this became less apparent. And, to have a way to display the "parallel" menu's, Baan had to think of a way to make them all visible, hence the new level in the menu structure.

Hope this clarifies it a bit!
