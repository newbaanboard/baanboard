So what I did once was I sent a sort key with the data(k.of.data). So I passed like a 1 for those in detail.1, then a 2 for those in detail.2, The report used k.of.data as the sort key - so as it organized the data with the 1's first, then the 2's. So then as the report prints the 1's the output expression on detail.1 is k.of.data=1. So detail.1 only prints for the 1's. Same for detail.2 (output expression is k.of.data=2). So detail.1 would get printed only for the 1's and detail.2 got printed only for the 2's.
