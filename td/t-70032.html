You can import the designs into the Prod LN dictionary with ttrpi1200m100. Then they will be stored as additional files. You can make an export of the additional files with ttiex4500m000 or ttiex1280m000. Then import those into the Test environment and export to the test IR server with ttrpi1200m000.
You can also define the Prod IR server on your Test environment, hardcode the host and package combination in the folder path and do the import directly into test from prod.
