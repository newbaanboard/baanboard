If you use the command
select t$tdat,to_char(t$tdat,'dd-mon-yyyy') from ttdsls030111 where rownum=1;

the output is (01-JAN-99) (00-000-0000)

hence what is the better way to detect null date values..
1) to_char(t$tdat,'DDMMYYYY')='00000000'
2) t$tdat <= to_date('01/01/-4712','DD/MM/SYYYY')
