Deepak:

Baan is not known for good error messages, but this one is kind of obvious:
"Can't open input file '${BSE_REM}!${BSE}/include6.2/bic_tiv'"

The missing file is one of the bic include files. As far as I remember, TIV is a recently introduced version number, so that the application can check for specific porting set features.

Anyway, it depends on the porting set that you are using, but solution 114812 provides you with the latest bic include files for B50_b, B50_c, 73a, B40c, B40b, and 61b. I am just missing the latest porting set release for ERP LN in the list, so you might want to ask SSA Global (now Infor) support for the latest files.
