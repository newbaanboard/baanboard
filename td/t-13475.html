You will have to start with creating a gwindow object (see create.object (http://www.baanboard.com/programmers_manual_baanerp_help_functions_user_interface_objects_create_object)). Then, within this gwindow, you will have to create a subobject, with type DsCgpArc (see create.sub.object (http://www.baanboard.com/programmers_manual_baanerp_help_functions_user_interface_objects_create_sub_object) and DsCgpArc (http://www.baanboard.com/programmers_manual_baanerp_help_functions_user_interface_objects_dscgparc)). The minimum needed attributes would be DsNx, DsNy, DsNwidth, DsNheight (to specify the bounding rectangle), and DsNangle=0, DsNrotation=360.
