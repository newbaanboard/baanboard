You have to customize the form and the script of the calling session.

First, you have to define an user option to use for zooming your new session, like user.N where N is the first number free for use in the form.
If you need a button also, just define it in the buttons of the form.

In the script, add:


choice.user.N:
on.choice:
.... set the variables for find data in your session ...
zoom.to$(.....)

To add a session, you could also open the form and have a look at the choice field. This field is sometimes linked to a menu so that you could add a session to it (and dump it).

But if you like to start the session with a given record, it might be necessary to use some code for the calling session or qkey.
