If you know have the permission to the session ttadv2530m000 then run the above session it will take you to the programs scripts. Now find the script which you want to identify the level of customization now select the option print there you can find the optioin "Print Differences Between Prgroam Scripts/Libraries" and choose the original standard VRC and vrc in whcih you want to check for level of customization. Then it will give a report with the differences in the program script, by seeing that report you can come to know the level of customizations done. But for doing the above you need to have both standard and customized scripts.
