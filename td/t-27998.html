Hi,

If you are trying to retrive the data from only tiitm001 table then you can try following query:

select tiitm001.item from tiitm001, tdpur041, tdsls041
where tdpur041.item = tiitm001.item
OR tdsls041.item = tiitm001.item
group by tiitm001.item
selectdo
endselect

Here, don't select any field from tdpur041 and tdsls041 tables.
