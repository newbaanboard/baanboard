Hi,

Declare a variable
extern domain tcbool print.detail

Set it to TRUE in SELECTDO
Set it to FALSE in SELECTEOS.

In the DETAIL LAYOUT declare it as the print condition "print.detail = TRUE".

For the SPECIAL DATA create an after report layout and put them there.otherwise you can declare one more detail with the print condition as print.detail = FALSE".

Hope this helps.
