Example code-

choice.text.manager:
before.choice:
if zmmig200.stat > zmmig.stat.qa.approved then
attr.textmode = 1
else
attr.textmode = 3
endif

Refer to the predefined variables -

long attr.textmode
1 display mode
3 read/write mode

Used by the Text Manager.
