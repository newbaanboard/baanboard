Multibyte strings overview and synopsis
Overview
Use these functions for handling multibyte strings.

The character sets used in languages all over the world require often more characters than the 128 characters available in the ASCII table (C0 Controls and Basic Latin) character set. BAAN software provides foreign character set support via the BAAN Super Set ( TSS Encoding). In TSS, a character occupies one or four bytes. Multibyte TSS characters always start with the hexadecimal byte 0x9B. So they can always be recognized within a string.

See also String operations overview.
