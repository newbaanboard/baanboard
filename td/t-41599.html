Hi Bey rnd,

one other way to acheive what you want ist to use an endless loop. Following is the code skeleton that I've running by a customer for entering delivered goods just like a POS-Terminal:

declaration:
|this is the form field:
extern domain tcmcs.str20 form.barcode

form.1:
init.form:
main.loop()
...
function main.loop()
{
reset.all()
quit = false
repeat
input("form.barcode")
on case form.barcode
case "COMMAND1":
do.something()
break
case "COMMAND2":
do.something.else()
break
....
case "QUIT":
quit = true
break
endcase
until quit
execute(end.program)
}
