I have seen the following problem on windows 2000 desktops...here is a solution suggested in worktop technical documentation.

A common Windows 2000 problem is that when Baan sessions are launched from within Worktop, they appear in back of the Worktop application (I've also noticed that session icon on taskbar blinks instead). This problem is not restricted to Worktop but many applications will show this behavior. The reason is that Windows 2000 by default does not allow applications to "steal the focus". To solve this, download the TweakUI application from the Microsoft site. Extract the executable. After extraction, go to that folder and find the "tweakui.inf" file. Right click on this file and select the "Install" option from the context menu. After installation, double click the "Tweak UI" icon located in the Control Panel and go to the tab called "General". There is a checkbox with the label "Prevent applications from stealing focus". Uncheck this box.
