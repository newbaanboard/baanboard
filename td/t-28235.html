Hi,

There are COM interfaces available for Baan application that you can use in VB or VC++, create an instance as CreateObject("Baan.Application"), then call the method "ParseExecFunction" of created object and send library or session name , calling function etc as parameters. The developer machine should have Baan client installed to instantiate the Baan application object.

Search on forum, you will find more detail.
