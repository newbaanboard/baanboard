Is there a function or command in Baan that will reverse a string?
I don't think so. But is here the code:

function extern long strrev( const string sourcestr(),
ref string result())
{
DLLUSAGE
Reverse the input string
Warning: The variable "result" in the calling program should be declared
as based else the bshell gets disconnected.
ENDDLLUSAGE

long i, length

length = len(sourcestr)
e = alloc.mem(result, length)
if e <> 0 then
for i = 1 to length
result(i) = sourcestr(length + 1 -i)
endfor
endif
return(e)
}
