If you sending data from session to report using rprt_send() function then you can actually define the field in the session itself. Initialize it to 0 first and then increment it by 1 as and when you are sending the records to report. Increment it by 1 before rprt_send() function.

You need to define this field as the report input field than you can actually use it in the detail layout.
