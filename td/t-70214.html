Hi Sangeeta,

You can import images using Import Additional files(ttadv2270m000) session and use these images in IR Report. In IR Report you need to add image from Toolbox and select Source Type as "Report Expression" and give report expression as tt_getAdditionalFile(ParamDisplayValue('datafile'), 'ta', 'gen', 'image_name.gif')

I hope this will help you.

Regards,

You need to keep all images on server in any folder not necessary in $BSE/Application. Session ttadv2270m000 will upload all images at once to mentioned package and module.

We have imported logo images with gif extension in ta-gen(Package-Module) with same session and with help of report expression fetching these images on IR Report.
