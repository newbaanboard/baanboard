I don't know whether it is the same in LN... but in 4c4.

Run bwprint.exe from your local client Baan directory
e.g. c:\program files\Baan\bin\bwprint.exe


When it opens, click Help, then Font Test.
Fonts appear in the window.

Double click the line displaying an unused font e.g. font 11, old name pfont3
A box appears where you can map the Baan font (font 11) to a Windows font, e.g. Arial 10
Close bwprint. The mapping will be active for that client computer only.
You have to repeat his exercise/import the font mapping (from the Windows registry) on another computer if you want it to have the same settings.

Then use font 11 in your Baan reports. It should print as Arial 10 in this example.
