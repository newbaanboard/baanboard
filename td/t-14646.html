Berthe,

You can write a simple program to do the line counting -

num.lines = 0
fp = seq.open(path.script,"r")
if(fp < 1) then
mess("zmadms0036", 1, path.script)
|* File Opening Error: %1$s
return
endif
while not seq.gets(buff,1024,fp)
num.lines = num.lines + 1
endwhile
ret = seq.close(fp)
