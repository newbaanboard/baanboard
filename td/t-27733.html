Hi Jeffers,

In BaaN 4 you can use following code in your program script of your "Maintain Session"

main.table.io:
before.write:
update_sern()


The function update_sern() will find maximum of sern and then increment current one by one.
Unlike ERP LN which has DAL2 , In BaaN 4 you will have to port above code in all sessions wherein new records of your tables are inserted. Also remember the table must be the declared as the main table of the session.

Refer following link for more info on main table section,

http://www.baanboard.com/programmers_manual_baanerp_help_4gl_features_4gl_main_table_io_sections
