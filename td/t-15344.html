One idea is to use a API function server. You could bring up each report, read through the layouts, find like the next detail available, add it, then save the report. For the script you could check ttadv330 to see if a report script exists, if yes then open the script in append mode, add your layout code. If no script existed then set the ttadv330.scpt to yes, then open the script in write mode and add everything needed for the new layout code. Sounds feasible, but not really sure about the API code. I have never really written one for tools sessions. Plus you will have to track VRC's, directories and lots of other information. Just my thought for the day.
