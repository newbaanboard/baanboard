Hi Anubhav,

Goto BaaN IV Tools --> User Management --> Authorisations --> Session Authorisations. You will find host of sessions for authorisation purpose.

Use session ttaad2131m000 " Session Authorisations by user/session" , if the user has permission for all modules of a package then by entering your restricted session in this session will make it as "exclusive" (User is not authorised to run the session). Else if the authorisation indiactor is inclusive (User is allowed to run this session) just set the start time and end time to 00:00.

I hope it solves your purpose.
