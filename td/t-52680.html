Here is a brief description:
1. In session ttadv3131s000 "Maintain Report Layouts", select the appropriate layout and press the button "Edit Layout".
2. In the Editor, select the main menu item "Form" and then "Resize window".
3. (The display now contains to plus signs that show the upper/left and the lower/right corners ot the layout).
3. Use the cursor keys to increase/decrease the layout (window).
4. Use 'E' to end.
