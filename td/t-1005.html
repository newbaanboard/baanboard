The conversion to runtime updates the file

$BSE/lib/compnr6.1.

This is why it takes very little time.

Create tables - may take between 5 and 30 minutes. Depends a lot of the fragmentation in your database. Be sure you create the tables in the right tablespace , if you are using Oracle ! ($BSE/lib/ora/ora_storage)
