Try to clean the used field values from tdplc105 before the select. I guess the values from the previous record set are presented in the next record.

For example ( I don't know the field names you use from tdplc105 )

tdplc105.invn = ""

select tdpcl105.*
from tdpcl105
where tdpcl105.ttyp = :tdsls045.ttyp
and tdpcl105.invn = :tdsls045.invn
and tdpcl105.cono = :i
