I find bw72 to be very stable but there are a few bugs still:

the fatal error you are getting when accessing GTM is probably when you are entering the password - it is not really fatal as it still lets you in. If you are logged in as Baan (on an NT system) you shouldn't get the error.

Also there are problems with XP if the computer is logged on for a long time - error is somthing like "cannot draw the polygon" - it is also not really fatal - just annoying - and you'll get this with earlier BW clients as well.
