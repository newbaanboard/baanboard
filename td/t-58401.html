You can't grayed out in Baan IV, however you can removed it.

For example you have enum constant in DD as below
Constant |Constant value |Desc
1 |tcyesno.no |No
2 |tcyesno.yes |Yes
3 |tcyesno.na |N/A

And you want to remove N/A option then you can code

before.input:
set.enum.values(1,2)

To check the repeating field number use attr.element as the array index for the field . And to know the value of the enum field selected earlier , use etol function .

Finally attr.enum.mask$ is a variable which controls enum option to display . U can have a code like following


field.relative:
before.input:
long enmctr , selctr , selected
attr.enum.mask$ = ""
for enmctr = 1 to 5
selected = false
for selctr = 1 to attr.element -1
if relative(selctr) = ltoe(enmctr) then
selected = true
break
endif
endfor
if not selected then
attr.enum.mask$ = attr.enum.mask$ & chr$(enmctr)
endif
endfor
