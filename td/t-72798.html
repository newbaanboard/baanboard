I am not sure about Baan IV. In Infor LN you can create script for starting tmboaserever. This script can contain similar command as this below:

bshell6.2 -server tmboaserver -f components.txt -l log.startow -nogui &

My command is for Linux and Infor LN. File "components.txt" contains list of components which will be started.
