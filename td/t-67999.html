Hi,

1. Install Notepad++.
2. Create a new Development Parameters Template
3. Provide notepad++ in "Editor Read Only Command" and "Editor Read/Write Command"
4. Convert to DD
5. Attach the template to user Data as Development Parameters.

Done.
