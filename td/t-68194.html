Hi.

In ttadv2100m000 "Maintain Sessions" switch to form 2; you see Help Text Code. Note If that field is empty. Then press the [T] button and select "Help Information". You get a Default Help Text Code if it was empty; convention: SE + Module + Session, e.g. SEitm0101m000. Maybe you get asked for a Text Group; I think it should be "docu". Finally, you will get to the text Editor, where you enter your help text.

Now comes the trick: If you created a new help text, you have to run ttadv5230m000 "Create Runtime Help". Enter Package, VRC and language - and don't forget to select Sessions [x]. (This is only required once per Session Help.)

Now, you can start your session and press F1.
