Was having trouble myself, with the get.mwindow.size in Baan IV. The help says;
Example:
long wwidth, wheight
get.mwindow.size(DSMAXWIDTH+DSMAXHEIGHT, wwidth, wheight)

However, compiling the script gives an error of; "argument 1 should be called by reference". This was due to all three arguments, being reference variables. Which didn't make sense, as it wouldn't let you specify if you wanted the minimum, maximum or actual widths and heights

So I still can't figure out how to pass to the function, that I want the DSWIDTH and DSHEIGHT (to get actual width and height)...

In the end, I gave up using that function and used get.mwindow.attrs. EG:
long at.values(MWINMAXSIZE)
get.mwindow.attrs(at.values)

Now, the actual width and actual height are stored in elements 11 and 12 respectively, ie:
wwidth = at.values(11)
wheight = at.values(12)

Not sure if this was your problem. But would like to hear what issue you were having and how you resolved it?
