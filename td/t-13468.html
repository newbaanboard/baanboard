What we did was a simple fix.
We got the specific codes for our printer and modified the lines of the printinf file for the printer.
this is located in $BSE/lib/printinf/manufactures initials "h" for hp

there are lines in this file for the user fonts per printer.

I know there are multiple ways to do this and you could lose your changes if you install a patch that overwrites the file.
