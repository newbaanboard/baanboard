Since that very tight requirements make it almost impossible to achieve the desired goal, it is now changed to a fully customized session where I can write my own script to make it work.
So the way it works now is just as the following:

declaration:
table ttcibd001
extern domain ttdyf.picture item.image

after.form.read:
bind.image("item.image", "tcibd001.imag")

plus, of course, the image field in the DFE. I am amazed it's that simple.
