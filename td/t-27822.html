Hello,

you have some options:
1.) print-preview with bwprint and then file-print (so you have exactly the same output).
2.) Create an new ascii-printer and generate the ascii-file to an pdf with external tools.
3.) try MailMerge from Nazdaq (you can define logos and it will collect the mail-adress automatically from the report.) - this is probably the easiest solution.
