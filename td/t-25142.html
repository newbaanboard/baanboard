If you have your .bwc data store on a central place (the files that baan uses when you start, to determine what ip-adress/port/bshell to use, the ones you see in BECS) you could add a new one, e.g. password.bwc. In this configuration, add the session name of the session you created to change the password to the "command" field, it will then start automatically when the user uses this configurationfile to start baan.
