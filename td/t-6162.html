Hi,

Session ttdsk1300m000 (Browse Data Entry) never really works
if the session is deep down the Menu Structure.
Location : Menu Browser >> Find >> By Description

Here is one way to make it search through the entire Menu
Structure.

Step 1 : Edit the user file
Location : $BSE/lib/user
Filename: u<login>

Go to the line which says
startprogram_gui: ottdskmbrowser -n1 - mmtccom000000001

By default the value should be -n1

Change it to -n0 for infinite depth
startprogram_gui: ottdskmbrowser -n0 -mmtccom000000001

Save the file.

If anybody knows another way of fixing it, please post it on the
Board. Coz the problem with this method is that, the user file
gets overwritten every time a runtime is done for the user.
At my client they do it everyday for all users (as a job) !!!!

Cheers,
Popeye!
