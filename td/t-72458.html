General steps while deploying a BDE -

1. Create Definition
2. Create Implementation
3. Generate Implementation
4. Login to C4WS Console and Select Deploy
5. Select the environment and Deploy the BDE you have developed.
6. Get the WSDL URL and use it to test using S/W like SOAP UI.
