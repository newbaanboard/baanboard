So what's the problem? Just
1) go to the relevant report (i.e. not the main language, but the one in other language which you want to link to the main language).
2) Delete the report
3) Go to the report layouts in the main language
4) Via 'Specific' click on 'copy report to other language'
5) choose the language and click the option 'layout linked to language'
6) Compile the new report
