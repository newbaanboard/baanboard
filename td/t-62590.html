On BW the changes on menu are automatically seen when the user log out and log in again.
On Worktop the menu is cached locally, so the user have to refresh the data.
Simillary on webtop in option/Infor erp options there is a menu item, the first, that states 'memorize menu in cache'. If flagged, every change on menu isnt seen by the user, because is cached. Simply unflag this.
