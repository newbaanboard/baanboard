Check the log.bshell6.1 file, 99% of the time it will tell which field was the problem. Other wise use the dbs.log option in the BW config command setting

-- -set DBSLOG=0100 tdilc4202m000 (session code)

when done search the dbs.log file for the DBMS Update Input it will show the exact data that was inserted or tried to be inserted in the table(s).
