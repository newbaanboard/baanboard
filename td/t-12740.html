From Baan you can't set system environment variables.
From Baan you can read system variables that are set from the shell where the bshell is started (in case of using BA) or set in the BW configuration file.

The command to read the variable is getenv$ (http://www.baanboard.com/programmers_manual_baanerp_help_functions_system_and_user_information_getenv)
