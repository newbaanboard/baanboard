Here is what I would do if I was you - I do not have source code either:
(1) Validate both companies are set-up the same.
(2) Validate the SAME OBJECTS for the session are the same in both companies.
(3) Make sure you are receiving the same type of data. I am not sure if the session handles menus based off of the type of purchase order (subcontract, MRO, etc.)
(4) Find the menu that is being activate - you can use the print where used on some of the session in the zoom. Displaying which menus they are in.
(5) Compare the menus to each session. If the menus line up then it is probably sometype of setup issue. At that point there might not be anything to do but contact baan.

Good Luck!
