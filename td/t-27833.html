Hi,

I assume the domain requires time to be stored in hhmm format for which you can use following code,



domain string str.dte(12)

str.dte = dte$()
tdinv120.time = lval(str.dte(7;2))*100 + lval(str.dte(9;2))


dte$ will return current date & time in MMDDYYHHmmSS format.
lval is used to convert string into long value.
