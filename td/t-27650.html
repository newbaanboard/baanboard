Hi,

Just go thro following code,

long fp,buffsize
string buff(80)
buffsize = 80
fp = seq.open.local("c:\test.txt","r",false)
if fp >= 1 then
while seq.read.local(buff,buffsize,fp) >0
message("%s",buff)
endwhile
endif
seq.close.local(fp)


The problem with seq.read.local is that, depending on buffer size it may read more than one line at one go and you will have to parse the line for CR/LF characters to separate out individual lines.
Hence i would recommend you to use client2server to get the file onto server and then read the file thro seq.gets.
