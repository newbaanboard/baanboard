On a report layout (Layout type = After.field) I have field 'Qty' which is a total field (Aggregate function = Total).

In the corresponding report script I would like to use the value of this total field in a calculation but I'm not sure how to code this in the report script. (If I just refer to the field 'Qty' it uses the value from the last detail line)


Example Report :

Group Item Qty

AAA 001 10 <--- Layout type = Detail
AAA 002 25 <--- Layout type = Detail
AAA 003 30 <--- Layout type = Detail

Total Group 'AAA' 65 <--- Layout type = After.field

BBB 012 12
BBB 014 20

Total Group 'BBB' 32 <--- Layout type = After.field

I want to use the Total field in the After.field in a subsequent calculation.

Report script:

after.group.1:
before.layout:

[New field] = Qty x 0.01

I think I need to replace 'Qty' in the report script code with something that tells the system to use the total 'Qty' but I have no idea what the syntax of this is.

I am grateful for any help. Thanks.

Compiling the report in debugger mode u can check the v right total.. variable for field qty . If qty is the only field being totalled in the report the variable would be total..0 .

Due to maintenance considerations,it's not advised normally to use these variables. Instead report variables declared by programmer (not report input variables) in report scripts are recommended.
