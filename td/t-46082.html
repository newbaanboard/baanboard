Hi,

The function handle.session.interface() does contain the following:
function void handle.session.interface()
{
handle.importing.fields()
handle.first.event()
handle.setting.filter()
handle.set.synchronize.dialog()
}
If you want to change the query extend of this session then you should modify handle.setting.filter() function.
