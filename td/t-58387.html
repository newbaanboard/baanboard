Hi,
One more alternative is:
use = chr$(34) & string & chr$(34)

Here 34 is the Decimal value for the character "

Similarlly use can use it for single quote also. Decimal value for it is 39. You can use the ASCII table (http://www.baanboard.com/programmers_manual_baanerp_help_misc_ascii_table) for more ASCII values.
