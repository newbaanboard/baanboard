As far as I know, NO_WORKTOP=1 is not officially supported by Infor (see for example https://www.inforxtreme.com/espublic/EN/AnswerLinkDotNet/SoHo/Solutions/SoHoViewSolution.aspx?SolutionID=949728). One of the problems of this setting is the message "There are still running sessions. Close all sessions?" that you get even if you closed all sessions. This happens if you printed something and you have also an external report server defined (either SSRS or Infor Reporting).
