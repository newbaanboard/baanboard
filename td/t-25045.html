I don't know what are the properties and methods of Baan4.application Class.

But you can create a bwc file without saving the password and use the following vb code in order to execute BW:

Sub Main()
Dim BWpath As String, BWCpath As String
Dim FullPath As String, pass As String

pass = "baan" ' your password
BWpath = "C:\Program Files\Baan\bin\bw.exe " ' the path of you bw
BWCpath = "C:\Program Files\Baan\lib\user\nt.bwc" 'the path of wour bwc file

FullPath = BWpath + Chr(34) + BWCpath + Chr(34)
retval = Shell(FullPath, vbNormalFocus) ' executing bw.exe

SendKeys pass + vbCrLf, 1 ' sending the password!

End Sub



I hope it helps
