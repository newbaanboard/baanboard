At least in Baan4 you could do it like this:

function check.input.itbp(domain tcdesc rel.field)
{
if strip$(rel.field) = "," then
retval = zoom.to$("tccom4512m000",
Z.SESSION,
zoom.name,
"tccom112.itbp",
0)

else
...




zoom.name is a string variable containing the name of the program where you are zooming from (can be used eg. in the zoom.from section of the subsession)

value of the variable "tccom112.itbp" is returned.
