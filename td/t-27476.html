Hi,

You cant use functions in where clause( except perhaps in selectbind clause which is not possible in query extensions)

One way out is to to code in following way,

str.query = "table1.balc-table1.bala > 0.0001 or table1.bala - table1.balc > 0.0001"
query.extend.where(str.query)
(I assume you want to select records wherein bala & balc amounts are not equal)

Ive not compiled the above query but it should work.
