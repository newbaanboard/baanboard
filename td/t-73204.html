Hi,

any variable - in our case "rep.doc.date" - with tfgld.date domain ist not in UTC but in number of days since 01-01-0001.

So I would use the standard function date.num() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_date_time_zones_date_num) to get the number of days since 01-01-0001 for today and use it to get the number of days since document date.
Something like this:

long today
long no.of.days

before.program:
today = date.num()

detail.1:
no.of.days = today - rep.doc.date
