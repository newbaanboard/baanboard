You can also change your font when you use BWprint, but this is done for all reports which you want to print.

You can do it with the following steps:
- start BWprint (found in e.g. c:\Baan\BW4\bin)
- go to Help, fonttest
now you can see all the available fonts and the options for them.

By clicking with the right mouse button on the fontline, you can change the size.

Go into regedit and export the section for Baan bwprint:

HKEY_LOCAL_MACHINE\software\baan\bwprint\fonts

This will create a .reg file which you can run on another machine and this will import the settings.
On the machine to import into, always back up the registry first though - just in case something goes wrong.

Incidentally, in the bwprint white paper it tells you even more of how to do things with fonts etc.
