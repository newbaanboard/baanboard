Refer to the thread - Import objects from Baan 5c to Baan 5b (http://www.baanboard.com/baanboard/showthread.php?t=23362&highlight=import)

The Tools table structure for the Forms definitions are different with DFE in Baan-5 compared to the static editor of Baan-4. While, import might work for some components it is not generally advised to do downward software migration. Typically, the higher releases are made downward compatible, that is you can import the older version dumps. The dump format might also be different from Baan-5 tools version to Baan-4.
