Hello skamat,

there are two ways to built an Easy-SQL Query:

1.) by textmanager
2.) by Easy-SQL

This two ways are options in the Session "Maintain Query Data" ttadv3180m000


I always use the textmanager as shown in my example.

1.) Add new Query
2.) Leave owner empty (everybody are able to run this query)
3.) Description for the query
4.) User - Date - Reportlist !
5.) Report (Name is - package + sql + queryname (is filled by BaaN-Tools))
6.) Chose Text-Manager an write down your "select-from-where"
7.) Leave Textmanager an choose "Query-Report"
8.) The Report, based on your query will be created by the system.

Last "RUN QUERY"

That's all
