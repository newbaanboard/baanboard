Try this ->

SSA ERP LN 6.1 Programmers Guide

seq.getc$()
--------------------------------------------------------------------------------
Syntax
string seq.getc$( long fp )

Description
This returns the next character from a specified file. fp is the file pointer returned by seq.open() when the file was opened. An empty string indicates either that the end-of-file has been reached or that an error has occurred.

Context
Bshell function.

Probably you have the trouble with the variable when you are keeping the results. I have read files greater of this size without trouble of this way.


string file(128)
string bufr(1) based
long buflen, fp

fp = seq.open(file, "r")

buflen = seq.seek(0, 2, fp)
alloc.mem(bufr, buflen)
seq.rewind(fp)
seq.read(bufr, buflen, fp)
seq.close(fp)
