When you create a report, normally you have to create first in the std language (2 = english), then copy to the other language you use (in your case o = chinese).
Doing this, you have two options:
1) link the report to the language 2 - this means that both the reports have the same structure, changing one in one language reflects the changes in the other.
2) don't link to the other language - this means that both the reports can be different and you have to maintain them separately.

In both cases, you have to compile the report in the language 2 and in the language o.
You can have the report message if you haven't compiled the report in language o.
