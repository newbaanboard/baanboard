Hi Robby,
When you customized this report, did you by any chance change the id of the report to a different one?
If so,
a) you would need to go to Maintain Sessions {ttadv2100m000},
b) look up session tdsls4404m000 (Print Sales Invoices) and then click on the Reports button.
c) if (as suspected) you have renamed the report id, then you would need to put this new id in place of the old one. Usually its on Position no. 1.

hth
sk

Note: If you had earlier customized the Print Sales Invoices session to your Custom VRC then the above steps need to be carried out for that (customized) session instead of the Standard VRC...
