Its a way to select the printed detail layouts. If one datasat is send to the printer every detail layout will be triggert and printed (if its print condition is true). If you want to print specific data for some datasets only, you can use a "let's print this data" variable like "itemline" (or "line").
The print program will set the variable prior of sending the data to the report.

When I write new print programs I often use "line=1" for the basic position data. So I will set Line = 1 in skript before the first rpt_send(). Then if I have to read additional data I will set Line = 2 before the second rpt_send(). The Layout will use "line = 2" in the print condition, so it will only be printed if the additional data is used.
