I am using dal.update in my customised program script. I would like to update the value of field tdipu001.otbp in table tdipu001. Following is the code , but it returns with Error 112(No Current Record) on tdipu001810 in db_update. Please guide.


function update.item.purchase.data()
{
long ret, ret1
db.retry.point()
select tdipu001.*
from tdipu001 |for update |* Item Purchase Data
where tdipu001._index1 = {:curr.item}
selectdo
| tdipu001.otbp = odm.buy.frm.bp
| db.update(ttdipu001, db.retry)
ret1 = dal.set.property("tdipu001",ttdipu001,"tdipu001.otbp",odm.buy.frm.bp,DAL_UPDATE)
dal.update("tdipu001",ttdipu001,ret,FALSE)
selectempty
endselect
commit.transaction()
}
