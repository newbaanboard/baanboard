You cannot resize a dialog window. If you observed carefully, the resize icon is disabled.

Are you using DFE ? From the BaaN-5 versions onwards you can have a single form for both single and multi-occ sessions and if the window type is then setup the multi-occ/overview session to list-window and then use a synchronized dialog to the same sessions for details/single-occ then you can still have a resizeable window.
