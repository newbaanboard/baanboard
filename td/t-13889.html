Hi,

In a sessionscript I have a sql select statement with ...

and tpppc305.cdoc inrange :cdoc.f and :cdoc.t

... in the where clause. The fields cdoc.f and cdoc.t are on the form with domain tpppc.cdoc (right aligned).

In the tpppc305-table in Oracle some cdocs are right-aligned and some are left-aligned. The problem is when a user selects a document number it will only be found when this cdoc is right-aligned in the database. How should I change the where-clause so that it will find the document independent of the aligment in the database?

Kindly regards,

Douwe.

Hi,

Create 2 variables: cdoc.f1 and cdoc.t1. Assign values of cdoc.f and cdoc.t with left allignment. Change where statement:

and ((tpppc305.cdoc inrange :cdoc.f and :cdoc.t)
or
(tpppc305.cdoc inrange :cdoc.f1 and :cdoc.t1)

Regards,
Klaas
