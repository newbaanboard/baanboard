Our coordinators are restricted to change the line with PO link due to DAL Script.(tdsls401ue)
Is there a way from the script that they can modify only the Warehouse.

Below is the logic.

|Check for Purchase Order Link |SD 21/05/2015
if tdsls094.reto = ltoe(2) then
e.flag = 0
check.for.polink.to.soline()
if e.flag <> 0 then
if e.flag = 9 then

function check.for.polink.to.soline()
{
if logname$ <> "t.rajah" and
logname$ <> "a.almasr" then
select tdsls402.*
from tdsls402
where tdsls402._index1 = {:tdsls401.orno, :tdsls401.pono, :tdsls401.sqnb}
selectdo
if tdsls402.**** <> " " then
if (tdsls401.oltp = tdsls.oltp.total or
tdsls401.oltp = tdsls.oltp.detail)then
|tdsls401.qbbo = 0 then
e.flag = 9
endif
endif
endselect
endif

}
