Hi,

In the path of the device you can specify a full path of the file and not just a directory.

I advice you to write your own script that will take a directory and copy the file to the directory with anyname you want. You can do that in the tools script or using a batch file.

You will have to print the report to your script (device) instead of ASCIF

There is a nice example of a batch file here:
http://www.baanboard.com/baanboard/showthread.php?t=29134
