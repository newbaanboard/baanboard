No screen shots - you can check the help attached to the board. Or maybe even run help on sesssion ttadv4120m000. Basically create a new table in your development VRC. Add fields desired, add indexes required(at least one unique one). Then convert to runtime data dictionary. Next log out of system, log back into system. I usually run ttaad4100 and create the table. Now it can be used.
