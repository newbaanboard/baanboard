Refer to: alloc.mem() (http://www.baanboard.com/programmers_manual_baanerp_help_functions_memory_operations_alloc_mem)

This allocates memory space to the specified variable at run time (up to a maximum of 5Mb). The variable can be a string or an array (of any type). It must be declared as BASED. With multi-dimensional arrays, each dimension must be initialized to 1 at declaration.
