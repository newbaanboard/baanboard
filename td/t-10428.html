DDC is one option.
Another option is to write a code in VB, wherein it reads the ASCII file generated by the scanner, and eventually populates the various fields of the activated BaaN session.
Some vendors might also offer to write this code in the Portable Terminal (kind of a scanner, which scans as well as has the logic of passing the scanned data further), and leave VB aside. But, the kind of controls you have in VB are much more than in the PT.
Good Luck.
