Since there was mention of another editor, I would like to tell you that I have been using the subject editor for baan development for many years.

The new version of the editor does all that you would expect, for example

color matching.
auto indenting and outdenting
templates for easy code writing
begin end matching if/endif, while/endwhile, select with endselect, on case.... "{" with "}" etc....
It even correctly handles matching of nested code.

They have baan support as a language. The standard editor includes support for a dozen other languages as well.

Extras include a programming language with macro processor to extend functionality

Interface to many source code packages.
Up to infinite undo (You set the level)

It is a legacy code type of product (very very very clean and error free).
