hpng98,

If the array has an unknown content, try this:


domain ttadv.cpac cpac(2)
long test
long i

cpac(1,1) = "tt"
cpac(1,2) = "tc"

test = 0
for i = 1 to 2
SELECT *
FROM ttadv100
WHERE ttadv100.cpac = :1
WHEREBIND(1,cpac(1,i))
SELECTDO
test = test + 1
ENDSELECT
endfor


if the array fields are always fixed, do this:


SELECT *
FROM ttadv100
WHERE ttadv100.cpac in ( 'tt', 'tc' )
SELECTDO
test = test + 1
ENDSELECT
