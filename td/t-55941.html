Andreas
26th May 2009, 13:37
Hi,
we had the same problem and solved it the following way:
Define a special font only for the amounts in the report layouts resp. the report fields. We used font 17 (old Name Font 9).
In bwprint assign the Arial Font to this Font. Additionaly set the Font spacing for that Font.
Untick the Field 'Use Font Metrics' for Horizontal spacing and define a character width of your choice.
The disadvantage is that the Font is not proportional anymore, but if you use it only for amounts it's reasonable.
Hope this helps,

Andreas
