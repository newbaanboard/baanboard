Hitesh Shah
7th December 2003, 07:20
You need to write a program which activates session tiitm0101m000 (if u need to upload items) using AFS and inserts items throgh stpapi calls in your program.

If you know sufficient about the database triggers , checks and validations then u can well design an exchange script with all such validation . So the knowledge of Baan validations is very crucial when using Exchange schemes / sequential import .

There are many master tables (like mcs tables, some com tables ) which do not have very extensive validations and Baan sessionsjust updates tables with basic validations . In such case exchange schemes (or sequential import if possible) are very handy, useful and the quickest method to upload data.

In complex cases like item master AFS script (as aforesaid) or manual data entry itself may be the best option.
