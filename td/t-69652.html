Hi shradha.d,

These steps is my own way of grouping but maybe there is some way of grouping.

Grouping is created in the layout. For example I want to group the "details" or the population of data by order number, normally we use the detail layout but for grouping you need to create another one. As I want to group it by order number then I will change the sort mode of for example the tdlsls401.orno to ascending (change the sort mode in input fields) after this I will create a layout that corresponds to this. The layout will be After.field tdsls401.orno. Now you can code inside here what you want to display and grouped by Order Number.

Sample script:


after.tdsls401.orno.1:
before.layout:

|some codes here to display what you want and the out will be |order/grouped by Order Number

after.layout:



I hope this will help you,
