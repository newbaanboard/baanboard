george7a
29th August 2006, 18:26
Syntax:
<condition> ? <expr_1> : <expr_2>

<condition> is tdilc000.iilc = tcyesno.no
<expr_1> is tdsls041.cwar
<expr_2> is also tdsls041.cwar ! (are yo sure it is not a typo)?:confused:

In other words:

if tdilc000.iilc = tcyesno.no then
return (the value of <expr_1>)
else
return (the value of <expr_2>)
endif

If there is no typo it will always return the same value.
