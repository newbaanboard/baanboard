The session tfgld0111m000 checks the input to field tfgld011.ses1 with the function tt.session.present(tfgld011.ses1).

This function checks if the specified session within the current package is present in the runtime dictionary. This function returns 0 if the session is not present, 1 if the session is present but not in the configuration of the user and 2 if the session is present and in the configuration of the user.

So I think in your case the function did not return 2.
