Probably a better way to do this - but before.layout of each section you could test if the invoice value is 0 and just skip that layout. Something like this in the report script - probably better methods for ln, but I did this in a few 4c4 reports to get them to print the way we wanted.


Header.1:
before.layout:
if inv.amnt = 0 then
attr.print =false
endif

datail.1:
before.layout:
if inv.amnt = 0 then
attr.print =false
endif

One easiest solution is, You can use this condition in Output Expression of Report Layout (Report layout -> Detail -> Output Expression)
Value <> 0

On above expression, Only those lines will display who has not equal to Zero value.
