Texts are stored in table tttxt010

You can use this:


select tttxt010.*
from tttxt010
where tttxt010._index1 = tdsls400.txta | For header Text

or

select tttxt010.*
from tttxt010
where tttxt010._index1 = tdsls400.txta | For footer Text
