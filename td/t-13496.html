Add this entries to the profile and this will run using oracle 7 driver. We had performance issue with the batch jobs but after using oracle7 driver our jobs completeing successfully.

BDB_DRIVER=oracle7 ;export BDB_DRIVER
ORACLE_SID=baaniv ;export ORACLE_SID
ORACLE_HOME=/oracle8/app/oracle/product/8.1.6; export ORACLE_HOME

ps -ef|grep oracle8 would show you whether you are running oracle 8 driver or not.
Hope this helps

Nagesh,

You can specify different database drivers/versions in the Database definitions. You would also need to have 2 different Oracle binaries in the $BSE/bin and then you need to change the $BSE/lib/ipc_info to make 2 tags/entries for Oracle-7/8 drivers and then the $BSE/lib/tabledef6.2 has to be modified accordingly.

I am not sure if BaaN Support will support this configuration or not. You can explore at your own risk.

Good Luck! :p
