"You have no authorization to change the standard"
As the message means you cannot set your current package VRC or modify the BaaN Standard VRCs.
You can change your current Package VRC using the session - ttadv0140m000.
You can set the developer's authorization template from - ttams1511m000.

I suggest you to refer to the BaaN Administrator's guide to configure a development environment, VRCs, Companies etc.
