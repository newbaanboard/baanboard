Hi Maria,

Try using ask.enum() function available in Baan 5 or infor Ln help file.

To use this function you will have to create a Question with tcyesno domain and set priority type = Warning and default value = No.

Then try using ask.enum() function in the same way as the example mentioned in help file and you will have to use exit() function to stop the execution.

May be it will be helpful to you.
