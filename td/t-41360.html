Wrappers are (probably) tools that help you 'wrap' a Baan session program script, without having the source code, and add in your own changes. This is known as several products:
Fullscope: QKEY
RMCis: RMCgen
Baan Development: Bolt-Ons

Fullscope & RMCis (my company...) both sell these as products. I can't speak for FullScope, but a link to the RMCis product description is:

http://www.rmcis.com/BaanTools.htm

Or let me know directly I can send you a user doc.

There are also some free version of similar wrapper ideas posted on these boards, search under 'rush' in the Code & Utilities section.
