We too have integrated the mobile application with Baan4 as well as other version of BaaN, but for integration you need to integrate it using Openworld connector, there are other ways to integrate as well like you can connect to the db at backend, can use OLE or can use other third party connectors, there is one opensource connector too which I had used sometime ago can't recollect the name, but the best you can use is Openworld connector supported and provided by Infor.

Regarding documentation, you can find the documentation about openworld in the media itself, or you can check on Infor website.
