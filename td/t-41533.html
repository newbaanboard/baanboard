I have the following problem with the command spool.pr.line:

In the report script, I assign the Bin2 of the printer:

before.program:
spool.pr.line = pc$(2)

I remarked that this command generate a special character in line 1 of my tmp-file
This character is the following "

it's a big problem because I use an external application to print purchase orders and
this application doesn't run with this character.
I seek a solution to assign the bin2 without this wrong character.

As the external application runs with a specific printer, the
solution is to create a condidion in the report script with
the variable spool.device. (to test the printer name)

before.program:

if spool.device = "AAAAA" then
spool.pr.line = (pc$2)
endif
