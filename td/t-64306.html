Hi !

Importing BOM and routing into the system from excel file in doable. I've been doing this quite some time for customers using exchange module.

You need to setup new scheme, adjust asci file fields, map them into fields in a new batch, compile scheme, and execute.

You need to be aware to set up the right data type for input fields, use the correct character encoding, separate char.
