Folks, "use a source code extender hack" is not an awnser that will help a beginner.

Without using any hacks (that I strongly discourage to use), there is only one awnser:

It is not possible to extend a print session with new form fields without having the source code of the session and most likely also other source codes of (Baan) Dynamic Link Libaries. It is further not possible to extend a session with new fields without having licensed the development environment, since you will have to modify the session script.

If you buy an ERP Software without insisting the vendor makes the sources available to you, you will run into that sort of problem, sorry. You will have to ask your implementer to do this software change for you.
