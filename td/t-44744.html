Hi,
I am trying to understand where the problem lies. So please check the following ... and try doing as suggested below.

Let us consider the same session that you have attached. I presume the session is based on a customized table.

1. Go to table definitions --> select fields --> then go to field details for all these three fields in the customized table.
2. Now, tell me what do you see in "Help Information", Help Input" and Help Display". Do you see any help codes filled in there.
3. If yes, then try running the session "Create runtime help" for this current session. Include forms too. If this does not help then run "Create Runtime Data Dictionary" for the sessions only. DO not forget to check "Run time help option".
4. If no then please define help codes for these fields.
5. Add these help codes to fields in the table definition under Help Input and Help Display fields.
6. Then run "Create Runtime Data Dictionary" for the sessions only. DO not forget to check "Run time help option".

I have just created a new session based on new table with three new fields. At first I have not added the help codes. So I did not get any help. AFter that I have added field help codes then tried. I get all the information that you are looking for.

Please try this for your current session. It should work.
