Hi,

While searching I found the pstat() function (http://www.baanboard.com/programmers_manual_baanerp_help_functions_processes_pstat) but it returns the info about running processes on the client. I don't know if you can do this using only Baan tools. What you can do is to write an exe/bat that will run on the server and bring all the PIDs of the running processes, then you will have to look for "ntbshell.exe" and print them on a temp file.

After that you can run this exe from your Baan function and read all the PID from the temp file.

I hope it helps,
