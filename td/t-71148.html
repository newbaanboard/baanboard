There is a guide in InforXtreme, under LN 10.6, Tools, Reporting.

It is not required to create a report extension before you can personalize the report. However, if you want to add fields from (other) tables or calculated fields, that are not yet in the report input fields of the report to be personalized, a report extension is needed.
