From the programmer's manual -

22 EINVAL - Invalid argument
Description:
This error indicates that an invalid argument was used. For example:

Dismount a nonmounted device.
Mention an undefined signal in signal or kill.
Read or write a file for which lseek has generated a negative pointer.
This error can also occur by the math functions.

Solution:
Most of the time this error occurs when you try to initialize the shared memory or a database. In that case, you must maintain the kernel.
