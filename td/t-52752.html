You can use SUM in a query like :

select sum(tdsls041.amta),
tccom010.nama,
tccom010.cuno
from tdsls041, tccom010
where tdsls041.cuno refers to tccom010
group by tccom010.nama, tccom010.cuno
order by tdsls041.amta desc
as set with 10 rows
