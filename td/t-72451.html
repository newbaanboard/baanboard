Hello,

The debug workbench in extensions halts the process only, if
functions are called or values are set.
Other halts (declaration of variables or comments) are ignored.

For debugging the extensions in Infor LN you can use:
•
Debug Workbench, which runs within LN UI.
Use this debugger for simple extensions debuggingor
when you do not have Infor LN Studio installed.
•
LN Studio debugger. Use this debugger for more complex .........
