I'm not sure what other components you had checked for export, but, unfortunately, Baan doesn't nicely export all of the components associated with a customization part/project from this session. Checking that box will only export the customization parts themselves.

Fortunately, there is a library to allow you to do this -- with a little work. We have written our own as have a number of other people including Gordon Fasbender. Check out this link (http://www.baanboard.com/baanboard/showthread.php?s=&threadid=1533&highlight=customization+parts) for some more information.

I hope that helps.
