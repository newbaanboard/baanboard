If you download the help, into windows help (tt_mp2.hlp), or use the Baan help viewer, there is a list of functions that are available in Baan 5.

However, NOT all functions are documented in the help. In an effort to find missing functions, I am starting this thread, so that others can also post functions that they have come across. So far I have encounted the following;

refresh.parent(long element)
- Used in a synchronised session, to refresh the parent session. If you pass -1 to the element, all occurrences on the parent session are refreshed. Otherwise, the element is the occurrence number on the parent session to refresh.

get.message.type(string mess_code [, language])
- Returns the message type of the message code given, where;
1 = Information or Message Code not found
2 = Warning
3 = Critical
- Have tested the returns of this function and YES, for some reason, it returns 1 if the message is an 'Information' type AND if the message does not exist... Not sure why they did that.
