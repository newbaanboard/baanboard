Try this peace of VB Script works on a 2000 Machine.
Only issue you might have is the permissions.


strComputer = "COMPUTER NAME"


Call MappingW(strComputer)
'Wscript.Sleep 2000
Wscript.Echo "FILE RUN"

Sub MappingW(strComputer)
Set objWMIService = GetObject("winmgmts:" _
& "{impersonationLevel=impersonate}!\\" & strComputer & _
"\root\cimv2:Win32_Process")
errReturn = objWMIService.Create("PATH AND FILE TO RUN", null, null,intProcessID)
If errReturn = 0 Then
Wscript.Echo "MappingW.bat was started with a process ID of " _
& intProcessID & "."
Else
Wscript.Echo "File did not launch " & _
errReturn & "."
End If
End Sub


Hope it helps.
