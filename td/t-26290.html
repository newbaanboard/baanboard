he sample directory contains the following files:


Logon:

baanlogon.java Standard Java source code for Baan logon.


Item:

pitmdll88000 Standard Baan 4GL DLL source code.
itmdll8800.java Standard Java DLL communication source code.
item.java Standard Java source code.


Sample:

pnetdll88000 Sample Baan 4GL DLL source code.
netdll8800.java Sample Java DLL communication source code.
samptext.java Sample Java source code.



The following steps are needed to develop a sample Baan Internet application:



1. Create Baan 4GL DLL source (pnetdll88000) using Baan development tools.

2. Create Baan 4GL DLL object (onetdll88000) using Baan development tools.

3. Create Java DLL communication source (netdll8800.java) using
$BSE/bin/bic_jstub -d onetdll88000 -c Onetdll88000.

4. Create Java object code (netdll8800.class)using Java compiler.

5. Create Java application code (samptext.java) using Java development tools.

6. Create Java object code (samptext.class) using java compiler.

7. Modify Java logon source (baanlogon.java) to call (samptext.class) file.
See commented out code for sample.

8. Create Java object code (baanlogon.class)

9. Copy all Java object code to http server machine.

10. Test sample application.
