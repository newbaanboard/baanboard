In the Option Dialog you can switch on all kind of BW loggings ("Output to file"). The file will come in the $BSE/tmp file on your local machine, because there also a Baan Software Environment is defined, there where the client software is installed. You can add extra loggings to the file by checking more log-options in the Option Dialog.

The bshell messages are actually server messages (the bshell runs on the server) and they are logged into the $BSE/log files too or in the event viewer...
