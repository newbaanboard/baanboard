In Baan 5 you already have the possibility to use the dynamic form editor.
It's not new for SSA ERP LN 6.1.

What's new is that you can edit the dynamic form directly at session level. You don't need to go to the forms level.

I suppose you have already the form editor from CD?
After installing, you can simply start the form editor and via the menu you can start the Help. All information you need is there available.
