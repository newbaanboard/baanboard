Crystal Reports XI R2 supports XP, I don't know about Vista, but if it doesn't there is probably a service pack. Yes, you have to install it on a client, and then connect it to Baan. Crystal (now part of SAP) have stopped developing the Integration Kit for Baan and their own dedicated Baan database driver. However, you can use Crystal on the underlying database through normal ODBC/ADO etc database access. It is not quite as convenient as the dedicated driver, but it works...
