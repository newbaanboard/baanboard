I guess I do not understand completely yet. But what you could do is declare all the fields you need for both types of records. Then do two detail layouts (detail.1 and detail.2) Then for the output expression use something like k.of.data = 1 and k.of.data = 2. In your program script just set k.of.data to 1 or 2 right before sending the records. Since there will be mutliple records for what you are calling "headers" I do not think you can use a header layout(well probably could with a layout.again() statement, but I don't like that). So two detail layouts each printing for the different type of records. This is the type of logic Baan uses for printing BOM's.
