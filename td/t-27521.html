You need to either copy the report to the development VRC to modify the report. Or if you have already copied the report to the development VRC, set your VRC to the package VRC of the report. Do this before trying to edit the report.
MariaC
6th July 2006, 14:49
You have to be in the same Package VRC as the report to make changes. Go to Maintain Reports and on the top right hand side is the VRC that your report is in. The bottom left of the screen will show the VRC your user is linked to. Normally you would have a customisation VRC where any changes that you made will be stored. If this is the first time you are making changes to the report then you will have to copy it to your customisation VRC.
