If I understand you correctly, you print your invoices in a job, and want the output to be in .bpf format?

Very simple:
In 'Sessions per job' (ttaad5101s000) in the output details, you enter a device type that writes to a file - or even better, you create a new device for output to a bpf-file - and in the file name you enter a .bpf file name. The files will then be saved as bpf files.

Good luck!!

You can try following code before rprt_open. This will have your invoices open in bwprint in the next morning.

if job.process then
spool.device = "windows" |Name of windows type printer device
spool.preview = true |to get the output in bwprint
endif

While creating job also u may specify the windows type printer .

Further u can always reprint the invoice later on also and view the same in next morning if required. It's baan standard functionality.
