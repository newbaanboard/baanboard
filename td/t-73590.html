Make it simple.
You have written functionality of report writer in report script !!
First of all, you need to generate an update session (type 4). This will auto generate a program script with update properties. In the update function open an SQL query and within the selectdo, create a temp file using creat.tmp.file$(). Open this file and dump your header (one time only) and keep writing all your data columns. Close this temp file and capture the directory and filename. After closing, bring the file from server to client using server2client() and while dumping to client locatin rename this file to .csv and open csv file using app_start(). This will open your generated file in MS-Excel. You can convert it to .xlsx after making changes and saving locally. There is no need to use any report script and if so required, you can do it in program script itself when you are preparing the data columns.
