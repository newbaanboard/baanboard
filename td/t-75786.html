Use End Activity from the Team context menu. You can commit all changed components or a selection of them; the latter only if you checked the Partial End Allowed option for the project (in Project Server perspective).
BTW, the component is shown in LN. In session ttadv2530m000 you will see the libraries, but should must uncheck the Show Current Package VRC Only box in Change Current Package VRC.
