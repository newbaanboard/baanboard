You can use lattr.pageno:

if (lattr.pageno / 2) = (lattr.pageno / 2.0) then
lattr.print = TRUE
else
lattr.print = FALSE
endif

This example would print on even pages. To print on odd pages, just change the TRUE/FALSE.
