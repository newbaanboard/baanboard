Hi Baxajaun!

The directory path must point to where the exe files are. You path of
PATH=c:\Office97;c:\ghostscript;c:\Adobe\bin
means that excel.exe is directly under the c:\office97 directory and the same is true of ghostscript and adobe. Usually excel is under something like office97\office.

Since we are running 98 I can start a MS-DOS window, type in excel.exe and excel launches. So you could use explorer and locate each one of the exe's and put those complete directories into your path. My very first version of this driver had the path to the excel.exe hardcoded.
