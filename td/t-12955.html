You will have to write a script that goes through all Tools objects, and do a bic_info6.1 on all of these! Something in the form of:
for i in $BSE/tools/tt*/ott*; do
bic_info6.1 ${i}|nawk -v FILE=${i} '{print FILE ":" $0}'
done|grep "create.extra.toolbar"If the function exist, this will give you the name of the object.
