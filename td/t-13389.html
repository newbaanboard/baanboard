goto session ttadv4120m000... find your table gtcat202, and on form 1 you should see the index button. Clik on that and it shows you the fields in that index....
If you intend on changing the fields there... be careful that if this index is used in other program scripts or sessions, those will start giving errors.... so you might effectively need to change in those scripts as well.

I'm sure you know that you need to do a convert to run time for that table... (log off and log back into baan to see the effective change).

Hi Gavin

A very easy, quick and safe way is to use General Table maintenance or Display session ttaad4500m000 and set display mode to 'programmer' (in the application drop down menu). When this has been set the index componebts will show with a number indicating their position in the index. If the table has more thna one index this can be set from the 'browse drop down menu (change order)

Hope this helps
