Hi,

the problem is function tcedi0051. In this function some variables (e.g. edi.typedll) were used which are not known when compiling the report.
This variables were declared in function tcedi0051.h

Please check if solution 113768 is installed on your system. With this solution the new variables were introduced.
The other possibility is that you have customised function tcedi0051.h and the already installed solution is not valid.
