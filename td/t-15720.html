U can take the item and double values in an dynamic string array of
say 32 characters . Fill the each array 1st 16 character with item code
and other 16 characters with the double value using edit$ function.

Then u can use the qss.sort function on this array to sort in ascending
or descending manner based on the double value in 17th position to 32th
postion.

This may eliminate the boolean variable.
