( table is DAL based i.e. it has DAL script)
Sample code for DAL based -
|***************************** Object Hooks ***********************************
function extern long before.open.object.set()
{

tcmcs.dll0095.read.parm("fmfmd000")
query.extend.select("fmlbd010.* ")
query.extend.from( "fmlbd010")

}
