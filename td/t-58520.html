Hi,

Infor told me enable.fields() will only work on the first form of a multi occurrence session. On all other forms fields defined as input will never be enabled. So in fact all form fields on these other forms will be display fields.

To check if this is the case in YOUR version of Baan:

Generate a maintain session (multi-occurrence) on the tcmcs010 table (countries). The generated session will have three parallel forms. On all forms the fields are defined as input fields. When you start the session you will see that you can only enter data on the first form.

So: enable.fields does only work on the first form. I found a work-around for this problem. If you want to know about this work-around: post a reply or send an email to info@quint-ict.nl
