There is a function remove.mark(), and, although the help says something about not removing marks set by the standard program, try it!

Alternatively, try the following:

choice.mark.occur:
on.choice:
| Save the variables you need to use for selecting
...
| Clear the memory of the table
db.set.to.default(t......)
| Reread the data
execute(find.data)
| If this does not work, try:
execute(first.set)

remove.mark() just removes the marked occurence from the multi-occurence form, when it is called anywhere in the program script.

This would really help you loose values of fields when it is called before performing any DB actions.
