gget79,

We had the same issue with the client2server() function. The return value was -1, which indicated an error, but the transfer was actually successful. This only happened with WebUI (Webtop) and Worktop was fine.

The issue was resolved in WebUI 10.6, build BL0075. If you are using an earlier version/build, you will have to upgrade to get this problem resolved.
