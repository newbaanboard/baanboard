Hi,

In following we set the data language to german:


string l.new.data.lang(10)
string l.save.data.lang(10)

|1- Save actual data language
l.save.data.lang = ml_get_datalang()
|2- Get data language for language 3 (german)
l.new.data.lang = ml_get_datalang_of_softlang ("3")
|l.new.data.lang is now "de_DE"
|3- Set new data language
l.ret = ml_set_datalang(l.new.data.lang)
