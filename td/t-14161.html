Hi,

Refer to - http://www.baanboard.com/programmers_manual_baanerp_help_functions_formatting_io_sprintf

and also search for following functions :

DATE.NUM()
DTE$()

Here are some ex.:
1. In the report script :
select tdpur045.item,tdpur045.quap
from tdpur045
where tdpur045.date between :date.f and :date.t
2. In the SQL Query :
select tdpur045.item,tdpur045.quap
from tdpur045
where tdpur045.date >= tdpur045.date.f and
tdpur045.date <= tdpur045.date.t
