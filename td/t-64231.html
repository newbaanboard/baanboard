Ok. It worked for me. What I did:-

1. I went to session Paper Types (ttaad3110m000)
2. Created new paper type "USSTD" with Orientation as "Portrait" & Paper Size as "US Std Fanfold" and saved it.
3. I selected the created paper type and from Menu - > Specific - > I went to Fonts by Paper Type (ttaad3111m000).
4. I selected Font = "Small" and Paper Width = "239" and Height = "63" and saved it.
5. I went to Device, and created New Device With Name Test_New with following properties:-

Device Name:- Test_New
Description:- Testing Fanfold
Device Type:- Windows Printer
Paper Type:- USSTD
4GL Program:- ttstpconv

Thats it. It gives me reports in same format. When you print a report using this device, select preview check box and check if you are getting proper output.

If this doesnt work, may be you need to look your local printer settings.
