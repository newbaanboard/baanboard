Yes, the spool.device variable is a string of 14 position, so you indeed need to remove trailing spaces if you want to check on "D" or "P".

If you want the report to be printed to one specific device, why don't you just define the device in the report script by calling instead of rprt_open, the function brp.open, where as the second argument you specify the device name, and the third argument 0 (zero), indicating that the user will not be prompted a device selection (ttstpsplopen).
