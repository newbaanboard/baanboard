The Rounding is specified in the report layout (field definition).

Please check the used field and the format code e.g. A001.

This format code is specified in Maintain default formats by format code.
(Baan IV)

Maybe you have a currency in your record -> Check "Formats by generic unit", the format is specified per currency.

Solutions:

You can specify a proper Formatcode (rounding 4 digits) in the report field defnition "Print Format" after unlinking from domain.
