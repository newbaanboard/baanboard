20th May 2016, 10:14
Well, you can't change the size of a single label in a report layout. You can change the size of the layout itself, when you configure the report to "Fixed Device Font" or change the size of the whole report.

There's just one workaround I'm aware of and it has already been mentioned by srprks and bhushanchanda: use a normal report field, enter the text of the label as string constant and change the font of this new "label"-field in the layout.
Sometimes I'm using function form.text$() which will return the language dependent text of a message as string (replacing the string constant in the report field).
