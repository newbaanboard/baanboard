Well, everyone has hoped. But Infor development seems to be busy developing all kind of Web 2.0/SOA/Cloud mumbo jumbo to be integrated with ERP LN.

So, as up to now, from Infor there is nothing else but the plain old Baan IV report generator, which could be called a stable technology. From ERP LN Feature Pack 7 there seems to be a tighter integration with the microsoft reporting services, and recently Infor started to deliver a couple of solutions for FP7 that indeed contained external reports (like invoices, order acknowledhements etc.) implemented with this technology. Still, from what I've seen so far, I do not recommend to use it, since it's still only an interface and not an integrated graphical report generator. You need a seperate technology stack (Sharepoint, Reporting Services, Microsoft SQL Server), you still cannot use Baan 4GL with these reports, you will have to learn a new programming language and an ugly Visual Basic API to use it, so: No real integration.

I share your disappointment and can only urge you to communicate your concerns to the Infor sales guys. Eventually those guys wanting to sell software might be the best channel to make the Infor tools product managers finally deliver an integrated solution after 10 years of fiddling around with badly integrated third party reporting packages.

Still, you might want to have them present to you thais Microsoft reporting integration. Probably it's something you can live with. You might also want to check out the Hidox FormDesigner for output management, from what I'm hearing the value for money seems to be quite ok there.

Hope that helps you a little
