In a Baan program script I would like to assign a program variable 'A' = a field from a Baan table (actually field ecedi701.orno from table ecedi701).

Field 'A' is numeric (domain = tf gld.docn, Data Type = 'Long')

Field ecedi701.orno has domain = tc docn, Data Type = 'String'

I am expecting the field ecedi701.orno to always contain a numeric value and currently have the following code:

A = val(ecedi701.orno)

This works OK, but is there a way in the program script to check ecedi701.orno is numeric, along the lines of:

If ecedi701.orno is numeric then
A = val(ecedi701.orno)
else
A = 0
endif

How do you actually code 'If ecedi701.orno is numeric' ?

I'd be grateful for any help.

you can also you use the function isdigit for testing whether a string contains a long:

if isdigit(ecedi701.orno) then
A = lval(ecedi701.orno)
else
A = 0
endif
