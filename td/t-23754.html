Hi,

Annotate as the word says adds comments to yours script.

It adds comments to the following:
1. Table names to Tables
2. Message descriptions when Message codes are used
3. Question descriptions when Question codes are used.

Its a cute tool which will help the programmer identify messages when debugging without going back again to see what the message is for the code. Also brings perfection to your code.
