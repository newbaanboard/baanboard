You worte:

Set BaanObj = CreateObject("Baan.Application")

Please have look at the Object Name. I think you should write something like:

Set BaanObj = CreateObject("Baan.Application.Name_of_.bwc_file")

Then Excel will be able to create the object.

You can check the name of Baan.Application on the 4th page of BW Config.
