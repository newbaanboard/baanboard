How to export the current Version of the DEM Model
from the Dev Box to Prod Box
lroeleveld
10th October 2002, 06:51
Dear Mraja,

In both Baan IV and iBaan ERP, there are sessions for the export of the version (tgbrg1235m000) "Export Version Dump". The dump will be copied to a directory that is indicated in the Export session.
If your production environment is on another box, you can copy the dump file to the production box and import the file with session "Import Version Dump" (tgbrg1245m000). In this session, you will be able to select the dump file.
You will need proper authorizations on Unix or NT level for the file copy.
