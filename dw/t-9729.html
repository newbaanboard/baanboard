Paul,

Starting a session with filter on

Yes certainly you can do that.
You have to call - execute(start.query) from the 4-GL choice sections.

46 start.query Start query by form 1234 ---4 1234

Beware- This works well in the BaaN-5 series as you can maintainly only one filter per session and the tools program used is ttstpqbf (Query Browser).

This works different in our Reger, BaaN-5.2 version onwards as we can maintain multiple filters per session and the tools program is now changed to ttstp9593m000 (Filter).

But when you set a particular filter as Default, whenever the session starts that particular filter is automatically picked up.
