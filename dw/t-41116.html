You must be using Menu Browser for DEM creation....

In that case, you can backup directory $BSE/application/<your VRC>/mtgxxx2/ where normally the DEM menus are stored. Whenever any user faces a problem, you can restore the menu attached to his DEM (in maintain user data session) and everything will be fine.
