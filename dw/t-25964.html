You can find out the other way around, what is a certain user/role authorized for.
Using in BaanIVc4 session tgbrg4450m000.

Role print session by BP in BaanIVc is tgbrg4440m000, which is started from within Print option in session roles by BP by Project Model.
You can also print information to Excel based file about Role links.
Go to session print project models, make selection and then you get an option for an Export report. One of them is for roles by activities and BP.
Maybe that helps your info request.

From BaanERP on, we have a session within the projectmodel to print the role-matrix, which gives you more insights in linked roles to BP's.
