In Baan IV, thses are the major activitires to be performed to create a model:

MODELER SEQUENCE OF MAJOR ACTIVITIES


Check off Activity Session
· Create Version tgbrg1500m000

· Authorize yourself for Version

· Activate yourself in the Version tgbrg1110m000

· Business Functions (Repository) tgbrg2101m000

· Create Wizards tgwzr1100m000

· Business Processes (Repository) tgbrg5101m000

· Write Rules

1. Transformation
2. Static Condition value
3. Static Condition rule
4. Attach SC value to BP path

· Copy BF from Repository to
Project level tgbrg4160m000
· Phasing
1. Define Phases tgbrg4100m000
2. Assign phases to BF tgbrg4101m000
· Execute Transformation rules tgbrg4160m000

· Create Employees tgbrg4100m000

· Assign Roles to Employees tgbrg8140s000

· Assign Responsibilities to Roles tgbrg3150s000
Or Maintain Roles by BP tgbrg4141s000

· Attach Roles to BP tgbrg4165m000

· Link/select current phase (project) tgbrg4100m000
(process) tgbrg4160s000
(link phase) tgbrg4140s000
(current phase) tgbrg4530s000

MODELER SEQUENCE OF MAJOR ACTIVITIES


Check off Activity Session



· Evaluate Static Conditions tgbrg5195s000
(ensure that you Evaluate Static
Condition values to rules in BF model first)
· Change to Phase 2
· Reverse SC values
· SAVE and EXIT (DO NOT EVALUATE)

· Generate desktops (project) tgbrg4100m000
(generate) tgbrg4520s000


NOTE: Phasing and Static Conditions ONLY apply if the project
Implementation is divided into phases. These are OPTIONAL and DO NOT need to be performed.
