If you set up a windows printer and driver on your Baan application on the network, you will be able to print DEM flows from any workstation logged into Baan on the network.

This is done in Tools/ Device Manager/Maintenance/Maintain Device Data. You need to complete the entries for: Device, Description, Device type, Locale, Driver, Device Queue, Paper type, Left margin and Page length.

You also need to ensure that the proper entries have been made in: Maintain paper types, Maintain fonts by paper types and Maintain device preference list by user (if you want to set a default printer).
