Hi,

To obtain the same type of access as LN (both the menu browser and the process browser) you have to connect to Baan4 using the Worktop or Webtop clients.
Both clients need some patches for the Baan Tools to be installed (you can find the list on the support site) or a full upgrade of the Tools.
You can check the Users sessions to see if the flags needed to activate the Worktop functionalities are already available.

Remember also to install the latest versions of the BW and Worktop.
Webtop has to be installed and configured on the server, instead and as the name suggests is a web interface that connects to a Baan environment through Internet Explorer.
