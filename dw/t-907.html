Use tgbrg1235m000 - Export Version Dump and tgbrg1245m000 - Import Version Dump.

I'm not sure what version you're running but before you start it may be wise to check with support for patches before starting this process.

We are running 4c3 which required a couple of sizeable patches to get DEM behaving properly.
