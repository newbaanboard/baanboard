Hi,

go to tgbrg9140 and define an external program for each document type (xls, doc, etc.).

tgbrg940.appl must be like "winword.exe"

then goto your bp repository, then to your bp.

here you can assign your tgbrg940.extn code (like doc) in field "code".
then specify your doc incl path in field "argument".

when clicking on "run program" then your word doc should appear.

make sure winword.exe is in your windows path.

When you are working in iBaanERP 5.0c, then you have 2 ways to reach your goal.
1. define your Activity as type File based and place a reference to the document in the file-field.
2. define a Component to startup Word and link that component to your activity and specify the file in the argument field.

When you are working in iBaanERP 5.0b, then you can also use the Helper Applications as explained above.
