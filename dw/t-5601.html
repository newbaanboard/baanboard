There are few more issues to consider -

1. Shell access & Time zone - handled by the User Data Template.

2. If she need access to only tf menus/sessions, you need to create your own customized menus and also a new Role Template which gives her access to only the TF package with Insert/Modify/Print/Display or Full Authorization and for the other packages only Print/Display. You can choose the authorizations on the integration sessions, individually at sessions level, or module/package level.

Hi,

Another way would be to use the Shortcutbar in the Worktop user interface for this. Worktop is a new User Interface replacing the old menu browser, desktop manager, DEM browser and Process Viewer and is also available on BaanIVc4.
Using the Shortcutbar of Worktop you can create the menu by simply dragging the sessions from the menu to the shortcutbar. You can organize the shortcuts into groups, you can change the icons assosiated with the icons, etc.
If you use central deployment offered by Worktop you can even maintain her shortcuts remotely.
