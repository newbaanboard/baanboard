If you work with DEM authorization, you can change the permissions (e.g. "write", "read", "no permission") for a subsession, see also this topic:
http://www.baanboard.com/baanboard/showthread.php?t=39476&highlight=session+dem

You can for example set the main session to "write" and some sub sessions to "read". You can moreover set different permissions for different DEM roles.


Edit: However, we are using ERP LN 6.1, so I can't tell you if it works with your environment.
