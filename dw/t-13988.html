Hi Brendan,
we faced a similar problem when we put our Live-Data to our Test Box. Normally this is done by 'create tables from sequential Dumps'.
After that we had the license data message.
As i remember Baan once told us not to move brg-Data via this way but to export DEM-Version and import it on the other side.

So what i do is exporting Versiondump via tgbrg1235m000 and import it again via tgbrg1245m000. After that the message is gone.

Hope this helps,

Andreas
