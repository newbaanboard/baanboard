Hello,

That is what I meant, that the role names are used for menu names. So when you generate menus in the same company different project models, but the same role name, then the last one generated will overwrite the first one.

So best would then to be, to generate in the different project models menus with a different module code. So then the menus will not overwrite each other.
In session tgbrg4250s000 on second form you can specify a module code.

Best regards,

Ada
