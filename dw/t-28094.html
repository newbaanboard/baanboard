You can use Support Applications and attach them to an activity.
The support application should then contain component applications, which refer to the documents you want.

You can also attach a document in the URL field, when starting with \\.

You could also define the activity as a File Based activity, with a reference to open a document.

To every activity you can link help text in the repository and project model. That text is then not a document, but stored in ERP tables.

Hope ones of these options can help you.
