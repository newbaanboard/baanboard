Descriptions for the activities linked to Component Applications are stored in the Comp Appl. data tables.

Probalby you filled them at the moment, that on your system only 1 language was installed.
When you would run the session to Import ..for Running ERP environment (tgbrg5155m000-> specific -> Import Applications) again at this moment, then it would add descriptions in both languages and then you should be able to see also the activities with comp.appl. in your new language.
