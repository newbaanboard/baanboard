Jeff Ball
27th February 2014, 18:00
I'm not sure DEM is the tool for this. It's authorisations are based on session level access, so main sessions and any fields that are validated against a table, such as those with a zoom button, are controllable. Fields that allow free entry are not.

If using webui or workspace, it is possible to hide the field, but a better way would probably be to allocate a specific Tools role to the users you want to limit, then use ttams3243m000 to change the authorisation of the table field to Read.
