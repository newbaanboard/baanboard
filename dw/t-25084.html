you can directly get a picture of the complete DEM processus by (from editor):

file / save as metafile...

The file has a .emf extension.

This function save the picture as a file. You can also include this file in a word document for exemple.

Snagit is of course a second way to print a processus if you are not in editor.

If you are using IVc4 you won't be able to save as a metafile. If you want to print DEM diagrams directly from Baan the device you print to has to be a postscript printer.
