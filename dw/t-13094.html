In BaanIVc and 5.0b that is possible by using Helper Applications.

In Baan 5.0c and higher this is done by defining a Component Application for Internet Explorer and then the URL can be an application, but also an argument on actvitivity level.

Next to that from 5.0c on there is an additional field in the BP and activity properties for linking an URL.
