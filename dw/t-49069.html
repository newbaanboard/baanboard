Jenny, those are "form commands" which you can find in the "maintain Session" session. Those form commands have an "authorization group" (display, modify, use...). So if a user has display access to a session, it has access to all form commands with "display" authorization group and so on. In other words: You cannot switch on/off specific form commands by DEM, you can only put them into a particular group using Baan Tools.
