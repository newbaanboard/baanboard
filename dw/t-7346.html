When you did this in 5.0c it can be a Component Application problem.

Because from 5.0c on in the DEM architecture there is no direct connection anymore with the Tools session description, but they are taken from the DEM component Application tables.

You could have had a different Component Application on your source box then you have on your target box.

Changing the Component Application name in a Version can be done with session tgbrg5220m000.
