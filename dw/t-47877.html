The easiest way is to print out the original processes and logon with a user in the other language. Then doubleklick your activity and fill in the description in this other language. Only the original description of the LN-Sessions, where labels for differenet languages are present, the descriptions are available in standard.
In our international projects we start with modelling in english or german (where the main project is located) and the translations should be done by the local key users.
