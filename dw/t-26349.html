The desc field is an internal field.
All DEM descriptions are stored as labels within package ED to enable multi language support.
You will find also a field with label-code and that referes to a label in the label table.
Be aware that the first 3 numbers need to be the company number in which you store the DEM data.
DEM data should only be transferred from one company to another company by import / export sessions within DEM, so labels can be re-created correct.
