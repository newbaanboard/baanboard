Before you can restrict subapplications, they must be generated. From Enterprise Modeller, select Master Data, Application Components, Applications by Component and then generate Sub-Applications by Application. That will generate a table holding a list of the recognised sub-applications for each session. If the particular sub-session you want to restrict isn't generated, you can also add it manually with the same session.

Once the subapplications are generated, do a right-click on the activity in EME and Insert/Subapplications. This will open the list of sub-apps that are available to restrict. You select the ones you want as modelled subapplications, then can change the access level of the sub-session.
