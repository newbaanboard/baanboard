Thanks for your info, Beni. Unfortunately, ocavazosg is asking about DEM, not Tools, role.

ocavazosg, you can use tgbrg8110m000 to create DEM role. Additional info:
You'd then need to record DEM employees (use tgbrg8535m000). This will be different from BaanERP Common Data employees.
After this you'd need to connect the role to DEM employee (tgbrg8535m000) as well as to the business processes you created
Then you'd have to link up business processes and employees to project model. Use tgbrg4500m000
Then specify which project model applies to your company number. Use tgbrg0105m000
Then make sure the user uses DEM Browser as its default user interface

Please feel free to ask if you still have more questions
