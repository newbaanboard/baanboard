Hi,

If a user has some role attached , do this:-

1. Go to ttams2100m000 and find that role.
2. Go to specific and click on Session Authorization by Session (ttams3132m000)
3. Add the new session to the list.
4. Specific and convert to runtime

Also, if you have a table attached to it (in case of maintain session) , go to Table Authorization by Table (ttams3142m000) and add that table in there and do a convert to runtime.

Relogin.


Now, let the user log out and log in again.


Note:- This is not for DEM.
If you use DEM flow, just add the session to the Business Process attached to that user.
