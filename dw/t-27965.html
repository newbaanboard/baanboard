Hi

Rather than have the activity as an Application, select File and put the path and filename in the Argument field, leaving the URL field blank. For commonly mapped file extensions, Windows will recognise what application to launch - peobably Notepad for a .txt file.

Whether this will work will still depend on the user having the correct UNIX permissions for the target file.

If this doesn't work, try storing the document on a shared Windows volume with read-only access.
