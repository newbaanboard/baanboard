From DEM you can create a process linked to ttdskmbrowser

The menu started will be the one defined in Maintain user session.

From menu browser you can open userÂ´s dinamic menu:

Yes. You can do it using AFS (function server) from tgbrg4100m000 session.

Or using this code:


#pragma used dll "otgbrg0008"

function main()
{
domain tgbrg.user usuario
usuario = logname$
tgbrg0008.start.screen(usuario, FALSE)
}
