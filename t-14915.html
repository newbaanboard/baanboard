Here is my connection code:

'--------------------------------------------------------------------------
' LogonBaan
'--------------------------------------------------------------------------
Function LogonBaan()
If BaanObj Is Nothing Then
ConnectBaan
End If
End Function

'--------------------------------------------------------------------------
' ConnectBaan
'--------------------------------------------------------------------------
Function ConnectBaan()

On Error GoTo CannotCreateBaan

If BaanVersion = 4 Then
Set BaanObj = CreateObject("Baan4.Application")
Else
Set BaanObj = CreateObject("Baan.Application")
End If

Set BaanObjS = BaanObj
BaanObj.Timeout = BaanTimeout
Exit Function

CannotCreateBaan:
MsgBox "ConnectBaan: Can not connect to Baan with User/Password"
End Function

'--------------------------------------------------------------------------
' DisconnectBaan
'--------------------------------------------------------------------------
Function DisconnectBaan()
If Not BaanObj Is Nothing Then
BaanObj.Quit
Set BaanObj = Nothing
End If
End Function


I have two questions:
1) This code hangs when I am already logged into Baan. What code do I need so it connects to Baan again and doesn't hang?
2) Is there a way to use the existing Baan connection if I am already logged in?
