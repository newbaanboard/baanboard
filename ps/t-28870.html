If your company is running b40_c4 then I would do this there.
In case you have an opportunity to implement in LN this would be the best option. SSA is supporting LN since 2004 so there is also no problem here.

I have worked with BAAN Project since 1996 and have been involved with many implementations. In case of Project there is usually no straight forward solution because Project is used by companies that are highly involved in a "customized" environment.
Perhaps you can start by checking out this post: http://www.baanboard.com/baanboard/showthread.php?t=10155

Do you plan on implementing Project yourself or do you require help from an external person as well?
If you have never been exposed to BAAN Project and you intend to do this all by yourself you will be in for a difficult time and a lot of headaches.

In any event, good luck!
