Ashish

you could create a Weeks Period Table for a each Month (4 Weeks). Inserting this Table in the Project Progress Parameter / Peoples Parameter Table.
This excercice needs to be done per Month.

In Peoples Time Recording, it only possible to insert data for a valid period.

The first week is always 1 and you can insert data up to 5. But for week 6 its all blocked!

Try / test this, maybe it will solve your problem.
