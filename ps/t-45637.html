Hi,

the equivalent to HRS is now part of the People module. The specific session to look for: bptmm1800m000. This is an umbrella session that will allow you to select what kind of hours you would like to enter. Select the Project option, bptmm1111m000. Don't forget to flag Project as an implemented software component in tccom0500m000.

Good luck!
