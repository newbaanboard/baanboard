Hi Nadeem,

It is standard functionality in LN that when you create an Estimated Labor Costs Line, the line number typically will be line 10.
Then when you do hours accounting and the Actual Labor Costs are updated you will get a separate line number (line 20). Line 10 coming from the Estimated Labor Costs line will also show up in the Actual Labor Costs session with a value of zero.

However, if during hours accounting you select the Estimated Labor line then the system will create only one Actual Labor Cost Line (line 10). In that case the Estimated Labor Cost line has been "copied" to the Actual Labor Cost line.

Please try to select the Estimated Labor Cost line in the Hours Accounting session and let us know if it worked.
