After selecting package TP in Implemented Software Components (tccom0500m000 / tccom0100s000), you'll also need to run Intitialize Mapping Scheme Data (tcfin0210m000). On the Project (TP) tab in tcfin0210m000, select the appropriate IDTs and Business Object Elements.

The Intitialize Mapping Scheme Data session makes the following available in your system: Integration document types; Business objects; Mapping elements; and Reconciliation elements. Without these, you cannot process any transactions in Projects.
