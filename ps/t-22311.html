Hi,

You can only use Budget Adjestments in tpptc when the status is "Final". As long as this is still "Actual" you will not be able to use Budget Adjustments.
This is standard functionality and will also give the correct transactions in Project, Finance and other areas in BAAN IV.

Hi,

To work with adjustments you need to specify the elements being effected by a change (assuming, you work with an element structure)

Then, change the element status to final (one or more elements in the structure)

Maintan the Adjustment and select the elements hat need to be changed.

You will be able to see the budget effect the change has coused.

But, in finance you will only see increased costs for changed costs. You will not see an adjustment code in finance. Only if you can add an Extension (additional contract!!) you would see the change there.

If you can not use extensions but you like to see changes in finance, you might consider operating with different cost objects (control code!)
But this would increase the number of cost objects!
