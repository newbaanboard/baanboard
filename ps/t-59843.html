Hi Suki,

As far as I can see it you are using BAAN V and in that case you can install a new solution from Infor Support.
This solution will allow you revert the status of the service order from "Costed" to "Completed" and then add the late costs/charges for labour, material, etc. The additional cost lines should be costed and can then be invoiced if needed. After that the service order can be set to "Costed" again.
When reverting the status from "Costed" to "Completed" you cannot add a new service order activity.

This solution was originally available in ERP LN but has been backported to BAAN V as well.
