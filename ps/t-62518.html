There are a number of fields that are disabled when an item is set to anonymous. This is because if an item is anonymous, there can be a quantity of it within a breakdown or cluster, and without a serial number it is not possible to tell them apart, including when they were installed if they were not all delivered at once. If you want an asset to have its own maintenance plan, warranty expiry date and so on, it ought to be serialised.
