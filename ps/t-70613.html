Process Transactions will set the status to Closed if successful. This assumes the financial result status on the project has been set to Determine Result and while using the process session the option Final Result has been flagged. The help says: 'The financial result status. To close such a project, you must use the Process Transactions (tpppc4802m000) session, then the final result is calculated.'
