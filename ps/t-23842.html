Hi Mahadevan,

This is not an error message.
Please check (for your particular project) the statusses of the elements, the relationships between the elements, the budgetlines associated to the elements and the surcharges.
In case one of these still has the status "Free" then you need to change these to "Actual". After that you can run tpptc1230m000 again and the 'error' message will no longer occur.
