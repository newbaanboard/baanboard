Wesselin,

if you purchase an item in TP, it does not create inventory in any warehouse, only hard committment or costs can be seen (in standard)!
Project inventory is available only since Baan ERP.

Iin order to avoid any problems with pricing use project items! No average prices, only direct purchase costs!

In Baan IV this is quite easy, since the project item is defined with very few fields.

Ulrich
