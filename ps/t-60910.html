Hi David,

In case you want to prevent a configuration to be used on a new service order, make sure to expire the configuration and all of its versions.
If you want to expire the Objects linked to a configuration (entered at Service Order Activity level) then expire the Objects and all of the Object relations.

Entering a signal code at the item level in tcibd0501m000 will have no impact on what you want to do in BAAN V Service.
