Hi Mahadevan,

The project module helps the Senior Management by analysing the projects based on the Cost component ( i.e analysis across various projects ).
Helps the Quanityt surveyor to analyse the project by Elements.
Helps the Planner to analyse the project based on Activity progress and related budget Vs actual etc..

Upto Project progress the user will be reporting to the system, then the key analysis is given by the system to control the project by three different roles ( Project Manager and above, Quantity surveyor, Planning Engineer ).

The key analysis like Earned Value are very critical for project monitoring.

As you browse through various reports in project monitoring module, U will find it very useful for each role in the project. You have the facility to analyse by drill down from the project to the last activity or element which is effecting the project. you can also derive the profitability analysis of projects/elements/activity.

Hope the above note is useful.
