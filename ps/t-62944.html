Is there an existing report for Working Capital by Project? (Current Assets less Current Liabilities)

Such a report would enable project managers to work with the Finance/Treasury to best manage their company's working capital.

In particular, this report would help industrial equipment companies that sell valuable goods with multi-year payment plans.

The report I am looking for would show the information, ideally by period.

1) The status of the open A/R entries generated by Project (i.e. from Installments)
2) The status of the open A/P entries generated by Project (i.e. from project-specific Purchase Orders)
3) A summary of the aggregate working capital for all projects selected.

The report would also have specific filters by data on the project header. (i.e. category, geographical area, project manager, etc.)
