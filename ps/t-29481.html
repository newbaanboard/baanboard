Hi,

the information you are looking for is already available via tpppc4421m000. That way the cost information is already rolled up/aggregated. If you want to create your own report you will have to access tpppc215, 235, 255, 275, and 295. If you need to include commitments a few more are relevant. Alternatively use tpppc400 (or other tables in the 400 range).
Good luck!
