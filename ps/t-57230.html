Hi,

There is a way to get Performance Measurement with bottom up budget, by generating Top Down budget from Bottom Up budget the same can be done but it is must to use activities in Project.
Use the session Generate Structure and Top Down Budget (tpptc5210m000) select the project and select the check box 'copy including budget lines'. and click Generate. This will generate Topdown budget from Bottomup budget.
further by creating TimePhased Budget from Top down Budget with Base Lines and entering Progress, the performance Measurement can be obtained.
It is important that for the Activities with Earned Value methods 'percentage Complete' the Progress entry should be at Activity/Cost Type Level. Topdown budget must be actual and Base Line should be approved for getting Performance in Performance Measurement.
