Sridhar,

Make sure that in the session Maintain Elements, the element for which you want to register progress and book revenue, needs to have the parameter Progress on Yes. If this parameter is on No, revenue will automatically be posted to the parent-element of the element concerned.

In session Maintain Element Progress (tpppc1150) you can check this. There should be a Y(es) in the column I for those elements that have the above mention Progress parameter in maintain elements on Yes.

I hope this makes it clear. If not, just send me an e-mail with further questions/information.

Good luck,
