Well any change like this requires a lot of work. I would first ask infor support before attempting and baan seeded domain. When we first went live with Baan we changed the item default to 32 characters. To this day we still have standard reports that only print 16 characters. Not only do you have to change all the tables that might be related to this, but you also need to change forms, reports and maybe even some baan code. Luckily the other domain changes are have been rejected since we do not own code and this would impact what happens with future SP's and patches. If you are lucky enough to have a play ground you can try doing this and just see what all needs to change - plus what works and does not work. This is not something I would do lightly - my opinion only.
