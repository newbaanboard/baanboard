Included a screenshot of our Inventory program.

The program draws a real-life like grid in Internet Explorer of our company's warehouse, filled with the actual BaaN inventory data. Different colors represent on-hand, blocked, inbound and outbound stock. Inventory transfer in BaaN can be done through drag-and-drop operations in the Internet Explorer window (a pop-up window allows you to (de-)select specific lots and change quantities).

Programming done in VB.NEt and BaaN IV / Boi's / AFS.

(next step: to use the NetList method to trace actual transactions and show warehouse staff / forklifts in IE moving from one location to another :-)
