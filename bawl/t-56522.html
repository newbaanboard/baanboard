PA-RISC Linux

The PA-RISC project provides a native port of Linux to the PA-RISC architecture. The port is merged into linux-2.6 mainstream kernel tree (http://www.baanboard.com/kernel/index.html). PA-RISC linux-2.4 is stable but not fully merged into the mainline tree nor well maintained these days.
