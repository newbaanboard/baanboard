Question
What is the process to change the tcitem domain from 16 to 32?

Clarifying Fact
BaanIVc4

Resolution
In order to alter the domain from 16 to 32 in Baan, you'll want to run session ttadv4107m000 (Modify Code Alignment).

Convert the change to runtime. Also, ALL USERS must be out of the system when this is done.
