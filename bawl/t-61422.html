First Create the User Data and select the User Type as Normal User.

Check On Menu Browser and Process Browser.

Convert CRDD.

Make the same user in Database.
Link ERP User and Database User.
Create Employee.
Create Roles.
Attach Particluar Roles to employee.
and also that role to Business Process.
Maintain Employee by Project module.

And then Configure Users by DEM Employees
