GTM means General Table maintenance.

For any version of Baan you need to run the session ttaad4100.

We need to have the password to enter into it.

note: It is very useful & critical session, where we can change the data of any table manually. It changes the data without checking the integrity with the other tables. so be cautious.

General Table Maintenance (ttaad4100). You must know the password to use this session.

This session is used to manipulate table field data without using sessions (which is the usual way). This session is meant only for developers. Inappropriate usage is liable to result in data inconsistency!

Enter the code of package, module, table and company for which you want to maintain the table data.
Press [Y] in the choice filed to access the table fields.

When you press the space bar, a pull-down menu with standard options appears at the top of your screen. The options are grouped in 7 menus.

It is possible to call help information by table field. Press [?] to get table field help.
