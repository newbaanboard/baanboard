There might have been a typo in the doc-number. You'll find the "Infor Web UI Installation and Configuration Guide for Infor Web Help" when searching for docno.: U8934*
The DocLink-search never worked for me when using the complete Infor docno (i.e. "U8934I US").
Please refer to the guide. It includes a section about how to setup the administration of webhelp to enable an existing user for webhelp customization (he may change existing webhelpfiles). I already did helpcostomization myself using the given method.

There's also a "Infor XML 6.4 Help Customization - Writers Guide and Reference" available: U7850*
This guide explaines some ways of creating and modifiying the LN webhelp. I'll still need to read this guide myself.
