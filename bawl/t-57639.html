Once Estimated Materials get frozen then these can not be modified, however modification as required is possible through session "Material to Issue for Production Orders" ticst0101m100. But these modifications will not get reflected in estimiated materials costs because it has already got frozen based on value of "Moment Freezing Estimates" field maintained in "Default Production Order Data" session tisfc0502m000

It's not possible to change the estimated materials after the Production Order has got the status Released. Status "Active" means Production process has already started and some of the operation in the routing already completed.

For your further info, you can execute Reprocess Production Order - tisfc1203m000. This will automatically include changes that are made in basic data such as BOM and routing. The session regenerates the production orders by deleting the existing production order data and hence the BOM and routing data for the new Reference Date are used.

You can only use this session for production orders whose status is:
Planned
Scheduled
Documents Printed.
