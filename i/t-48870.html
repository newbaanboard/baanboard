Hi,

Please find the steps in finding which level driver you are operating on.

1) The main difference between SQL LEVEL 1 and 2 is, level 1 will have hash columns,
ie when you do "select * from t<package><module><table><company>" in the result set u will find hash1 hash2 hash3 ..... depending on the table.

If you dont find hash columns in the result set then u r running on level 2.

2) If you observe tabledef6.x,

for level1 it will be MSQL_LEVEL1=1
for level2 it will be MSQL_LEVEL1=0

3) If you observe msql_storage file for level2 it will have the format as
*:*:T:group:00400:5:NOIDXHINT

4) The db_resource file will be of the following format msql_join_hint:NONE


Hope this information would be useful for any user who is looking for level1 and level2 differences.
