Lets clarify this a bit for newbies:
- There is no certification on Baan, like the one you can have for Cisco, Microsoft, Oracle, etc. and, currently, nothing is mentioned on SSA webpages regarding a future certification path...
- You can, however, attend Baan trainings and these come with a certificate of completion of the course.

Now, as recommendation for future Baan Tools wanabee admins, I would do:
1) get some training/knowledge on:
- Microsoft Windows and, at least, one UNIX flavour (like Linux)
- A [Baan supported] database
- Baan Tools module (by a good trainer)
2) Do a test installation of Baan to play [heavily] with;
3) Read all Baan Tools and OS/database related threads of Baanboard :cool: and test the things mentioned there on the system of 2).
