1. You need to use Location Control Module
2. In Item Data the field Inventory Valuation has to be Lot Price.
3. In Item Data the field Outbound Priority has to be FIFO.

Please Note: FIFO is not a Inventory Valuation Method. It is only used for the consumptions / issues. Which in turn means the items purchased first will be used first. Which finally means inventory valuation is at ACTUAL COST.
