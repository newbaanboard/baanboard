Hitesh's is the only solution outside of modifying the program script. You need to establish a reference between the combined 3 fields of the child and parent tables. You can only do this if there's a combined field in both the parent and child containing the same 3 fields. I am not sure why you're unwilling add a combined field to table xxx. Adding a combined field does not change the physical structure of table xxx, as far as I know.
