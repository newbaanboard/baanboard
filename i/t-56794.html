Hi Indianidol,

Here is what the "Release Notes" of portingset 8.5 says:

"Support for VMware ESX in production environments
For the Linux and Windows production environments, it’s now supported to
have Enterprise Server running on a VMware ESX virtual environment.
Check the “Infor Enterprise Server - Technical Notes for Porting Set 8.5a”
(U9546 US) for the conditions."

However - running the SLM-Server in a virtual environment is not supported, so you have to install the SLM-Server on a real hardware.
