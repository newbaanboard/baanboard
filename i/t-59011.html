Hi,

I do not know about any standard session to do it.

Another way using Maintain Tables Session:

* First of all check if the non finalized lines are not duplicated. It is possible the entire batch has been finalized but the lines have not been cleared.
* If the lines exist only as non finalized I would try to create a new empty batch and use it to finalize the lines. In order to get it the batch number has to be changed in the pending lines.

Hi Dinesh,

Sometimes it happens that some lines of transactions get updated on finalzed tables and some remain on the non finalize table during finalization process.

Easiest way to slove this problem from user end is make a new entry, if user can do so and delete the non finalized and finalize from table and maintain session.

Or, if batch is not modifiable, then you need to update tables or run the same process again.
