hi rajesh,
You could use Scan Solution File {ttpmc2200m000} session for scanning the 530 solutions. In the [Solution File] field (of this session) just give the Folder name (without the solution file name) where you have the solution files stored. On pressing Continue, the system would ask you if you want to scan all the files in the folder..

After this you could run session Process Solutions {ttpmc2101m000} or straight away run Check and Install Solutions {ttpmc2204s000}...to install the solutions scanned.

hth

sk

Note: Using the above session (for scanning solutions) might depend on what service pack you are on.
