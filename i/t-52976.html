For Dealer Invoice â€“ Reciept, the Excise Duty set up can be done as follows in BanIV c4, with India Localization set up.

Dealer Quotes the Basic Price ( Rs.150 ), which includes the Excise Duty ( unknown at Order placement time ), which is being passed on to you as Cenvat claimable amount.
The value or the percentage of the Excise Duty is not knows till the Dealer Invoice Copy is received by you.
Hence while placing the Purchase Order, the Basic price ( Rs.150 ) quoted by Dealer is to be put into the Purchase Order Line Basic Price ( Rs.150 ). In the Localization Line the Excise Applicable is kept as NO, but Cenvat applicable is maintained as YES.
When the material is received from the Dealer, Before the Receipt entry,
The changes are made into the Purchase Order by looking into the Dealer Invoice.
For Example, The Excise Duty available is (Rs.30), then
Split the Basic Price (Rs.150) into two parts ( Rs.120 + Rs.30 )
Change the PO Line Basic Price to Rs.120 &
In the Localization Line maintain
Excise Applicable YES
Tariff Code as applicable in your case
ASV as ( Rs.182 ) which can be arrived at by calculation,
Excise applicable is 16.48% then
(Excise Rs.30 as required X 100 ) / ( Ex Percentage 16.48 ) = Rs182 as ASV

And as such the calculation is correctly maintained.

You can also change the calculation after Receipt maintenance, thru Change Prices & Discounts after Reciept, if you have already maintained the Reciept without any Change. The Cenvat will be always taken as per the Cenvat value available on the Dealer Invoice.

Dealer Invoice is mapped in such way in our case.
