Dear akshay,

The feature used for return the inventory from staged location to bulk location in case

1) Material has been picked and sales /production order is canceled .
2) If Material is picked X pcs but you want to ship only X-Y pcs so for return of Y Pcs from staging location to Bulk location the functionality is used.


Step 1:- Create one sales order for material “0088-0020” for quantity 10 pcs.
Release the order to warehouse.




Step 2:- In warehouse perform the activity a) Generate outbound Advice
b) Release outbound Advice
c) Generate picking List
Step 3:- generate picking list, System automatically create shipment for that Picking

Step 4:- While shipment you decided that you are not going to ship all 10 materials , you will only allow to ship 5 material , for this you need to double click on shipment line ( Shipment line shouldn’t be freeze)

You need to change the shipped quantity.

Now freeze the shipment line and capture the GPD. And confirm the shipment .

Once you confirm the shipment system automatically activate the option Return Not shipped goods.


Step 4:- Click on the option system generates one transfer order from staged location to bulk location ( To generate the transfer order automatically you need to set the order series and order type into “Default order Type by Origin” )



Hope it will Solve your problem
