Hi Vishal,

The BaanIV FST functionality was brought back to LN in feature pack 4. In Baan5.0b/Baan5.0c the FST functionality in Baan itself only provides some non-normalized tables where the transactional data is stored. To use this for reporting you need an external report writer and knowledge how to use the data from these tables. I know there have been plans to backport the FST functionality from FP4 and higher to FP2 and the Indian localization, but you better ask you local support organization whether this has been realized in the end or not.
