From my limited experience of a single project in china:

1) Yes. Actually installing it in both languages is often the only option, so that a non chinese admin can operate the system.

2) Yes

3) You need to install the chinese language on your existing installations and (very important!) you need a multi byte installation. If your currentinstall is not multibyte, then you need to start fresh!

4) Costs are depending on your support contract, I guess. But even if you need a second installation of baan, then it can reside on the same server, if it is a UNix system. If it is a windows system, then it needs to be a chinese windows, otherwise it will not work.

Good luck! Implementing Baan in china is a challange for sure.

Regards

Markus
