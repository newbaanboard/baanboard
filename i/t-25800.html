Hi,

First in the mapping scheme details set the set up for FAM_ASSE select element group for FA category. Set the ledgers for the types of FA Categories u have set . This FA_ASSE should be Debit. Then in the default accounts u can set the mapping for FAM_CAPA for the credit (preferably "Asset Invest A/c" ) which will have FA integration in the COA. FAM_DEPR,FAM_gAIN , FAM_LOSS,FAM_LOSS can be set in the default a/cs also.
