Hi, Please find steps:
(1) Donwload and Install Ruppes font from internet.
(2) In you report add a new variable of string at your desire position (e.g., String of 1 character and initialize it with value "R") and assign this variable with some unique font(E.g., "User Font 11")
(3) Compile your report
(4) Print Report in BW Printer and check for the value "R" at your desired location. Double click on value "R" and change its font to the newly installed font with size 48 (you might have to change the font size depending upon the font that you had installed).
