How to move an Infor ERP environment to another host
Infor ERP LN 6.1
Document code: U9189A US
Release: Infor ERP LN 6.1
Publication date: May 07

This document describes how to move an Infor ERP environment to another machine.
The term Infor ERP contains the following versions
* ERP Baan IV
* ERP Baan 5.0
* ERP LN
The document is intended for Infor Support department, consulting and system administrators who are specialized in the system administration of UNIX and Windows Operating Systems. An elementary understanding is needed about the way Infor ERP software is structured. High administration skills are required.
The document contains the following chapters:
Chapter 1, "Introduction", provides a brief introduction about cloning and how the document is structured.
Chapter 2 “Steps to Clone”, explains how to move an environment from one server to another.
Chapter 3 “Post installations steps”, explains the additional steps to take after moving an environment to another server.
