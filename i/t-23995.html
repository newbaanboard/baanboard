Hi,
Transaction schedule is use to create some template voucher, like monthly expense allocation, etc.
1. Create a transaction schedule header, with schedule type "amount"
2. maintain detail information in detail, like Debit A, Credit B, etc. can set amount to 0
3. use it in maintain transaction, transaction type must journal voucher, after maintain header information, select "create transaction from schedule", after transaction created, you need modify net amount manually.

Hope it is helpful for you!
