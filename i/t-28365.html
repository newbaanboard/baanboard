No. According to BaaN Help:
Expired If this check box is selected, the table has expired for this package VRC and the package VRCs derived from it.
Expiring in BaaN means, you keep the component but set a status that its not in use or execution. This way you have a copy of source code, session or table definition which was developed/used in history of time.
