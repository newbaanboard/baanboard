The easiest way to install the India Localization for ERP LN FP7 is using the Installation Wizard. See Infor ERP LN 6.1 FP7 India Localization - Installation Guide (U9712A). This guide can be found in the Documentation area of InforXtreme.
This Installation Wizard automatically creates the PVRC with the correct derivation and creates also a Package Combination.
