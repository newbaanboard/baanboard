Hi Raman,

Say today u hv raised a Sales order for X Item Qty 10 pcs. The same had been invoiced.

Now after some time u come to know that u have to raise invoice some further amount on the same 10 Pcs by Rs 50.00 per Pcs.

So You have got a Cost Item (say "OTHCHARGES"). You add this item in the same sales order in 2nd line for 10 Pcs @ Rs. 50.00. Process Sales Order and Raise Invoice. For releasing Cost Item we dont need any inventory.

In case u want a different description on Invoice, Use Line Text.

Hope the same is clear to you.
