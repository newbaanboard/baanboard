28th May 2003, 17:06
There is a BaaN solution (81837) that allows pmc patches to be imported into BaaN IVc3. This will then allow you the easier option of 'just' upgrading the porting set version rather than the BaaN version.

BaaN always recommend upgrading to the latest porting set. The current version (6.1c.06.07) allegedly has some fixes to known problems with the Oracle 8 drivers / libraries. As usual, do as full a test as possible to determine what problems you may encounter before installing into production (function server calls spring to mind).
