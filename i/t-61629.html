HI,
The best way would be the use of AFS, which will automatically take care of all the tables to be updated.

there are 4 different kind of opening balances that you will need to update as follows:
1) Open entries relating to Creditors (invoices / Debit notes) (AFS)
2) Open entreis relating to Customers (Invoices / Debit notes) (AFS)
3) Advances paid and advance received.(AFS for advances)
4) Fixed assets details where applicable (AFS for Fixed asset and capitalisation process)
5) Transactions relating to inventory if applicable (through Adjustment order logistics)
6) Other balances. (by using Import JV functionality)

Hope this should answer your query.

The Opening balance transaction type updates the table tfgld203, ie the ledger opening balance history.. As we are aware that at the time of closing, only balance sheet items are transfered to the next financial year, the transaction type of categoty "Opening balance" considers the same fundamental. and hence, while using this transaction type, system only allows the Balance sheet items, and not the P & L items.
