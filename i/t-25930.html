Hi,
Here are the steps:
Creating New Company in BaaN
·Maintain Companies - Session ttaad1100m000
Enter new company code, name and package combination.
- Convert to RDD
- Convert to runtime
·Create tables

Ensure that package combination of user is same as that of new company. Use session ttaad4230m000 to create tables for packages tc, td, tf, ti, tp, tr, ts, ps.

·Change company to new company. Run initialize parameters (session tcmcs0295m000). Take down all error codes for each initialize run and create all those tables individually. This is interactive process.

·Maintain units - Session tcmcs0101m000
Maintain basic units of

- Length - mm
- Weight - kg
- Area - m2
- Volume - m3
- Pieces - no
- Time - sec

·Assign tables to database (ttaad4111m000)  Special - Load mandatory tables
Special - Convert to runtime
·Maintain base units  session tcmcs0100m000
·Maintain signal code  session tcmcs0118m000

Add one signal code with all blocking status blocked.
Maintain languages  session tcmcs0146m000
Add ENG for English.
·Maintain countries - session tcmcs0110
IND, India, -, 91, -, -, No (Considering India as a conuntry)
· Maintain currencies  session tcmcs0102m000
· Maintain system configuration parameters  session tisfc0000m000
· Maintain MPS plan codes  session timps1101m000
· Maintain hourly labor types  session tihra1130m000
· Maintain cost price calculation codes  session ticpr1101m000
· Maintain all manufacturing parameters.

Go on pressing tab keys and enter something wherever cursor stops.

ITM parameter  prefix - $$$$$$, Signal code  blocked signal code
BOM parameter  mm, No.
PCF parameter  form2 Production Rate per: Should be pr_rate or production_rate
·After successful company formation you should be able to go to maintain item data.
Session tiitm0101m000. This completes the process.
