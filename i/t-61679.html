Yes, I deployed Webtop on W2K8 several times.
If I remember correctly: You should install JDK 32bit. Then set the JAVA_HOME environment variable on OS level. Then edit the installtomcatservice.bat file and adjust JAVA_HOME there as well. Finally execute installtomcatservice.bat.

If you have already installed the InforESE service you have to uninstall it first before you apply what I wrote above.

Good luck !
