Gurmeet, Refer to the documentation for SSA License Manager (SLM) and contact BaaN/SSA support site for more information.

Most SSAÂ® software must be validated prior to use. The SSA License Manager was developed to facilitate one license file per customer covering all products with various license types.

We strongly recommend that you start your license key request process prior to beginning your implementation of SSA products. The License key request process may take up to five business days to complete, please take this into consideration during your implementation planning. The SSA License Manager (SLM) key request process for SSA software installations listed under the below mentioned product group links consists of the following steps:

1. Install the SLM software and the purchased software applications
2. Enter the applications specific information in the Customer License File of the SLM
3. Send your Customer License File to the Validation Specialist using the submission form

Revalidation of existing software installations is required for the following reasons:

* Hardware changes
* Additional Software applications installed which are supported by the SLM
* Updates/changes in versions
* Hardware or operating system upgrades
* Increase/decrease in numbers of users
* Catastrophic failure of system requiring a complete reload

For further information please contact SSA Validation
