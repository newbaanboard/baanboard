I was tasked with producing reports & documents in PDF format including graphics, logo2, watermarks etc so that they could be automatically e-mailed to customers, supopliers, users etc.

I found some great software at www.fytek.com that enables this.

text2pdf takes the text output from a report & converts it into a PDF file.
pdfmeld allows you to use a take a predesigned PDF file containging logos, watermarks etc & use this as a template, overlaying the converted PDF file from text2pdf therefore creating a combined PDF document that looks like it has been printed on preprinted stationary.

I created a new device & print spooler that incorporates the software & e-mails the PDF to the recipient. Maybe this could be a start to the mythical paperless office.
