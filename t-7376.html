Ulrich,
The test function server session you speak about does not go through the DDC. It is only for testing the function server capabilities. I have seen several instances where function servers will operate in test function server but not in the DDC. You also cannot acquire certain characteristic which are available such as default values. I am not saying that you shouldn't jump in, just make sure you know where you are jumping.
