I've yet to come across any book/reference you can find on the Internet or in a bookstore for Baan Tools.

Other than the Baan Tools help, and the course books you get at Baan training, I have found the BaanERP Application Performance Guide to contain very useful information. You should be able to get an electronic copy from Baan. The document # is M2017B US. Although it's for BaanERP, a lot of it also applies to Baan IV.

I believe there is another document on UI guidelines for BaanERP. However, I have not looked closely at it since we're running Baan IV.

Hi Redwards,

I havn't seen any book so far.
But on your system there are the Ban Help Files, (windows help files) located at $BSE/lib/help/DIRNAM, where DIRNAM is tfB40_c4 or suchlike. The most interesting dir here is ttB40_c4, where you will find a file called "tt3.hlp". Transfer this file to a windows box (if you do that with ftp, don't forget to use the "bin" option, otherwise you can't use the file) and doubleclick it. In this tools help file you can find a lot of interesting topics. Search for "executables", if you look for information about the programs of your porting set.
