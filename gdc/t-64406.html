The documentation team delivers a 'Difference Study' for each new feature pack, so in these document you will find the major feature changes.
Does the following help you?

For each major release and feature pack we deliver a documentation portal published as a CD image (.iso):
1. Go to http://www.infor.com/inforxtreme, and log on
2. Click Downloads > Products
3. In the Product List, select Infor > Enterprise Resource Planning > ERP LN > ERP LN
4. From the list of downloads for ERP LN, you can select Documentation Portal Infor LN 10.2.1 (FP8.1)
5. Select FTP Download or configure the other download options to download the image
6. The downloaded image (.iso) can be burned on a CD-ROM, or mounted on a virtual CD-ROM drive using a utility program
