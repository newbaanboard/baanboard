...and the format would be "ZZZZZZZZZZ9VD99" without the double quotes. Number of digits for this format is 14 and so since you have taken 2 digits off after the decimal, add the two digits before the decimal (as shown above)... If you want the display format to be only changed in one place, i would recommend taking the "%AQ20" format off the form and replace it with the above format so that the change is not affecting any other forms/reports that uses Q20....

sk

P.S: Dont forget to run "Convert to Runtime" after changing (if you are changing the format in "Maintain Default Formats by Format Code {ttad1108m000} session)...
