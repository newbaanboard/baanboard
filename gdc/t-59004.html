It is the method by which logistics transactions (for example, receipting a Purchase Order) causes related financial transactions.

Typically a purchase order receipt is set up to create this kind of financial transaction:

Credit to Liabilities (how much you owe the supplier)
Debit to Assets (usually inventory) or Costs (if it is just an expense to be written off immediately).

In Baan 4, you can set up different integrations depending on Supplier Group, Warehouse going into, Item Group, Item and Cost Price Component.

Hope that helps.
