EDM as part of Baan is a way to manage revision control on your BOM. The functionality is very basic but usefull.

Now if you have an engineering department that have drawings on a CAD system you would need PLM. PLM have PDM and ECM. PDM can be integarted to your CAD system and allowes such functionality as Red Lining etc. that is needed in an environment where you have strict engineering control. There is also workflow functionality to help with the authorization of revisions.

ECM (Enterprise Content Management) is the documentation management part of PLM. This allowes web based documentation control of the engineering documents in PDM as well as any documents that can be linked to Baan sessions. These documents are revision controlled as well.
