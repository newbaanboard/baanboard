The Knowledge Quest installation will put the .dll files under:

Program Files\Common Files\Folio Shared

You must copy these .dll files into the Knolwledge Quest directory.

Then try to run it again.

I don't know why, but that is what the small Post-It note in our Knowledge Quest CD case says to do (I had to recently install it for a new Cost Accountant at our company so that he could learn Baan costing.)
