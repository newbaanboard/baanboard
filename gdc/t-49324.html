ERP LN FP4 Development will be done internally by June 2008. There will be a limited availability of FP4 for selected and interested customers and must be willing to move to FP5. FP5 will be released near the end of 2008.

ERP LN FP4 has many functional enhancements like Vendor managed inventory, new standard currency system in Finance, completely enhanced financial statements etc
Apart from that many technical and usability enhancements are there, like 10+ home pages, new ERP LN reporting with BIRT etc

FP5 will also functional enhancements at the same time SOA is the main focus. FP5 will be released with a pre built integration with some of the infor Extent and Evolve products. For example it will be integrated with supplyweb, EAM like Extend products and New Multi Book Ledger, Role Based home pages, like Evolve products...

FP5 with come out not only with many functional enhancements but also with a new SOA enabled integration for ERP LN using Infor Bus which can help to integrate ERP LN with any product using generic open standards
