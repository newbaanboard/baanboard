You should nevere need to kill inactive users, you should use instead the existing built-in functionality (Windows or unix) of SESSION_TIMEOUT, see this thread: http://www.baanboard.com/baanboard/showthread.php?s=&threadid=1735&highlight=TIMEOUT

For killing users when needed (different than the recommended above!) you use the bshcmd6.1 kill command, see this thread as example: http://www.baanboard.com/baanboard/showthread.php?s=&threadid=6115&highlight=bshcmd6.1+kill (search for bshcmd6.1 in tools admin for more threads about this)
