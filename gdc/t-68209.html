did you see your btw (not sure how it is called now) file in your b2bar directory? it's just a formatting file. it needs a text file to print. this text file is from LN, so you can probably feed through a print session or a custom script. the text file is moved based on your windows services in your bartender. in a nutshell, LN feeds the txt or xml file. the bartender is just processing and printing those files by detecting that there are new files in your input directory. it doesn't really care whether the file is from LN or not. i don't do this anymore but this is as far as i can remember.
