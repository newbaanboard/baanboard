I can't tell you about the functional things, only the technical stuff since I have done these before.

The main thing is that tfgld410 is splitted up into a few other tables. Baan helps you doing this by delivering a session for it. The other tables can easily be migrated with a normal change package combination for the company.

However if you have changed the standard tables of Baan than more can be involved. Furthermore you have to consider your customizations.

The docu that comes with c4 explains how you should do these things (if I remember well).
