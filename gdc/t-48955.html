There are three places to get documentation. Unfortunately the tools sessions are not available outside of training.

The system is delivered with two sets of documentation. One is in webhelp. If you use a browser and go to the webhelp server and look around, there is a section with PDF files on some of the subjects.

There is a second CD delivered with the system called the documentation portal. This cd autoruns Tomkat and has many of the Module Descriptions for different areas.

The Programming Manual is installed as a help file when you install the Dynamic Form Editor.

Not all of the documention mentioned so far can you always find on the website. Some of it is in the form of patches to be loaded into webhelp. Other individual PDF can be found on the documentaiton sites. Cookies need to be enabled not only for infor but SSA and Baan also.
