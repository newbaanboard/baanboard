I can't say anything about the package that you're referring to. If you only need some key metrics calculated though, then it's also possible to develop those specifically (or have them developed) in the "Enterprise Performance Manager" in Baan (have a look at the Baan main menue).

It's a package in Baan that offers a framework for performance indicator calculations. It's necessary to develop the scripts for the calculations though. That requires some programming skills. Once developed, a process runs to calculate the metrics. For better visibility I upload the results into an Excel sheet or word document.

Maybe something you want to look into.
