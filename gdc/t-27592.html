About This Guide
This document describes how to upgrade an existing SSA® BaanERP 5.0b or
SSA® BaanERP 5.0c environment from the 7.1d porting set line to the SSA®

Chapter 3 Delayed Move to 8.2b Porting Set
Windows cluster
SSA® Enterprise Server 8.2 does not support Windows clustering at this time.
Therefore, SSA® BaanERP 5.0c customers with this setup must delay their
upgrade to the SSA® Enterprise Server porting set until that support is
available.
Windows and DB2
When you deploy ERP 5.0c on Windows with DB2 and you are ready to
move to the new porting set line, contact your SSA OnePoint Support
representative for SSA Global to enable support.
