anubhav,

After installing iCRM4.3 Server Database, you must create a database account (name 'aurum' and password 'aurum'), before you can install the iCRM4.3 Client. Then you can log in with Attached Mode.

After installing iCRM4.3 Client, you must not change your ODBC setting, becauser you do not know the password of 'sa' for the demo database. If you change the password, you will can not log in with Deatched Mode.
