What we have done is added an extra field on the sales order line where the original delivery date (Cust Req Date) is saved and can not be chaned once you leave the order. This will be used to monitor delivery performance. We can then change the planned delivery date by line.

Also found by using the margins the system can plan delivery into the history if you do not delete the delivery dates by Std Route. This caused an issue as you only want to deliver today onwards so need to delete the dates by Std Route on a regular basis.
