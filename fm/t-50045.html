FM planning engine checks if the carrier/TMG specified on the Freight order line is able to achieve the required transportation. The check is done on many parameters such as speed of the trucks (TMG), the load capacity of the TMG, Carrier capability of operating in different transport modes etc. If the carrier on the order does not satisfy any of the requirements, the search for a other suitable carrier is done. Only the carriers which have been defined under "Carriers by Shipping Office and Planning group" are considered for further planning.

Would suggest, check the following in the set-up to make sure atleast one carrier/TMG is selected.
1) Define multiple carriers under Shipping office / Planning group.
2) If you are using combination codes and/or Transport types for the items then make sure there is a TMG defined with similar values.
3) Specify a average speed at TMG such that the transport can be achieved between the desired planned load and unload dates.
4) If Means of transport is being used, make sure there are sufficient number of those defined under a Carrier/TMG combination. Also make sure planning percentage is given as 100%.
5) Specify the capacity on the TMG such that it is able to handle the items which are being transported on the freight order line.

Hope this helps.
