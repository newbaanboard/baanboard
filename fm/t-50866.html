The question is now a bit confusing :confused: .

If I understand correctly, you are trying to use one freight order to ship goods to different addresses. This is not possible, there have to be different freight orders for each of the different address (even if they all belong to the same business partner).
Each of the freight order has to go in a different shipment (if addresses differ), but all these shipments can be consolidated under the same load. Also, all shipment lines under the shipment go to the same address only.
