Are you keeping the /SI/cache directory and /Windows/temp directory cleaned out? Too many files in these directories will register as memory errors because they're to saturated. I keep mine cleaned out regularly and don't have these problems. The Windows/temp directory is especially bad when full when running other apps like Word or Excel. These apps complain about low memory when space gets tight there.

In 1999 Crystal introduced their Baan driver in beta release and it was free to download and use. It is still in circulation, but is not available from Crystal since they introduced the production version in 2000. The production version is fully supported where the beta version is not. The production version is only available from Baan or Crystal, and is only licensed for use with Seagate Info or Crystal Enterprise. You will not get any support if you use the beta version and have problems with it.
