Well. This is the answer of Infor concerning BIRT :
- Can you give me more information about BIRT for LN ?

Check solution No 229031

- On which release of Baan it can be used ?

Enterprise Server 8.4.1, but I suggest ti aooky the latest version (SOL No 213717).

- Can it modify built in baan reports ?

Yes, you can.

- Is the modification of fields done visually or it needs coding ?

No, you only need to enable the report to print via Web. The procedure is very simple, but you need to make sure you have all the proper portingset, Webtop, Report Viewer versinos.

- Is it difficult to install ?

One thing is to activate a BIRT in Baan, and other issue is to develop. So, to be able to activate it, you only need a proper ES (SOL No 213717), portingset (SOL No 148218) and Webtop version (SOL No 142977). The procedure to activate a report to print in BIRT is pretty much easy. And the other hand, if you want to develop you need Eclipse, Java Connectior (JDBC) etc.

- what are the requirements to install it ?

To create graphs/reports, Infor Reporting Studio 8.4 has to be used, which is based on BIRT Report Designer 2.1.2.
