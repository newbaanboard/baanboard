BaanInOhio
31st March 2014, 06:58
You can call a standard DLL function instead of using AFS to release a production order:

domain tcbool short.found
#pragma used dll otisfcdll0221 | Release production order

ret = tisfc.dll0221.release.production.order(
inp.pdno, | order number to release
tcyesno.no, | release without operations (no)
true, | commit when done (yes)
short.found) | shortages found (returned)

I have run into a few sessions that provide 'report not found' when successful even when I supply the report information. You can check the API error and ignore (assume success) when this message is returned.
