From the support.crystaldecisions.com knowledge base:

The CESK for Baan version 2.5 is compatible with CE 8.5, both the Windows and Solaris editions. The CESK for Baan 2.5 should not be installed together with CE 8, CE 9 or Seagate Info 7.5. This solution kit is compatible with these versions of Baan products: Baan IVc, Baan 5b, Baan 5c, and Baan 5.2

The CESK for Baan version 2 is compatible with Crystal Enterprise 8.

I scanned through the installation guide for the solution kit, but couldn't find anything that differentiates the installation between Baan4 and Baan5.
