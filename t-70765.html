You need to import the Custom BOD into the ION Registry, Both the xml file as well as the Schema, xsd file.

Suppose you want LnMessageNew to be imported , You need to create a Zip File which will contain. LnMessageNew.xsd as well as LnMessageNew xml file and then import this Zip File from Registry.
