How to add printer to a new user in erp ln

Hello,

1.)
Define a output setting with ttams1140m000 and a the printer the group/role should use with the priorities.
2.)
Add the setting from 1.) to the user (ttams1100s00 Form 2) in "settings output device"

User is able to use the devices for the group/role.
