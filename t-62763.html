ION abbribiation stands for Integrated Open Network platform. Infor10 ION enables Infor and customer applications to configure a routed infrastructure, setup Workflows, design and activate Business Event monitors and manage the tasks and alerts.
The Infor10 ION suite is a set of services built by Infor that simplifies connectivity and data sharing across all applications.
The Infor10 ION suite consists of:
• ION Desk
• ION Connect
• ION Process divided in: Following 3 Catagories
•-- ION Event Management
•-- ION Workflow
•-- ION Pulse
