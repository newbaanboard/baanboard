The integration transactions for this project still have to be posted to Finance with session "Post Integration Transactions to Finance (tfgld4201m000)".

If this session can't process a transaction then the transaction will show up in session "Display Integration Transaction Log (tfgld4532m000)" together with a reason why. Check if you'll find anything in there.

To investigate the transactions further, have a look at the finance transactions for the project in session tfgld4510m000 "Display integration Transactions by Project/Order" or General Table Maintenance, table tfgld418 and make sure they have status "Posted".
