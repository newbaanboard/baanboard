Sorry, forgot to say how to connect to informix.
If you select a application connector type "oledb" you can simply choose Database Vendor Informix and the provider e.g. Ifxoledbc.
....enter you other db properties and you should have a connection with Informix. (i'm using BCP 4.1 but I have done this with earlier versions also)
