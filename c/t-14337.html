3.1.3 Thin-client
A thin-client is not allowed to access local resources. Local resources are for example the Windows registry, files, directories etc. Baan offers functionality that depends on accessing local resources that can not be supported in a true thin-client solution.
The following functionality is not supported because it depends on local resources:
 BW print
BW print is a software component that makes it possible to print via the locally installed printers.
 File transfer ()
