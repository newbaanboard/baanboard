Internet? 3 party apps? Normally people means

1. Can I use internet browser as the Baan ERP client?
The architecture of BaanIV and BaanV is similar, still traditional three tier client-server base. In order to let Baan ERP runs on internet browser, Baan develops webtop, which is an add-on component and the current versions works for BaanV only.
I would say this is a nice add-on but the setup is quite difficult.

2. Can I exchange information with Baan ERP through internet applicatons (3rd party apps)?
In the old old decades, we can only write the data directly to BaanERP's database from our internet application.
In the not so old decades, we are supposed to call Baan BOI to write data into Baan ERP
In this new decade, we are advised to plug on OpenWorld and exchange information with XML, following the standard defined by Rossetanet or Microsoft(?).
Baan supports OpenWorld both on BaanIV and BaanV.
