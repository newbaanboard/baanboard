hi,

this is a expamle to connect to a baan table (baan IV) /oracle 8, select some rows and display the rows in a html-table.

for the connection you need a odbc-connection to the oracle database and a oracle-user and password.

<html>
<head>
<title></title>
<head>
<body text="#000000" bgcolor="#FFFFFF" link="#FF0000" alink="#FF0000" vlink="#FF0000">
<h3>Test-Aufruf Oracle-Datenbank über Apache-Server mit ODBC</h3>
<br><br>
<?


$Link_ID=odbc_connect("Odbcname", "orausername", "orapwd");
$Query_ID = odbc_do($Link_ID,"Select T\$TYPE,T\$DSCA from baan.tspssl001XXX");
// T\$TYPE TYPE is datafield name of table spssl001 from company XXX
// name of the oracle table baan.tspssl001001XXX (this is not a baan standard table)
$i=1;
$res=odbc_fetch_row($Query_ID,$i);

echo "<table border=1 align='center' bordercolor=#000000 bgcolor=#FFFFFF>";
echo "<tr bgcolor= #00C3FF>";
echo "<th width=50>Nr</th>";
echo "<th width=120>Name</th>";
echo "<th width=120>Vorname</th>";
echo "</tr>";
while($res)
{

echo "<tr>";
echo "<td><div align='right'>$i.</div></td>";
echo "<td>".odbc_result($Query_ID,"T\$TYPE")."</td>";
echo "<td>".odbc_result($Query_ID,"T\$DSCA")."</td>";
echo "</tr>";
$i++;
$res=odbc_fetch_row($Query_ID,$i);
}
echo "</table>";
odbc_free_result($Query_ID );
odbc_close($Link_ID);

?>
</body>
</html>

xesklu

If possible, it would be better to setup one of the web service (SOAP) frameworks for Baan. Then, the PHP/Perl script can do a SOAP request to get data, call a DLL, insert into a session, and so on.

We have written Perl scripts (similiar to PHP) that do this, using Cordys.

Using Oracle directly works, but isn't the best way as you probably know, since you can't reuse existing session/script logic.

8th January 2004, 15:28
We created a php tcp/ip-server application with one baan connection using COM. From php-script we can do requests on a tcp-port to Baan functions, for example:

$werktijdentabel = baan_aanvragen("/otihrac001?get_werktijdentabel($bedrijf, $medewerker)", "");

- otihrac001 = Baan dll
- get_werktijdentabel(parameters) = function in Baan dll

It works like a web-service to Baan.
We have already created several application with this, like:

- Web-application for Hour Accounting (with barcodes).
- Web-interface for productconfigurator.

Let me know if this could be interesting for you.
