You need to write a dataobject in java or VB(depends on your platform requirements and strategy), and have that acess the BOIS you write.

Dont try to access BOI directly on JSP's or ASP's, because you will have issues regarding java compatibility and stuff like that.

So, always ensure that, the design pattern of data object wrapping a BOI is followed.

J2EE Connector Architecture (JCA) is the Sun Microsystems standard for writing adaptors especially for heterogeneous ERP Systems. You can download J2EE Connector API's from http://java.sun.com/j2ee/download.html#connectorspec

Import the JCA along with the Java code to access the BOI's. You can also use JCA for Cordys based BOI integration.
