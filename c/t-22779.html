When you import the Dem models from your dump file a new version is created. You can see the name of this version from Organization->Settings->Versions in cordys studio.

What you need to do is set the design time user preferences so you can use this new version. You can either set your current version to this version or set Show Repository content for all version in Organization->Settings->Design Time User Preferences

This will now show the list of all the models generated from the Dem import.

--------------------------
Once these are imported you will need to generate the objects for the sessions in the dem model.
