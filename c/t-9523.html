We have written some programs for the Intermec wireless devices using Perl. Works very well. We first wrote custom telnet servers, but dcBrowser is a lot easier to develop with.

Menus are easy to do, since they are just a bunch of HTML links.

These perl scripts connect to Baan function servers, using web services provided by the SOAP toolkit in Cordys. Works fine.
