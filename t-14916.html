I have not been using Word but like Mark i used Excel. It is a bit quick and dirty but you can copy the connection string from Excel to word.

To create the connection string, record a new macro. Click on the Data menu, choose import external data, choose new database query. Here you can select an existing data source or create one, finally MS Query starts an lets you select tables and fields. Stop the macro recording process. Start the vba editor and copy and paste.
