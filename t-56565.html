Baan4U is the answer to questions like:

1. Which non-standard software components are there on my system, when & why were they modified, which software components have been removed etcetera
2. Is the test environment in line with the production environment (servicepacks, developments, customizations)
3. Which patches / servicepacks are installed, which software components are updated
4. What is the 'structure' of my Baan environment (companies, packagecombinations, packagevrcs, packagevrc structure, devices, users etcetera)
5. Did my Baan jobs run smoothly (in all companies)
6. Etcetera

Baan4U was developed in MS/Access 2003. Implementation will take one day & is included in the price of the product.

Baan4U provides you with a precise & actual snapshot of the current situation of your Baan environment(s). No need anymore to hire expensive consultants to do a project intake; you know it already!

Baan4U works with all versions of the Baan software (Triton, BaanIV, ERP-LN) & on all operating systems.
