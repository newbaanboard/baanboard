We were successful getting the Baan Integration Kit to work with our porting set using the shdbapi.dll version 7.6a03. Contrary to one of the other posts, only this one file is needed in the \BusinessObjects\Integration Kit for Baan 11.5\Native\bin\SSA_ERPLN61\ folder.

Different versions of this dll are required for different Baan porting sets. We are in the process of upgraded our porting set and observed the old shdbapi.dll will not allow the Baan Integration Kit to work with the more recent Baan porting set.
